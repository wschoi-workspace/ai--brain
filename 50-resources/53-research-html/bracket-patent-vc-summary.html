오프라인 매장이
데이터 플랫폼이 되다

Bracket은 매장 외부 파사드부터 내부 모듈까지 통합하여, 고객의 노출→행동→구매→CRM→피드백을 하나의 풀퍼널로 추적하는 리테일 미디어 스토어입니다.

Retail Media Store Bracket — 마케팅 효과 측정 및 데이터 수집이 가능한 콘텐츠 중심의 리서치 특화 리테일 미디어 스토어

01 / 08
Problem

오프라인 마케팅, 왜 측정이 안 되는가

연간 수십조 원이 오프라인 마케팅에 투입되지만, 효과를 정량적으로 증명할 수 없습니다.

효과 측정 불가

옥외 광고·매장 프로모션의 실제 전환율을 수치로 증명할 방법이 없음

온·오프 데이터 단절

오프라인 고객 행동과 온라인 CRM이 분리되어 통합 추적이 불가능

단편적 솔루션

동선·매출·CRM 각각 별도 시스템으로 운영, 풀퍼널 인사이트 부재

Differentiation

기존 리테일테크 vs Bracket

개별 솔루션 조합이 아닌, 하나의 매장에서 풀퍼널 데이터를 통합 수집합니다.

구분기존 리테일테크Bracket
데이터 범위 단일 데이터 (동선 OR 매출 OR CRM) 통합 노출→행동→구매→CRM→피드백 풀퍼널
온·오프 연결 단절 (별도 시스템 운영) OMO 데이터 풀 프로세싱으로 실시간 연결
고객 분류 수동 또는 기본 인구통계 기반 AI C.C 기반 자동 세분화
광고 효과 노출 수만 측정 가능 풀퍼널 시선→진입→구매→CRM 전환 추적
공간 유연성 고정 인테리어 모듈 유닛 구조 → 브랜드별 즉시 교체
리서치 별도 설문/포커스그룹 필요 내장 매장 내 서베이 + 행동 데이터 동시 수집

Patent Scope

특허 개요

Bracket의 특허는 매장 설계·데이터 수집·분석 프로세스 전반을 포괄합니다.

Retail Media Store Bracket

by PROJECT-RENT — 마케팅 효과 측정 및 데이터 수집이 가능한 콘텐츠 중심의 리서치 특화 리테일 미디어 스토어 설계 개발

  • 외부 디지털 파사드(DID + 센서) 기반 트래픽·시선 추적 및 진입 전환 측정 시스템
  • 내부 모듈형 유닛 구조 설계 — 브랜드별 가변 공간 운영 방법
  • AI 기반 고객 카테고리화(C.C) — 센서·CCTV 행동 패턴 자동 세분화 알고리즘
  • OMO 데이터 풀 프로세싱 — 오프라인→온라인 CRM 실시간 연결 프로세스
  • 6단계 Consumer Flow 기반 풀퍼널 데이터 수집·분석 프레임워크
  • 매장 내 서베이 시스템 — 정성·정량 리서치 데이터 통합 수집 방법
