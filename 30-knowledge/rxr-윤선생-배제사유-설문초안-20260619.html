설문 목적 · 검증 가설

가설(분석에서 도출)설문 검증 문항
H1. 배제 1순위는 전집·패드가 아니라 '혼자독학·관리부담'E1(랭킹)
H2. 전집·패드는 단독 장벽이 아니라 관리부담과 결합될 때 배제로 전환E1·E2(가정 제거 질문)
H3. 약정 없음·가성비는 배제사유가 아니라 방어 자산E1(약정 보기 하위 검증)
H4. 전환처는 특정 1곳이 아니라 병렬 비교(튼튼·구몬·눈높이·엄마표)E3
H5. 배제사유는 BM(교실·베이직·영어숲)·시작시점별로 다름B1×E1, D1×E1 교차
💡 핵심 산출물: 배제사유 위계(가중 N) + 전집·패드 순수 장벽 효과(%) + 전환처 Top + BM·세그먼트별 배제 차이. 자유응답은 RXR Micro-Psyche 텍스트 분석으로 보강.

3

스크리닝 (S)

S1단일선택 · 쿼터
자녀(영어 학습 대상)의 만 나이는?
○ 만 4~5세 · ○ 만 6~7세 · ○ 초1~2 · ○ 초3~4 · ○ 초5~6 · ○ 중등 이상 · ○ 해당 없음
로직: '해당 없음' → 종료
S2단일선택
자녀 영어 사교육 결정에 본인이 관여하시나요?
○ 내가 주 결정자 통과 · ○ 배우자와 공동 통과 · ○ 거의 관여 안 함
로직: '관여 안 함' → 종료
S3단일선택 · ★핵심 라우팅
'윤선생영어'에 대해 어디까지 경험하셨나요?
① 들어봤고, 상담·체험·비교까지 했지만 결국 선택하지 않았다 → 배제군(본설문)
② 현재 사용 중이다 → 비교문항
③ 과거에 썼다가 그만뒀다 → 이탈사유(별도)
④ 이름만 들어봤고 고려한 적 없다 → 종료
⑤ 잘 모른다 → 종료
로직: ①=메인 배제군 / ③=이탈군(보유 문제, 별도 집계) / ②=현사용 대조군. 배제(획득)와 이탈(보유)을 처음부터 분리한다.
4

고려 상품(BM) · 깊이

B1복수선택
어떤 윤선생 상품을 고려/상담/체험하셨나요? (아는 만큼)
□ 방문 1:1(윤선생영어교실) · □ 화상(윤선생베이직) · □ 학원·등원(윤선생영어숲) · □ 유아 놀이(스마트랜드) · □ 구분 잘 모르고 '윤선생'으로 봄 기본값
분석: B1 × E1 교차 → BM별 배제사유 차이(H5)
B2단일선택 · 고려 깊이
윤선생을 얼마나 진지하게 고려하셨나요?
○ 검색·후기만 봄 · ○ 상담 전화/방문까지 · ○ 무료체험·레벨테스트까지 고관여 · ○ 등록 직전까지 갔다 취소
분석: 깊이가 깊을수록 배제사유 신뢰도↑(가중치 부여 가능)
5

배제군 본설문 (E) — ★핵심

E1랭킹(1~3순위) · ★위계 측정 랭킹형 추천
윤선생을 최종 선택하지 않은 이유를 영향이 큰 순서대로 3개까지 골라주세요. (1순위·2순위·3순위)
a. 결국 아이 혼자 해야 하고/엄마가 다 봐줘야 할 것 같아서 (관리부담) H1
b. 가격이 부담되거나 가성비가 별로여서
c. 영업·상담이 부담스러웠다(강매 느낌)
d. 전집·패드(전용기기)를 꼭 사야 해서 H2
e. 효과·아웃풋(말하기·실력)이 의심돼서
f. 화상/비대면이라 집중이 안 될 것 같아서
g. 원어민이 아니라서(한국인 교사)
h. 약정·해지·위약금이 걱정돼서 H3 역검증
i. 다른 브랜드가 더 나아 보여서
j. 기타(직접 입력)
분석: 가중점수 = 1순위×3 + 2순위×2 + 3순위×1 → 배제 위계 확정. 보기 순서는 랜덤화 추천(순서 편향 제거). h(약정)가 하위면 H3(방어자산) 확정.
E2단일선택 · 분기(E1에서 d 선택자만) · ★H2
만약 전집·패드(전용기기) 구매가 필요 없었다면 윤선생을 선택하셨을까요?
○ 그래도 선택 안 함 · ○ 선택했을 것 같다 전집·패드 순수 장벽 · ○ 모르겠다
분석: '선택했을 것' 비율 = 전집·패드의 순수 배제 기여도(%). 낮으면 "전집·패드는 결합 요인일 뿐" 확정.
E3복수선택 + 단일(최종) · ★전환처 H4
윤선생 대신 무엇을 선택하셨나요? (해당 모두 + 최종 1개)
□ 구몬 · □ 눈높이 · □ 튼튼영어 · □ 캠블리키즈 · □ 시원스쿨 · □ 윙크 · □ 천재(밀크티) · □ 잉글리시에그 · □ 영어유치원 · □ 동네 영어학원 · □ 엄마표 · □ 아직 안 정함 · □ 기타
분석: 전환처 Top → 경쟁 구도(H4). '아직 안 정함' 비율 = 미획득 잠재시장.
E4복수선택 · 역질문(획득 레버)
윤선생이 어떤 점을 갖췄다면 선택을 고려하셨을까요?
□ 선생님이 더 직접 봐주는 관리 H1 대응 · □ 가격 인하·요금 유연화 · □ 원어민 수업 · □ 전집·패드 구매 없이 시작 · □ 무료체험·레벨테스트 확대 · □ 약정/위약금 없음(이미 있음 인지 여부 확인) · □ 눈에 보이는 아웃풋(말하기) 증거
분석: E1(왜 안 했나)과 E4(있었다면)의 정합성 → 획득 메시지 우선순위 직접 도출.
6

이탈군 분기 (C) — S3=③ 전용 · 별도 집계

C1랭킹(1~2순위)
윤선생을 그만둔 가장 큰 이유는? (보유 문제)
선생님·원장 교체 / 효과 안 보여서 / 아이가 싫어해서 / 가격 부담 / 더 좋은 곳으로 이동 / 시간·이사 / 기타
분석: 이탈(보유)은 배제(획득)와 분리 집계 — 처방이 다름(운영·콘텐츠 vs 광고·체험).
7

공통 세그먼트(D) · 자유응답(O)

D1 자녀 영어 시작(예정) 시점: 4~5세 / 6~7세 / 초1~2 / 초3~4 / 초5+ 하부② 교차

D2 시작 동기: 흥미·습관 / 기초 만회 / 결과·선행·내신 / 기타 하부② 매트릭스 교차

D3 윤선생 고려 전 경험: 처음 영어 / 엄마표 / 타학습지 / 영유·어학원

O1 (자유) 윤선생을 한 단어로 표현한다면?

O2 (자유) 안 고른 결정적 이유를 한 문장으로 적어주세요. Micro-Psyche 텍스트 분석

분석 계획 (수집 후)

  • 배제 위계: E1 가중점수 → 막대(가설 H1 검증: 관리부담 1위?). 멘션 분석 위계와 교차검증
  • 전집·패드 순수 장벽: E2 '선택했을 것' % → 결합요인 여부(H2)
  • 전환처 맵: E3 Top → 경쟁 구도(H4), 멘션 '병렬비교'와 대조
  • 획득 메시지 우선순위: E1×E4 정합 → "관리해준다" vs "3무" vs "원어민" 중 무엇을 전면에
  • 세그먼트 교차: BM별(B1)·시작시점별(D1)·동기별(D2) 배제 차이(H5)
  • 자유응답(O2): 텍스트 클러스터링 → 정량 보기 밖 신규 배제사유 발굴
  • 3중 검증: 설문 위계 ↔ 카페 멘션 위계 ↔ 블로그 천장. 세 소스 수렴 시 결론 확정
이 설문이 닫는 마지막 한 칸
멘션(스니펫·거짓양성 92%)이 못 준 배제 위계의 정량 N
설문 위계가 카페 위계(관리부담 1위)와 수렴하면 → 획득 전략을 "혼자 안 해도 되게 관리"로 확정. 어긋나면 멘션 편향을 보정하는 새 가설 도출.

배포 전 체크리스트 추천

  • 보기 순서 랜덤화(E1·E3·E4) — 순서 편향 제거
  • E1은 '3개까지'로 제한(과다선택 방지), 1순위 필수
  • 유도 문구 금지 — "전집이 부담스럽죠?" ❌ / 중립 표현 ⭕
  • 모바일 1문항=1화면, 진행바 표시(완료율↑)
  • 개인정보 최소수집(연락처는 추첨용 별도 동의)
