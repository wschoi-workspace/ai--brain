왜 이 리서치인가 (메우는 4대 공백)

기존 20260615 보고서는 윤선생을 단일 브랜드로 보고 구몬/빨간펜/청담과 비교 — 아래가 빠져 있었다
  • 1BM 미분리 — 교실(방문1:1)·베이직(화상)·영어숲(학원)이 한 덩어리로 분석돼 타겟·메시지 차별화가 불가능했다.
  • 2영어수준 근거 부재 — "관리형 입문 영어" 결론이 실제 학생 행동 데이터(시작시점·사전경로·동기)로 받쳐지지 않았다.
  • 3배제사유 없음 — 이탈사유(이미 쓴 사람=보유 문제)만 있고, "고려했는데 안 한" 잠재고객의 배제사유(획득 문제)가 없었다.
  • 4시간흐름 없음 — 24/25/26 추이가 없어 인식 변화·신규/소멸 토픽을 볼 수 없었다.

3대 분석 모듈 구체화

①BM분리 · ②영어수준 정량추정 · ③배제사유 + 시간흐름
①

BM별 분리 분석

목적 — BM별 마케팅 메시지 분리 (동일 브랜드 산하, 타겟 다르게)
  • BM별 감성분포 — 긍/중/부 비율 + 전체 대비 차이
  • BM별 토픽 Top8 — 교실=방문관리/1:1, 베이직=가성비/화상편의, 영어숲=학원성과/레벨 가설 검증
  • BM별 이용자 프로파일 — 학년·동기·상황(워킹맘 등), ②와 연결
  • 각 BM 카드 하단 "메시지 차별화 한 줄 제언"
②

이용자 영어수준 정량추정

목적 — 타겟이 "지연 시작층"인지 "평균층"인지 확정 → 카피·채널 전략 결정
  • (a) 시작시점 분포 — 초1이전/초1~2/초3~4/초4이후 → 막대(윤선생 추정 vs 통계청 평균), BM별
  • (b) 사전경험 경로 — 영유졸업→ / 영유미경험→ / 타학습지이탈→ 3분류 + 대표 인용 데이터 잘 잡힘
  • (c) 시작시점×동기 매트릭스 — (늦게/평균/빨리)×(흥미·기초만회 / 결과·선행) 히트맵
  • 가설 판정 — "윤선생=지연시작층 vs 평균층"을 BM별로 명시
③

배제사유 TOP (이탈 ≠ 배제)

목적 — 진입장벽 식별. 이탈=보유 문제 / 배제=신규 획득 문제
  • 전용 쿼리 — 윤선생 고민 윤선생 vs OO 윤선생 안 시키는 이유 윤선생 거른 우리 애한텐
  • 배제사유 클러스터 TOP N — 예상: 가격·약정 / 아웃풋 약점 인식 / "학원이 낫다" / 방문 부담 / 디지털 거부감
  • 경쟁사 대비 — 삼성·뮤엠 / 눈높이·재능의 배제사유도 추출 → "윤선생 고유 장벽 vs 카테고리 공통 장벽" 구분
⏱

시간흐름 24 / 25 / 26

목적 — 인식 변화·신규/소멸 토픽 추적
  • 멘션 발행연도 버킷팅 → 연도별 멘션량·감성추이·신규/소멸 토픽
  • 적용 위치 — ①감성분포(연도축), ③배제사유(연도별 장벽 변화)
  • 산출 — 3개년 sparkline + 연도 감성 스택바
5

데이터 현실성 · 리스크 보정

웹 멘션의 한계를 명시하고 항목마다 대비책을 내장
⚠️
②는 설문 수치가 아니다웹 멘션은 모집단 대표 표본이 아니다(후기 작성자=만족/불만 극단 편향). 모든 %에 N + 신뢰도 라벨 + 통계청 사교육비조사 벤치마크 대비를 병기해 과대해석을 차단한다. 형식: 멘션 관찰값(N=__) · 통계청 대비 ±__
📉
연도별 N 불균형24년 멘션은 최신순 편향으로 적게 잡힘 → 절대량 비교 자제, 비율 중심 해석. 연도별 N을 차트에 표기.
🔀
교실·베이직 신호 혼동학부모가 브랜드명을 혼동 → 두 BM은 N이 적을 수 있음. 단순화(항목 축소)는 하지 않되 "방향성 참고용" 라벨로 신뢰도를 노출.

실행 단계

/rxr-analysis 단순분석 모드 기반, 시간축·BM축 확장

STEP 1데이터 수집

윤선생 멘션을 연도×BM신호로 태깅(트랙당 30건+). 배제사유 전용 쿼리, 영어수준 신호(시작연령·사전경로), 경쟁사 멘션 동시 수집. 검증 앵커: 통계청 2024 사교육비조사.

STEP 2① BM별 분리 분석

BM별 감성·토픽Top8·프로파일 + 메시지 차별화 제언.

STEP 3② 영어수준 정량추정

(a)시작시점 (b)사전경로 (c)동기 매트릭스 + 가설 판정. N·벤치마크 보정 필수.

STEP 4③ 배제사유 TOP

배제 클러스터 + 경쟁사 대비 "고유 장벽 vs 공통 장벽".

STEP 5시간흐름 24/25/26

연도 버킷팅 → 멘션량·감성·신규/소멸 토픽. N 불균형 리스크 명시.

STEP 6경쟁사 풀 RXR 2-Layer

트랙별 경쟁사 감성+토픽+심리계층+배제+시간흐름.

STEP 7HTML + MD 리포트 생성

R디자인가이드(#6666FF, Pretendard), Chart.js, card-explain 해설, 부록 로우데이터. 헤더/푸터 가이드 적용. 외부본 필요 시 v3 용어+보안세팅.
