# 윤선생 배제사유 검증 설문 (폼 입력용 · 추천안) · 2026.06.19 · by Project Rent

> 목적: "윤선생 고려→미선택" 배제 위계를 N으로 확정. 멘션(스니펫 거짓양성 92%)이 못 준 마지막 한 칸.
> 표본 추천: 배제군 N≥100 / 이탈군 N≥30 / 현사용 N≥50 · 맘카페+페북+인스타 모집 · 3~4분 · 구글폼/네이버폼.
> ⚠️ 보기 순서 랜덤화(E1·E3·E4), 유도문구 금지, 파일럿 10명 후 본조사.

## [스크리닝]

**S1. 자녀(영어 학습 대상) 만 나이?** (단일·쿼터)
- 만 4~5세 / 만 6~7세 / 초1~2 / 초3~4 / 초5~6 / 중등 이상 / 해당 없음
- 로직: '해당 없음' → 종료

**S2. 자녀 영어 사교육 결정에 관여하시나요?** (단일)
- 내가 주 결정자 / 배우자와 공동 / 거의 관여 안 함(→종료)

**S3. ★'윤선생영어'를 어디까지 경험?** (단일·핵심 라우팅)
- ① 상담·체험·비교까지 했지만 선택 안 함 → **배제군(본설문 E)**
- ② 현재 사용 중 → 비교문항
- ③ 과거 썼다가 그만둠 → 이탈사유(C, 별도)
- ④ 이름만 들음, 고려 안 함 → 종료
- ⑤ 모름 → 종료

## [고려 상품·깊이]

**B1. 어떤 윤선생 상품을 고려/체험?** (복수)
- 방문1:1(영어교실) / 화상(베이직) / 학원·등원(영어숲) / 유아놀이(스마트랜드) / 구분 모르고 '윤선생'으로 봄

**B2. 얼마나 진지하게 고려?** (단일)
- 검색·후기만 / 상담까지 / 무료체험·레벨테스트까지 / 등록 직전 취소

## [배제군 본설문 E] ★핵심

**E1. ★윤선생을 최종 선택하지 않은 이유 (영향 큰 순 3개, 1·2·3순위)** (랭킹)
- a. 결국 아이 혼자 해야/엄마가 다 봐줘야 할 것 같아서 (관리부담)
- b. 가격 부담·가성비
- c. 영업·상담 부담(강매 느낌)
- d. 전집·패드(전용기기) 필수 구매
- e. 효과·아웃풋(말하기·실력) 의심
- f. 화상/비대면 집중 우려
- g. 원어민 아님(한국인 교사)
- h. 약정·해지·위약금 걱정
- i. 다른 브랜드가 더 나아 보임
- j. 기타(직접입력)
- 분석: 가중점수 1순위×3+2순위×2+3순위×1 → 배제 위계

**E2. (E1에서 d 선택자만) 전집·패드 구매가 필요 없었다면 선택?** (단일·분기)
- 그래도 선택 안 함 / 선택했을 것 같다 / 모르겠다
- 분석: '선택했을 것' % = 전집·패드 순수 장벽 기여도

**E3. ★윤선생 대신 무엇을 선택?** (복수 + 최종 1개)
- 구몬/눈높이/튼튼영어/캠블리키즈/시원스쿨/윙크/천재(밀크티)/잉글리시에그/영어유치원/동네학원/엄마표/아직 안 정함/기타

**E4. 윤선생이 어떤 점을 갖췄다면 고려?** (복수·역질문)
- 선생님이 더 직접 봐주는 관리 / 가격 인하·유연화 / 원어민 수업 / 전집·패드 없이 시작 / 무료체험·레벨테스트 확대 / 약정·위약금 없음 / 눈에 보이는 아웃풋(말하기) 증거

## [이탈군 C] (S3=③ 전용·별도 집계)

**C1. 윤선생을 그만둔 가장 큰 이유 (1~2순위)** (랭킹)
- 선생님·원장 교체 / 효과 안 보여서 / 아이가 싫어해서 / 가격 부담 / 더 좋은 곳 이동 / 시간·이사 / 기타

## [공통 세그먼트 D · 자유응답 O]

- **D1.** 영어 시작(예정) 시점: 4~5세/6~7세/초1~2/초3~4/초5+
- **D2.** 시작 동기: 흥미·습관 / 기초 만회 / 결과·선행·내신 / 기타
- **D3.** 윤선생 고려 전 경험: 처음영어 / 엄마표 / 타학습지 / 영유·어학원
- **O1.** (자유) 윤선생을 한 단어로?
- **O2.** (자유) 안 고른 결정적 이유 한 문장 → Micro-Psyche 텍스트 분석
