## 1. Claude Code 기본 사용법

- 대화하듯 쓰는 게 가장 좋다
- 대화 내용이 마음에 들면 "오늘 파일로 저장해놓자"라고만 하면 된다
- 지금 한 일을 저장해놓자라고만 하면 충분

## 2. AI와 엑셀 파일 다루기

- AI는 엑셀 파일을 그렇게 선호하지 않는다
- 엑셀 파일은 복잡한 구조(병합셀, 다단 헤더 등) 때문에 AI가 정확하게 읽기 어렵다
- 해결책: 엑셀을 CSV로 변환하여 AI가 정확하게 읽게 만드는 것이 핵심

### excel-to-csv 스킬
- 병합셀 해제, 다단 헤더 정리, 불필요한 행 건너뛰기
- 표의 형태로 깔끔하게 만들어주는 스킬
- 엑셀의 "그릇"을 바꾸는 작업

### CSV 클린업 (2단계)
- 합계행 제거 (잘못하면 한 라인으로 인식되어 데이터 오류 발생)
- 숫자 서식 정리, 날짜 통일
- 가로형을 세로형으로 변환
- 엑셀→CSV 변환 후 CSV 클린까지 하면 정말 잘 정제된 데이터로 분석 가능

## 3. 스킬(Skills) 개념 소개

- 이미 워크스페이스에 다양한 스킬이 세팅되어 있음
- 내게 필요한 도구를 만들어 사용할 수 있게 하는 것이 6주간의 가장 큰 미션
- 노무법인 해지할 수 있을 정도의 스킬을 만드는 것이 목표

## 4. 참여자 반응

- 컨텍스트 스위칭이 줄어드는 느낌이 확실히 있다
- 웹이나 어딘가에 가지 않고 하나의 공간에서 모든 일을 하는 감정이 좋다
- 할 수 있는 게 너무 많아지다 보니 "눈치우기"처럼 쌓인 일들을 처리할 수 있는 시대
- "어떻게 써먹지?"보다 우선 기본부터 익히는 것이 중요
