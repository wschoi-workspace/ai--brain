01
Step 1
WHY 선언
Simon Sinek — Golden Circle  |  1장

첫 슬라이드는 "회사 소개"가 아니다. 우리의 신념, 존재 이유, 이 프로젝트에 임하는 철학을 한 문장으로 선언한다. 변연계(감정+의사결정)를 먼저 자극하면, 이후 논리에 수용적 상태가 된다.

"우리는 [신념/가치]를 믿습니다."
"우리는 [대상]이 [이상적 상태]에 도달할 수 있다고 확신합니다."
✕ 기존 방식

"안녕하세요, 저희 회사는 2015년에 설립된 공간기획 전문 회사로..."

회사 연혁 타임라인, 조직도, 수상 이력

✓ 하이브리드

"우리는 모든 공간이 미디어가 될 수 있다고 믿습니다."

한 문장. 풀 블리드 이미지 또는 미니멀 타이포.

02
Step 2
Big Shift — 맥락 설정
Andy Raskin — 5-Part Narrative  |  1~2장

우리 제품이 아닌 세상의 변화로 시작. 클라이언트가 이미 느끼고 있는 변화를 명명한다. "당신의 제품을 믿으라"가 아닌 "이 변화를 인정하라."

"[산업/시장]에 [거부할 수 없는 변화]가 일어나고 있습니다."
+ 이 변화를 증명하는 1개의 충격적 수치

안티패턴: 업계 상식의 반복 ("디지털 전환이 가속화되고 있습니다"), 데이터 3개 이상 (1개면 충분. 3개 이상은 기억 안 됨)

03
Step 3 — 승부처
시장 문제 해부
Miller SB7 + Raskin Winners/Losers + Gallo Villain First  |  3~5장

제안 PT의 승부처. 클라이언트의 문제를 그들보다 더 깊이 이해하고 있음을 증명하는 구간. 목표: "맞아, 바로 이게 우리 문제야."

4단계 해부 구조

3-1
현상 진단
클라이언트가 이미 체감하는 표면적 증상. 통계표가 아닌 감정적 증거 (사진, 스크린샷, 실제 사례).
3-2
원인 분석 (3중)
표면 아래의 구조적 원인 규명. 3중 Problem 구조를 적용하여 External + Internal + Philosophical 층위.
3-3
구조적 한계
기존 접근법이 왜 불충분한지. 빌런 = 낡은 관행 (경쟁사를 직접 공격하지 않음).
3-4
방치의 대가
이 문제를 방치하면? Loss Aversion 활용. 소금처럼 — 과하면 공포 마케팅.

3중 Problem 구조 — 가장 강력한 무기

External
표면적 문제
매장 방문객 감소
체류시간 단축
재방문율 하락
Internal — 실제 구매 동기
감정적 문제
"우리 브랜드가
잊혀지고 있다는 불안"
"뭔가 해야 하는데 방법을 모르겠다"
Philosophical
도덕적 부당함
"좋은 브랜드가 공간 경험
하나 때문에 저평가되어서는
안 된다"

핵심: External은 의식적이지만, Internal(감정)이 실제 구매 동기. 클라이언트의 감정적 고통을 정확히 짚는 것이 승부처다.

슬라이드 설계 원칙

원칙실행
감정적 증거 우선데이터표 < 실제 고객 후기, 매장 사진, 소비자 반응 캡처
클라이언트 언어 사용RFP/미팅에서 클라이언트가 쓴 단어를 그대로 인용
숫자의 인간화"매출 15% 하락" < "10명 중 1.5명이 다시 오지 않는다"
대비 활용Before/After, 승자/패자, 이상/현실 나란히

안티패턴: 문제를 클라이언트 탓으로 돌리기 (빌런은 구조/관행이지 클라이언트가 아님) · 너무 많은 문제 나열 (3개 이하) · 문제 분석 없이 바로 솔루션으로 점프

Phase B
비전과 해법
문제를 깊이 공감한 뒤, 아직 제품을 말하지 않고 "도달해야 할 미래"를 그린다. 갈망을 먼저 생성하고, 해법은 그 갈망 위에 얹는다.

Step 4
Promised Land — 약속의 땅
Andy Raskin — 5-Part Narrative  |  1~2장

우리 제품/서비스를 아직 언급하지 않고 매력적 미래상을 제시. 클라이언트가 "저기 가고 싶다"고 느끼게. 제품 소개를 지연시켜 갈망(desire)을 먼저 생성.

"만약 [이상적 상태]가 가능하다면?"
"[구체적 장면/경험]이 현실이 되는 세계"
✕ 기존 방식

"저희 솔루션은 A, B, C 기능을 제공하며..."

아직 제품을 소개할 때가 아님

✓ 하이브리드

"방문객이 매장을 나서며 SNS에 올리고 싶어지는 공간이 현실이 된다면?"

구체적 장면. 열망적 이미지. 밝은 톤.

Step 7 — 핵심
차별화 강점 어필
Sinek WHY→HOW + Miller Guide + Gallo Messianic Purpose  |  2~3장

강점을 "기능"이 아닌 "자격"으로 프레이밍. StoryBrand Guide 원칙: 공감(Empathy) + 권위(Authority) 동시 필요. 공감만 있으면 약해 보이고, 권위만 있으면 거만해 보인다.

3단계 어필 구조

♥
7-1. 공감 증명
"우리는 이 문제를 깊이 이해합니다"
• 클라이언트와 같은 고민을 해본 경험
• 업계 인사이트, 소비자 심리에 대한 이해
• Internal Problem을 정확히 짚는 것 자체가 공감 증명
• 클라이언트 언어 인용: "말씀하신 '브랜드 피로감'이 정확합니다"
⚖
7-2. 권위 증명
"그리고 해결할 역량이 있습니다"
• 성과 수치: "이 접근법으로 재방문율 12%→34%"
• 외부 검증: 수상, 미디어, 파트너십
• 방법론: "저희만의 [프레임워크명]으로 접근"
• 1개의 깊은 케이스 > 10개의 로고 나열
7-3. 관점의 차별화
"우리는 이 문제를 다르게 봅니다"
경쟁사와의 차이를 접근 방식/관점으로 설명. 직접 비교보다 프레임 전환이 효과적.

관점 전환: "매장이 아닌 미디어로 봅니다"
방법론 차별: "RXR 분석으로 소비자 감정을 수치화합니다"
경험 차별: "47개 공간 프로젝트에서 발견한 패턴"
구조 차별: "기획-디자인-운영이 분리되지 않는 원팀 구조"

안티패턴: 회사 연혁부터 시작 (연혁은 부록) · 자화자찬 ("업계 최고") — 수치로 보여줘라 · 경쟁사 직접 비하 — 빌런은 "관행"이지 "경쟁사"가 아님

Step 8
Magic Gifts — 기능 프레이밍
Andy Raskin — 5-Part Narrative  |  2~3장

기능을 나열하지 않고, 약속의 땅(Step 4)에 도달하는 도구로 프레이밍. 각 기능 = 영웅(클라이언트)에게 주어지는 마법의 도구. Rule of Three: 핵심 3개로 묶어라.

"공간 디자인 서비스"
→
"브랜드 메시지를 공간 언어로 번역하는 도구"
"소비자 분석 리포트"
→
"보이지 않는 소비자 감정을 수치로 만드는 렌즈"
"운영 컨설팅"
→
"오픈 후 6개월간 성과를 보장하는 가드레일"
[약속의 땅의 특정 측면]을 실현하기 위해
[우리의 기능/서비스]가 [구체적 방법]으로 작동합니다.
