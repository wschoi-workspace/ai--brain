## 테스트 케이스 (대화 내 완료, 미저장)

### 테스트 1: 올리브영 샵인샵 업세일링
- **과제**: 올리브영 매장 내 샵인샵 매대에서 업세일링 강화
- **분석 결과**:
  - 고객 심리 진단: 의사결정 피로 + 비주의 맹시 + 신뢰 부족 + 인지 부하
  - 문제 재정의: "멈추게 → 믿게 → 따르게 → 더 사게" 4단계
  - 5개 아이디어 도출:
    1. 주의 포착 (Gruen Transfer, 감각 마케팅, 프라이밍, 색채 대비)
    2. 3초 신뢰 (사회적 증거 뱃지, 권위, 대표성 휴리스틱)
    3. 선택 설계 (BEST 3 + 디폴트 + 미끼 효과 + 앵커링)
    4. 업세일링 트리거 (디드로/번들링/손실회피/소유효과/상호성/자이가르닉)
    5. BA 화법 설계 (ELM 주변경로, Yes 사다리, 프레이밍, 반발 방지)
  - 매대 설계 종합 프로토타입 (감각트리거→신뢰→앵커→메인→체험→업세일→종결)

### 테스트 2: 회사원 대상 북 큐레이션 서비스
- **과제**: 2030 직장인 대상 책 추천 + 구매 유도 서비스 설계
- **데이터 기반**: 기존 리서치 (`50-resources/2030-book-recommendation-2026-03.md`) 활용
  - 종합/소설/경제/자기계발/에세이/인문 각 30권 + 2030 추천사 완비
- **분석 결과**:
  - 5개 욕구 클러스터 발견: 의미/사유, 감정 회복, 돈/생존, 자기 리셋, 지적 쾌감
  - 문제 재정의: "서점"이 아니라 "북 처방전" 서비스
  - 5개 전략 도출:
    1. "오늘의 처방전" — 감정/상황 기반 5개 클러스터 × 2권 추천 (선택 설계)
    2. "페어링 세트" — 2권 번들 업세일링 5종 (디드로 + 심적 회계 + 앵커링)
    3. 세일즈 메시지 — 심리 레버별 카피 8종 (사회적 증거/손실회피/프레이밍/권위 등)
    4. 채널별 설계 — 오프라인 팝업 / 온라인 구독 3티어(미끼 효과) / B2B 기업복지
    5. 리텐션 — Hook 모델 습관 루프 + 자이가르닉 + 쾌락 적응 방지
