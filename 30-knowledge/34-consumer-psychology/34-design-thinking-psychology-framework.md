# 비즈니스 디자인씽킹 × 소비자 심리학 프레임워크
## Psychology-Powered Design Thinking Framework

> 디자인/서비스 아이디어를 구상할 때, 어떤 심리 원리를 적용할지 바로 찾아 쓰는 실전 참조 도구.
> 학습용 가이드는 → `34-consumer-decision-psychology.md`

---

## 사용법

**아이디어가 있을 때 이 순서로 활용:**

```
1. [Part A] 디자인씽킹 5단계에서 지금 내 단계를 찾는다
2. [Part B] 디자인 영역별 심리 레버를 확인한다
3. [Part C] 해당 이론의 핵심 원리와 적용 질문을 읽는다
4. [Part D] 심리 감사 체크리스트로 아이디어를 검증한다
```

# Part A. 디자인씽킹 5단계 × 심리학 도구

## Stage 1: 공감 (Empathize) — "고객이 진짜 원하는 게 뭔가?"

**목표**: 고객의 의식적 니즈 너머, 무의식적 동기와 비합리적 행동 패턴을 발견한다.

### 이 단계에서 활용할 심리 렌즈

| 렌즈 | 질문 | 핵심 원리 |
|------|------|-----------|
| 이중 처리 이론 | 고객이 이 결정을 System 1(직관)으로 하는가, System 2(분석)로 하는가? | 대부분의 소비 결정은 직관적. 고객이 "논리적으로 결정했다"고 말해도 실제로는 직관이 먼저 |
| 제한된 합리성 | 고객은 최적 선택을 하는가, 충분한 선택을 하는가? | "최고"가 아니라 "충분히 괜찮은" 것을 선택한다(satisficing) |
| 매슬로 욕구 위계 | 이 서비스는 어떤 욕구 층위를 건드리는가? | 기능적(생리/안전) vs 감정적(사회/존경) vs 자아실현적 |
| 조절 초점 이론 | 고객은 촉진(성장/열망) 동기인가, 예방(안전/손실방지) 동기인가? | 동기 유형에 따라 메시지와 경험 설계가 완전히 달라짐 |
| 소비자 여정 (CDJ) | 고객이 지금 여정의 어디에 있는가? | 인지→평가→구매→사후 각 단계에서 필요한 것이 다름 |
| 문화 심리학 | 타겟 고객의 문화적 소비 패턴은? | 한국: 집단주의+체면+사회적 증거 특히 강력 |
| 신체 표지 가설 | 고객이 이 카테고리에 대해 갖고 있는 감정적 기억(somatic marker)은? | 과거 경험이 형성한 무의식적 감정 반응이 선택을 필터링 |

### 공감 단계 실전 팁
- **고객이 말하는 것 vs 실제 행동의 차이**를 관찰하라 — 인지 부조화 때문에 자기 행동을 합리화하므로, 인터뷰보다 관찰이 더 정직하다
- **확증 편향 주의**: 내 가설을 확인하는 정보만 수집하지 않도록, "이 아이디어가 실패할 이유 10가지"를 먼저 적어본다
- **스키마 파악**: 고객이 이 카테고리에 대해 가진 기존 정신적 틀(스키마)은 무엇인가? 그 스키마를 활용할 것인가, 깨뜨릴 것인가?

## Stage 2: 정의 (Define) — "진짜 문제가 뭔가?"

**목표**: 심리학적 통찰을 바탕으로, 표면적 문제가 아닌 행동의 근본 원인을 재정의한다.

### 문제 재정의 프레임워크

```
[표면 문제] 고객이 우리 서비스에 가입하지 않는다
     ↓ 심리 렌즈 적용
[심리적 진단]
 - 인지 부하: 가입 절차가 너무 복잡한가? (S2 강제 호출)
 - 현상 유지 편향: 기존 서비스에서 바꾸기 싫은 건가?
 - 손실 회피: 전환 시 잃는 것(기존 데이터, 익숙함)이 두려운가?
 - 사회적 증거 부재: "다른 사람들도 쓰는가?"를 모르는 건가?
 - 신뢰 부족: 첫인상(디자인)에서 신뢰를 못 느끼는 건가?
     ↓
[재정의된 문제] 고객은 전환의 손실이 이득보다 크다고 느낀다
     ↓
[디자인 방향] 손실을 최소화하고, 이미 소유한 느낌을 먼저 준다
```

### 이 단계에서 활용할 심리 렌즈

| 렌즈 | 재정의 질문 |
|------|-------------|
| 인지 부하 / 의사결정 피로 | 고객이 우리 서비스 앞에서 멈추는 이유가 "복잡해서"는 아닌가? |
| 손실 회피 / 현상 유지 편향 | 고객이 안 바꾸는 이유가 "현재가 좋아서"가 아니라 "바꾸는 게 무서워서"는 아닌가? |
| 스키마 불일치 | 고객의 기존 기대(스키마)와 우리 서비스가 어긋나는 건 아닌가? |
| 계획된 행동 이론 | 태도/규범/통제 중 어느 것이 행동 전환을 막고 있는가? |
| 심리적 반발 | 우리가 너무 세게 밀어서 오히려 반발을 사고 있진 않은가? |

### JTBD × 심리학 통합 문장

```
"[상황]에서 [고객]이 [Job]을 해결하려 할 때,
[심리 원리 X] 때문에 [비합리적 행동 Y]를 하고 있다.
기존 대안은 [Z]이지만, [심리적 한계]를 해소하지 못한다.
→ 우리의 디자인 기회: [심리 원리를 활용/교정하는 솔루션]"
```

## Stage 3: 아이디어 (Ideate) — "심리학이 열어주는 솔루션은?"

**목표**: 62개 심리 원리를 조합하여, 인간 행동에 기반한 창의적 솔루션을 발상한다.

### 아이디어 발상 4가지 전략

| 전략 | 설명 | 심리 원리 조합 예시 |
|------|------|-------------------|
| **편승 (Ride)** | 기존 비합리적 행동을 활용하여 가치 창출 | 앵커링 + 미끼 효과 → 가격 전략 |
| **교정 (Correct)** | 비합리적 행동을 교정하여 새로운 서비스 | 선택의 역설 → 큐레이션 서비스 |
| **설계 (Design)** | 새로운 행동 패턴 자체를 설계 | 넛지 + 습관 루프 → 행동 변화 앱 |
| **방어 (Defend)** | 경쟁사의 심리 전략에 대한 방어 | 매몰비용 + 심리적 소유감 → 전환 장벽 |

### 아이디어 증폭 질문 (Amplification Questions)

**경험 설계 아이디어일 때:**
- 절정-종결 규칙: "고객 경험의 절정 순간과 마지막 순간을 의도적으로 설계했는가?"
- 감각 마케팅: "5감 중 아직 활용하지 않은 감각 채널은?"
- 프라이밍: "고객이 첫 접점에서 받는 인상이 이후 경험을 어떻게 프라이밍하는가?"
- 쾌락 적응: "시간이 지나도 신선함을 유지하는 장치가 있는가?"

**서비스 설계 아이디어일 때:**
- 넛지/선택 설계: "디폴트 옵션은 고객에게 최선인가?"
- 습관 형성: "트리거→행동→보상→투자 루프가 완성되는가?"
- 사회적 증거: "다른 사람들의 선택을 보여주고 있는가?"
- 쌍곡선 할인: "장기 혜택만 말하고 있진 않은가? 즉시 보상은?"

**가격/수익 모델 아이디어일 때:**
- 심적 회계: "고객의 어떤 심리적 계좌에서 지출되는가?"
- 프레이밍: "가격을 어떤 프레임으로 제시하고 있는가? (일 단위? 커피 한 잔?)"
- 번들링: "비용은 합치고, 혜택은 분리해서 보여주고 있는가?"
- 공정성: "고객이 이 가격을 '공정하다'고 느끼는가?"

**브랜드/커뮤니케이션 아이디어일 때:**
- ELM: "중심 경로(논리)와 주변 경로(감성) 중 어디로 설득하는가?"
- 브랜드 개성: "브랜드의 성격(성실/흥분/유능/세련/강인)은?"
- 사회적 정체성: "이 브랜드를 쓰면 '나는 어떤 사람'이 되는가?"
- 단순 노출: "브랜드를 반복 노출할 자연스러운 접점은?"

**SNS/온라인 커뮤니케이션 아이디어일 때:**
- eWOM 논거 품질: "양 대신 품질 높은 리뷰를 유도하는 구조가 있는가? (69개 연구: 논거 품질 ★★★★★ > 리뷰 양 ★★)"
- 행동적 vs 태도적 사회증거: "'N명 구매'(행동) vs '인기 1위'(태도) 중 어떤 증거를 보여주고 있는가?"
- 파라소셜 관계: "마이크로 인플루언서의 팔로워와의 심리적 거리가 가까운가? 메가보다 마이크로가 전환율 높은 이유"
- 부정성 편향: "부정 리뷰를 삭제하고 있는가? 공개 응대가 더 효과적(감정적 진단성)"
- 정체성 시그널: "이 콘텐츠를 공유하면 사용자가 어떤 정체성을 표현하게 되는가? (취향/라이프스타일/가치관/전문성)"
- 한국 맥락 사고: "장면→맥락→제품 순서로 보여주고 있는가? 제품 스펙보다 사용 상황이 먼저(전체적 사고)"

## Stage 4: 프로토타입 (Prototype) — "심리적으로 작동하는가?"

**목표**: 아이디어를 구체적 설계로 전환할 때, 심리 원리가 올바르게 작동하도록 세부 설계한다.

### 프로토타입 심리 설계 체크포인트

#### 4-1. 첫인상 설계 (First 3 Seconds)
| 체크 | 적용 원리 |
|------|-----------|
| 3초 안에 신뢰감을 주는가? | 신뢰성 휴리스틱: 디자인 품질 = 서비스 품질로 인식 |
| 첫 시각 자극이 의도한 감정을 프라이밍하는가? | 프라이밍 효과 |
| 기존 스키마를 활용하여 즉시 이해 가능한가? | 스키마 이론: "○○의 우버" 식 포지셔닝 |
| 색상이 의도한 감정/카테고리를 전달하는가? | 색채 심리학 |

#### 4-2. 선택 구조 설계 (Choice Architecture)
| 체크 | 적용 원리 |
|------|-----------|
| 옵션이 3-5개 이내인가? | 선택의 역설 + 인지 부하 |
| 추천 옵션이 명확히 표시되어 있는가? | 넛지/디폴트 효과 |
| 미끼 옵션이 전략적으로 배치되어 있는가? | 미끼 효과(Decoy) |
| 가장 중요한 것이 첫/끝에 있는가? | 초두/최신 효과 |
| 앵커 가격이 먼저 제시되는가? | 앵커링 효과 |

#### 4-3. 행동 유도 설계 (Behavioral Triggers)
| 체크 | 적용 원리 |
|------|-----------|
| 핵심 CTA가 1개로 명확한가? | 인지 부하 최소화 |
| "잃을 수 있는 것"이 표현되어 있는가? | 손실 회피 |
| 사회적 증거가 보이는가? | Cialdini 사회적 증거 |
| 즉시 보상이 있는가? | 쌍곡선 할인 |
| 미완결 요소가 다음 행동을 유도하는가? | 자이가르닉 효과 |

#### 4-4. 경험 곡선 설계 (Experience Curve)
| 체크 | 적용 원리 |
|------|-----------|
| 감정적 절정(peak) 순간이 의도적으로 설계되어 있는가? | 절정-종결 규칙 |
| 마지막 인상이 긍정적인가? | 절정-종결 규칙 |
| 오감 중 2개 이상을 활용하는가? | 감각 마케팅 |
| 고객이 직접 참여/만드는 요소가 있는가? | IKEA 효과 + 심리적 소유감 |
| 시간이 지나도 신선함을 유지하는 변화 요소가 있는가? | 쾌락 적응 방지 |

#### 4-5. 디지털/SNS 설계 (Digital & eWOM)
| 체크 | 적용 원리 |
|------|-----------|
| 결제까지 System 2가 개입할 이유가 제거되었는가? | System 1 마찰 제거 (Amazon 1-Click, 토스 3탭) |
| 리뷰 시스템이 다층 신뢰 구조(별점→텍스트→인증→2차 증거→프로필)인가? | eWOM 다층 신뢰 아키텍처 |
| 리뷰 요청 타이밍이 구매 후 2-3일인가? | 구매 후 합리화 골든 타이밍 |
| "N명 구매"(행동적 증거)를 보여주고 있는가? | 행동적 > 태도적 사회증거 |
| 희소성+FOMO+사회적 증거가 3중 결합되어 있는가? | Booking.com 3중 결합 = 전환 2-3배↑ |
| 목적 구매→탐색 구매 전환을 유도하는 디지털 그루엔 장치가 있는가? | 추천 알고리즘, 무한 스크롤, "함께 본 상품" |
| SNS 공유 시 정체성 시그널이 작동하는가? ("이걸 공유하면 나를 어떻게 볼까?") | 정체성 시그널 + 사회적 정체성 |
| 한국 소비자에게 맥락(장면/상황)이 제품보다 먼저 보이는가? | 전체적 사고 (Nisbett) |

## Stage 5: 테스트 (Test) — "실제로 행동이 바뀌는가?"

**목표**: 프로토타입이 의도한 심리적 효과를 실제로 만들어내는지 검증한다.

### 테스트 프레임워크

#### 관찰해야 할 것
| 관찰 포인트 | 관련 심리 원리 | 측정 방법 |
|-------------|---------------|-----------|
| 첫 반응 (3초) | 신뢰성 휴리스틱, 프라이밍 | 첫인상 평가, 이탈률 |
| 선택 패턴 | 디폴트, 미끼, 앵커링 | 옵션별 선택률 |
| 체류/몰입 | Gruen Transfer, Atmospherics | 체류시간, 이동 경로 |
| 감정 반응 | 신체 표지, 감각 마케팅 | 표정, 언어, NPS |
| 구매 전환 | 손실 회피, 소유 효과, 사회적 증거 | 전환율, 객단가 |
| 재방문/습관 | Hook 모델, 쾌락 적응 | 재방문률, DAU/MAU |
| 구전/공유 | 사회적 정체성, Cialdini(Unity) | 추천률, UGC |

#### 주의할 편향
- **확증 편향**: "잘 됐다"는 증거만 찾지 않기. 실패 신호를 적극 탐색
- **가용성 휴리스틱**: 극단적 사례(대성공/대실패) 하나에 전체 판단을 의존하지 않기
- **매몰비용**: "여기까지 왔으니" 계속하지 않기. 데이터가 안 되면 피벗

# Part B. 디자인 영역별 심리 레버 매트릭스

## B-1. 공간/경험 디자인

> 공간을 설계하거나 오프라인 경험을 기획할 때

| 공간 요소 | 핵심 심리 레버 | 설계 원칙 |
|-----------|---------------|-----------|
| **입구/파사드** | 프라이밍 + 신뢰성 휴리스틱 + 스키마 | 첫 3초에 브랜드 정체성 전달. 기대감 형성. 시각적 품질 = 서비스 품질 인식 |
| **동선** | Gruen Transfer + 의사결정 피로 + 초두/최신 | 목적→탐색 전환 유도. 고가품은 초입, 충동품은 후반. 핵심 상품은 동선의 시작과 끝 |
| **진열/배치** | 게슈탈트 + 앵커링 + 미끼 효과 | 근접성으로 그룹핑. 프리미엄→스탠다드 순서. 전략적 미끼 옵션 배치 |
| **체험존** | 소유 효과 + IKEA 효과 + 감각 마케팅 | 터치&트라이, DIY 체험. 오감 자극 설계. "만지면 내 것이 된다" |
| **분위기** | Atmospherics + 색채 심리 + 프라이밍 | PAD 모델(쾌감-각성-통제). 목적에 맞는 감각 조합. 시그니처 향/음악 |
| **체류 유도** | 쾌락 적응 방지 + 리테일 테라피 | 계절별 변화, 발견의 요소. 쇼핑 자체를 치유 경험으로 |
| **계산/퇴장** | 절정-종결 + 인지 부조화 + 자이가르닉 | 마지막 감동 순간 설계. "좋은 선택" 확인. 재방문 동기(미완 과제) |
| **연쇄 구매** | 디드로 효과 + 번들링 | "이것에 어울리는..." 크로스셀링. 통일된 미학으로 라이프스타일 제안 |

## B-2. 서비스/앱/플랫폼 디자인

> 디지털 서비스, 앱, 온라인 플랫폼을 설계할 때

| 서비스 요소 | 핵심 심리 레버 | 설계 원칙 |
|-------------|---------------|-----------|
| **온보딩** | 인지 부하 + 디폴트 + 쌍곡선 할인 | 3단계 이내. 최적 디폴트 세팅. 가입 즉시 보상(첫 혜택) |
| **핵심 기능** | System 1 설계 + 습관 루프 | 핵심 행동은 직관적으로. 트리거→행동→보상→투자 루프 설계 |
| **가격 페이지** | 앵커링 + 미끼 + 프레이밍 | 최고가 먼저(앵커). 전략적 미끼 플랜. "하루 ○원" 프레이밍 |
| **전환 유도** | 손실 회피 + 소유 효과 + 사회적 증거 | 무료 체험 후 "이 기능을 잃게 됩니다". "N명이 이용 중". 먼저 체험시켜 소유감 |
| **리텐션** | 매몰비용 + 심리적 소유감 + 쾌락 적응 방지 | 사용 데이터 축적(전환비용). 커스터마이징. 주기적 새로움(추천, 업데이트) |
| **알림/재참여** | 자이가르닉 + 가변 보상 + FOMO(손실 회피) | 미완료 과제 알림. 예측 불가한 보상. "놓치고 있는 것" |
| **구독 갱신** | 현상 유지 편향 + 디폴트 | 자동 갱신 디폴트 (단, 윤리적으로). 해지 시 손실 시각화 |
| **리뷰/추천** | 구매 후 합리화 + 사회적 정체성 | 구매 직후(합리화 동기 최고) 리뷰 요청. 커뮤니티 소속감 |
| **리뷰 신뢰 설계** | eWOM 다층 신뢰 + ELM + 부정성 편향 | 5층 구조: ①별점(주변경로) → ②텍스트(중심경로) → ③구매인증(권위) → ④"N명 도움됨"(2차 사회적 증거) → ⑤리뷰어 프로필(출처 신뢰). 논거 품질 > 리뷰 양(Ismagilova 메타분석). 부정 리뷰 삭제보다 공개 응대가 ROI↑ |
| **SNS 공유 유도** | 정체성 시그널 + FOMO + 행동적 사회증거 | "이걸 공유하면 나를 어떻게 볼까?" 설계. 행동적 증거("N명 구매") > 태도적 증거("인기 1위"). 희소성+FOMO+사회적 증거 3중 결합 시 전환 2-3배↑ |

## B-3. 가격/수익 모델 디자인

> 가격을 책정하거나 수익 모델을 설계할 때

| 가격 요소 | 핵심 심리 레버 | 설계 원칙 |
|-----------|---------------|-----------|
| **가격 수준** | 가격-품질 휴리스틱 + 베블런 효과 | 경험재는 높은 가격 = 높은 품질 인식. 럭셔리는 가격이 곧 가치 |
| **가격 표시** | 앵커링 + 프레이밍 + Weber-Fechner | 정가(앵커) 먼저 표시. 저가: % 할인, 고가: 원 할인. 시간 단위 분할 |
| **옵션 구조** | 미끼 효과 + 선택의 역설 | Good-Better-Best 3단계. 중간 옵션으로 유도하는 미끼 |
| **번들링** | 심적 회계 + 전망 이론 | 비용은 통합(하나로), 혜택은 분리(여러 개로). 여행/경험 계좌 활용 |
| **할인 전략** | 참조 의존 + 희소성 + 공정성 | 할인 이유 제공. 과도한 할인은 품질 의심 유발. 기간/수량 한정 |
| **결제 설계** | 결제 고통(뇌섬엽) + 쌍곡선 할인 | 결제를 경험과 분리(선불, 구독). BNPL로 미래 비용 과소평가 활용 |
| **멤버십/구독** | 매몰비용 + 손실 회피 + 현상 유지 | 연회비 → 본전 동기. 해지 시 잃는 혜택 강조. 자동 갱신 |

## B-4. 브랜드/커뮤니케이션 디자인

> 브랜드를 만들거나, 마케팅 메시지를 설계할 때

| 브랜드 요소 | 핵심 심리 레버 | 설계 원칙 |
|-------------|---------------|-----------|
| **브랜드 인지** | 단순 노출 + 가용성 휴리스틱 | 반복 노출로 친숙함 확보. TOM(떠오르는 브랜드 = 시장 리더) |
| **브랜드 신뢰** | 신뢰성 휴리스틱 + 권위 원칙 | 디자인 품질, 인증/수상, 전문가 추천, 언론 보도 |
| **브랜드 개성** | 브랜드 심리학(Aaker) + 사회적 정체성 | 5개 차원(성실/흥분/유능/세련/강인) 중 포지셔닝. 타겟 집단의 정체성 반영 |
| **광고 메시지** | ELM + 조절 초점 + 프레이밍 | 관여도에 맞는 경로(중심/주변). 동기 매칭(촉진/예방). 이득/손실 프레임 선택 |
| **스토리텔링** | 신체 표지 가설 + 기억 시스템 | 감정을 유발하는 스토리 → somatic marker 형성 → 무의식적 선택 우위 |
| **바이럴/구전** | 사회적 정체성 + 반발 이론 | "이 브랜드를 공유하면 나는 ○○한 사람" 설계. 과도한 Push는 반발 유발 |
| **로열티** | 확증 편향 + 브랜드 애착 + 일관성 | 팬은 장점만 수집. 커뮤니티로 소속감 강화. 공개 선언/참여 유도 |
| **온라인 리뷰 관리** | eWOM 부정성 편향 + 감정적 진단성 + 설득 지식 | 부정 리뷰 삭제보다 공개 응대가 ROI↑(비대칭 효과). 소비자의 리뷰 조작 탐지 능력 증가(persuasion knowledge) — 별5만 있으면 오히려 의심. 리뷰 요청은 구매 후 2~3일(합리화 골든타이밍) |
| **SNS 콘텐츠** | 전체적 사고(한국) + 정체성 시그널 + 참여형 일관성 | 장면(맥락) 먼저, 제품 나중에. 숫자 증거→맥락 증거 전환("10만 판매"→"30대 워킹맘 재구매 이유"). 댓글 참여 = 일관성 압력 + 무료 시장조사. 공유 확률 = 품질 × 정체성 부합도 × 사회적 안전성 |

## B-5. 커뮤니티/멤버십 디자인

> 멤버십 서비스나 커뮤니티 공간을 설계할 때

| 커뮤니티 요소 | 핵심 심리 레버 | 설계 원칙 |
|---------------|---------------|-----------|
| **가입 동기** | 사회적 정체성 + 매슬로(사회/존경) | "이 커뮤니티에 속하면 나는 ○○한 사람이 된다" |
| **진입 장벽** | 인지 부조화 + 매몰비용 + 스놉 효과 | 적절한 가입 허들 → "노력해서 들어왔으니 가치 있다" 합리화 |
| **소속감** | Cialdini(Unity) + 사회적 증거 | "우리"라는 정체성. 멤버 활동 시각화. 공동 의식/전통 |
| **활동 유지** | 습관 루프 + 가변 보상 + 자이가르닉 | 정기 이벤트(트리거). 예측 불가 혜택. 진행 중인 프로그램 |
| **배타성 유지** | 밴드왜건 vs 스놉 균형 + 럭셔리의 역설 | 접근성(성장)과 배타성(가치) 사이 긴장 관리 |
| **이탈 방지** | 심리적 소유감 + 매몰비용 + 손실 회피 | 축적된 관계/데이터/히스토리가 전환 장벽. "잃게 되는 것" 시각화 |

# Part C. 62개 이론 실전 레퍼런스 카드

> 각 이론을 한 눈에 파악하고, 아이디어에 바로 적용할 수 있는 요약 카드

## 인지 기반 (How the mind works)

### C-01. 이중 처리 이론 (System 1 / System 2)
- **한 줄**: 직관적 사고(S1)가 대부분의 소비 결정을 주도한다
- **디자인 적용 질문**: "이 경험이 System 1으로 처리 가능한가, 아니면 고객에게 생각을 강요하는가?"
- **활용**: S1에 맡길 것(동선, 기본 선택)과 S2를 의도적으로 호출할 것(프리미엄 차별화)을 구분

### C-02. 인지 부하 (Cognitive Load)
- **한 줄**: 작업 기억 용량(7±2)을 초과하면 판단 품질이 급락
- **디자인 적용 질문**: "한 화면/한 순간에 고객에게 몇 가지를 요구하고 있는가?"
- **활용**: CTA는 1개, 카테고리는 7개 이내, 정보는 progressive disclosure

### C-03. 의사결정 피로 (Decision Fatigue)
- **한 줄**: 결정을 많이 할수록 이후 결정의 질이 떨어짐
- **디자인 적용 질문**: "고객이 우리에게 도달하기 전에 이미 몇 개의 결정을 했는가?"
- **활용**: 복잡한 결정은 여정 초반에, 충동 구매는 후반에

### C-04. 인지 부조화 (Cognitive Dissonance)
- **한 줄**: 행동과 신념이 모순되면 불편해서 합리화한다
- **디자인 적용 질문**: "구매 후 고객이 '잘한 선택'이라고 느끼게 돕고 있는가?"
- **활용**: 구매 확인 메시지, 멤버십 가입비의 활용 동기, ESG 죄책감 해소 포지셔닝

### C-05. 제한된 합리성 (Bounded Rationality)
- **한 줄**: 고객은 "최고"가 아니라 "충분히 괜찮은" 것을 고른다
- **디자인 적용 질문**: "모든 스펙을 나열하고 있진 않은가? 핵심 3개만 말하고 있는가?"
- **활용**: "이 가격대에서 가장 무난한" 포지셔닝이 때론 "절대 최고"보다 효과적

### C-06. 선택적 주의 (Selective Attention)
- **한 줄**: 한 번에 하나의 정보 채널에만 깊이 집중 가능
- **디자인 적용 질문**: "핵심 메시지가 고객의 주의를 독점하고 있는가, 경쟁하고 있는가?"
- **활용**: 이메일 제목 첫 5단어가 전부. 매장에서 조명/색상으로 핵심 상품에 주의 집중

### C-07. 비주의 맹시 (Inattentional Blindness)
- **한 줄**: 주의 밖의 것은 눈앞에 있어도 보이지 않는다
- **디자인 적용 질문**: "고객이 목적지를 향해 이동 중일 때, 중간 메시지를 인지할 수 있는가?"
- **활용**: 배너 맹시 우회(네이티브 광고). 멈춤 지점에 메시지 배치

### C-08. 게슈탈트 지각 원리 (Gestalt Principles)
- **한 줄**: 전체 패턴을 부분보다 우선 인식
- **디자인 적용 질문**: "시각적 그룹핑이 의도한 정보 구조와 일치하는가?"
- **활용**: 근접성(관련 상품 묶음), 유사성(가격대별 색상), 폐합(미니멀 로고), 연속성(동선)

### C-09. 프라이밍 효과 (Priming)
- **한 줄**: 먼저 접한 자극이 이후 판단에 무의식적 영향
- **디자인 적용 질문**: "고객이 첫 접점에서 받는 자극이 이후 경험을 어떤 방향으로 프라이밍하는가?"
- **활용**: 매장 입구 아트워크(고급감 프라이밍). 카탈로그 첫 페이지에 프리미엄 상품

### C-10. 단순 노출 효과 (Mere Exposure)
- **한 줄**: 반복 노출만으로 호감도가 증가한다 (친숙함 = 안전함)
- **디자인 적용 질문**: "런칭 전에 로고/이름을 충분히 노출하는 사전 작업이 있는가?"
- **활용**: 티저 캠페인. 공간 내 브랜드 패턴 반복. 일상 접점에서의 자연스러운 노출

### C-11. 기억 시스템 (Memory Systems)
- **한 줄**: 감각→작업(15초, 4±1 chunk)→장기 기억, 감정과 결합 시 강화
- **디자인 적용 질문**: "핵심 메시지가 3개 이내의 chunk로 구성되어 있는가?"
- **활용**: "가볍다, 빠르다, 싸다". 감동적 순간은 에피소드 기억으로 강하게 각인

### C-12. 스키마 이론 (Schema Theory)
- **한 줄**: 새 정보는 기존 정신적 틀에 맞춰 해석된다
- **디자인 적용 질문**: "고객의 기존 스키마에 기대어 설명하는가, 아니면 새로운 카테고리를 강요하는가?"
- **활용**: "에어비앤비 = 숙박의 우버". 스키마를 따르되 한 가지를 의도적으로 위반하면 기억에 남음

### C-13. 초두/최신 효과 (Serial Position)
- **한 줄**: 처음과 마지막이 가장 잘 기억된다
- **디자인 적용 질문**: "가장 중요한 것이 첫 번째 또는 마지막에 있는가?"
- **활용**: 메뉴 첫/끝에 마진 높은 메뉴. 프레젠테이션 시작/끝에 핵심 메시지

## 경제적 판단 (Why decisions are irrational)

### C-14. 가용성 휴리스틱 (Availability Heuristic)
- **한 줄**: 쉽게 떠오르면 자주 일어난다고 오판
- **디자인 적용 질문**: "고객 머릿속에 우리 브랜드가 쉽게 떠오르는가?"
- **활용**: 생생한 고객 후기 > 통계 데이터. TOM(최초 상기) 전략

### C-15. 대표성 휴리스틱 (Representativeness)
- **한 줄**: "고급 브랜드처럼 보이면" 고급으로 인식
- **디자인 적용 질문**: "우리 디자인이 목표 카테고리의 '전형'에 부합하는가?"
- **활용**: 패키징의 시각적 코드(무광, 무게감, 미니멀 = 프리미엄). 타겟과 닮은 모델

### C-16. 확증 편향 (Confirmation Bias)
- **한 줄**: 기존 신념을 확인하는 정보만 수집
- **디자인 적용 질문**: "내가 이 아이디어에 확증 편향에 빠져 있진 않은가?"
- **활용**: 사업 기획 시 pre-mortem("실패 이유 10가지"). 고객 리서치 시 중립 질문

### C-17. 앵커링 효과 (Anchoring)
- **한 줄**: 처음 제시된 숫자가 이후 판단의 기준
- **디자인 적용 질문**: "고객이 가장 먼저 보는 숫자가 무엇인가?"
- **활용**: ~~129,000원~~ 89,000원. 협상에서 먼저 숫자를 제시. 프리미엄을 앵커로 제시 후 스탠다드

### C-18. 전망 이론 (Prospect Theory)
- **한 줄**: 절대 가치가 아니라 참조점 대비 변화에 반응. 손실 > 이득
- **디자인 적용 질문**: "고객의 참조점(기대치)은 어디인가? 그것 대비 이득/손실로 프레이밍하고 있는가?"
- **활용**: 환불 정책으로 참조점을 "이미 소유"로 이동. 손실 프레임 vs 이득 프레임 전략적 선택

### C-19. 손실 회피 (Loss Aversion)
- **한 줄**: 잃는 고통 ≈ 얻는 기쁨의 2배
- **디자인 적용 질문**: "고객에게 '얻는 것'만 말하고 있는가? '잃을 수 있는 것'도 말하고 있는가?"
- **활용**: 무료 체험 후 "이 기능을 잃게 됩니다". 한정 기회 "놓치면 6개월 뒤"

### C-20. 소유 효과 (Endowment Effect)
- **한 줄**: 내 것이 된 순간 가치가 올라간다
- **디자인 적용 질문**: "구매 전에 '이미 내 것'이라는 느낌을 줄 수 있는가?"
- **활용**: 매장에서 직접 만지기. AR 가상 배치. 커스터마이징으로 심리적 소유감

### C-21. 심적 회계 (Mental Accounting)
- **한 줄**: 돈을 심리적 계좌별로 분리 관리한다
- **디자인 적용 질문**: "이 지출이 고객의 어떤 심리적 계좌에서 나오는가?"
- **활용**: 비용 합치기 + 혜택 쪼개기. 여행/선물/자기계발 계좌는 지갑이 열린다

### C-22. 프레이밍 효과 (Framing)
- **한 줄**: 같은 정보도 표현 방식에 따라 완전히 다른 판단
- **디자인 적용 질문**: "이 메시지를 이득/손실/중립 중 어떤 프레임으로 전달하고 있는가?"
- **활용**: "80% 무지방" vs "지방 20%". "하루 1,600원" vs "월 49,000원"

### C-23. 현상 유지 편향 / 디폴트 (Status Quo / Default)
- **한 줄**: 디폴트가 인간 행동의 가장 강력한 예측 변수
- **디자인 적용 질문**: "디폴트 옵션이 고객과 우리 모두에게 최선인가?"
- **활용**: 자동 갱신(윤리적으로). 추천 설정. 옵트아웃 > 옵트인 설계

### C-24. 쌍곡선 할인 (Hyperbolic Discounting)
- **한 줄**: "지금"의 가치를 비합리적으로 높게 평가
- **디자인 적용 질문**: "장기 혜택만 강조하고 있지 않은가? 즉시 체감되는 보상은?"
- **활용**: 가입 즉시 사용 가능한 혜택. BNPL. "첫 주에 느낄 수 있는 변화"

### C-25. 매몰비용 오류 (Sunk Cost)
- **한 줄**: 이미 투자한 것이 아까워서 비합리적으로 지속
- **디자인 적용 질문**: "고객이 우리 서비스에 투자한 시간/노력/돈을 느끼게 하고 있는가?"
- **활용**: 연회비(본전 동기). 게임화(레벨/포인트 축적). 커스터마이징 투자

### C-26. Weber-Fechner / 참조 의존
- **한 줄**: 변화 인지는 원래 크기에 비례 (비율이 중요)
- **디자인 적용 질문**: "할인을 %로 보여줄지, 절대액으로 보여줄지 어느 쪽이 더 크게 느껴지는가?"
- **활용**: 저가 상품은 % 할인, 고가 상품은 원 할인. 가격 인상 시 JND(최소 인지 차이) 이내로

### C-27. 미끼 효과 (Decoy Effect)
- **한 줄**: 열등 옵션을 추가하면 특정 옵션의 선택률이 올라감
- **디자인 적용 질문**: "3개 옵션 중 미끼 역할을 하는 옵션이 의도적으로 설계되어 있는가?"
- **활용**: SaaS 가격표의 전략적 미끼 플랜. 부동산 투어에서 비교 매물

### C-28. 선택의 역설 (Paradox of Choice)
- **한 줄**: 선택지가 많으면 만족도 하락 + 선택 마비
- **디자인 적용 질문**: "고객에게 몇 개의 선택지를 주고 있는가? 7개를 넘지 않는가?"
- **활용**: SKU 줄이기. 메뉴 7±2개. "당신을 위한 추천 3개" 큐레이션

### C-29. IKEA 효과
- **한 줄**: 직접 만든 것에 비합리적으로 높은 가치를 부여
- **디자인 적용 질문**: "고객이 직접 참여하거나 만드는 요소가 있는가?"
- **활용**: DIY 워크숍. 밀키트. 대시보드 커스터마이징. 단, 완성까지 도달해야 효과

### C-30. 절정-종결 규칙 (Peak-End Rule)
- **한 줄**: 경험 평가 = 가장 강렬한 순간 + 마지막 순간
- **디자인 적용 질문**: "경험의 절정(감동/놀라움)과 끝(마지막 인상)을 의도적으로 설계했는가?"
- **활용**: 서프라이즈 디저트(절정). 체크아웃 선물(종결). 불만 처리 후 감사 쿠폰(종결)

### C-31. 쾌락 적응 (Hedonic Adaptation)
- **한 줄**: 어떤 좋은 것도 시간이 지나면 당연해진다
- **디자인 적용 질문**: "이 경험의 신선함이 시간이 지나도 유지되는가?"
- **활용**: 경험 > 물건 (적응이 느림). 구독 서비스의 지속적 새로움. 계절별 공간 변화

### C-32. 공정성 / 불평등 회피 (Fairness)
- **한 줄**: 불공정하면 자기 손해를 감수하고도 거부
- **디자인 적용 질문**: "이 가격/서비스가 고객에게 '공정하다'고 인식되는가?"
- **활용**: 다이나믹 프라이싱의 정당화. 투명한 보상. 가격 차별의 합리적 프레이밍

### C-33. 넛지 / 선택 설계 (Nudge)
- **한 줄**: 자유를 유지하면서 더 나은 선택을 유도하는 환경 설계
- **디자인 적용 질문**: "선택의 자유를 제한하지 않으면서, 최선의 옵션을 자연스럽게 유도하고 있는가?"
- **활용**: 건강한 옵션을 눈높이에. 추천 옵션 하이라이트. 사회적 비교 넛지

### C-34. 자유주의적 개입주의 (Libertarian Paternalism)
- **한 줄**: 넛지의 윤리: 다크 패턴이 아닌, 고객 이익을 위한 설계
- **디자인 적용 질문**: "이 디폴트/넛지가 고객의 최선의 이익에 부합하는가, 우리 이익만을 위한 조작인가?"
- **활용**: 윤리적 디폴트 기준. 단기 매출 vs 장기 신뢰. ESG 넛지

## 소비 행동 (What consumers do)

### C-35. 소비자 의사결정 여정 (CDJ)
- **한 줄**: 구매는 깔때기가 아니라 순환적 여정
- **디자인 적용 질문**: "각 여정 단계(인지→평가→구매→사후→충성)에 맞는 경험을 설계했는가?"
- **활용**: JTBD 결합 — 각 단계에서 고객이 고용하는 Job이 다름

### C-36. 정교화 가능성 모델 (ELM)
- **한 줄**: 관여도 높으면 논리로, 낮으면 감성으로 설득
- **디자인 적용 질문**: "이 제품/서비스에 대한 고객의 관여도가 높은가 낮은가?"
- **활용**: 고관여(B2B, 고가) → 백서, ROI 데이터. 저관여(일상품) → 감성 이미지, 연예인

### C-37. 계획된 행동 이론 (TPB)
- **한 줄**: 행동 = 태도 + 주관적 규범 + 인지된 통제
- **디자인 적용 질문**: "태도/규범/통제 중 어느 것이 가장 약한 고리인가?"
- **활용**: 태도 OK + 규범 약 → "N명이 이미..." 통제 약 → "초보자도 쉽게" 메시지

### C-38. 매슬로 욕구 위계 (Maslow)
- **한 줄**: 기능적(생리/안전) → 감정적(사회/존경) → 자아실현
- **디자인 적용 질문**: "이 서비스는 어떤 욕구 층위에 호소하고 있는가? 한 단계 위를 노릴 수 있는가?"
- **활용**: 같은 자동차: 안전(볼보), 소속(미니), 존경(벤츠), 자아실현(테슬라)

### C-39. 신체 표지 가설 (Somatic Marker)
- **한 줄**: 과거 경험이 만든 감정적 반응이 선택을 자동 필터링
- **디자인 적용 질문**: "우리 브랜드/공간이 어떤 감정적 기억을 형성하고 있는가?"
- **활용**: 브랜드-감정 결합(안전, 흥분, 향수). 공간의 감각 자극으로 긍정적 marker 형성

### C-40. 습관 형성 (Hook / Habit Loop)
- **한 줄**: 트리거→행동→보상→투자의 반복이 습관을 만든다
- **디자인 적용 질문**: "외부 트리거에서 내부 트리거(습관)로 전환되는 설계가 있는가?"
- **활용**: 습관 형성 초기 21-66일에 보상 빈도를 높이고 마찰 최소화

### C-41. 브랜드 심리학 / 애착 (Brand Psychology)
- **한 줄**: 브랜드 = 기억 속 연상 네트워크 + 자아와의 유대
- **디자인 적용 질문**: "고객이 이 브랜드를 떠올릴 때 어떤 연상이 활성화되는가?"
- **활용**: 기능적+감정적+자기표현적 편익 통합. 팬덤 기반 커뮤니티

### C-42. 신뢰성 휴리스틱 (Trust Heuristics)
- **한 줄**: 첫 3초에 70%의 신뢰 판단이 완료됨
- **디자인 적용 질문**: "첫인상(디자인, 인터페이스, 공간)이 전문적 신뢰감을 주는가?"
- **활용**: 디자인 품질 = 서비스 품질 인식. 인증/수상 뱃지. 실명+사진 후기

### C-43. 심리적 소유감 (Psychological Ownership)
- **한 줄**: 통제/지식/자아투자 세 경로로 "내 것" 느낌 형성
- **디자인 적용 질문**: "법적 소유 전에 심리적 소유감을 형성할 수 있는 접점이 있는가?"
- **활용**: 커스터마이징(자아 투자). 데이터 축적(친밀한 지식). 직접 조작(통제)

### C-44. 가격-품질 휴리스틱 (Price-Quality)
- **한 줄**: 비싼 것 = 좋은 것 (특히 판단이 어려운 경험재에서)
- **디자인 적용 질문**: "가격이 품질 신호로 작동하는 카테고리인가? 의도적으로 높여야 하는가?"
- **활용**: 와인, 화장품, 컨설팅에서 가격 인상 = 품질 인식 상승. 과도한 할인의 역효과

### C-45. 조절 초점 이론 (Regulatory Focus)
- **한 줄**: 촉진(얻는 것)과 예방(잃지 않는 것), 두 가지 동기 체계
- **디자인 적용 질문**: "타겟 고객이 촉진형(도전/성장)인가 예방형(안전/보호)인가?"
- **활용**: 여행=촉진 메시지, 보험=예방 메시지. 고객 세분화 기준

### C-46. 소비자 신경과학 (Consumer Neuroscience)
- **한 줄**: 도파민은 보상 자체가 아니라 "기대"에 반응
- **디자인 적용 질문**: "기대감을 극대화하는 설계가 있는가? (언박싱, 카운트다운, 예고)"
- **활용**: 럭셔리 언박싱. 사전 예약 기간의 기대감 콘텐츠. 결제 고통 분리(선불/구독)

### C-47. 구매 후 합리화 (Post-Purchase Rationalization)
- **한 줄**: 구매 직후 "잘한 선택"을 확인하고 싶어한다
- **디자인 적용 질문**: "구매 직후에 합리화를 도와주는 콘텐츠가 있는가?"
- **활용**: "5가지 활용법" 이메일. 구매자 커뮤니티. 구매 직후 리뷰 요청(긍정 확보)

### C-48. Cialdini 설득 6+1 원칙
- **한 줄**: 상호성 / 일관성 / 사회적 증거 / 권위 / 호감 / 희소성 / 통합
- **디자인 적용 질문**: "7가지 원칙 중 몇 개를 활용하고 있는가? 빠진 것은?"
- **활용 체크리스트**:
  - [ ] 먼저 주고 있는가? (상호성)
  - [ ] 작은 Yes에서 시작하는가? (일관성)
  - [ ] "N명이 선택"을 보여주는가? (사회적 증거)
  - [ ] 전문가/인증이 보이는가? (권위)
  - [ ] 호감 가는 요소가 있는가? (호감)
  - [ ] 한정/제한 요소가 있는가? (희소성)
  - [ ] "우리"라는 소속감이 있는가? (통합)

### C-49. 심리적 반발 (Reactance)
- **한 줄**: 자유를 제한하면 더 하고 싶어진다. 과도한 설득은 역효과
- **디자인 적용 질문**: "너무 세게 밀고 있진 않은가? '선택은 당신의 것' 자율성을 존중하고 있는가?"
- **활용**: 조작으로 느껴지는 희소성은 반발 유발. "진입 금지" 대신 "VIP 전용"(긍정 프레임)

### C-50. 자이가르닉 효과 (Zeigarnik)
- **한 줄**: 미완료 과제가 완료된 과제보다 더 강하게 기억에 남는다
- **디자인 적용 질문**: "다음 행동을 유도하는 '열린 루프'가 설계되어 있는가?"
- **활용**: 프로그레스 바(70% 완료). "다음 편 예고". 장바구니 리마인더

### C-51. 분위기 이론 (Atmospherics)
- **한 줄**: 환경의 시각/청각/후각/촉각이 구매 행동에 직접 영향
- **디자인 적용 질문**: "공간의 4가지 감각 채널(시/청/후/촉)이 의도한 감정을 일으키는가?"
- **활용**: PAD 모델(쾌감-각성-통제). 느린 음악=체류 증가. 향기=품질 인식 상승

### C-52. 감각 마케팅 (Sensory Marketing)
- **한 줄**: 다감각 경험이 단일 감각보다 기억에 강하게 각인
- **디자인 적용 질문**: "5감 중 몇 개를 의도적으로 활용하고 있는가?"
- **활용**: 시그니처 향기(후각 브랜딩). 제품 촉감(브랜드 가치의 촉각적 전달). 멀티감각 공간

### C-53. 색채 심리학 (Color Psychology)
- **한 줄**: 색상은 감정/인지/행동에 즉각적 영향 (단, 문화 맥락 주의)
- **디자인 적용 질문**: "선택한 색상이 의도한 감정과 카테고리 코드에 맞는가?"
- **활용**: 빨강=긴급/식욕. 파랑=신뢰. 검정=럭셔리. CTA 버튼 색상 실험

### C-54. 사회적 정체성 / 소비 (Social Identity)
- **한 줄**: 소비 = "나는 이런 사람이다"라는 정체성 표현
- **디자인 적용 질문**: "이 브랜드/서비스를 이용하면 고객은 어떤 집단에 속하게 되는가?"
- **활용**: 트라이브 마케팅(파타고니아=환경주의자). 멤버십=정체성 소속감

### C-55. 과시 소비 / 밴드왜건 / 스놉 (Veblen / Bandwagon / Snob)
- **한 줄**: 유행 따라가기 vs 차별화 추구, 영구적 긴장
- **디자인 적용 질문**: "타겟이 밴드왜건(동조)형인가 스놉(차별화)형인가?"
- **활용**: 대중 라인(밴드왜건)과 컬렉션(스놉) 분리. 매스티지(접근 가능한 프리미엄)

### C-56. 그루엔 전이 (Gruen Transfer)
- **한 줄**: 목적 쇼핑에서 충동적 탐색 모드로 무의식적 전환
- **디자인 적용 질문**: "목적→탐색 전환이 일어나는 지점을 설계했는가?"
- **활용**: IKEA 일방통행 동선. 코스트코 보물찾기. 이커머스 추천 알고리즘

### C-57. 디드로 효과 (Diderot Effect)
- **한 줄**: 새 소유물과 기존 것의 불일치 → 나머지도 교체 충동
- **디자인 적용 질문**: "하나의 구매가 연쇄 구매로 이어지는 크로스셀링 구조가 있는가?"
- **활용**: "이 소파에 어울리는..." 추천. 통일된 미학의 라이프스타일 브랜드

### C-58. 문화 심리학 / 소비 (Cultural Psychology)
- **한 줄**: 개인주의(자기표현) vs 집단주의(체면/사회적 승인)
- **디자인 적용 질문**: "한국 시장의 집단주의 + 체면 문화를 고려한 설계인가?"
- **활용**: 한국: 사회적 증거 특히 강력, 후기/랭킹 중시. 체면 소비(선물용, 과시용)

### C-59. 해석 수준 이론 (Construal Level Theory)
- **한 줄**: 멀면 추상적(Why), 가까우면 구체적(How) 사고
- **디자인 적용 질문**: "구매까지 시간이 먼 상황인가 가까운 상황인가?"
- **활용**: 멀리: 비전/가치/꿈 이미지. 가까이: 가격/배송/스펙 상세

### C-60. 주의 경제학 (Attention Economics)
- **한 줄**: 정보 과잉 시대, 주의가 가장 희소한 자원
- **디자인 적용 질문**: "고객의 주의를 확보하기 위해 무엇을 교환하고 있는가?"
- **활용**: 콘텐츠 가치 = 주의의 양 × 질. 구독 = 반복적 주의 확보 모델

### C-61. 리테일 테라피 (Retail Therapy)
- **한 줄**: 쇼핑으로 부정적 감정을 일시적으로 완화 (통제감 회복)
- **디자인 적용 질문**: "쇼핑/서비스 이용 자체가 감정적 치유 경험이 되는가?"
- **활용**: 작은 사치(립스틱 효과). 에이솝의 핸드 워시 의식. 감정 기반 타이밍

### C-62. 고객 경험 심리학 (CX Psychology)
- **한 줄**: 경험 경제의 진화: 상품→서비스→경험→변환
- **디자인 적용 질문**: "우리는 상품/서비스/경험/변환 중 어느 수준에서 가치를 제공하는가?"
- **활용**: Kano 모델(기본/성능/매력적 품질). 감정 곡선 매핑. JTBD + 경험 결합

# Part D. 아이디어 심리 감사 체크리스트 (Psychology Audit)

> 디자인/서비스 아이디어가 나왔을 때, 아래 체크리스트로 심리적 완성도를 점검한다.

## D-1. 진입 장벽 감사

- [ ] **인지 부하**: 첫 접점에서 고객에게 요구하는 정보/결정이 3개 이내인가?
- [ ] **스키마 호환**: 고객의 기존 기대와 일치하는가? (의도적 위반이라면, 그 이유가 명확한가?)
- [ ] **신뢰 3초**: 첫인상이 전문적 신뢰감을 전달하는가?
- [ ] **System 1 친화**: 직관적으로 이해 가능한가, 분석적 사고를 강요하는가?

## D-2. 동기 감사

- [ ] **욕구 층위**: 어떤 욕구(기능/안전/사회/존경/자아실현)를 충족하는가?
- [ ] **조절 초점**: 촉진(얻는 것) or 예방(잃지 않는 것) 중 어떤 동기에 소구하는가?
- [ ] **JTBD 정합**: 기능적 + 감정적 + 사회적 Job을 모두 고려했는가?
- [ ] **즉시 보상**: 장기 혜택만 있는가, 즉시 체감되는 보상이 있는가?

## D-3. 선택 구조 감사

- [ ] **옵션 수**: 3-5개 이내인가? 추천 옵션이 명확한가?
- [ ] **앵커링**: 가격/가치의 기준점(앵커)이 의도적으로 설정되어 있는가?
- [ ] **디폴트**: 디폴트 옵션이 고객의 최선의 이익에 부합하는가?
- [ ] **프레이밍**: 이득/손실 프레임 중 상황에 맞는 것을 선택했는가?

## D-4. 경험 곡선 감사

- [ ] **절정 설계**: 의도적으로 감동/놀라움의 순간을 설계했는가?
- [ ] **종결 설계**: 마지막 인상이 긍정적인가?
- [ ] **감각 활용**: 5감 중 2개 이상을 의도적으로 활용하는가?
- [ ] **적응 방지**: 시간이 지나도 신선함을 유지하는 장치가 있는가?

## D-5. 행동 유도 감사

- [ ] **손실 회피**: "잃을 수 있는 것"이 적절히 표현되어 있는가?
- [ ] **사회적 증거**: 다른 사람들의 선택/만족이 보이는가?
- [ ] **소유감**: 구매 전에 심리적 소유감을 형성하는 접점이 있는가?
- [ ] **습관 루프**: 반복 사용을 유도하는 트리거→행동→보상 구조가 있는가?
- [ ] **미완결**: 다음 행동을 유도하는 열린 루프가 있는가?

## D-6. 디지털/SNS 감사

- [ ] **eWOM 품질 설계**: 리뷰 양보다 논거 품질을 유도하는 구조인가? (보상 남발보다 구체적 리뷰 유도가 전환율↑)
- [ ] **리뷰 다층 신뢰**: 별점→텍스트→구매인증→"도움이 됨"→리뷰어 프로필, 최소 3층 이상인가?
- [ ] **부정 리뷰 대응**: 삭제가 아니라 공개 응대 전략이 있는가? (부정 리뷰의 감정적 진단성 활용)
- [ ] **행동적 사회증거**: "인기 1위"(태도)가 아니라 "N명 구매"(행동)를 보여주는가?
- [ ] **FOMO 3중 결합**: 사회적 증거+희소성+FOMO가 결합되어 있는가? (단일 < 결합 효과)
- [ ] **정체성 시그널**: 공유/리뷰가 사용자의 정체성 표현(취향/전문성/가치관)이 되는가?
- [ ] **한국 맥락 설계**: 제품 스펙보다 사용 맥락(장면/상황/동행자)이 먼저 보이는가?
- [ ] **리뷰 타이밍**: 리뷰 요청이 구매 후 2-3일(합리화 골든 타이밍)인가?
- [ ] **파라소셜 활용**: 메가 인플루언서보다 마이크로(팔로워 1만 미만)의 전환율이 높은 카테고리인가?

## D-7. 윤리 감사

- [ ] **다크 패턴 확인**: 고객의 이익이 아닌 우리 이익만을 위한 조작은 없는가?
- [ ] **반발 위험**: 과도한 설득/제한으로 심리적 반발을 유발하지 않는가?
- [ ] **공정성**: 가격/서비스 차별이 공정하다고 인식되는가?
- [ ] **투명성**: 숨기고 있는 비용/조건은 없는가?
