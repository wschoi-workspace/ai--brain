🔍 Skill 1: 원본 분석 — 진짜 반응을 걸러내는 심리학
Sincerity Filter + 2-Layer 분석의 모든 지표가 소비자 심리학 이론에 기반합니다. 10개 이론이 6개 지표에 매핑됩니다.
PSYCHE LAYER Authenticity (Auth) — 진정성 지표
Auth 산출 공식 (0~100)
기본 60점
- 협찬: -15 ← 인지 부조화 보정
- 과잉긍정어 초과: ×-3
- 느낌표 10개 초과: ×-2
- 표현 모방 의심: -10 ← 프라이밍 효과 보정
+ 개인경험 2건+: +10
+ 구체적 디테일 3건+: +10
+ 균형 평가(긍+부 공존): +15 ← ELM 중심경로 처리 신호
정교화 가능성 모델 (ELM)Petty & Cacioppo
장단점을 모두 언급하는 것은 중심 경로 처리(높은 인지적 노력)의 행동적 증거. 주변 경로(감정적 "좋아요!")보다 진정성이 현저히 높다.
RXR 적용: 균형 평가 가점을 +8에서 +15로 상향. 긍정과 부정을 모두 언급한 리뷰는 가장 신뢰할 수 있는 진정성 신호.
프라이밍 효과John Bargh
블로그 리뷰어가 다른 리뷰를 먼저 읽으면 유사한 표현/감정 톤을 무의식적으로 모방. "비슷비슷한 긍정 리뷰"가 쏟아지는 원인.
RXR 적용: 다른 리뷰와 유사 문장 구조 3건+ 감지 시 -10점으로 프라이밍된 모방 리뷰를 보정.
인지 부조화Leon Festinger
무료 체험·초대 참여자는 "내 시간을 쓸 만큼 좋은 것이었다"고 합리화하는 부조화 해소 패턴을 보일 수 있다.
RXR 적용: 협찬 -15점은 이 메커니즘을 부분 보정. 무료 체험(비협찬)에서도 패턴이 나타날 수 있으므로 과잉 긍정 감지와 병행 해석.
PSYCHE LAYER Clout — 추천 강도 지표
Clout 산출 공식 (0~100)
기본 40점
+ 추천 표현("추천", "꼭 가", "강추"): 건당 +15
+ 복수형 > 1인칭: +10
+ 제3자 경험 인용: +12 ← 사회적 증거
+ 예방 초점 추천: +8 ← 손실 회피 활용
+ 조건부 추천: +10 ← 높은 신뢰도
+ 정체성 선언: +12 ← 사회적 정체성
사회적 증거Robert Cialdini
"친구도 좋아했어"라는 제3자 경험 인용은 본인의 추천보다 설득력이 높다. 독립적 경험 확인 = 사회적 증거 + 권위의 결합.
RXR 적용: "친구가", "같이 간 사람이", "데려갔는데" 패턴 감지 시 +12점
조절 초점 이론 + 손실 회피Tory Higgins / Kahneman & Tversky
"안 가면 후회해요"(예방 초점)는 손실 회피를 활성화하므로 "꼭 가봐요"(촉진 초점)보다 실제 전환율이 더 높을 수 있다.
RXR 적용: "안 가면 후회", "놓치면", "없어지기 전에" 패턴 감지 시 +8점
사회적 정체성 이론Henri Tajfel
"나는 ___팬/덕후"라는 정체성 수준의 추천은 경험 수준("이거 괜찮더라")보다 지속성이 현저히 높다. 자기 정체성은 쉽게 안 바뀐다.
RXR 적용: "~덕후", "~팬", "~러버" 패턴 감지 시 +12점. Advocacy Decay 해석 시 정체성 선언 비율도 참조.
PSYCHE LAYER Freshness — 신선함 지표
Freshness Index 공식
Freshness = "이/가" / ("이/가" + "은/는") × 100

"이/가" = 초점/발견 (새로운 정보)  |  "은/는" = 주제/설명 (이미 아는 것)
해석 수준 이론 (CLT)Trope & Liberman
"이/가"는 저수준 해석(구체적, 심리적으로 가까운 대상)을 반영. "은/는"은 고수준 해석(추상적, 이미 스키마에 통합된 정보)을 반영.
RXR 적용: Freshness가 높다 = 대상을 아직 구체적·감각적으로 처리 중. BEI Discovery 축에 직접 연결.
기억 시스템 이론Atkinson & Shiffrin
시간이 지나면 감각 기억 → 에피소드 기억 → 의미 기억(스키마)으로 전환. "이/가"(새로운 인상)가 "은/는"(이미 아는 것)으로 자연 전환.
RXR 적용: Freshness 자연 하락은 "나빠진 것"이 아니라 "감각 기억이 의미 기억으로 통합 완료"된 정상적 인지 과정.
eWOM 감성 분석 — 부정 리뷰의 숨겨진 가치
부정성 편향 + 감정적 진단성Lee et al., 2026
부정적 감정이 약간만 증가해도 리뷰 유용성(helpfulness)이 크게 증가. 시선 추적 연구에서도 부정 댓글 주시 시간 > 긍정 댓글. 부정 리뷰 1건의 매출 영향 > 긍정 1건 (비대칭 효과).
RXR 적용: Gate 3 통과 "혼합" 감성이 높은 비율이면, 부정 신호가 아니라 정보 밀도가 높은 양질 리뷰의 증거. ELM 중심경로 처리의 산물.
eWOM 메타분석 (69개 연구)Ismagilova et al., 2020
구매 의도에 가장 강한 영향: 논거 품질 ★★★★★. 리뷰 양은 ★★☆☆☆ (약함). 리뷰 수를 늘리는 것보다 구체적 리뷰를 유도하는 구조 설계가 전환율에 더 효과적.
RXR 적용: RQL(Review Quality Level) 5단계 가중치의 학술적 근거. Q5 서사형(×2.0)이 Q1 간단형(×0.2)보다 10배 가치 있는 이유.
eWOM Content Class — 리뷰 동기별 심리 분류
eWOM 생성 동기 ↔ Content Class 매핑 (King et al., 2014)

리뷰 작성에는 4가지 심리적 동기가 작동합니다. Content Class는 이 동기를 텍스트 패턴으로 역추정하는 구조입니다.

이타적 동기 → Class A타인을 돕고자
가장 높은 Auth. 구체적 디테일과 경험 공유가 풍부. "다른 사람도 알면 좋겠다"는 동기에서 출발.
RXR 적용: A등급 = 진성후기. 2-Layer 핵심 분석 대상.
자아 향상 동기 → Class A~B전문가로 보이고자
분석적 톤, Q4~Q5 비율 높음. Clout도 높을 수 있음. "내가 이 분야를 잘 안다"는 자기 표현.
RXR 적용: 자아 향상 동기의 리뷰는 깊이가 있되 과시적일 수 있음. Auth와 Clout를 함께 확인.
경제적 보상 동기 → Class C포인트/협찬
동기가 외재적이므로 Auth 스펙트럼이 넓음(10~63). 진심 여부를 별도 판단해야 함.
RXR 적용: C등급 = 협찬. 제외하지 않고 진심 스펙트럼으로 분석 — 이것이 RXR의 핵심 차별화.
감정 해소 동기 → 부정 리뷰불만 표출
부정 리뷰이지만 감정적 진단성이 높아 정보 가치가 큼. 구체적 문제점을 포함할 확률↑.
RXR 적용: 부정/혼합 리뷰를 "실패 신호"로 보지 않고 "브랜드가 귀 기울여야 할 피드백"으로 프레이밍.
SOCIAL Account Trust — 파라소셜 관계와 유대 강도
파라소셜 관계Horton & Wohl / SNS 연구
팔로워가 인플루언서를 "가까운 지인"처럼 느끼는 일방적 친밀감. 마이크로 인플루언서(T2)는 심리적 거리가 짧아 추천 설득력↑. 매크로(T3)는 "광고"로 인식될 확률↑.
RXR 적용: T1(×1.0) > T2(×0.8) > T3(×0.6) > T4(×0.3) 가중치가 심리적 거리 순서와 일치.
행동적 증거 > 태도적 증거eWOM 연구
"많은 사람이 실제로 갔다"(행동)가 "많은 사람이 좋아한다"(태도)보다 진정성이 높다고 인식됨. 행동은 위조하기 어렵다고 느끼기 때문.
RXR 적용: T1(순수 개인)의 방문 후기는 행동적 증거이므로 가중치 최대(×1.0).

eWOM BEI의 핵심 철학: 행동적 증거 > 태도적 증거
69개 연구 메타분석이 증명한 원칙

"좋다고 말했으니까 좋아하는 거다" (X) → "시간과 노력을 들여 500자짜리 후기를 쓰고, 구체적 경험을 공유하고, 친구한테 꼭 가보라고 추천했다 → 이것은 좋아한다는 행동적 증거다" (O). 행동은 과장이 어렵다. 이것이 BEI가 단순 건수/긍정률보다 정확한 이유.

⚖ Skill 3: 비교 분석 — 왜 A 팝업이 B보다 효과적인가
5축 비교 프레임워크의 각 축에 심리학적 해석 근거가 내장되어 있습니다.
동사화 테스트컨셉 침투율
IKEA 효과(Norton): 직접 참여→높은 가치 부여. 심리적 소유감: 능동적 참여→"내 것" 느낌. Peak-End Rule: 행동은 절정을 만들고, 개념은 못 만듦.
"오늘 ___했다"로 완성되는 컨셉(동사형)이 개념형보다 침투율·Auth·기억 지속성 모두 높은 이유.
긍정률 거품 해석앵커링 + 손실 회피
앵커링 효과: Gate1 높은 긍정률이 앵커가 되어 Gate3 하락이 실제보다 더 나빠 보임. 손실 회피: 클라이언트가 +12%p 상승보다 -12%p 하락을 약 2배 강하게 부정적으로 체감.
리포트에서 하락 제시 시 "왜 하락이 긍정적 신호인지"(과장 긍정 제거 = 더 정확한 현실)를 선제적으로 설명해야 함.
협찬 신뢰도인지 부조화 + 사회적 정체성
인지 부조화: 가치 공감형은 일관성 유지(Auth 유지), 의무적은 부조화 상태(과잉 긍정으로 보상→Auth 급감). 사회적 정체성: 브랜드를 정체성 일부로 인식하면 격차 축소.
Auth 격차 10pt 이하 = 가치 공감형 협찬(양호). 20pt 이상 = 의무적 협찬(브랜드 훼손 수준).
진정성 유지력기억 미화 + Peak-End Rule
기억 미화(Rosy Retrospection): 장기 운영 시 구체적 디테일이 사라지고 "좋았다"라는 추상적 평가만 남으면서 과잉 긍정어로 빈 공간을 채움→Auth 하락.
"행동적 참여"가 절정 경험을 만들면 에피소드 기억이 강하게 인코딩→기억 미화 적음→Auth 유지.
한국 소비자 맥락 해석Nisbett / 정체성 시그널
전체적 사고: 한국 소비자는 제품이 아니라 "그 제품이 놓인 맥락 전체"를 봄. "공간/분위기"가 "제품/맛"보다 높은 비율은 정상적 인지 패턴이며 실패가 아님.
정체성 시그널: "이걸 공유하면 나를 어떻게 볼까?"가 강한 팝업일수록 Q5 비율과 Advocacy가 높아짐.
리뷰 깊이 학술 근거Ismagilova et al., 2020
69개 연구 메타분석: 논거 품질 ★★★★★ > 리뷰 양 ★★. Q4+Q5 비율이 높다는 것은 논거 품질이 높은 리뷰가 많다는 의미.
RQL 기반 비교가 건수 기반 비교보다 정확한 이유. 단순 건수로 팝업을 비교하는 것은 학술적으로 약한 영향(★★) 기준으로 판단하는 것.
💰 Skill 4: 가치 분석 (EMV) — BEI를 돈으로 바꿀 수 없는 이유와 우회 경로
BEI → ₩ 직접 전환은 불가능합니다. 3가지 행동경제학 원리가 그 이유를 설명하고, 3단계 전환 체인이 우회 경로를 제공합니다.
심적 회계Richard Thaler
소비자는 "팝업 체험"과 "제품 구매"를 다른 심리적 계좌에서 처리. 팝업에서 감동(BEI↑)해도 구매는 별도 의사결정이며, 다른 예산·맥락·동기에서 작동.
EMV 적용: BEI→행동 효과 단위→₩의 3단계 전환 체인으로 우회. 심리적 계좌 간 전환 마찰을 반영.
현상 유지 편향Samuelson & Zeckhauser
기존 브랜드/제품을 쓰던 관성이 새로운 브랜드로의 전환을 막음. BEI가 높아도 전환 장벽(습관, 전환 비용, 접근성)은 별개 변수.
EMV 적용: Depth(재구매)와 Advocacy(신규 획득)의 전환 계수에 업종별 전환 마찰을 반영.
쌍곡선 할인George Ainslie
팝업 직후의 구매 의도(T+0)와 1주 후 실제 구매(T+1w)는 시간 할인으로 급격히 차이남. "꼭 사야지" → "나중에" → "잊음"의 패턴.
EMV 적용: Advocacy Rapid Decay와 연동. 축별 감쇠 함수(Decay Factor)로 4주 누적 가치 보정.
Sincerity → 전환율 경로의 학술적 근거

리뷰 있는 상품 전환율 +270% (Spiegel Research Center, 2017). 진성 리뷰의 전환 효과가 협찬/의무 리뷰보다 현저히 큼. Gate 3 통과 비율(Sincerity)이 높은 캠페인일수록 V_sincerity의 실제 정확도 상승. 구매 후 2-3일 리뷰 = 만족감 + 인지 부조화 해소가 동시에 작동하는 골든 타이밍.
