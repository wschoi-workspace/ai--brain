# 윤선생 하부③ 배제사유 심층 (RXR · by Project Rent · 2026.06.18)

> 본문 700건 전수 정독. 핵심 발견 = "블로그로는 배제를 못 잰다"는 천장 확정.

## 보강이 밝힌 것 — N을 키운 게 아니라 거짓양성을 걷어냈다
- 배제 본문 700건 크롤(성공 100%) → 1차 배제후보 209건 → **본문 전수 정독 진짜 배제 N=3**
- 209후보 실체: 가맹 원장홍보~110·협찬후기~45·공식블로그~35·"고민→윤선생 선택" 긍정담~15·**진짜 배제 3**·무관~5
- 스니펫 단계 N≈16은 **거짓양성**(100자에서 '고민/비교'만 보고 분류 → 본문 결론은 "그래서 윤선생 선택")

## 구조적 천장
네이버 블로그 윤선생 멘션 **98.6%가 공급자(가맹·공식·협찬) 콘텐츠** → "안 고른 학부모" 목소리 구조적 부재. **배제는 블로그로 정량화 불가.** (이탈 글도 이 엄격 집합엔 0건 — 배제≠이탈, 소스부터 다름)

## 살아남은 가설 — 전집·패드 필수구매 (정성 🔴)
세 갈래 단서로 일관 생존:
1. 진짜 배제 3건: ②2018 윤선생 비교군에만→DWE 선택 / ②2024 무료체험 상담만→엄마표 회귀 / ②2023 부스 순회 미구매
2. **내부자 증언**: 윤선생 스마트랜드 5년연속 전국1위 센터장이 가맹탈퇴·창업하며 "몇백만원 전집 구매해야 수업 가능" 구조 비판
3. 스니펫 테마 반복: 배제보강 1,256건서 '전집·패드' 클러스터가 타 테마보다 반복(광고혼입 → 테마 존재 방증)
→ **전집·태블릿 필수구매 = 윤선생 고유 진입마찰점(가설)**. 경쟁 학습지엔 약함. 정량 미검증.

## 고유 vs 공통
- 전집·패드 필수구매 = 윤선생 고유(가설) / 방문약함→부모관리·가격 = 카테고리 공통 / 화상거부·약정 = 신호 거의 없음

## 카페 검증 (방법 A 실행) — 블로그가 못 본 배제를 채움
네이버 카페 API로 배제쿼리 21종 2,701건 수집 → 자사·홍보 제외 정제 1,750 → 배제신호 **679건(39%)** (블로그 본문 0.4%와 비교 불가). 단 스니펫이라 정독 시 진짜 배제 ~23건(거짓양성 92% — "대신/말고"는 배제 아님).
- **배제사유 위계 재정렬**: ①혼자독학·관리부담(7) → ②가격·영업강매(6) → **③전집·패드(4)** → ④효과의심(4). 약정·해지 0·화상거부 0
- **전집·패드 가설 재판정**: 유효하나 단독 1위 아님(3위). 본질="비용 치렀는데 셀프 부담 남음"(가성비-노력 불일치). 윤선생 광고의 "전집 강요❌" 반복 = 거부감 실재 방증
- **약정없음·가성비 = 배제 아닌 방어 자산**(약정 이탈 ~0)
- **경쟁 구도**: 전환처보다 병렬 비교 디폴트(튼튼116·구몬90·눈높이88·화상79). 실제 전환 소수=잉에·무무·리틀링고(관리·전집 회피)
- 대표인용: "윤선생은 엄마가 일일이 봐줘야"(상위1%카페) / "너무 부담돼 저렴이로"(레몬테라스) / "패드만 4개 부담"(맘이베베)

## 결론 (블로그→카페 갱신) + 제언
- **배제 1순위 = '혼자독학·관리부담'**(전집·패드 아님). 차별 메시지 수정: "전집 강매 없음(3무)" < **"혼자 안 해도 되게 관리해준다"**(관리·아웃풋 전면)
- 보유(이탈) 처방=원장교체·효과회의는 운영·콘텐츠(하부①). 배제와 섞지 말 것
- **남은 검증=설문(B)**: 카페도 스니펫 한계(거짓양성 92%) → "윤선생 고려→미선택" 직접 설문으로 위계 N 확정 + 전환처 검증(/rxr-survey-analysis)
