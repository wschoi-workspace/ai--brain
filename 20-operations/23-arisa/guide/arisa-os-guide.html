1시작하기

1
접속 — https://arisa-os.com (PC·모바일 모두 가능, 즐겨찾기 추천)
2
로그인 — 아이디 = 본인 이름(한글), PIN = 개별 안내받은 번호
3
PIN 변경 — 로그인 후 우측 상단 PIN변경 클릭 → 현재 PIN 입력 → 새 PIN(4자 이상) 설정. 첫 로그인 후 기억하기 쉬운 것으로 변경을 권장합니다

첫 화면오늘 Brief이번 주문서 시뮬레이터
대표 내 업무 — 🧾 결재·확인 필요 / ⏰ 지연 업무 / 업무 분장(리더에게 배분) 전체 병합 브리프 — 최상단 [프로젝트 단위|팀별 단위] 우선 파악존 + 팀별 상세·개인카드 전사 주간 집계 (팀 드롭다운으로 팀별) 전체 + 회의분석 Pro(질문 루프·실행 패키지 풀버전)
리더 팀 홈 — 📥 받은 업무(대표 지시) / 팀 Todo / 업무 분장(팀원에게 배분) / 진행중 프로젝트 / 팀원 오늘 보고 담당팀 브리프 — 우선 파악존 + 팀원별 보고 상세·분장 현황

💡 프로젝트 탭은 모두 공통 — 배정된 프로젝트의 상세(업무·간트·이슈·분장 업무 자동 연동)를 볼 수 있습니다.

3업무분장 사용법 (대표·리더)

방법 A — 자유 텍스트로 빠르게

1
업무 분장 입력창에 이번 주 업무를 자유롭게 적습니다 (예: "봉은사 마스터플랜 검토, KBO 굿즈 발주 확인")
2
[AI로 항목 정리] 클릭 → 프로젝트별 그룹으로 항목이 자동 분해됩니다
3
그룹마다 프로젝트명 확인 → 항목별 담당자 지정 (그룹 "일괄 담당자"로 한 번에 채우고 개별 수정 가능)
4
[최종 승인 · 등록] → 담당자의 화면·브리프에 자동 반영

방법 B — 주간업무계획 엑셀 업로드 (주간회의용 추천)

📥 주간업무계획 템플릿 다운로드 (.xlsx)
1
템플릿의 프로젝트명 · 금주 진행업무 · 차주 진행 업무 열을 채웁니다 (기존 주간회의 엑셀 그대로 써도 됩니다 — 헤더 이름만 유지되면 인식)
2
분장 섹션의 [📄 주간업무계획 업로드] 클릭 → 파일 선택
3
프로젝트별 검토 초안이 자동 생성 → 담당자 지정 → 등록
⚠️ 표에 "프로젝트명" / "금주" / "차주"가 들어간 헤더 행이 반드시 있어야 합니다. 날짜가 문장에 있으면(예: "3월 3주 납품") 마감일로 자동 계산됩니다.

리더 전용 — 대표 지시를 팀원에게 쪼개기

1
팀 홈 최상단 📥 받은 업무에서 대표가 배분한 업무 확인
2
[→ 팀원에게 상세 분장] 클릭 → 해당 업무가 입력창에 자동으로 채워짐
3
이어서 실행 단위 상세 업무를 적고 AI 정리 → 팀원별 배분 → 등록
💡 배분 범위는 위계로 제한됩니다 — 대표는 리더에게만, 리더는 자기 팀원에게만 배분할 수 있습니다.

4일일보고와 자동 반영

  • 저녁 보고 — 텔레그램 일일보고 봇으로 그날 업무 보고 (20:00·22:00 리마인드)
  • 아침 07:30 — 브리프가 자동 생성·발송: 대표는 전체, 리더는 팀 브리프 링크를 텔레그램으로 수신
  • 분장 상태 자동 갱신 — 보고 내용이 분장 업무와 매칭되면 상태가 미착수→진행중→완료로 자동 변경
  • 반영 표면 — 브리프 우선 파악존, 팀원 개인카드, 프로젝트 상세 "분장 업무" 섹션, 주간 달성률 게이지
  • 대표 화면 — 마감 지난 미완료 분장은 ⏰ 지연 업무(D+N일), 의사결정·승인 요청은 🧾 결재·확인 필요로 자동 집계

5문서 시뮬레이터 — 전 직원 사용 가능

상단 문서 시뮬레이터 탭에서 3가지 모드를 쓸 수 있습니다. 완료된 결과는 PDF·워드(.doc)로 바로 저장됩니다.

모드무엇을 하나요이럴 때
일일보고보고를 쓰고 AI 리뷰(ReportScore 100점 채점 + 개선 팁)를 받습니다저녁 보고 전에 셀프 점검
기획안 (1P-Brief)기획안 7항목을 채우고 AI 리뷰를 받습니다대표 보고 전 기획안 다듬기
회의분석회의 전사·회의록을 넣으면 AI가 실행 중심 회의록(써머리·결정·할 일·다음 단계)으로 정리합니다회의 끝나고 3분 안에 정리·공유
1
입력 2가지 — ✍ 텍스트 붙여넣기 또는 📎 파일 업로드(끌어다 놓기 가능, .txt·.md·.html·.docx)
2
AI 분석 클릭 — 30~60초 뒤 결과가 나옵니다
3
빈칸 채우기 — 담당·기한 등 "확인 필요"로 표시된 점선 값은 클릭해서 직접 입력하면 문서에 반영됩니다
4
저장 — [🖨 PDF로 저장] 또는 [문서(.doc)] 버튼으로 내보내 공유하세요
💡 PDF 파일은 아직 직접 업로드가 안 됩니다 — 내용을 복사해 텍스트로 붙여넣어 주세요. 대표 화면에는 회의분석 Pro(AI가 부족한 정보를 질문하는 풀버전) 모드가 추가로 보입니다.

7FAQ

Q. PIN을 잊었어요
A. 자가 재설정은 현재 PIN이 필요합니다. 분실 시 대표에게 재발급을 요청하세요.
Q. 사이트가 안 열려요
A. 주소가 https://arisa-os.com인지 확인하세요 (구 주소는 폐기됨). 계속 안 되면 대표에게 알려주세요.
Q. 등록한 분장을 수정/삭제하고 싶어요
A. 현재 버전에서는 등록 후 수정은 대표에게 요청하세요 (시트에서 직접 수정). 상태(진행중/완료)는 일일보고와 매칭되어 자동 갱신됩니다.
Q. 보고는 언제까지 해야 하나요
A. 당일 저녁까지 하면 다음날 아침 브리프에 반영됩니다. 밤 12시 이후 지각 제출은 브리프에서 빠질 수 있습니다.
Q. 팀원인데 오늘 Brief에 왜 동료 내용이 안 보이나요
A. 의도된 설계입니다 — 직원은 본인 카드+팀 헤드라인만, 팀 전체는 리더·대표가 봅니다.
