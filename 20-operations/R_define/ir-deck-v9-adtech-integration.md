## V9 Core 15장 상세

### s01 — Cover ★변경

```
V8: "우리는 모든 매장이 미디어가 될 수 있다고 믿습니다."
    오프라인 경험 데이터 플랫폼

V9: "The Google AdSense for Offline"
    데이터로 도시의 가치를 증명하는 오프라인 애드테크 플랫폼

    PROJECT RENT · IR DECK 2026 · CONFIDENTIAL
```

WHY: "매장이 미디어"는 좋지만 투자자에게 스케일이 작게 느껴짐.
"오프라인의 구글 애드센스"는 **즉시 시장 규모($450억 Amazon Ads)를 연상**시킴.

### s02 — Problem ★변경

```
V8: "소비자 지출 80% 오프라인, 측정 90% 온라인"
V9: + 쿠키리스 시대

[1] 쿠키가 사라진다
    · 구글 3rd Party Cookie 완전 폐지
    · iOS ATT로 앱 추적 차단
    · 온라인 타겟팅 광고의 정교함이 급격히 하락

[2] 오프라인은 여전히 블랙박스
    · 소비자 지출 80%는 오프라인
    · 마케팅 측정 90%는 온라인만 가능
    · 10조원 시장이 '감'으로 집행

[3] 해답은 오프라인에 있다
    · 검색어는 속일 수 있지만, 발자국은 거짓말하지 않는다
    · 오프라인의 '실제 행동'이 가장 순수한 퍼스트파티 데이터
    · 이 데이터를 수집·분석·활용하는 인프라 = 아직 없다
```

### s03 — Market (유지, 표현 강화)

TAM에 **디지털 광고 시장** 추가:

```
TAM: 글로벌 디지털 광고 $7,400억 + 리테일미디어 $3,120억
SAM: 한국 오프라인 마케팅 OOH 4.3조 + 카페 6조 = 10조+
SOM: 3년 내 매장 10개소 × 연 3억+ = 30억+

"Amazon이 온라인 쇼핑 데이터로 $450억 광고 사업을 만들었다.
 오프라인에는 아직 그 Amazon이 없다. 우리가 만든다."
```

### s04 — Solution ★변경

```
V8: RaaS → RaaM → RXR → Bracket (서비스 모듈)

V9: 3계층 기술 아키텍처

Layer 1: Physical AI (The Tracker)
  · Bracket 노드에서 고객의 스타일·행동을 실시간 페르소나로 치환
  · LAS(Lifestyle AI Segmentation): 32종+ 취향 세그먼트 자동 분류
  · 비식별 기반 — 개인정보 보호법 완전 준수

Layer 2: RXR Data Bridge (The Connector)
  · 오프라인 비식별 데이터를 모바일 AD-ID와 매칭
  · 온라인 리타겟팅 경로 확보
  · 특허 제10-2545502호의 핵심 기능

Layer 3: OMO Attribution (The Proof)
  · 오프라인 경험 → 온라인 매출 연결을 "증명"
  · "성수동 팝업 3번 매대 경험이 온라인 매출 X%를 만들었다"
  · 브랜드에게 압도적 ROAS 제공

+ Bracket Node (The Infrastructure)
  · 유휴 공간을 48시간 내 지능형 미디어 노드로 변환
  · 도시 전체가 광고 인벤토리가 되는 인프라
```

### s05 — Why Now (미세 강화)

3번째 카드에 **쿠키리스** 추가:

```
① 팝업 폭발: 3,077개 (+80%)
② RMN 공백: $7,090억 시장, 오프라인 RMN = 0
③ 쿠키리스 위기 + AI 시대: 온라인 타겟팅 한계 → 오프라인 데이터가 대안
   + 바이브코딩으로 6명이 SaaS를 만드는 시대
```

### s06 — Product ★변경

```
V8: RXR (분석 도구) + 성과 숫자 4건

V9: Physical AI — 오프라인의 구글 애널리틱스

[좌측] 기술 스택
  · Physical AI + LAS Engine (Alpha/Beta 구축 중)
  · RXR Data Bridge (특허 제10-2545502호)
  · OMO Attribution Algorithm
  · CTO: 전 퓨리오사 출신 개발자 영입

  OMO 저니맵:
  매장 진입 → AI 취향 식별 → 제품 체험 데이터 수집
  → 귀가 후 리타겟팅 → 온라인 구매 → Attribution 확정

[우측] 검증된 성과 (RXR 기존 데이터)
  · 마케팅 효율 4,500% (가나초콜릿)
  · 재방문 희망률 100% (비욘드)
  · 시청 희망률 88% (Apple TV+)
  · 방문객 활성화 +44% (바스켓_삼성스토어)

  모 대기업 POC(전용 분석 개발) 논의 중
```

### s08 — BM ★변경

```
V8: Revenue Flywheel (6개 수익원 순환)

V9: J-Curve 3단계 수익 모델

Phase 1: Ad Inventory (광고 지면)
  · Bracket 노드를 전국에 보급
  · 브랜드가 오프라인 광고 지면으로 Bracket을 구매
  · 수익: 광고비 + 공간 서비스 수수료
  · → 현재 여기 (매출 40억)

Phase 2: First-Party Data SaaS (데이터 구독)
  · LAS 취향 데이터를 브랜드에 구독 모델로 제공
  · "당신의 오프라인 고객이 어떤 사람인지" 리포트
  · 수익: 월 구독료 (Recurring Revenue)
  · → 2026~2027 진입 (RXR SaaS, POC 시작)

Phase 3: OMO Commerce (커머스 플랫폼)
  · 오프라인 방문자 전용 혜택/결제 앱
  · 체험 → 즉시결제 → AI 추천 → 재방문
  · 수익: 판매 수수료 (Transaction Revenue)
  · → 2027~2028 런칭

J-Curve 논리:
  노드 ↑ → 데이터 ↑ → AI 정교화 ↑ → ROAS 증명 ↑
  → 브랜드 유입 ↑ → 노드 ↑ (비선형 폭발)

현재(2024): Service 59% / Product 31% / Space 10%
미래(2028): Ad-Tech 30% / SaaS 25% / Commerce 25% / Service 20%
```
