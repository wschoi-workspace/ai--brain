## 2. 비즈니스 모델

### 핵심 정의
> **리테일 미디어 스토어 플랫폼** — 매장을 "판매처"가 아닌 "미디어"로 설계하는 오프라인 마케팅 서비스 플랫폼

### 3대 축
| Value | 설명 |
|-------|------|
| **Retail as Media** | 공간이 곧 콘텐츠이자 채널. 매장을 미디어 자산으로 운용 |
| **Editorial Discipline** | Dezeen·Wallpaper* 톤의 매거진 문법. 브랜드 스토리를 에디토리얼로 전달 |
| **Modular Scalability** | 단발 팝업이 아닌 반복 가능한 모듈형 플랫폼. 시즌·브랜드·지역별 재조합 |

### 운영 모델
- 1 브랜드 = 3~4주 단독 팝업 (공간 독점)
- 공간 기획 + 콘텐츠 설계 + 운영 + 성과측정까지 원스톱
- OMO (Online Merge with Offline) 커머스 확장
- RXR 행동데이터 분석 (특허 제10-2545502호) — 팝업 경험의 잔존을 정량 측정

## 3. 핵심 자산·역량

### 공간 자산
| 위치 | 상세 |
|------|------|
| 성수동 3개소 | 서울숲길 PR Bracket 본점 포함 |
| 롯데백화점 영등포 | 백화점 내 팝업 공간 |
| 이화여대 | 캠퍼스 팝업 공간 |
| 논현·답십리 | 추가 Bracket 자산 |

### 실적 규모
- **누적 250+ 팝업** 기획·운영
- **누적 계약 실적 약 60억원** (20개 사업 수행)
- 현대자동차, 매일유업, 오비맥주, 롯데웰푸드, 배달의민족 등 대기업 포함
- 공공: 한국관광공사, 해남군, 공주시, 남해 독일마을, 신용보증재단
- 글로벌: Apple TV+ 파친코2, 더셀렉츠(KOCCA K-패션 글로벌)

### 대표 프로젝트
| 프로젝트 | 의미 |
|---------|------|
| **가나초콜릿하우스 (2019)** | 성수동 팝업 붐의 분기점. 이전 분기당 3~4개 → 이후 40~50개로 급증. 성수 팝업 시대를 연 프로젝트 |
| **매일유업 CAFÉ AMAZING OAT** | 브랜드 경험 설계 + F&B 운영 역량 입증 |
| **더셀렉츠 글로벌 (KOCCA)** | K-패션 16개 브랜드의 도쿄·파리 팝업. 정부사업 + 글로벌 동시 수행 |
| **Apple TV+ 파친코2** | 글로벌 IP 기반 체험형 팝업 |
| **남해 독일마을 팝업홍보관** | 공공·관광 분야 지역 팝업 실적 |

### 특허·IP
- **RXR Analysis Solution** — 특허 제10-2545502호
- 팝업 1회당 EMV(Estimated Media Value)·BEI(Brand Engagement Index) 산출
- 행동데이터 기반 경험의 잔존·전환 측정
- 한국 유일의 리테일 미디어 성과측정 IP
- RaaS(Retail as a Service) 매뉴얼 보유 — 300+ 팝업 트랙레코드 기반

## 4. 경쟁 포지셔닝

### "성수동의 스티브 잡스" (KBS 보도)
- 성수동 팝업 생태계를 만든 1세대 기획자로 인정
- 6.5평 빈 공간에서 시작 → 팝업 성지로 변환

### 차별화 포지션
| 경쟁사 유형 | 그들의 접근 | 프로젝트렌트의 접근 |
|------------|-----------|------------------|
| 일반 공유오피스·공간대관 | 공간만 빌려줌 | 공간+기획+콘텐츠+운영+측정 원스톱 |
| BTL 행사대행사 | 부스 설치+운영 | "매장의 미디어화" — 경험이 콘텐츠가 되는 설계 |
| 팝업 전문 에이전시 | 트렌디한 공간 디자인 | 비즈니스 성과(매출·데이터)까지 연결 |
