## 1. 이 회사가 하는 일의 본질

프로젝트렌트는 **"팝업스토어 대행사"가 아니다.**

겉보기에는 공간을 빌려주고 팝업을 운영하는 것처럼 보이지만, 실제로 이 회사가 하는 일의 본질은 **"오프라인 공간을 미디어 채널로 전환하는 것"**이다.

비유:
- 유튜브 = 영상을 미디어로 만드는 플랫폼
- 인스타그램 = 이미지를 미디어로 만드는 플랫폼
- **프로젝트렌트 = 매장을 미디어로 만드는 플랫폼**

대부분의 팝업 대행사는 "예쁜 공간을 만들어주는 시공사"에 가깝고, 프로젝트렌트는 **"공간에 메시지를 싣는 편집자(에디터)"**에 가깝다. 최원석 대표가 성수동을 "잡지 같은 거리"라고 말한 이유가 여기에 있다 — 한 매장이 한 페이지이고, 넘기면 다른 브랜드의 이야기가 나오는 구조.

### 한 줄 정의
> **"오프라인 공간을 미디어로 전환하는 에디토리얼 플랫폼"**
> 팝업 대행사가 아니라, 공간에 메시지를 싣고 그 효과를 측정하는 "오프라인 미디어 회사".

## 2. 핵심 가치 — 3개 층위

| 층위 | 가치 | 설명 |
|------|------|------|
| **1층 (겉)** | 좋은 팝업을 만들어준다 | 250+ 팝업 실적, 성수 팝업 성지를 만든 1세대 |
| **2층 (진짜)** | 경험을 설계하고 측정한다 | "이 공간에서 소비자가 어떤 경험을 하고, 그것이 어떤 행동(구매·공유·재방문)으로 전환되는가"를 설계. RXR 특허로 숫자 증명 |
| **3층 (미래)** | 오프라인 데이터의 독점적 해석력 | 온라인은 GA/픽셀로 추적됨. 오프라인은 아직 블랙박스. RXR 300+ 케이스 데이터 = 한국 유일의 "오프라인 경험→비즈니스 성과" 연결 데이터셋. 쌓일수록 해자(moat)가 됨 |

## 3. 비즈니스 모델 구조 — 현재 4 레이어

```
Layer 1: 공간 대관 (Bracket)
  — 성수·논현·답십리 공간을 브랜드에 대여
  — 안정적 고정 수입, but 공간 수에 물리적 한계

Layer 2: 기획·운영 대행 (프로젝트 단위)
  — 브랜드/기관의 팝업을 기획~운영~정산까지
  — 건당 수천만~수억원, 핵심 매출원
  — 더셀렉츠 18.2억, 섬세한식탁 4억 등

Layer 3: 성과측정 IP (RXR)
  — 특허 기반 분석 서비스
  — 아직 독립 매출원으로 성장 전이지만, 차별화의 핵심

Layer 4: 교육·컨설팅 (Do Better Things)
  — AI 업무전환 교육
  — 새로운 수익원 탐색 단계
```

### 현재 모델의 구조적 문제
- Layer 1~2가 매출의 거의 전부이고, 둘 다 **프로젝트 단위**
- 일을 하지 않으면 매출이 0 — **반복적 구독형 수입(recurring revenue) 없음**
- 대표 의존도 극도로 높음

### 시장 맥락
- 글로벌 리테일 미디어 시장: 2025년 $184B → 2030년 $312B (Forrester)
- 글로벌 팝업 리테일 시장: 2025년 $95B (IBISWorld)
- **소비자 지출의 80%는 오프라인**, 리테일 미디어 광고의 90%는 온라인 — 이 불일치가 거대한 기회

## 5. 미래 비즈니스 가능성 — 5가지 시나리오

### 시나리오 1: 오프라인 리테일 미디어 네트워크 (RMN) ★가장 유망

아마존이 온라인 쇼핑 데이터로 광고 사업을 만든 것처럼, PR이 오프라인 경험 데이터로 광고/미디어 사업을 만드는 구조.

- RXR 데이터 + 오프라인 공간 네트워크 = "오프라인 RMN"의 원형
- 필요 조건: 공간 네트워크 확대 (5개 → 20개+), RXR 표준화, 광고주 직접 연결 플랫폼
- 시장 규모: 오프라인 리테일 미디어는 아직 초기 — 선점 기회

### 시나리오 2: RaaS 프랜차이즈화

300+ 팝업의 RaaS 매뉴얼을 패키지 상품으로 만들어 라이선스.

- "성수 PR Bracket" 모델을 부산·대구·제주에 복제
- 지자체·관광공사에게 "팝업 운영 매뉴얼 + RXR 측정 세트"를 SaaS처럼 판매
- **프로젝트 단위 → 반복 구독형 매출로 전환**

### 시나리오 3: 정부·공공 팝업 전문 브랜드

더셀렉츠, 섬세한식탁, K-존, 한국관광공사 등 공공 실적이 축적 중. "정부사업 팝업 = 프로젝트렌트"로 포지셔닝.

- 건당 4~20억, 연간 5~10건 = **20~100억 매출 궤도**
- RXR로 "다른 대행사는 방문객 수만 보고하지만, 우리는 경험의 잔존까지 증명"

### 시나리오 4: 브랜드 인큐베이터

250+ 팝업을 운영하면서 "어떤 브랜드가 되고 어떤 브랜드가 안 되는지"를 가장 많이 본 회사.

- 초기 브랜드에 투자·인큐베이팅 (공간 + 데이터 + 마케팅)
- "섬세한식탁"처럼 자체 브랜드를 만들어 운영하는 것도 같은 맥락
- 팝업 성과 데이터 기반 투자 판단 → VC적 기능

### 시나리오 5: AI + 오프라인 경험의 교차점

AI가 온라인을 먹고 있는 시대에, "AI가 대체할 수 없는 것 = 물리적 경험"이 프리미엄.

- AI로 팝업 기획 프로세스 자동화 (페이지네이션, 콘텐츠 생성, 성과 예측)
- RXR 데이터를 AI로 분석 → "이 브랜드에 최적인 공간·시기·콘텐츠" 추천
- **최종 형태: "AI 기반 오프라인 경험 설계 플랫폼"**

## 6. 솔직한 평가 — 강점과 리스크

### 진짜 강점
1. **최원석 대표의 "디자이너의 눈 + 사업가의 머리"** — 한국에서 이 조합은 매우 희소
2. **RXR 측정 IP** — "감(感)"이 아니라 "데이터"로 말하는 습관이 체질화
3. **"잡지 같은 거리"라는 비전의 구체성** — 추상적 비전이 아니라 6.5평에서 검증된 실체
4. **250+ 케이스의 암묵지** — 매뉴얼화된(RaaS) 것과 아직 안 된 것 모두 자산

### 리스크
1. **6명 조직의 스케일 한계** — 대표 의존도 극도로 높음. 최원석이 빠지면 핵심 역량(기획·컨셉)이 급격히 약화
2. **프로젝트 단위 매출** — 반복 수입 없음. "다음 달 매출"이 항상 불확실
3. **공간 확장의 자본 부족** — Showfields, Leap은 수십 M 달러 투자. PR은 자본 유치 없이 자력 성장
4. **팝업 시장 포화** — 2024년 3,077개 팝업(전년 +80%). "팝업 피로" 시작되면 플랫폼도 영향

## 7. 핵심 질문 — 다음 논의를 위한 씨앗

1. **RXR을 독립 사업으로 성장시킬 수 있는가?** — SaaS 모델로 전환하면 recurring revenue 문제 해결
2. **"프로젝트렌트 없는 프로젝트렌트"가 가능한가?** — 대표 의존도를 낮추는 시스템화/매뉴얼화의 수준은?
3. **공간 확장 vs IP 확장, 어디에 집중할 것인가?** — Leap처럼 공간을 늘릴 것인가, 아니면 RXR+RaaS를 IP로 파는 경량 모델로 갈 것인가
4. **정부사업은 안정적 매출원인가, 성장 발목인가?** — 규모는 크지만 마진이 낮고 정산이 느림
5. **"섬세한식탁" 같은 자체 브랜드는 얼마나 진지한가?** — 클라이언트 사업 vs 자체 브랜드의 자원 배분
