1기본업무 — 운영 원칙 (우선순위)

모든 업무는 이 우선순위로 판단한다. 위 순위가 아래 순위에 앞선다.

  • 1순위 · 고객 응대 및 고객 경험 — 모든 업무보다 우선. 입장 시 즉시 인사, 주문·문의·요청 신속 처리, 매장 상태 수시 확인. 고객 경험을 최우선 가치로.
  • 2순위 · 대표·매니저가 전달한 당일 업무 — 당일 업무 확인 → 우선순위 파악 → 운영 지장 없는 범위에서 수행 → 완료 후 공유·보고.
  • 3순위 · 메뉴 및 매장 개선(디벨롭) — 메뉴 퀄리티 개선점 발굴, 고객 피드백 수집·공유, 운영 효율·매장 환경 개선 제안.
  • 4순위 · 정리정돈 및 기타 — 청결 유지, 정리정돈 생활화, 기타 운영 업무.

6업태별 SOP 부록

일일/주간에 녹지 않는 상황형 절차. 발생 시 펼쳐 본다. (현장 운영 SOP 가안 충실 이관)

① 고객 응대 카페대관

STEP1 맞이

  • 5초 이내 인사
  • 이용 목적 확인
  • 공간·서비스 안내

STEP2 서비스

  • 주문 접수
  • 예약·대관 여부 확인
  • 시설 이용·유의사항 안내

STEP3 퇴장

  • 만족도 확인
  • 다음 방문 안내
  • 분실물 확인·배웅
컴플레인 대응 — 경청 → 공감·사과 → 사실관계 확인 → 관리자 보고 → 해결방안 제시 → 조치 결과 기록.  VOC 기록 — 발생일시·고객명·내용·처리결과·담당자.

② 대관 운영 대관

예약 접수

  • 행사 목적·일정·인원·시간
  • 필요 장비·이용 금액 안내
  • 예약 확정 → 계약·결제

행사 전 준비

  • 공간 청소·의자/테이블 배치
  • 프로젝터·음향·와이파이 점검
  • 안내문 비치·냉난방

행사 종료

  • 파손·쓰레기·원상복구 확인
  • 추가 비용·퇴실 확인
  • 만족도 확인·기록

③ 카페 제조·위생 카페

제조 전→후

  • 손 씻기·위생복·머신/원재료/냉장고 온도 확인
  • 레시피 준수·정량 계량·주문 재확인
  • 음료 상태 점검·전달·제조공간 정리·장비 세척

위생 점검

  • 머신·그라인더·싱크대 청결
  • 냉장고 청결·원두 보관 상태
  • 유통기한 점검

④ 편의점 상품 운영 편의점

입고

  • 발주 내역·수량·파손 확인
  • 유통기한 확인
  • 재고 등록

진열

  • 정렬·가격표·행사 표시
  • 품절·청결 확인

유통기한

  • D-30 / D-14 확인
  • D-7 할인 판매 검토
  • 폐기 분류·등록

⑤ 시설·장비 / ⑥ 위생·안전·보안

시설·장비 공간/시설

  • 일일 점검: 냉난방·조명·음향·프로젝터·와이파이·POS·CCTV·출입문
  • 고장 시: 확인 → 관리자 보고 → 사용중지 → AS 접수 → 수리일정 → 완료 점검

위생·안전·보안

  • 위생: 화장실·바닥·테이블 소독·쓰레기·냉장고 온도·조리공간
  • 안전: 비상구·소화기·비상등·전기·누수
  • 사고 시: 응급조치 → 관리자 보고 → 기록 → 재발방지

7재고·상품 운영 (편의점·리테일·도서)

소모품 발주 · 유통기한(한도 2개월) · 입고/등록 · 도서 반품. 초안 = Basket 실제 기준은 담당자 작성 시트(basket-재고상품운영-담당자작성시트.xlsx)로 채운 뒤 확정.

① 소모품 재고확인 및 발주오픈 점검 + 마감 발주리스트
  • 대상 — 카페 비품(종이컵·뚜껑·빨대·냅킨·물티슈·세제) / 화장실 비품(화장지·페이퍼타올) / 운영 비품 카페편의점
  • 기준 — 품목별 최소재고(par) 도달 시 발주. 최소재고 수치 확인 필요
  • 실행 — 오픈 시 부족분 메모 → 마감 시 일일 발주리스트 확정 → 담당자 카톡 전달(§2-3 연계)
  • 채널 — 발주 채널·발주일 확인 필요
② 상품 유통기한 확인 — 한도 2개월(D-60)오픈 점검 + 주간 전수
단계조치
D-601차 확인 — 회전 느린 품목 식별, 진열 앞으로(FIFO), 발주량 조절
D-302차 확인 — 소진 계획(프로모션·번들) 검토
D-143차 확인 — 할인 판매 검토 할인율 확인
D-7할인 판매 실행 → 미소진 시 폐기 분류·등록
  • 일일 오픈: D-60 이내 품목 체크 1줄 / 주간 1회: 전수 점검
  • 추적 방식(수기/엑셀/POS)·폐기 등록 담당 확인 필요
③ 상품 입고 및 등록 프로세스입고 시
  • 1. 입고·검수 — 발주서 대조(품목·수량·파손) / 유통기한 확인
  • 2. 등록 — 바코드 등록(미등록 품목) → 상품명·카테고리·원가·판매가 입력 → 재고시스템 반영 등록 시스템·바코드 방식 확인 필요
  • 3. 유통기한 등록 → D-60 체크리스트 연결
  • 4. 진열(디피) — 선입선출 위치 / 가격표·POP
  • 원가·판매가 측정·입력 주체 확인 필요 (현재 '금액 미측정' 이슈 해결)
④ 도서 확인 및 반품주기적 + 반품 주기
  • 1. 재고·판매 확인 — 주기적 실사, 베스트/슬로우(회전) 분류 리테일
  • 2. 반품 대상 선정 — 회전 안 되는 도서·파본 위탁/사입·반품 가능 기한·반품률 확인 필요
  • 3. 반품 처리 — 반품 리스트 작성 → 거래처(출판사/총판/유통사) 반품 요청 → 발송·정산 확인 → 재고 차감 거래처·반품 주기 확인 필요
＊ 위 확인 필요 항목은 basket-재고상품운영-담당자작성시트.xlsx의 시트 1~5에 대응. 담당자가 채워오면 초안의 placeholder를 실제 값으로 확정한다.
