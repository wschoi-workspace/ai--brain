Basket 운영봇 이렇게 쓰세요 🧺

이제 일일업무보고와 요일별 체크리스트를 텔레그램 봇 하나로 합니다. 따로 양식 안 열어도 되고, 그냥 적으면 봇이 정리해 줍니다.

봇 찾기 → @basket_ops_bot

1처음 한 번 — 시작하기

  • 텔레그램에서 @basket_ops_bot 검색
  • 대화 열고 /start 누르기 (또는 아무 인사나 입력)
  • 끝! 이제부터 보고·체크리스트를 봇에서 합니다.

꼭 — 두 분 다 한 번씩 /start 해주셔야 봇이 인식합니다.

2매일 — 일일업무보고

퇴근 전(또는 오후 4:30쯤), 오늘 한 일을 편하게 한 번에 적어 보내세요. 항목 나눠서 안 써도 됩니다 — 봇이 알아서 정리하고, 빠진 핵심만 되묻습니다.

나오늘 매출 평범했고 지출 없음. 머신 점검함, E61 견적 100만원 받음. 대관 사진전 문의 와서 회신 대기. 종이컵·빨대 부족해서 발주 필요. 단골손님이 신메뉴 맛있다고 함.
봇📋 정리했습니다 — 장비(E61 견적 100만), 대관(사진전 회신대기), 구매(종이컵·빨대 발주), 특이(신메뉴 호평)… 대관 일정/금액은 정해졌나요?
나일정 미정, 금액 소통 중
봇요약 보여드릴게요 → ✅ 등록 / ✏️ 다시

적을 때 생각나는 대로 — 이런 게 들어가면 좋아요:

지출송금·승인 필요특이사항장비(고장·견적·AS) 업무(완료/진행/예정)대관스태프구매입점 제안

자동 알림 — 송금·승인, 장비 견적, 입점 제안처럼 결재가 필요한 건은 봇이 매니저에게 바로 전달합니다. 그러니 빠뜨리지 말고 적어주세요.

3출근하면 — 요일별 체크리스트

출근해서 /todo 만 치면, 오늘 요일의 할 일이 오픈 / 미들 / 마감으로 쭉 뜹니다.

나/todo
봇🗓️ 화요일 TO-DO · 근무 2인
■ 오픈 — 전원·조명, 바 세팅, 재료 제조, 머신 세팅…
■ 미들 — 이슈노트 인수인계, 청결…
■ 마감 — 머신 약품청소, 🔧 그라인더 호퍼 청소…
나완료 (또는: 호퍼 청소 약품 부족해서 못함)
봇✅ 기록 완료. (못한 항목은 매니저에게 전달돼요)

약품 사이클 주의 — 화(호퍼 담금)→수(호퍼 결합) / 수(그룹헤드·포터필터 약품 밤새 담금)→목 오전 머신 세팅 때 헹궈 결합. 결합 전엔 작동 금지(화상 주의). 봇이 요일마다 알려줍니다.

4꼭 기억할 것

  • 발생 즉시 메모 — 마감 때 몰아 쓰지 말기 (까먹어요)
  • 결재 필요 건은 꼭 — 송금·견적·입점은 매니저에게 자동 전달
  • 커피 머신 전원은 끄지 않기 — 마감 시 그라인더만 OFF, 에스프레소 머신은 유지
  • 이슈 노트 인수인계 — 오전→오후 교대 때 봇 미들 항목 확인
