현재 자유서술로 작성되던 일일보고를, 실제 보고에 늘 등장하던 10개 항목으로 표준화했다. 매일 16:30 · 바스켓 운영방에 같은 틀로 올리면 누락·편차가 사라진다.

실제 일일보고 10회분 구조 기반 매일 16:30 · 운영방 by Project Rent
작성 원칙 — ① 오픈 이후 발생한 사항은 즉시 기록(마감 때 한 번에 몰아쓰지 않는다) ② 업무 완료 시 즉시 체크 ③ 16:30에 최종 정리하여 바스켓 운영방 업로드.

YYMMDD (예: 260609)

우리매장 ___ 원바스켓 로얄 ___ 원GS타워 ___ 원
② 지출 총합
___ 원 (지출 내역 있으면 함께)
③ 송금·승인 급한 건 1차 승인자 피드백 필요
- 지출결의서 / 송금 / 환불 등 즉시 결재 필요 항목
④ 특이사항
- 운영 중 특이점·이슈·외부 연락·문의 진행 상황
⑤ 장비
- 고장·수리·설치·견적·AS (대표님과 논의하여 픽스)
⑥ 업무 보고 완료 / 진행 / 예정
- 미팅·기획·재고·자료 작업 등 진행 상태
⑦ 대관
- 문의·일정·인원·금액·회신 상태
⑧ 스태프
- 근태·근로계약·추가근무·면접·교육
⑨ 구매

⑩ 입점 제안
- 입점·테스트 제안 및 일정 조율(대표님 피드백)
⑪ 복기 선택 · REM
- 오늘 손님이 멈춘 곳 / 반복한 말 / 내일 바꿔볼 것 (1줄)

③ 송금·승인 급한 건
- 블러썸 커피머신 설치비 미입금 → 빠른 처리 요망(업체 문의) · 1차 승인자 피드백 필요
- 바스켓 선불카드 환불건 지출결의서 작성 필요 · 1차 승인자 피드백 필요

- 식음료 교육 관련 찬호님 문의 완료, 회신 없어 AI 활용 서칭 중 · MASI RAW 기획안 진행
⑤ 장비
- 0610 GB5 바스켓 설치 완료 · 커피머신 당근판매 예정(E61/이글원 2그룹 점검·수리 후) → 현재 블러썸 인계 완료

⑦ 대관 사진전 / 필라 키즈 촬영 — 연·월·일·시 회신 대기
