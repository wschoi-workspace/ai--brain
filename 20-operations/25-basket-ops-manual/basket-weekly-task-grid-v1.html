1매일 공통 업무 (월~토 동일)

아래는 영업하는 날마다 똑같이 하는 업무. 요일에 상관없이 매일 수행한다.

🌅 오전 (오픈, 10시 전)
  • 매장 준비 — 조명 ON·POS 영업중·에어컨/난방·DID·음향(갤탭)·와이파이 / 홀 청소·화장실 / 건물 체크 / 소화기·비상구 / 제빙기 얼음 정리·작동 / 온수기·냉장냉동고 온도
  • 바·머신 세팅 — 그라인더 예비추출·입자조절 / 머신 셋팅(19~21g·32~40g·30~36초) / 브루비 예열 / 그룹헤드 샷 1회·브루잉 온도 / 워터디스펜서·서비스바 물통
  • 식자재·제조 — 식자재 정리(택배/쿠팡) / 소분(소비기한·선입선출·라벨) / 베이스 제조·디저트 해동·가니쉬 소분 / 식재료·원두 수량 체크
  • 재고·소모품·위생 — 유통기한·진열·부족·발주필요 / 리테일 DP·비품 세팅·종이컵 채우기 / 손·보건증
  • 환경·메뉴·이슈 — 행주 빨아 널기 / 테이블·유리문·동선·테라스 / 당일 재료·제조동선 / 전일 인계·이슈 파악
🌙 오후 (영업 중 → 마감)
  • 고객 응대 우선 + 수시 청결·설거지·매대 정리·VOC 기록
  • 데일리 보고 작성 — 매일 16:30 · 바스켓 운영방 매일
  • 머신·그라인더 마감 — 1·2차 약품 마감(샤워스크린·블라인드·린싱·배수트레인) / 그라인더 호퍼·챔버·날 / 브루비 청소
  • 식자재·집기 — 디저트·가니쉬 상태 / 통 집게·소량 재료 합치고 용기 세척
  • 발주·소모품 — 일일 발주리스트 → 담당자 카톡 / 식재료·원두·비품·화장실 비품·리테일 발주 체크 / 소모품 보충
  • 클리닝·기물 — 사용 기물 세척·린넨 핸들링 / 작업대·냉장고·고무패킹 / 린넨 세제 담그기 / 분리수거
  • POS·정산 — 정산 / 현금 시재 / 갤탭·진동벨 충전
  • 테라스·전원·보안 — 테라스 의자 내부 / 조명·에어컨·DID·핫디스펜서 OFF(머신 유지) / CCTV·문단속
절대 원칙 — 커피 머신 전원은 끄지 않는다(마감 시 머신 제외 전원만 OFF). · 마감은 클로즈 30분 전(18시 종료 → 17:30) 시작.
