L0 → L2: ENTRY FUNNEL

노출 → 시선 → 진입: 전환율 퍼널

주목률 & 진입률

L0 → L1: 주목률 (Attention Rate)
파사드 앞 2초+ 정지 / 전체 유동인구
기술: YOLOv8 + BoT-SORT 좌표 정지 판정 | 정확도 90%+
L1 → L2: 진입률 (Entry Rate)
매장 진입 인원 / 파사드 주목 인원
기술: 입구 라인 교차 감지 | 정확도 94~97%
DID 콘텐츠 A/B 테스트
시간대별 다른 콘텐츠 송출 → 각 콘텐츠의 주목률 × 진입률 비교
→ 콘텐츠 "진입 유도력" 자동 측정

진입 품질 분류

유형행동 패턴의미
목적형주목 없이 직진 진입기존 고객/재방문
유인형파사드 주목 후 진입콘텐츠에 의한 신규
동반형2인+ 그룹 진입사회적 방문
통과형진입 후 30초 내 퇴장오인입/호기심
핵심: 진입 유형별 이후 인게이지먼트 패턴이 다름
유인형 진입 → 평균 체류 3.2분, 구매 전환 12%
목적형 진입 → 평균 체류 1.8분, 구매 전환 34%
(파일럿에서 검증할 가설)

L3: DWELL QUALITY SCORE

체류 품질 스코어 — 실제 구현 설계

DQS = (실제 체류 / 기대 체류) × 밀도 계수 × 세그먼트 보정

① 기대 체류 기준표

상품 진열대45초~1.5분
체험존2~4분
전시/스토리텔링3~7분
포토존1~3분
파일럿 데이터 축적 시 자동 업데이트

② 체류 밀도 계수

같은 3분이라도:

1 Zone × 3분 정지
= 수동적 관람 (밀도 낮음)

3 Zone × 각 1분 순회
= 능동적 탐색 (밀도 높음)

밀도 = (방문Zone/전체Zone) × (1 + 상호작용 × 0.1)

③ 세그먼트 보정

아웃핏 세그먼트 평균 대비 편차:

미니멀 모던 평균 2.5분
→ 이 고객 5분 = 보정 2.0
→ 강한 관심 신호

캐주얼 컴포트 평균 4분
→ 이 고객 4분 = 보정 1.0
→ 보통 수준
PROJECT RENT · R-lab04 / 08
L4: CONVERSION QUALITY

구매 전환의 질적 분류

같은 구매라도 어떻게 샀는가에 따라 의미가 다릅니다. 아웃핏 세그먼트 × 구매 패턴 교차로 취향 DB를 축적합니다.

구매 유형 4분류

확신 구매
DQS 높음 + 다구역 탐색 후 구매 → 재방문 가능성 높음
충동 구매
DQS 낮음 + 단일 Zone 즉시 구매 → 취향 반응 신호
목적 구매
최단 동선 구매 → 기존 고객 / 재방문자
동반 구매
그룹 진입 + 구매 → 사회적 소비, 추천 기반 전환

결제 세부항목 → 취향 분석

데이터인사이트
구매 카테고리관심 영역 (식음/굿즈/체험/클래스)
가격대지불 용의(WTP) 세그먼트
구매 시점 (체류 몇 분 후)의사결정 속도 → 확신형 vs 탐색형
동반 구매 여부사회적 소비 패턴
옵션 선택세부 취향 (사이즈/컬러/향)
아웃핏 × 구매 교차 예시
"미니멀 모던 → 객단가 35K, 디저트+굿즈 세트 선호"
"스트릿 트렌드 → 포토존 5분+ 후 굿즈 구매 전환 42%"

L5 → L6: RETENTION & CRM

재방문 · 회원 — 하이브리드 접근

L5: 재방문 전략

방법정확도개인정보판정
얼굴 인식85~95%✗ 위반제외
의류 Re-ID (당일)80~90%✅ 안전사용
의류 Re-ID (일간)30~50%✅ 안전불안정
회원 앱/카드99%+✅ 동의최우선
핵심 전략
L5 정확도를 높이려면 L6(회원 전환)을 적극 유도.
회원 되면 재방문 추적이 자동 해결.
비회원 재방문률은 서베이 표본으로 추정.

L6: 회원 = Stage 1 → 2 전환점

회원가입 시 발생하는 데이터 연결
익명 아웃핏 세그먼트 → 회원 프로필에 태그
인게이지먼트 이력(L0~L4) → 개인 기준선 설정
구매 세부항목 → 개인 취향 프로필
체류 DQS → 개인 관여도 기준
OMO 통합 가치
오프라인 충동구매(감성) + 온라인 비교구매(이성)
= 한 사람의 이성/감성 비율

오프라인 아웃핏 세그먼트 + 온라인 장바구니
= 라이프스타일 정밀 프로필
