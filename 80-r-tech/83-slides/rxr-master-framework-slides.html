CORE PRINCIPLE

리테일테크의 본질

핵심 자산은 기술이 아니라 분석과 해석의 프레임이다. 기술은 오픈소스일 뿐이다.

🔍

프레임이 자산이다

RetailNext가 비싼 건 센서 때문이 아니라 해석을 못 해서. 같은 데이터를 다르게 읽는 틀이 경쟁력.

해석 프레임 + 변환 테이블 = Moat
🔄

양적 → 질적 전환

숫자를 세는 게 아니라 의미를 읽는다. "1만명, 8%"가 아니라 "왜 샀고, 진심이었고, 다시 올 것인가".

데이터의 성격 자체를 바꾼다
🏪

공간 = 데이터 플랫폼

쿠키가 사라진 시대, 오프라인 매장이 유일한 1st party 데이터 채널. 매장 자체가 쿠키를 대체한다.

Cookieless era → Store = Cookie

ENGAGEMENT MODEL

멘탈코스트 기반 인게이지먼트 레벨

모든 행동 전환은 심리적 비용의 산물이다. 전환의 깊이 = 브랜드와의 관계 깊이.

L0
통과
매장 앞 지나감 — 무관심
L1
시선
파사드에 시선 멈춤 — 인지
L2
진입
매장 안으로 들어옴 — 관심
L3
체류 ★
DQS 측정 — 핵심 가중 변수
L4
구매
결제 완료 — 확신적 관여
L5
재방문
다시 찾아옴 — 관계적 관여
L6
회원
회원 가입 — 커밋먼트
L7
추천 ★
RQL 측정 — 핵심 가중 변수
★ L3 체류 — Dwell Quality Score
단순 "몇 분"이 아닌 복합 변수
DQS = (실제/기대 체류) × 밀도 계수 × 퍼스널리티 가중
바쁜 직장인 5분 ≠ 주말 방문자 5분
단순 진열 3분 ≠ 체험형 공간 3분
★ L7 후기 — Review Quality Level
별점만(Q1) → 한줄(Q2) → 구체적(Q3) → 비교분석(Q4) → 설득형 서사(Q5)
가중 L7 = RQL × Authenticity × Clout
Q5+진심 = 최고 옹호 / Q5+과장 = 기만 패턴

같은 데이터에 여러 렌즈를 교차 적용하여 다른 인사이트를 추출합니다.

Lens 1 — 인구통계 × 라이프스타일

AI 아웃핏 패션 타입 → 라이프스타일 군
영상 AI 연령대 · 성별 추정
결제 구매 항목 · 가격대 → 소비 성향
동선 탐색 유형 분류

Lens 2 — 인게이지먼트 레벨

전체 유입을 L0~L7로 분류
각 레벨 간 전환율 = 핵심 KPI
멘탈코스트 기반 관계 깊이 측정
시계열 추적 → 인게이지먼트 변화 감지

Lens 3 — 교차 분석

Lens 1 × Lens 2 교차
"20대 스트릿 군 중 L4 전환율 = 12%"
구매 고객만 추출 → 결제 세부항목 취향 분석

같은 공간에서 브랜드별 패턴 비교
DID 콘텐츠별 진입 전환율 비교
→ 브랜드 인텔리전스 DB 축적
PROJECT RENT · R-lab05 / 12
CORE IP

행동 ↔ 심리 변환 테이블

리워드 기반 마이크로 서베이로 행동 데이터의 심리적 의미를 매핑합니다. 이 테이블이 RXR의 핵심 지적 자산입니다.

구축 프로세스

STEP 1. 행동 수집
체류 시간, 동선 패턴, 터치포인트, 구매 여부 자동 기록
↓
STEP 2. 서베이 교차
리워드 기반 마이크로 설문 → 만족도, 추천 의향, 재방문 의사
↓
STEP 3. 패턴 매핑
"추천하겠다" 응답자의 행동 패턴 → 기준점 설정
↓
STEP 4. 자동 추론
서베이 없이 행동 데이터만으로 인게이지먼트 레벨 추론

변환 테이블 예시

행동 패턴추정 레벨신뢰도
체류 <1분, 동선 1구역L2 통과형85%
체류 1~3분, 동선 2구역L3 탐색형72%
체류 3분+, 상품 터치 2회+L3+ 관심형76%
체류 4분+, 3구역+, 구매L4 확신형78%
2회+ 방문, 구매 이력L5 관계형82%
회원 + 리뷰 + 높은 CloutL7 옹호형88%
이것이 Moat가 되는 이유
경쟁사가 같은 센서를 설치해도, 이 변환 로직이 없으면 "체류 4분"이 무엇을 의미하는지 모른다.
PROJECT RENT · R-lab06 / 12
QUALITY ENGINE

2-Layer 질적 전환 엔진

텍스트 데이터가 존재하는 지점에서 양적 데이터를 질적 인사이트로 변환합니다.

Content Layer — "무엇을 말했는가"

• n-gram 기반 토픽 추출
• 감성 분석 (긍정/부정/중립)
• 키워드 빈도 및 연관 분석
• → 의식적/명시적 반응 포착
예: "분위기 좋다", "디저트 맛있다", "가격 비싸다"
→ 토픽: 공간(42%), 메뉴(35%), 가격(23%)

Psyche Layer — "어떻게 말했는가"

• FWA: 대명사, 조사, 접속사 패턴
• LIWC 4대 변수 수치화
• 한국어: "이/가" vs "은/는" 조사 패턴
• → 무의식적/잠재적 심리 포착
예: 같은 "좋다"라도
Authenticity 78 = 진심 vs Authenticity 28 = 과장

산출 지표

BAS
Brand Advocacy Score
진정한 옹호자 비율
FI
Freshness Index
브랜드 신선함 소진 감지
ED
Engagement Depth
질적 관여도 종합 점수
SG
Sincerity Gate
가짜/과장 리뷰 자동 필터

PROJECT RENT · R-lab08 / 12
KEY INSIGHT

오프라인 결제 데이터의 숨겨진 가치

같은 사람의 온라인 구매와 오프라인 구매는 전혀 다른 자아를 보여줍니다.

🛒

온라인 구매

이성적 자아
"필요해서 산다"
비교 검색 → 리뷰 확인 → 최저가 → 결제
계획적, 합리적, 기능 중심
🏪

오프라인 구매

감성적 자아
"끌려서 산다"
공간 경험 → 감각 자극 → 충동 → 결제
즉흥적, 감각적, 취향 중심
두 개를 합치면
한 사람 안의 이성과 감성의 비율까지 보인다 → 기존 어떤 고객 DB에도 없는 차원
결제 세부항목을 알면 → 취향의 구체적 방향까지 파악 가능
