03
STEP 2 — SINCERITY FILTER

모든 데이터가 같은 가치는 아닙니다

수집된 데이터를 7단계로 자동 분류하여, 분석할 가치가 있는 데이터만 추출합니다.

A. 진성 후기 — 직접 경험, 구체적 디테일, 개인 감상
핵심 분석 대상
B. 일반 포스트 — 이벤트 관련 일반 언급
분석 대상
C. 협찬/체험단 — #광고, 제공받음 표기
진심 스펙트럼 분석
D. 나열형 정보 — "3월 팝업 총정리" 식 리스트
버즈량만 카운트
E. 리그램/단순 공유 — 텍스트 없이 퍼가기만
버즈량만 카운트
F. 비즈니스/보도자료 — 보도자료 복붙, 위장 계정
제외
G. 노이즈 — 이벤트와 무관한 동명이인 데이터
제외
핵심: 협찬(C)도 버리지 않습니다. 협찬 안에서도 "진심으로 좋았던 글"과 "의무적으로 쓴 글"을 구분합니다.
04
STEP 2 — SINCERITY GATE

3단계 Gate를 통과하면
진짜 숫자가 보입니다

같은 데이터를 세 개의 렌즈로 보면, 각각 다른 질문에 답합니다.

Gate 1 — 전체
"이 이벤트가 시장에 얼마나 노출됐는가?"
532건
▼ 노이즈 + 리그램 + 위장 제거
Gate 2 — 정제
"유료 홍보를 빼면 실제 반응이 얼마인가?"
~150건
▼ 협찬 제외 + 진정성 필터
Gate 3 — 진심
"진심으로 좋았던 사람이 몇 명인가?"
~80건
532건 중 진심 반응은 약 80건 (15%) — 나머지 85%는 노이즈, 리그램, 협찬, 위장 계정
05
STEP 3 — 2-LAYER ANALYSIS

같은 "좋아요"도
깊이가 다릅니다

정제된 데이터에 두 겹의 분석을 적용합니다.

CONTENT LAYER

"무엇을 말했는가"

토픽 분석 — 무엇이 화제였는가
공간 체험? 제품 맛? 굿즈? 웨이팅?

감성 분석 — 긍정/부정/혼합/중립
기존 3단계 → 4단계(혼합 추가)

후기 깊이 — 한 줄 vs 500자 서사
5단계 가중 (Q1 ×0.2 ~ Q5 ×2.0)
PSYCHE LAYER

"진심이었는가"

진정성 (Authenticity)
과장인가, 진심인가? 느낌표 64개 = 과장

추천 확신도 (Clout)
"괜찮아요" vs "꼭 가세요!" = 확신의 차이

신선함 (Freshness)
"이/가"(발견) vs "은/는"(설명) 조사 패턴

08
PROVEN CASE

이클립스 월드 팝업에서 증명했습니다

2026년 3월, 성수동에서 열린 이클립스 팝업스토어 4일간의 SNS 반응을 분석한 결과.

발견 1 — 85%가 노이즈

Brand24이 보여준 532건 중 실제 유효 데이터는 약 80건. "이클립스"가 게임/앱/드라마 등과 겹쳐 67%가 팝업 무관 노이즈, 25%가 리그램/단순공유.

발견 2 — 협찬이 긍정률 부풀림

협찬 포스트 긍정률 88%(부정 0%) vs 자발적 41%. 협찬 8건이 전체 긍정률을 5%p 끌어올리고 있었음. 진정성 점수도 10.5점 차이.

발견 3 — "시식"이지 "체험"이 아니었다

토픽 68%가 "제품/맛" 중심. 공간 경험(6%), 게임(8%)은 부차적. 팝업이 "레트로 오락실 체험"이 아닌 "포도향 시식 이벤트"로 인식됨.

발견 4 — 시간이 지나면 과장된다

진정성(Auth)이 사전 59.5 → 팝업기간 56.7 → 팝업후 54.6으로 하락. 기억의 과장/미화 패턴. 반면 신선함(Freshness)은 직접 경험 시 최고.
