RXR Analysis Algorithm v2.0

2-Layer 분석 알고리즘
Content Layer + Psyche Layer

n-gram 기반 내용 분석(What)과 기능어 기반 심리 분석(How)을 구조적으로 분리하여,
각 레이어의 강점을 극대화하고 타 솔루션 대비 차원이 다른 인사이트를 생성하는 RXR 고유의 분석 아키텍처

2026.04 · Project RENT · R-lab · 기능어분석 + n-gram 복합 설계

1 왜 2-Layer인가 — 구조적 분리의 근거

핵심 원칙: n-gram(내용어)과 기능어는 측정하는 대상 자체가 다르다.
하나로 합치면 기능어의 기여가 n-gram에 묻힌다(Ott et al. 89.6% → 89.8%, +0.2%p).
분리해야 각각의 가치가 보인다.

학술적 근거: 왜 합치면 안 되는가

구성정확도출처시사점
LIWC(기능어) 단독 76.8% Ott et al. 2011, ACL 무의식 심리 포착에 특화, 내용 분류에는 약함
Bigram(n-gram) 단독 89.6% Ott et al. 2011, ACL 내용 패턴 분류에 강함, 심리 해석 불가
LIWC + Bigram 결합 89.8% Ott et al. 2011, ACL 기능어 기여분 +0.2%p — n-gram에 묻힘
DeBERTa fine-tuned (현 SOTA) 96~98% 2024-25 다수 연구 분류 정확도에서는 이길 수 없음
RXR 2-Layer (분리 운영) — RXR 독자 설계 "정확도 경쟁"이 아닌 "해석 깊이 경쟁"으로 전환
전략적 판단: 정확도(Accuracy) 싸움에서는 DeBERTa(96%)를 이길 수 없다. 그러나 "왜 긍정인가", "얼마나 진심인가", "추천 확신은 있는가"는 n-gram도 Transformer도 답하지 못한다. RXR은 이 영역에서 싸운다.

2-Layer 분리의 논리적 기반

Layer 1: Content Layer

측정 대상: "무엇을 말했는가" (What)
분석 단위: 내용어 (명사, 형용사, 동사)
기술: n-gram, TF-IDF, 감성사전, LLM
특성: 의식적 선택, 조작 가능, 대량 처리 용이
역할: 토픽/감성/키워드 분류 = 기본 자격(Table Stakes)

Layer 2: Psyche Layer

측정 대상: "어떻게 말했는가" (How)
분석 단위: 기능어 (대명사, 조사, 접속사, 어미)
기술: LIWC/K-LIWC, 조사 패턴 분석
특성: 무의식적 산출, 조작 극히 어려움
역할: 심리/진정성/확신도 측정 = 차별화 무기(Differentiator)

비유: Layer 1은 "마이크" — 고객의 말을 정확하게 수집한다. Layer 2는 "목소리 톤을 읽는 귀" — 같은 말 속에서 진심과 립서비스를 구분한다. 마이크 없이 귀만 있어도 안 되고, 마이크만 있고 귀가 없어도 안 된다. 단, 이 둘은 별개의 장치다.

2 알고리즘 아키텍처 — 전체 파이프라인

INPUT: 텍스트 데이터 수집
SNS 크롤링 (인스타그램, 블로그, 트위터) + 설문 개방형 응답 + 현장 인터뷰 + 리뷰 플랫폼
↓
Layer 0: 전처리 (Preprocessing)
텍스트 정규화 → 형태소 분석(KoNLPy/Mecab) → 내용어/기능어 분리 태깅 → 날짜/소스/프로젝트 메타데이터 부착
↓ 내용어/기능어 분리 ↓
Layer 1: Content Layer
Step 1-1. n-gram 추출 (uni/bi/trigram)
Step 1-2. TF-IDF 키워드 클러스터링
Step 1-3. 감성 분류 (긍정/부정/중립)
Step 1-4. 토픽 모델링 (공간/제품/서비스/분위기)
Step 1-5. LLM 맥락 요약 & 의미 추출
OUTPUT: 감성 스코어, 토픽 분포, 키워드 맵, 내용 요약
Layer 2: Psyche Layer
Step 2-1. LIWC/K-LIWC 4대 변수 산출
Step 2-2. 대명사 패턴 분석 (나/우리/여기)
Step 2-3. 조사 패턴 분석 (이/가 vs 은/는)
Step 2-4. 어미 패턴 분석 (확신/추측/감탄)
Step 2-5. 시계열 심리 변화 추적
OUTPUT: Authenticity, Clout, Analytical, Tone, Freshness, Distance
↓ 교차 분석 (Cross-Layer Integration) ↓
Layer 3: Insight Synthesis Engine
Layer 1 결과(What) × Layer 2 결과(How) = 2차원 인사이트 매트릭스
"긍정인데 진심인가?" · "추천했는데 확신이 있는가?" · "신선함이 아직 살아있는가?"
↓
OUTPUT: RXR 2-Layer Insight Report

5 Layer 3: 교차 분석 — 2차원 인사이트 매트릭스

교차 분석의 핵심: Layer 1의 "무엇"과 Layer 2의 "어떻게"를 곱해야 비로소 "왜"가 보인다.
같은 "긍정 81%"도, 심리 레이어에 따라 전혀 다른 전략적 의미를 갖는다.

2차원 매트릭스: 감성(What) × 진정성(How)

3주차 Freshness: 0.31 ("그 카페는", "그 메뉴는" — 구정보)
Freshness 0.84 → 0.31 급락
L2 진단: "신선함 소진. 2주차 이후 체험이 '새로운 발견'에서 '이미 아는 것'으로 전환"
2-Layer 통합 인사이트 (P-04 패턴):
"감성 하락(-13%p)의 원인은 품질 저하가 아니라 신선함 소진(Freshness 0.84→0.31). 소비자가 '이 공간이'를 '그 공간은'으로 지칭하기 시작 — 새로움의 감정이 사라짐. 액션: 2주차에 콘텐츠 리프레시(신규 메뉴/존 오픈)를 투입하여 Freshness 반등 유도. 향후 3주 이상 팝업에서는 주간 리프레시 캘린더 필수 적용."
시나리오 C: 라이프스타일 브랜드 상설 매장 (월간 추적, 리뷰 1,840건/6개월)

Layer 1 6개월 Content 추이

감성: 긍정 78~82% 사이 안정적 유지
키워드: 큰 변화 없음 (공간, 제품, 분위기)
언급량: 월 350건 → 월 280건 (서서히 감소)
L1 결론: "안정적. 특이사항 없음."

Layer 2 6개월 Psyche 추이

Tone: 76 → 72 → 68 → 65 → 61 → 58
Clout: 62 → 58 → 55 → 51 → 48 → 44
Distance: 0.41 → 0.38 → 0.35 → 0.32 → 0.29 → 0.25
모든 심리 지표 6개월 연속 하락
L2 경보: "브랜드 피로도 진입. 감정-확신-귀속감 동반 하락"
2-Layer 통합 인사이트 (P-06 패턴):
"감성 점수(L1)는 78~82%로 안정적이지만, 심리 지표(L2)가 6개월 연속 하락 중 — 전형적인 '조용한 이탈' 전조. 특히 Clout(추천 확신) 62→44, Tone(감정 강도) 76→58의 동반 하락은 '불만은 아니지만 더 이상 추천하고 싶지 않다'는 심리를 반영. L1만 보면 '안정적'이지만, L2는 '6개월 뒤 매출 하락'을 예고하고 있다.
액션: 공간 리뉴얼 또는 시즌 이벤트 투입. 기존 고객 대상 참여형 프로그램(커뮤니티/워크숍) 설계로 Distance 반등 유도."
3개 시나리오의 공통 교훈: Layer 1(Content)만으로는 시나리오 A를 "대성공", B를 "원인 불명의 하락", C를 "안정적"이라고 판단한다. Layer 2(Psyche)를 추가해야 비로소 A는 "립서비스", B는 "신선함 소진", C는 "조용한 이탈 전조"라는 진짜 진단이 가능하다.

8 학술적 검증 현황과 보강 전략

각 지표별 근거 수준 & 보강 필요도

지표학술 근거 수준한국어 검증보강 전략우선순위
Authenticity 강함 Newman 2003, Pennebaker 2011 ▵ K-LIWC 2016 일부 검증 한국어 리뷰 200건 자체 실험으로 보강 긴급
Clout 강함 Kacewicz 2014 ▵ 한국어 연구 제한적 동일 실험에서 함께 검증 높음
Analytical 강함 Pennebaker 2014 ▵ 한국어 접속사 체계 다름 한국어 인과 접속사 목록 자체 구축 중간
Emotional Tone 강함 LIWC-22 표준 O 한국어 감정사전 다수 기존 자원 활용 가능 낮음
Freshness Index ▵ 정보구조론 기반 (언어학) X 리뷰 맥락 직접 검증 없음 자체 실험 필수: 팝업스토어 리뷰에서 "이/가" vs "은/는" 비율과 실제 재방문/신선함 체감의 상관관계 검증 긴급
Psych. Distance 강함 대명사-거리 상관 다수 ▵ 한국어 대명사 빈도 낮음 대명사 + 지시어("여기/거기") + 어미("-잖아/-거든") 복합 지표로 확장 높음
보강 로드맵: ① 한국어 팝업스토어 리뷰 200~400건 수집 → ② K-LIWC + 조사/어미 분석 → ③ Authenticity, Freshness의 한국어 유효성 검증 → ④ 결과를 백서(White Paper)로 발행 → 피치덱에서 "자체 검증 데이터 보유"로 신뢰도 전환. 예상 기간: 3~4주, 리서치어시스턴트 1명.

★ 핵심 요약

Layer 1: Content (What)

  • "무엇을 말했는가"
  • n-gram, 감성분석, 토픽
  • 기본 자격 (Table Stakes)
  • 이것만으로는 "좋다/나쁘다"만 알 수 있음

Layer 2: Psyche (How)

  • "어떻게 말했는가"
  • LIWC, 조사패턴, 대명사
  • 차별화 무기 (Differentiator)
  • "진심인가, 확신인가, 신선한가"

Layer 3: Synthesis (Why)

  • "왜 그런 반응인가"
  • What × How 교차 분석
  • 6개 패턴 사전 기반 자동 진단
  • 액션까지 연결되는 인사이트
다른 솔루션은 "무엇을 말했는가"까지만 답한다.
RXR은 "얼마나 진심으로 말했는가"까지 답하는 유일한 구조다.
Layer 1은 마이크. Layer 2는 진심을 읽는 귀. 둘 다 있어야 대화가 된다.
