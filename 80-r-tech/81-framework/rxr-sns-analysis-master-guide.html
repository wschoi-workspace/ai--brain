MASTER GUIDE

RXR SNS Analysis

"긍정 50%입니다" 뒤에 숨겨진 진짜 이야기를 찾는 방법 | Project RENT · R-lab
SNS에 올라온 수백 건의 후기 중에서
"진짜 마음에서 우러나온 반응"만 골라내는 분석 방법입니다.

왜 이런 분석이 필요할까요?

팝업스토어를 열고, SNS 반응을 확인합니다. "300건 언급, 긍정 50%!" — 좋은 숫자처럼 보입니다. 하지만 이 숫자를 자세히 들여다보면 이런 것들이 섞여 있습니다:

🎭
과장된 긍정
"대박!!! 최고!!!" 느낌표 100개짜리 글. 뭐가 좋았는지는 안 써 있음.
💰
협찬 후기
돈을 받고 쓴 글. "긍정"이지만 진짜 좋아서 쓴 건 아닐 수 있음.
📋
노이즈
"새로 생긴 카페" — '새로'가 브랜드가 아니라 그냥 '새롭게'라는 뜻.
기존 도구의 한계: "긍정 50%"라는 숫자만 줄 뿐, 그 안에 과장이 몇 %이고, 진심이 몇 %인지 구분하지 못합니다. RXR SNS Analysis는 이 구분을 해냅니다.

기존 SNS 분석과 뭐가 다른가요?

기존 도구
숫자만 세기
"총 300건 언급됐습니다"
"긍정 50%, 부정 10%"
"워드클라우드: 팝업, 성수, 소주..."
"협찬? 있다 / 없다"
→ 그래서 어쩌라고?
→
RXR SNS Analysis
진심까지 읽기
"300건 중 진짜 유효한 건 204건"
"긍정 50%인데, 진심 긍정은 38%"
"방탈출이 핵심인데 7%만 언급"
"협찬도 진심의 깊이가 다름"
→ 다음에 이렇게 바꾸세요
쉽게 말하면, 기존 도구는 "몇 명이 좋아했는지"만 세지만, RXR은 "정말로 좋아했는지, 왜 좋아했는지, 얼마나 진심이었는지"까지 읽어냅니다.

핵심 개념, 쉽게 풀어보면

1. 진심 필터 — "찐 반응만 골라내기"

SNS에 올라온 모든 글을 3단계 체로 걸러냅니다. 점점 더 촘촘한 체로 걸러서, 마지막에 남는 것이 "진짜 반응"입니다.

1차 필터 — 노이즈 제거
관계없는 글 빼기
▼
2차 필터 — 협찬/광고 분리
돈 받고 쓴 글 따로 보기
▼
3차 필터 — 진심만 남기기
과장 없는 솔직한 글만
실제 사례: 새로중앙박물관 팝업 — 281건의 블로그 후기가 있었지만, 3차 필터를 통과한 진짜 반응은 204건이었습니다. 나머지 77건은 느낌표를 100개 넘게 쓰거나, 구체적 경험 없이 "대박!!"만 반복한 과장된 글이었습니다.

2. 진정성 지수 — "이 사람, 진짜로 좋아한 거 맞아?"

모든 후기에 "얼마나 진심인가"를 점수로 매깁니다. 같은 "좋았어요"라도, 구체적인 경험을 풀어쓴 글과 느낌표만 가득한 글은 전혀 다르니까요.

진정성 높은 후기
"방탈출 미션이 생각보다 몰입감 있었다. 힌트를 3번 써서 겨우 클리어했는데, 같이 간 친구랑 엄청 웃었다. 침착맨 굿즈 퀄리티도 의외로 좋더라."
→ 구체적 경험 + 솔직한 감상 + 디테일
진정성 낮은 후기
"대박!!!! 너무너무너무 좋았어요!!!!! 꼭 가보세요!!!!!!! 인생 팝업!!!!!!!!!"
→ 느낌표 132개, 뭐가 좋았는지 안 써 있음

3. 추천 확신도 — "주변에 추천할 만큼 좋았어?"

"좋았다"고 느끼는 것과 "주변 사람한테 꼭 가보라고 말하는 것"은 다릅니다. 추천 확신도는 "이 사람이 실제로 주변에 추천할 의향이 있는가"를 측정합니다.

"꼭 가보세요", "강추합니다", "같이 가자"처럼 추천 표현이 담긴 글은 추천 확신도가 높습니다. 반면 "나쁘진 않았어요" 같은 글은 긍정이지만 추천까지는 아닙니다. 진짜 브랜드 옹호자가 될 수 있는 사람을 찾는 지표입니다.

4. 신선함 지수 — "새로운 발견이 있었어?"

한국어에는 흥미로운 패턴이 있습니다. "분위기가 좋았다"(새로운 발견)와 "분위기는 좋았다"(이미 알고 있던 것)는 조사 하나 차이지만 뉘앙스가 완전히 다릅니다.

"분위기가 좋았다"
→ 처음 느낀 놀라움, 발견
신선함 높음
"분위기는 좋았다"
→ 이미 알고 있던 것 확인
신선함 낮음
팝업 초반에는 "이/가"가 많이 쓰입니다 (새로운 경험). 하지만 시간이 지나면 "은/는"이 늘어납니다 (이미 알려진 정보). 이 변화를 추적하면 팝업의 "신선함 유통기한"을 알 수 있습니다.

5. 후기 깊이 등급 — "한 줄짜리 vs 에세이급"

같은 1건의 후기라도, "좋았어요" 한 줄과 500자짜리 체험 서사는 가치가 다릅니다.

등급유형예시분석 가치
Q5서사형500자 이상, 경험을 이야기처럼 풀어씀매우 높음
Q4분석형300자 이상, 장단점을 구체적으로 분석높음
Q3경험형150자 이상, 간단한 체험담보통
Q2감상형사진 위주 + 짧은 감상낮음
Q1간단형"좋았어요" 한 줄매우 낮음
기존 도구는 Q5 에세이급 후기나 Q1 한줄 후기를 똑같이 "1건"으로 셉니다. RXR은 깊이에 따라 가중치를 다르게 적용하여, 진짜 인사이트가 담긴 후기에 더 큰 비중을 둡니다.

분석은 어떻게 진행되나요?

1
데이터 수집
네이버 블로그에서 관련 후기를 자동 수집하고, Brand24에서 Instagram/X/뉴스 등 멀티채널 데이터를 통합합니다. 중복은 자동 제거합니다.
2
콘텐츠 분류
각 글을 자발적 후기 / 협찬 / 정보성 / 노이즈 등으로 자동 분류합니다. 협찬이라고 버리지 않고, "협찬인데 진짜 좋아서 쓴 건가?"까지 분석합니다.
3
콘텐츠 분석 — "무엇을 말했는가"
어떤 토픽(제품/공간/체험/웨이팅 등)이 얼마나 언급됐는지, 감성은 어떤지, 후기의 깊이는 어떤지를 분석합니다.
4
심리 분석 — "진심으로 그렇게 느꼈는가"
글을 쓴 사람의 진정성, 추천 의향, 새로운 발견 여부를 글쓰기 패턴에서 읽어냅니다. 사람은 의식적으로 내용은 꾸밀 수 있지만, 글쓰기 습관(조사, 대명사, 느낌표 빈도)은 무의식적이라 조작이 어렵습니다.
5
진심 필터 3단계
노이즈 제거 → 협찬 분리 → 과장 긍정 탈락. 최종적으로 "진심으로 좋았던 사람"만 남깁니다.
6
해석 + 제언
"왜 이런 결과가 나왔는지" 해석하고, "다음에 무엇을 바꿔야 하는지" 구체적으로 제안합니다. 숫자만 던지지 않습니다.

실제로 어떤 발견이 있었나요?

사례 1: 새로중앙박물관 (16일 운영)

총 384건 멘션 (네이버 281 + SNS 103)

발견 1. 기존 도구: "긍정 50%" → RXR: "진심 긍정은 38%" (과장 77건 탈락)
발견 2. 방탈출이 핵심 체험인데 후기의 7%만 언급 — 소주(68%)에 묻힘
발견 3. 16일 장기 운영 → 후반부 후기의 진정성이 급락 (기억이 미화됨)
제언: 체험이 콘텐츠가 되는 설계 + 주차별 콘텐츠 리프레시 필요

사례 2: 이클립스 월드 (4일 운영)

총 676건 멘션 (네이버 299 + SNS 377)

발견 1. 기존 도구: "긍정 46%" → RXR: "진심 긍정은 33%" (과장 탈락)
발견 2. 신제품 버즈가 팝업의 4.5배 — 팝업 자체 화제성 부족
발견 3. 4일 단기 운영 → 진정성 하락이 적음 (-6점) (단기의 장점)
제언: 팝업 독자 화제화 + 체험 요소 SNS 공유 장치 설계
두 사례를 비교하면 일관된 패턴이 보입니다: 기존 도구의 긍정률은 실제보다 12~13%p 부풀려져 있고, 핵심 체험 요소는 제품 이야기에 묻히는 경향이 있습니다. 이건 하나의 팝업만 봐서는 몰랐을 인사이트입니다.

이 분석으로 무엇이 가능해지나요?

브랜드 담당자에게
• "진짜 성과"를 알 수 있습니다. 300건 언급이 아니라, 그 중 진심으로 좋아한 사람이 몇 명인지.

• 다음 캠페인의 방향을 잡을 수 있습니다. "체험이 제품에 묻혔다"를 알면, 다음엔 체험 중심으로 설계할 수 있습니다.

• 협찬 효과를 정밀하게 평가할 수 있습니다. "돈을 쓴 만큼 진심 반응이 늘었나?"를 수치로 확인합니다.
기획자에게
• 체험 설계를 데이터로 검증할 수 있습니다. "방탈출을 넣었는데 7%만 언급됐다" → 설계 수정 근거.

• 운영 기간의 최적 길이를 알 수 있습니다. 16일은 진정성이 급락했지만 4일은 유지됐다 → 기간 결정 근거.

• "진짜 팬"을 찾아낼 수 있습니다. Q5 서사형 후기를 쓴 사람들이 브랜드 옹호자 후보입니다.
