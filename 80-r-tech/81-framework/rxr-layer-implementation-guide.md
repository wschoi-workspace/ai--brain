# RXR Master Framework — 층위별 고도화 가이드

> 버전: 1.0 | 작성일: 2026-04-06
> Project RENT · R-lab
> 기준: 현시점 기술적으로 구현 가능한 것만 포함. 안 되는 건 제외.

---

## 0. 기술 원칙

1. **CCTV 기반 우선** — 추가 센서 최소화
2. **오픈소스 우선** — 기술은 자산이 아니라 프레임이 자산
3. **개인정보 세이프** — 얼굴 인식 없이 구현 (Re-ID는 의류+체형 기반)
4. **현시점 수집 가능한 데이터**에서 시작

---

## 1. L0~L1: 노출 · 시선 (Exposure & Attention)

### 현재 가능한 것

| 기능 | 기술 | 정확도 | 상태 |
|------|------|:---:|:---:|
| 유동인구 카운팅 | YOLOv8 nano | 92~95% | ✅ 구현됨 |
| 파사드 앞 정지 감지 | BoT-SORT + 좌표 정지 판정 | 90%+ | ✅ 구현 가능 |
| 시선 방향 추정 | MediaPipe Head Pose | 70~80% | △ 기본 구현 |
| DID 콘텐츠 연동 | 시선 시점 × 콘텐츠 스케줄 교차 | — | ⬜ 설계 필요 |

### 고도화 프레임

**L0 → L1 전환율 = "주목률(Attention Rate)"**

```
주목률 = 파사드 앞 2초+ 정지한 사람 / 전체 유동인구
```

**DID 콘텐츠 효과 측정:**
- 파사드 DID에 시간대별 다른 콘텐츠 송출
- 각 콘텐츠 시간대의 주목률 비교 → **콘텐츠 A/B 테스트 자동화**
- 주목 후 진입한 비율(L1→L2 전환율) = 콘텐츠의 "진입 유도력"

**데이터 산출:**
- 시간대별 유동인구 트래픽 패턴
- 콘텐츠별 주목률 / 주목 후 진입 전환율
- 요일 × 시간대 × 날씨(외부 API) 교차 분석

## 2. L2: 진입 (Entry)

### 현재 가능한 것

| 기능 | 기술 | 정확도 | 상태 |
|------|------|:---:|:---:|
| 진입 카운팅 | YOLOv8 + BoT-SORT (입구 라인 교차) | 94~97% | ✅ 구현됨 |
| 진입 시점 기록 | 프레임 타임스탬프 | ±0.5초 | ✅ 구현됨 |
| 직원 제외 | 스태프 등록 (고정 위치/의류 패턴) | 85~90% | △ 개선 필요 |

### 고도화 프레임

**L1 → L2 전환율 = "진입률(Entry Rate)"**

```
진입률 = 매장 진입 인원 / 파사드 앞 정지(주목) 인원
```

**진입 품질 분류:**

| 유형 | 행동 패턴 | 의미 |
|------|---------|------|
| **목적형 진입** | 주목 없이 직진 진입 | 이미 아는 고객 / 재방문 |
| **유인형 진입** | 파사드 주목 후 진입 | 콘텐츠에 의해 유도된 신규 |
| **동반형 진입** | 2인+ 그룹 진입 | 사회적 방문 (체험/데이트) |
| **통과형** | 진입 후 30초 내 퇴장 | 오인입/단순 호기심 |

→ 진입 유형별 이후 인게이지먼트 패턴이 다름. 이것이 변환 테이블의 첫 데이터.

## 3. 비주얼 아웃핏 AI — 라이프스타일 세그먼테이션

### 현재 기술 수준

| 기술 | 모델 | 정확도 | 라이선스 |
|------|------|:---:|:---:|
| 의류 감지 (Clothing Detection) | YOLOv8 + DeepFashion2 | 88~93% | MIT |
| 의류 카테고리 분류 | CNN (ResNet/EfficientNet) | 85~92% | Apache |
| 스타일 분류 | Fashion-Gen / YOLO-GARNet | 80~88% | MIT |
| 컬러 추출 | OpenCV HSV 히스토그램 | 95%+ | Apache |
| 체형 추정 | MediaPipe Pose Estimation | 85~90% | Apache |

**핵심:** 얼굴 인식 없이, **의류 + 컬러 + 체형 + 액세서리**만으로 라이프스타일 군 분류 가능.

### 아웃핏 분석 파이프라인

```
CCTV 프레임
  ↓ YOLOv8 사람 감지
  ↓ 상반신/하반신 크롭
  ↓
  ┌─────────────────────────────────────┐
  │  1. 의류 카테고리 (13개 기본 타입)    │
  │     상의: 티셔츠/셔츠/재킷/코트/니트  │
  │     하의: 진/슬랙스/스커트/쇼츠       │
  │     원피스/점프수트/아우터             │
  │                                     │
  │  2. 컬러 팔레트 (상위 3색)            │
  │     모노톤/뉴트럴/비비드/파스텔       │
  │                                     │
  │  3. 스타일 키워드                     │
  │     캐주얼/포멀/스트릿/스포츠/        │
  │     미니멀/프리미엄/보헤미안/빈티지    │
  │                                     │
  │  4. 액세서리 (감지 가능 시)           │
  │     가방 타입/모자/선글라스/시계       │
  │                                     │
  │  5. 연령대·성별 추정                  │
  │     체형+의류 조합으로 추정           │
  │     (얼굴 인식 아님)                  │
  └─────────────────────────────────────┘
  ↓
  라이프스타일 세그먼트 분류
```

### 라이프스타일 세그먼트 (초기 모델)

| 세그먼트 | 아웃핏 시그니처 | 예상 소비 패턴 |
|---------|--------------|-------------|
| **미니멀 모던** | 모노톤, 깔끔한 실루엣, 무채색 가방 | 선별적 구매, 품질 중시, 객단가 높음 |
| **캐주얼 컴포트** | 데님+티, 뉴트럴 컬러, 스니커즈 | 가격 민감, 충동구매 가능, 빈도 높음 |
| **스트릿 트렌드** | 오버사이즈, 로고 노출, 비비드 | SNS 공유 활발, 한정판 반응, 콘텐츠 소비 |
| **프리미엄 클래식** | 정제된 핏, 가죽 소재, 쥬얼리 | 높은 객단가, 브랜드 로열티, 재방문 |
| **스포티 액티브** | 애슬레저, 기능성 소재, 밝은 컬러 | 경험 중심, 워크숍/클래스 반응 |
| **아트/보헤미안** | 레이어드, 빈티지, 패턴 믹스 | 독특한 아이템 탐색, 공간 감성 반응 |
| **매스 노멀** | 특징 없는 일상복 | 가격 중심, 통과형 비율 높음 |

### 세그먼트 분류 정확도 전략

**Phase 1 (파일럿):** 7개 세그먼트, 수동 레이블링 500건 → 모델 학습 → 70~75% 정확도
**Phase 2 (고도화):** 세그먼트 세분화, 5,000건+ → Fine-tuning → 80~85% 정확도
**Phase 3 (성숙):** 구매 데이터 교차 검증 → 세그먼트별 소비 패턴 실증 → 자동 피드백 루프

### 개인정보 안전성

- ✅ 얼굴 인식 없음 (체형+의류만 분석)
- ✅ 세션 기반 (매장 내에서만 추적, 퇴장 시 ID 삭제)
- ✅ 개인 식별 불가 (의류 특징만 벡터화, 원본 이미지 미저장)
- ✅ GDPR/개인정보보호법 준수 가능

## 4. L3: 체류 DQS (Dwell Quality Score)

### 현재 가능한 것

| 기능 | 기술 | 정확도 | 상태 |
|------|------|:---:|:---:|
| 절대 체류시간 | BoT-SORT 프레임 범위 | ±2초 | ✅ 구현됨 |
| 구역별 체류 | Zone 정의 + 좌표 매핑 | 90%+ | △ Zone 설정 필요 |
| 동선 히트맵 | 좌표 밀도 맵 변환 | 공간해상도 의존 | △ 기본 구현 |
| 상호작용 감지 | 상품 터치/픽업 감지 | 70~80% | ⬜ 추가 개발 |

### DQS 구현 설계

**DQS = (실제 체류 / 콘텐츠 기대 체류) × 밀도 계수 × 세그먼트 보정**

**① 콘텐츠 기대 체류 (기준점 테이블)**

파일럿에서 수집할 기준값:

| Zone 유형 | 기대 체류 | 측정 방법 |
|----------|:---:|------|
| 상품 진열대 | 45초~1.5분 | 전체 방문자 평균 |
| 체험존 (터치/시향) | 2~4분 | 전체 방문자 평균 |
| 전시/스토리텔링 | 3~7분 | 전체 방문자 평균 |
| 포토존 | 1~3분 | 전체 방문자 평균 |
| 결제 대기 | 제외 | DQS 산정에서 제외 |

→ 이 기준값은 파일럿 데이터가 쌓이면 **자동으로 업데이트**

**② 체류 밀도 계수 (Dwell Density)**

같은 3분이라도:
- 1개 Zone에서 3분 정지 = 밀도 낮음 (수동적 관람)
- 3개 Zone을 각 1분씩 순회 = 밀도 높음 (능동적 탐색)
- 상품 터치/픽업 행위 감지 시 밀도 가산

```
밀도 계수 = (방문 Zone 수 / 전체 Zone 수) × (1 + 상호작용 횟수 × 0.1)
```

**③ 세그먼트 보정**

아웃핏 AI로 분류된 세그먼트의 평균 체류 대비 편차:

```
세그먼트 보정 = 개인 체류 / 해당 세그먼트 평균 체류
```
- "미니멀 모던" 세그먼트 평균 2.5분인데 이 고객은 5분 → 보정 2.0 → 강한 관심 신호
- "캐주얼 컴포트" 세그먼트 평균 4분인데 이 고객은 4분 → 보정 1.0 → 보통 수준

## 5. L4: 구매 전환 (Conversion)

### 현재 가능한 것

| 기능 | 기술 | 상태 |
|------|------|:---:|
| POS 데이터 연동 | REST API / CSV 수동 연동 | ⬜ 매장별 개발 |
| CCTV-POS 시간 동기화 | 타임스탬프 매칭 (±30초 윈도우) | ⬜ 설계됨 |
| 결제 세부항목 분석 | 구매 카테고리/단가/수량 | ⬜ POS 연동 후 가능 |

### 고도화 프레임

**구매 전환의 질적 분류:**

| 구매 유형 | 판별 기준 | 의미 |
|---------|---------|------|
| **확신 구매** | 체류 DQS 높음 + 다구역 탐색 후 구매 | 깊은 관여 → 재방문 가능성 높음 |
| **충동 구매** | 체류 DQS 낮음 + 단일 Zone에서 즉시 구매 | 취향 반응 → 아이템 기반 세그먼트 신호 |
| **목적 구매** | 진입 후 최단 동선으로 구매 | 기존 고객 / 재방문자 패턴 |
| **동반 구매** | 그룹 진입 + 구매 | 사회적 소비 → 추천 기반 전환 |

**결제 세부항목 → 취향 분석:**

Bracket 매장 자사 POS이므로 결제 데이터의 세부항목을 완전 통제 가능:

| 데이터 | 추출 인사이트 |
|--------|------------|
| 구매 카테고리 | 관심 영역 (식음/굿즈/체험/클래스) |
| 가격대 | 지불 용의(WTP) 세그먼트 |
| 구매 시점 (체류 몇 분 후) | 의사결정 속도 → 확신형 vs 탐색형 |
| 동반 구매 여부 | 사회적 소비 패턴 |
| 옵션 선택 (사이즈/컬러) | 세부 취향 데이터 |

**아웃핏 세그먼트 × 구매 교차:**
- "미니멀 모던 세그먼트가 평균 객단가 35,000원, 주로 디저트+굿즈 세트"
- "스트릿 트렌드 세그먼트가 포토존 체류 5분+ 후 굿즈 구매 전환 42%"
→ 세그먼트별 소비 프로필 자동 축적

## 6. L5: 재방문 (Return Visit)

### 기술적 현실

| 방법 | 정확도 | 개인정보 | 실현성 |
|------|:---:|:---:|:---:|
| 얼굴 인식 Re-ID | 85~95% | ✗ 위반 위험 | ✗ 제외 |
| 의류 기반 Re-ID (세션 내) | 80~90% | ✅ 안전 | ✅ 가능 |
| 의류 기반 Re-ID (일간) | 30~50% | ✅ 안전 | ⚠ 불안정 (옷 변경) |
| **회원 앱/카드 스캔** | **99%+** | **✅ 동의 기반** | **✅ 가장 현실적** |
| Wi-Fi/BLE 프로브 | 60~75% | ⚠ 애매 | △ 추가 장비 |

### 고도화 프레임 — 하이브리드 접근

**얼굴 인식 없이 재방문을 측정하는 현실적 전략:**

```
┌──────────────────────────────────────────────┐
│  세션 내 추적 (Same Day)                      │
│  → 의류 기반 Re-ID: 80~90% 정확도             │
│  → "같은 날 매장을 2번 방문했다" 감지 가능     │
│  → 개인정보 안전 (세션 종료 시 ID 삭제)       │
├──────────────────────────────────────────────┤
│  일간/주간 재방문 (Cross-Day)                  │
│  → 회원 앱/카드 스캔: 99%+ 정확도             │
│  → L6(회원) 전환을 적극 유도하여 해결          │
│  → 비회원 재방문은 서베이 표본조사로 추정      │
└──────────────────────────────────────────────┘
```

**핵심 전략:** L5 정확도를 높이기 위해 **L6(회원 전환)을 적극 유도**. 회원이 되면 재방문 추적이 자동으로 해결됨. 비회원 재방문률은 표본 서베이로 추정.

## 7. L6: 회원 CRM + OMO

### 현재 가능한 것

| 기능 | 상태 |
|------|:---:|
| 회원가입 시스템 (매장 내 태블릿/QR) | ⬜ 구축 필요 |
| 회원 DB (이름, 연락처, 가입일) | ⬜ CRM 설계 필요 |
| 오프라인 구매 이력 연결 | ⬜ POS-CRM 연동 필요 |
| 온라인 구매 이력 연결 | ⬜ OMO API 필요 |

### 고도화 프레임

**회원가입 = Stage 1 → Stage 2 전환점**

회원가입 시점에 발생하는 데이터 연결:

```
[익명 Stage 1 데이터]              [실명 Stage 2 데이터]
  아웃핏 세그먼트           →        회원 프로필에 태그
  인게이지먼트 레벨(L0~L4)  →        개인 인게이지먼트 이력
  구매 세부항목             →        개인 취향 프로필
  체류 DQS                 →        개인 관여도 기준선
                                    
                           +        온라인 구매 이력 (OMO)
                           +        이메일/SNS 리타겟팅
                           +        개인화 프로모션
```

**OMO 통합의 핵심 가치:**

| 오프라인 데이터 | 온라인 데이터 | 통합 인사이트 |
|:---:|:---:|------|
| 충동 구매 (감성) | 비교 구매 (이성) | 이 사람의 이성/감성 비율 |
| 체험 선호 (공간) | 검색 키워드 (관심) | 잠재 관심 영역 확장 |
| 아웃핏 세그먼트 | 장바구니 카테고리 | 라이프스타일 정밀 프로필 |
| DQS (관여 깊이) | 페이지 체류/이탈 | 채널별 관여 패턴 비교 |

## 8. L7: 후기 RQL + 2-Layer

### 현재 가능한 것

| 기능 | 기술 | 상태 |
|------|------|:---:|
| SNS 리뷰 크롤링 | 네이버/인스타그램 API | ⬜ 크롤러 구축 필요 |
| 감성 분석 (Content Layer) | KoBERT / Transformers | ⬜ 모델 학습 필요 |
| FWA 분석 (Psyche Layer) | K-LIWC + KoNLPy | ⬜ 파이프라인 구축 필요 |
| 매장 내 서베이 | QR 기반 모바일 서베이 | ⬜ 시스템 구축 필요 |

### 고도화 프레임

**RQL 자동 판정 파이프라인:**

```
리뷰 텍스트 입력
  ↓
  ┌─── Content Layer ───────────────────────┐
  │  단어 수 → 길이 등급 (Q1~Q5 초기 판정)   │
  │  고유명사 비율 → 구체성 점수              │
  │  비교/인과 표현 → 분석 깊이 점수          │
  │  감성 극성 → 감정 강도                    │
  │  토픽 추출 → 관심 영역                    │
  └──────────────────────────────────────────┘
  ↓
  ┌─── Psyche Layer ─────────────────────────┐
  │  대명사 패턴 → 관여 방식 (나/우리/여기)    │
  │  조사 패턴 → "이/가"(발견) vs "은/는"(평가)│
  │  Authenticity → 진정성 점수               │
  │  Clout → 추천 확신도                      │
  │  Analytical → 이성적 vs 감성적             │
  │  Tone → 감정 깊이                         │
  └──────────────────────────────────────────┘
  ↓
  가중 L7 = RQL × Authenticity 보정 × Clout 보정
  ↓
  Sincerity Gate: Authenticity < 30 → 기만 플래그
```

**서베이 × 리뷰 교차 분석:**

매장 내 리워드 서베이 응답자 중 SNS 리뷰도 남긴 사람을 교차 분석:
- 서베이 "추천하겠다" + 리뷰 Clout 높음 → **진짜 옹호자 확증**
- 서베이 "추천하겠다" + 리뷰 Clout 낮음 → **사회적 바람직성 편향 감지**
→ 이 교차 패턴이 서베이 보정 계수로 축적

## 9. AI 고객 카테고리 (C.C) — 통합 세그먼테이션

### 모든 층위 데이터가 합쳐지는 곳

```
┌─ 시각 데이터 ─────────────────────────┐
│  아웃핏 세그먼트 (패션 타입)            │
│  연령대·성별 추정                      │
│  그룹/단독 방문 여부                    │
└────────────────────────────────────────┘
         +
┌─ 행동 데이터 ─────────────────────────┐
│  진입 유형 (목적/유인/동반/통과)         │
│  체류 DQS (관여 깊이)                   │
│  동선 패턴 (어떤 Zone에 반응했는가)      │
│  인게이지먼트 레벨 (L0~L7)              │
└────────────────────────────────────────┘
         +
┌─ 거래 데이터 ─────────────────────────┐
│  구매 카테고리 · 객단가                 │
│  구매 유형 (확신/충동/목적/동반)         │
│  의사결정 속도 (진입→구매 소요 시간)     │
└────────────────────────────────────────┘
         +
┌─ 텍스트 데이터 (있는 경우) ─────────────┐
│  RQL 등급 (후기 깊이)                    │
│  FWA 프로필 (진정성/확신/분석성/감정)     │
│  Content 토픽 (관심 영역)                │
└────────────────────────────────────────┘
         ↓
┌─ AI C.C (Customer Category) ──────────┐
│                                        │
│  세그먼트 예시:                         │
│                                        │
│  "30대 미니멀 · 확신구매형 · DQS 상위  │
│   · 디저트+굿즈 선호 · Clout 72        │
│   → 핵심 팬 + 잠재 옹호자"             │
│                                        │
│  "20대 스트릿 · 유인진입 · 포토존 집중  │
│   · 구매 없음 · 인스타 후기 Q2          │
│   → 화제성 방문 + 콘텐츠 소비형"        │
│                                        │
└────────────────────────────────────────┘
```

### C.C의 활용

| 활용 | 내용 |
|------|------|
| **브랜드에게** | "당신 브랜드에 반응하는 핵심 세그먼트는 '30대 미니멀 확신구매형'이며, 이들의 DQS 평균은 1.8, 객단가 42,000원" |
| **공간 설계에** | "포토존에 스트릿 세그먼트가 집중, 디저트존에 프리미엄 세그먼트가 집중 → Zone별 콘텐츠 최적화" |
| **마케팅에** | "옹호자 세그먼트의 후기 RQL 평균 Q4, Clout 68 → 이 사람들의 리뷰를 마케팅 자산으로 활용" |
| **OMO에** | "오프라인 충동구매형이 온라인에서는 비교구매형 → 채널별 다른 커뮤니케이션 전략" |
