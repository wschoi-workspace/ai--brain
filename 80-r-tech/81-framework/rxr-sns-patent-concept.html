Patent Concept Document

소셜 미디어 데이터 기반
소비자 인게이지먼트 분석 시스템

온라인 소비자 반응 데이터에서 표면적 수치(건수, 긍정률)를 넘어, 심리적 진정성과 행동적 깊이를 다단계로 측정·추적·비교·환산하는 통합 분석 체계

PROJECT RENT · R-lab · 본 문서는 특허 개념 설명을 위한 것으로, 구체적 산출 공식·가중치·임계값은 포함되어 있지 않습니다.

1. 발명의 배경 — 기존 도구의 한계

기존 SNS 분석 도구의 한계

  • 총 언급 건수(mentions)만 집계 — "많이 언급됨 = 성공"
  • 긍정/부정 2-3단계 감성 분류 — "좋다"의 질적 차이 구분 불가
  • 협찬/광고와 자발적 후기를 동일하게 1건으로 취급
  • 리그램·단순 공유와 깊은 서사형 리뷰를 구분하지 못함
  • 시간에 따른 인식 변화(감쇠/강화) 추적 불가
  • 분석 결과를 금전적 가치로 환산하는 표준 체계 부재

본 발명이 해결하는 문제

  • "281건의 언급" 대신 "진짜 유효한 반응 204건"을 식별
  • 같은 "긍정"이라도 진심의 농도를 구분
  • 협찬 안에서도 진심 스펙트럼을 분석
  • 후기의 질적 깊이를 5단계로 가중 측정
  • 캠페인 효과의 지속성을 4축으로 주차별 추적
  • 심리 지표를 금전적 가치(추정 미디어 가치)로 전환

2. 발명의 핵심 구조 — 4단계 분석 파이프라인

STEP 1
원본 분석
수집 → 분류 → 정제 → 이중 계층 분석
→
STEP 2
추적 분석
4축 지수 산출 → 시계열 추적 → 감쇠 판정
→
STEP 3
비교 분석
다축 교차 비교 → 구조적 법칙 도출
→
STEP 4
가치 환산
심리 지표 → 행동 효과 → 금전 가치

각 단계는 독립적으로도 작동하며, 순차적으로 연결되어 종합 분석 체계를 구성합니다.

3. STEP 1: 원본 분석 — "진짜 반응"을 식별하는 다단 정제

3-1. 데이터 수집 및 콘텐츠 유형 분류

수집된 소셜 미디어 데이터를 7가지 유형으로 자동 분류하여, 분석 대상과 노이즈를 체계적으로 분리합니다.

유형설명분석 처리
진성 후기개인 경험 신호 + 구체적 디테일이 풍부한 포스트심층 분석 대상
일반 포스트이벤트 관련이나 특별 플래그 없음심층 분석 대상
협찬/체험단광고·협찬 마커가 감지된 포스트심층 분석 (진심 스펙트럼 분석)
나열형 정보"총정리", "리스트" 등 큐레이션 포스트건수만 집계
단순 공유리그램, 짧은 재공유건수만 집계
비즈니스/보도보도자료, 비즈니스 홍보제외 또는 집계만
노이즈이벤트와 무관한 콘텐츠제외

핵심 차별화: 협찬 포스트를 단순 제외하지 않고, 진심 스펙트럼으로 분석합니다. 같은 협찬이라도 진심의 농도가 다르며, 이 차이를 수치화하는 것이 본 발명의 고유 기능입니다.

3-2. 이중 계층 분석 (Dual-Layer Analysis)

각 포스트를 "무엇을 말했는가"와 "어떻게 말했는가"의 두 계층으로 동시에 분석합니다.

표현 계층 (무엇을 말했는가)

  • 토픽 분석: 이벤트별 주제 분류 + 비중 변화 추적
  • 감성 분석: 4단계(긍정/부정/혼합/중립) — "혼합"은 균형 잡힌 리뷰의 신호
  • 후기 질적 수준: 5단계 등급화 — 단순 감상에서 설득형 서사까지, 각각 다른 가중치 부여

심층 심리 계층 (어떻게 말했는가)

  • 심리 신뢰성: 텍스트 패턴에서 진정성 수준을 추정하는 독자적 지표
  • 발언 강도: 추천 의지와 사회적 영향력 수준을 추정하는 지표
  • 정보 갱신성: 새로운 발견인지, 이미 아는 정보의 재확인인지를 한국어 조사 패턴에서 추출하는 독자적 지표

3-3. 다단 정제 게이트 (Multi-Stage Refinement Gate)

수집된 전체 데이터에서 "진짜 반응"만을 단계적으로 추출하는 3단계 필터링 체계입니다.

1단계 정제"시장에 얼마나 노출됐는가?"
2단계 정제"유료 홍보 빼면 실제 반응은?"
3단계 정제"진심으로 좋았던 사람은?"

핵심 가치: 각 단계를 통과할 때마다 건수·긍정률·심리 지표 평균이 변화합니다. 이 단계 간 차이(Delta)가 가장 핵심적인 인사이트입니다. 예를 들어 1단계 긍정률 50%가 3단계에서 38%로 떨어지면, 12%p의 "과장 긍정"이 걸러진 것입니다.

4. STEP 2: 추적 분석 — 캠페인 효과는 얼마나 오래 가는가

4-1. 수용자 인게이지먼트 종합 지수 (4축)

1단계 분석 결과를 4가지 독립 축으로 종합하여 "브랜드를 얼마나 진심으로 좋아하는지"를 하나의 지수로 산출합니다.

관여 깊이

얼마나 정성껏 리뷰를 썼는가? 대충 쓴 건가, 진심을 담은 건가?

추천 강도

남에게 추천까지 했는가? 좋았다고만 한 건가, 가보라고 권한 건가?

진정 비율

전체 반응 중 진심이 몇 %인가? 노이즈를 빼면 진짜 목소리는?

발견 신선도

새로운 경험이었나? 예상대로였나, 의외의 발견이었나?

4-2. 2가지 관점으로 읽기

인게이지먼트 관점

"깊이 있게 좋아하는 사람이 얼마나 되나" — 관여 깊이와 발견 신선도에 더 큰 비중

예: 인게이지먼트 78 / 이슈성 52 = "조용하지만 깊은 팬을 만든 캠페인"

이슈성 관점

"얼마나 퍼지고 화제가 되었나" — 추천 강도와 진정 비율에 더 큰 비중

예: 인게이지먼트 48 / 이슈성 75 = "화제는 됐지만 깊은 팬은 못 만든 캠페인"

4-3. 시간에 따른 감쇠 추적 (Decay Tracking)

캠페인 직후(T+0)부터 4주간(T+1w ~ T+4w) 4축을 반복 측정하여 효과의 지속성을 판정합니다.

핵심 발견: 4축은 각각 다른 속도로 감쇠합니다. 이 차이를 추적하면 "무엇이 살아남고 무엇이 먼저 죽는지"를 알 수 있습니다.

관여 깊이

느리게 감쇠
체험 기억이 깊이 각인

추천 강도

빠르게 감쇠
"지금 핫한" 맥락에 의존

진정 비율

오히려 상승
노이즈가 먼저 사라짐

발견 신선도

급격히 감쇠
놀라움은 빠르게 적응

5. STEP 3: 비교 분석 — 왜 A 캠페인이 B보다 효과적인가

여러 캠페인의 분석 결과를 5가지 축으로 교차 비교하여, 단순 순위가 아니라 "왜 이런 차이가 나는가"에 대한 구조적 법칙을 도출합니다.

비교 축측정 내용핵심 질문
진정성 유지력시간에 따른 심리 신뢰성의 변화량"이 캠페인의 진정성은 얼마나 오래 유지되는가?"
컨셉 침투율핵심 컨셉이 소비자 토픽에서 차지하는 비율"브랜드가 의도한 메시지가 실제로 전달됐는가?"
리뷰 깊이분석형+서사형 리뷰의 비율"소비자가 단순 감상이 아닌 깊은 리뷰를 쓰는가?"
긍정률 진실성1단계 긍정률 대비 3단계 긍정률의 차이"표면적 긍정 중 얼마나 많은 거품이 있는가?"
협찬 신뢰도자발적 포스트와 협찬 포스트의 심리 지표 격차"협찬이 브랜드 신뢰를 훼손하고 있는가?"

법칙 도출 원칙: 2개 이상의 캠페인에서 패턴이 확인되면 "법칙"으로 격상. 1개에서만 발견되면 "시사점"으로 분류. 직관과 다른 결과(예: 규모 1위 ≠ 효과 1위)를 적극적으로 강조.

6. STEP 4: 가치 환산 — "이 캠페인이 돈으로 얼마짜리인가"

6-1. 전환 불가 원칙과 3단계 전환 체인

심리 지표를 매출로 직접 전환하는 것은 원칙적으로 불가능합니다. 소비자의 심리적 호감과 실제 구매는 서로 다른 의사결정 영역에서 작동하기 때문입니다. 따라서 검증된 중간 변수를 경유하는 3단계 전환 체인을 사용합니다.

Stage 1: 심리 지표 (4축)
↓ 축별 행동 전환 계수
Stage 2: 행동 효과 단위 (재구매율, 신규고객, 전환율, Earned Media)
↓ 업종별 단가
Stage 3: 추정 미디어 가치 (₩)

6-2. 4축별 경제적 경로

심리 축경제적 경로산출하는 가치
관여 깊이재구매/재방문깊이 관여한 사람이 다시 오는 가치
추천 강도신규 고객 획득추천으로 유입되는 신규 고객의 가치
진정 비율전환율 프리미엄진성 리뷰가 전환율을 높이는 가치
발견 신선도Earned Media자발적 공유가 광고비를 대체하는 가치

6-3. 시간 가중 누적 가치

4축의 감쇠 패턴이 다르므로 각각 독립적인 감쇠 함수를 적용하여 4주 누적 가치를 산출합니다. 업종(F&B/패션/공간/테크)과 규모에 따라 파라미터가 조정됩니다.

7. 발명의 특징 요약

1. 다단 정제를 통한 진성 반응 추출

7유형 콘텐츠 분류 + 3단계 정제 게이트로 "진짜 반응"만 식별. 협찬 포스트를 제외하지 않고 진심 스펙트럼으로 분석.

2. 이중 계층 동시 분석

표현 계층(무엇을)과 심층 심리 계층(어떻게)을 동시에 분석하여, 같은 "좋다"라도 진심의 농도를 구분.

3. 4축 인게이지먼트 지수 + 감쇠 추적

관여 깊이/추천 강도/진정 비율/발견 신선도 4축으로 종합하고, 축별로 다른 감쇠 패턴을 독립 추적하여 캠페인 효과의 지속성을 판정.

4. 심리 지표의 금전 가치 전환

직접 전환이 아닌 3단계 전환 체인(심리→행동→금전)으로 추정 미디어 가치 산출. 업종·규모별 파라미터와 감쇠 보정 적용.

5. 다축 교차 비교 및 법칙 도출

5축 비교 프레임워크로 복수 캠페인의 효과를 구조적으로 비교하고, 재사용 가능한 마케팅 법칙을 자동 도출.

6. 한국어 텍스트 특화 심리 분석

한국어 조사 패턴("이/가" vs "은/는")에서 정보 갱신성을 추출하는 독자적 지표. 한국 소비자의 맥락 중심 인지 특성을 반영한 해석 체계.
