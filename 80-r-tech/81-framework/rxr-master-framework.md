## 0. Executive Summary

RXR Master Framework는 오프라인 매장을 **쿠키리스 시대의 1st party 데이터 플랫폼**으로 전환하여, 고객의 행동·구매·인식 데이터를 **양적 수치가 아닌 질적 인사이트**로 변환하는 통합 체계다.

**핵심 명제:** 리테일테크의 자산은 기술이 아니라 **분석과 해석의 프레임**이다. 기술은 오픈소스일 뿐이다.

### 프레임워크 구성

| 레이어 | 역할 | 성격 |
|--------|------|------|
| **Bracket** | 데이터 수집 인프라 | 물리적 매장 + 센서 + 모듈 (특허) |
| **RXR Engine** | 해석 프레임 | 전환율 퍼널 + 다층위 렌즈 + 질적 전환 (핵심 지적 자산) |
| **2-Layer Algorithm** | 심리 분석 | Content(토픽/감성) + Psyche(FWA/무의식) |
| **OMO Integration** | 데이터 통합 | 오프라인 취향 + 온라인 구매 = 통합 프로필 |

## 1. Bracket — 데이터 수집 인프라

### 1-1. 본질

Bracket은 **매장 자체가 데이터 수집 장치**인 리테일 미디어 스토어다. 쿠키가 사라진 시대에, 오프라인 공간이 고객 데이터를 확보하는 유일한 1st party 채널이 된다.

### 1-2. 6단계 Consumer Flow

| Stage | Flow | 수집 데이터 | 의미 |
|:---:|------|-----------|------|
| 1 | **Exposure** (노출) | 유동인구, 시선 방향, 주시 시간 | 브랜드 도달 |
| 2 | **Engagement** (반응) | DID 콘텐츠 반응, 진입 시점 | 콘텐츠 효과 |
| 3 | **Exploration** (탐색) | 동선, 체류시간, 관심 구역 | 공간 몰입도 |
| 4 | **Conversion** (구매) | 제품, 금액, 전환율 | 세일즈 임팩트 |
| 5 | **Retention** (유지) | 회원 DB, 유입 로그 | OMO CRM 연결 |
| 6 | **Feedback** (피드백) | 서베이, SNS 리뷰, NPS | 인식/심리 데이터 |

### 1-3. 인프라 구조

**OUTDOOR — 디지털 파사드**
- DID 콘텐츠 송출 + 스케줄링
- 시선·주목도 Eye Tracking
- DID 시청 → 매장 진입 전환율
- 시간대·요일별 유동인구 패턴

**INDOOR — 모듈형 매장**
- 유닛 구조: 브랜드별 즉시 교체 가능
- POS 연동: 프로모션별 전환율, 결제 세부항목
- AI 고객 카테고리화(C.C): 행동 패턴 기반 자동 세분화
- CRM 확장: 멤버십 → 온라인 리타겟팅 자동 연결
- 서베이 시스템: 인지도·만족도·재방문 의사

### 1-4. Bracket의 비즈니스 모델

- **자사 매장**: 유닛화 + 가변성 기반, 데이터 수집 + 알고리즘 테스트
- **플랫폼**: 누구나 브랜드/상품 위탁 판매 가능
- **외부 판매**: 테스트 완료 후 타사 매장에 Basic 분석 서비스 제공 + DB 확충
- **특허 해자**: 매장 설계·데이터 수집·분석 프로세스 전반 특허 보호

## 2. RXR Engine — 해석 프레임

### 2-1. 핵심 원리: 멘탈코스트 기반 인게이지먼트 레벨

오프라인 매장에서의 모든 행동 전환은 소비자의 **멘탈코스트(심리적 비용)**를 수반하는 의사결정이다. 각 전환은 브랜드와의 관계가 한 단계 깊어지는 것을 의미한다.

```
지나감 → 시선 → 진입 → 체류 → 구매 → 재방문 → 회원 → 후기 → 추천
──────────────────────────────────────────────────────────────────
멘탈코스트:  극저  → 저  → 중  → 중  → 고  →  고  → 고  → 극고 → 극고
관계 깊이:  무관심 ──────────────────────────────────────────→ 옹호자
데이터 성격: ──── 양적(행동) ────────────── 질적(심리) ────→
```

**핵심 통찰:**
- "시선이 멈추다"와 "그냥 지나가다"는 큰 차이
- "매장 안으로 들어오다"는 더 큰 차이
- "구매 전환"은 확신의 신호
- "재방문"은 본질적 리테일 관계성으로의 전환
- "추천"은 자발적 옹호 — 가장 높은 멘탈코스트

**같은 행동이라도 "누가"에 따라 가중치가 다르다:**
- 내향형(I)의 추천 = 외향형(E)의 추천보다 멘탈코스트 높음
- 이 가중치 차이가 양적 데이터를 질적 데이터로 전환하는 핵심

### 2-1b. 체류시간 심화 — 핵심 가중 변수

체류시간은 단순히 "몇 분 머물렀는가"가 아니다. **누가, 어떤 콘텐츠/브랜드에, 얼마나, 어떻게** 머물렀는가의 복합 변수이며, 인게이지먼트 품질을 결정하는 핵심 가중 지표다.

**체류시간의 4가지 차원:**

| 차원 | 측정 | 의미 |
|------|------|------|
| **절대 길이** | 총 체류 시간 (분) | 기본 관심 수준 |
| **상대 길이** | 동일 콘텐츠/브랜드 평균 대비 편차 | 개인의 특이 반응 감지 |
| **체류 패턴** | 시간 흐름에 따른 집중도 곡선 | 초반 몰입형 vs 후반 발견형 vs 반복 순회형 |
| **체류 밀도** | 구역별 체류 분포, 터치 포인트 수 | 수동적 관람 vs 능동적 탐색 |

**사람에 따른 체류 의미 차이:**

같은 "5분 체류"라도:
- 바쁜 직장인의 5분 = 강한 관심 신호 (평소 체류 평균 1.5분 대비 3.3배)
- 여유로운 주말 방문자의 5분 = 보통 수준 (평소 체류 평균 4분 대비 1.25배)
- → **절대 시간이 아닌 "그 사람의 평소 대비 얼마나 더 머물렀는가"가 진짜 지표**

**콘텐츠/브랜드에 따른 체류 기대치 차이:**

| 콘텐츠 유형 | 기대 체류 | 의미 있는 초과 | 해석 |
|-----------|:---:|:---:|------|
| 단순 상품 진열 | 30초~1분 | 2분+ | 상품에 대한 진지한 고려 |
| 체험형 공간 (팝업) | 3~5분 | 8분+ | 깊은 몰입, 브랜드 애착 형성 |
| 전시/스토리텔링 | 5~10분 | 15분+ | 브랜드 세계관 수용 |
| 클래스/워크숍 | 20~30분 | 40분+ | 적극적 참여, 관계 전환 |

**체류시간 품질 스코어 (Dwell Quality Score):**
```
DQS = (실제 체류 / 콘텐츠 기대 체류) × 체류 밀도 계수 × 퍼스널리티 가중치
```
- 콘텐츠 기대 체류: 해당 콘텐츠 유형의 평균 체류 시간
- 체류 밀도 계수: 능동적 탐색(터치, 이동, 상호작용) 비율
- 퍼스널리티 가중치: 세그먼트별 평균 체류 대비 편차

→ DQS가 높을수록 해당 콘텐츠/브랜드에 대한 **진정한 관여**가 발생한 것으로 판단

> **실측 데이터 연결**: 오프라인 DQS의 실측 입력은 `funnel.xlsx` 시트 `30_dwell_agg` (일자별 평균체류분·평균방문구역수·평균인터랙션수)에서 받는다. 집계 버전에서는 `퍼스널리티 가중치 = 1.0` 고정(개인 단위 visit_id 매칭 없음). 개인별 personality_w 재도입은 v1.1(Bracket 센서 연동) 범위. 상세: `rxr-offline-funnel-spec.md` §3.

### 2-1c. 후기 질적 레벨 — 핵심 가중 변수

온라인 후기/추천(L7)은 단순히 "후기가 있다/없다"가 아니라, **후기의 질적 깊이**에 따라 인게이지먼트 의미가 완전히 다르다. 이 차이를 가중하는 것이 RXR의 차별화 핵심이다.

**후기 질적 레벨 (Review Quality Level, RQL):**

| RQL | 후기 유형 | 예시 | 멘탈코스트 | 가중치 |
|:---:|---------|------|:---:|:---:|
| **Q1** | 별점만 / 한줄 | "좋아요", "⭐⭐⭐⭐⭐" | 극저 | ×0.2 |
| **Q2** | 일반 감상 | "분위기 좋고 맛있었어요" | 저 | ×0.5 |
| **Q3** | 구체적 경험 서술 | "테린느가 특히 좋았고, 2층 헤리티지 존이 인상적" | 중 | ×1.0 |
| **Q4** | 맥락 포함 비교/분석 | "시즌2보다 구성이 나아졌고, 가격 대비..." | 고 | ×1.5 |
| **Q5** | 추천/설득형 서사 | "꼭 가보세요. 예약 어렵지만 그만한 가치가..." | 극고 | ×2.0 |

**2-Layer 알고리즘과의 연동:**

후기의 질적 레벨은 Content Layer와 Psyche Layer 양쪽에서 측정:

| 측정 | Content Layer | Psyche Layer |
|------|:---:|:---:|
| **길이/구체성** | 단어 수, 고유명사 비율, 디테일 밀도 | — |
| **진정성** | — | Authenticity 점수 (FWA) |
| **추천 확신도** | 추천 표현 유무 | Clout 점수 (FWA) |
| **감정 깊이** | 감성 극성 강도 | Tone + 감정어 다양성 |
| **분석적 깊이** | 비교/인과 표현 유무 | Analytical 점수 (FWA) |

**후기 가중 인게이지먼트 점수:**
```
Weighted L7 = RQL 가중치 × Authenticity 보정 × Clout 보정
```
- RQL Q5(추천형 서사) × Authenticity 78(진심) × Clout 72(확신) = **최고 가치 옹호**
- RQL Q5(추천형 서사) × Authenticity 28(과장) × Clout 85(강한 어조) = **의심 필요 (기만 패턴)**

→ 같은 "추천합니다"라도 FWA 분석을 통해 **진짜 옹호 vs 과장/조작**을 구분하는 것이 핵심

### 2-2. 다층위 교차 렌즈 모델

같은 데이터에 여러 렌즈를 갈아끼우며 다른 인사이트를 추출하는 구조.

**Layer 1 — 인구통계 × 라이프스타일 렌즈**

| 데이터 소스 | 분류 기준 | 산출물 |
|-----------|---------|--------|
| AI 아웃핏 인식 (CCTV) | 패션 타입 → 라이프스타일 군 | 미니멀/프리미엄/스트릿/클래식... |
| 영상 AI | 연령대, 성별 추정 | 인구통계 프로필 |
| 결제 데이터 | 구매 항목, 가격대 | 소비 성향 |
| 행동 패턴 | 동선, 체류 구역 | 탐색 유형 |

**Layer 2 — 인게이지먼트 레벨 렌즈**

전체 유입 고객을 인게이지먼트 깊이로 분류:

| 레벨 | 행동 기준 | 관계 성격 |
|:---:|---------|---------|
| L0 | 매장 앞 통과 | 무관심 |
| L1 | 시선 멈춤 / 파사드 주목 | 인지 |
| L2 | 매장 진입 | 관심 |
| L3 | 체류 (DQS 측정) — 콘텐츠/브랜드 기대 체류 대비 상대 길이 + 체류 밀도 + 퍼스널리티 가중 | 탐색적 관여 |
| L4 | 구매 전환 | 확신적 관여 |
| L5 | 재방문 | 관계적 관여 |
| L6 | 회원 가입 | 커밋먼트 |
| L7 | 온라인 후기/추천 (RQL 측정) — 후기 깊이 × Authenticity × Clout 가중 | 옹호 |

**Layer 3 — 교차 분석**

Layer 1 × Layer 2를 교차하여:
- "20대 스트릿 패션 군 중 L4(구매 전환)까지 간 비율 = 12%"
- "30대 프리미엄 군 중 L5(재방문)까지 간 비율 = 28%"
- "구매 전환 고객(L4+)만 추출 → 결제 세부항목으로 취향 분석"
- "브랜드 A에는 이 세그먼트가 반응, 브랜드 B에는 저 세그먼트가 전환"

**Layer 4 — 브랜드/콘텐츠별 렌즈**

Bracket의 모듈형 유닛 구조로 브랜드가 교체되므로:
- 같은 공간에서 브랜드별 인게이지먼트 패턴 비교 가능
- 콘텐츠(DID) 유형별 진입 전환율 비교 가능
- → **브랜드 인텔리전스 DB** 축적

### 2-3. 행동↔심리 변환 테이블 — 핵심 지적 자산

리워드 기반 마이크로 서베이로 인게이지먼트 레벨의 **기준점**을 설정:

```
[서베이 + 행동 데이터 교차 분석]

"이 브랜드를 추천하겠다" 응답자 → 평균 체류 4.2분, 동선 3개 구역+
"다시 방문하겠다" 응답자 → 평균 체류 2.8분, 상품 터치 2회+
"그냥 지나가다 들어왔다" 응답자 → 평균 체류 1.1분, 동선 1개 구역

→ 변환 테이블:
  체류 4분+ × 3구역+ = 추천 가능 세그먼트 (신뢰도 78%)
  체류 1분 미만 = 통과형 방문 (신뢰도 85%)
```

이 변환 테이블이 쌓이면 → **서베이 없이도 행동 데이터만으로 인게이지먼트 레벨 추론 가능**

변환 테이블 = RXR의 핵심 지적 자산. 경쟁사가 같은 센서를 설치해도, 이 변환 로직이 없으면 "체류 4분"이 무엇을 의미하는지 모른다.

## 3. 2-Layer Algorithm — 질적 전환 엔진

### 3-1. 구조

텍스트 데이터(서베이 응답, SNS 리뷰, 후기)가 존재하는 지점에서 작동:

**Content Layer — "무엇을 말했는가"**
- n-gram 기반 토픽 추출
- 감성 분석 (긍정/부정/중립)
- 키워드 빈도 및 연관 분석
- → 의식적/명시적 반응 포착

**Psyche Layer — "어떻게 말했는가"**
- FWA(기능어분석): 대명사, 조사, 접속사 패턴
- LIWC 4대 변수: Authenticity, Clout, Analytical, Tone
- 한국어 특화: "이/가"(초점) vs "은/는"(주제) 조사 패턴
- → 무의식적/잠재적 심리 포착

### 3-2. 2-Layer가 인게이지먼트 퍼널과 만나는 지점

| 퍼널 단계 | 2-Layer 적용 | 질적 전환 |
|---------|------------|---------|
| L6 (회원) | 가입 동기 서베이 → Content + Psyche | "왜 가입했는가"의 진심 측정 |
| L7 (후기) | 리뷰 텍스트 → Content + Psyche | 진짜 팬 vs 화제성 방문 구분 |
| L7 (추천) | 추천 텍스트 → Psyche의 Clout 분석 | 추천 확신도 수치화 |
| 서베이 응답 | 개방형 응답 → Content + Psyche | 의식적 답변 너머 무의식 포착 |

### 3-3. 산출 지표

| 지표 | 산출 방식 | 용도 |
|------|---------|------|
| **Brand Advocacy Score** | Clout + Authenticity 조합 | 진정한 옹호자 비율 측정 |
| **Freshness Index** | "이/가" vs "은/는" 비율 시계열 | 브랜드 신선함 소진 감지 |
| **Engagement Depth** | 인게이지먼트 레벨 × FWA 가중치 | 질적 관여도 종합 점수 |
| **Sincerity Gate** | 3단계 필터링 (아래 상세) | 노이즈→협찬→진심 단계적 정제 |

### 3-4. Sincerity Gate — 3단계 데이터 정제 프레임

이클립스 월드 파일럿(2026.04)에서 실증된 분석 프레임. 같은 데이터를 3개 Gate로 관통하며 "숫자 뒤의 진짜 의미"를 드러낸다.

**Gate 구조:**

```
Gate 1: 전체 데이터 ─→ Gate 2: 협찬 제외 ─→ Gate 3: Auth 60+ 필터
(시장 노출 규모)       (순수 마케팅 효과)     (진짜 팬의 목소리)
```

| Gate | 필터 | 답하는 질문 | 이클립스 케이스 |
|:---:|------|---------|---------|
| **Gate 1** | 노이즈 제거만 | "시장에 얼마나 노출됐는가?" | 90건, 긍정 46% |
| **Gate 2** | + 협찬/유료 제거 | "유료 홍보 빼면 실제 반응은?" | 82건, 긍정 41% (-5%p) |
| **Gate 3** | + Authenticity 60+ | "진심으로 좋았던 사람은?" | 진정한 옹호자 수 |

**이클립스 파일럿 실증 결과:**
- 협찬 8건이 긍정률을 **5%p 부풀림** (46%→41%)
- 협찬의 Authenticity **47.6** vs 자발적 **58.1** (차이 10.5점)
- 협찬의 긍정률 **88%**(부정 0%) — 비현실적 편향
- 협찬은 **사전 시딩 기간에 집중**, 팝업 기간은 100% 자발적

**각 Gate에서 다른 리포트가 나온다:**
- Gate 1 리포트: 클라이언트 보고용 (규모 중심, "이만큼 노출됐습니다")
- Gate 2 리포트: 내부 분석용 (실효 중심, "진짜 효과는 이정도입니다")
- Gate 3 리포트: 전략 수립용 (팬 중심, "이 사람들이 진짜 좋아했습니다")

### 3-5. 계정 신뢰도 분류 (Account Trust Level)

협찬 표기가 없어도 **고객을 사칭하는 비즈니스 계정**이 존재한다. 이를 구분하는 것이 Sincerity Gate의 정밀도를 결정한다.

**계정 분류 체계:**

| 등급 | 계정 유형 | 감지 방법 | 분석 가중치 |
|:---:|---------|---------|:---:|
| **T1** | 순수 개인 계정 | 팔로워 소규모, 일상 포스트 혼재, 비정기 포스팅 | ×1.0 (기준) |
| **T2** | 마이크로 인플루언서 | 팔로워 1K~10K, 카테고리 집중, 정기 포스팅 | ×0.8 |
| **T3** | 매크로 인플루언서 | 팔로워 10K+, 높은 영향력 점수, 협찬 빈도 높음 | ×0.6 |
| **T4** | 비즈니스 계정 (명시) | 브랜드/매장/서비스 계정, 프로필에 사업자 표시 | ×0.3 |
| **T5** | 비즈니스 위장 계정 | 개인 계정처럼 보이나 특정 브랜드/제품만 포스팅 | ×0.1 |

**T5 (위장 계정) 감지 신호:**
- 특정 카테고리/브랜드 포스트 비율 80% 이상
- 포스팅 패턴이 캠페인 시작과 일치 (연속 3건+ 동일 주제)
- 본문에 공식 보도자료 문구와 높은 유사도
- Authenticity 극저 + Clout 극고 (자신 있게 추천하지만 진심이 아닌 패턴)
- 다른 경쟁 브랜드에 대한 언급 제로

**구현 방법:**
- **네이버 블로그**: 블로거 프로필 + 최근 포스트 카테고리 분포 크롤링
- **인스타그램**: Brand24의 Influence Score + 팔로워 수 + 계정 bio 분석
- **1차**: 규칙 기반 (팔로워 수, 포스팅 빈도, 카테고리 집중도)
- **2차**: FWA Authenticity 점수와 교차 검증 (T5 위장 계정은 Auth 극저 예상)

**Sincerity Gate × Account Trust 교차:**

```
최종 신뢰도 = Gate 필터 × Account Trust Level

예: Gate 2(협찬 제외) + T1(순수 개인) + Auth 72 
   = 가장 신뢰도 높은 데이터 → 핵심 인사이트 추출 대상

예: Gate 1(전체 포함) + T5(위장 계정) + Auth 35
   = 노이즈 → 카운트만 하고 질적 분석에서 제외
```

## 4. 2단계 고객 데이터 모델

### 4-1. Stage 1 — 익명 표본조사 (오프라인)

**목적**: 세그먼트 분류 + 세그먼트별 성향 파악
**성격**: 통계적 표본조사 ("이런 유형의 사람들은...")

| 데이터 소스 | 유형 | 분석 |
|-----------|------|------|
| 행동 데이터 (동선, 체류, 전환율) | 자동 수집 | 인게이지먼트 레벨 분류 |
| AI 아웃핏 인식 | 자동 수집 | 라이프스타일 군 분류 |
| 결제 세부항목 | 자동 수집 | 취향/충동소비 패턴 |
| 리워드 서베이 | 표본 수집 | 인게이지먼트 기준점 + 심리 프로필 |
| SNS 리뷰 | 크롤링 | 2-Layer 심리 분석 |

**산출물:**
- 세그먼트 맵: 인구통계 × 라이프스타일 × 인게이지먼트 교차
- 세그먼트별 성향 프로필
- 행동↔심리 변환 테이블 (기준점 데이터)
- 브랜드/콘텐츠별 인게이지먼트 패턴

### 4-2. Stage 2 — 실명 개인 데이터 (회원가입 이후 + OMO)

**목적**: 고도화된 개인 프로필 → 개인화 커뮤니케이션
**전환 조건**: 회원가입 (L6 도달)

| 데이터 소스 | 유형 | 분석 |
|-----------|------|------|
| Stage 1 세그먼트 프로필 | 기존 | 소속 세그먼트 + 인게이지먼트 레벨 |
| 회원 CRM (실명, 연락처) | 실명 | 개인 식별 |
| 온라인 구매 행동 | OMO 연동 | 이성적/필요 기반 소비 패턴 |
| 오프라인 구매 행동 | POS 연동 | 충동적/취향 기반 소비 패턴 |
| 개인 리뷰/후기 | 텍스트 | 퍼스널리티 프로필 (FWA) |

**산출물:**
- 개인 통합 프로필: 이성적 자아(온라인) + 감성적 자아(오프라인)
- 퍼스널리티 기반 커뮤니케이션 전략
- CLV(고객 생애 가치) 예측
- 개인화 리타겟팅 시나리오

## 8. 핵심 원칙

1. **프레임이 자산이다** — 기술은 오픈소스, 해석의 틀이 경쟁력
2. **양적 → 질적 전환** — 숫자를 세는 게 아니라 의미를 읽는다
3. **멘탈코스트 = 관계 깊이** — 모든 전환은 심리적 비용의 산물
4. **같은 행동, 다른 무게** — 퍼스널리티에 따라 가중치가 다르다
5. **공간이 있어야 유의미** — 매장 = 쿠키리스 시대의 데이터 플랫폼
6. **표본에서 개인으로** — 1단계(세그먼트) → 2단계(개인 프로필) 점진적 고도화

## 9. BEI Value Conversion Layer (EMV)

BEI(0~100)를 금전적 가치(₩)로 전환하는 레이어. 4축이 각각 다른 경제적 경로로 가치를 만든다.

| 축 | 경제적 경로 | 가치 유형 |
|---|---|---|
| Depth | 관여 깊이 → 재구매 확률 | 재구매/재방문 가치 |
| Advocacy | 추천 강도 → 바이럴 유입 | 신규 고객 획득 가치 |
| Sincerity | 진성 비율 → 전환율 상승 | 전환율 프리미엄 |
| Discovery | 신선함 → 자발적 공유 | Earned Media 가치 |
