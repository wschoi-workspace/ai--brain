Part 01 — The Problem

기존 리테일 분석은
"그래서 뭐?"에 답하지 못한다

수천만 원짜리 센서를 달아도, 숫자만 나올 뿐 "이유"는 모릅니다.

비유로 이해하기

체중계는 몸무게를 알려주지만, 왜 살이 찌는지는 안 알려줍니다.
건강검진표는 수치를 보여주지만, 어떤 습관이 문제인지는 안 알려줍니다.

기존 리테일 분석도 마찬가지입니다. "방문 1만명, 전환율 8%"라는 숫자는 주지만,
"왜 92%가 안 샀는지"는 아무도 모릅니다.

📊

기존 솔루션이 알려주는 것

RetailNext / Sensormatic
"오늘 방문자 3,200명, 평균 체류 12분, 전환율 8%"

메이아이
"A구역에 사람이 많이 모였고, B구역은 한산합니다"

McKinsey NPS
"추천 의향 72점입니다"

❓

하지만 대답 못하는 질문들

왜 92%가 안 샀는가?
관심이 없어서? 가격 때문에? 공간이 별로여서?

왜 A구역에 모였는가?
진짜 관심이 있어서? 그냥 길이 막혀서?

추천 72점이 진심인가?
사회적 바람직성? 협찬? 진짜 좋아서?

RXR의 목표

숫자만 던져주는 "체중계"가 아니라, 원인과 심리까지 읽어주는 "주치의"가 되는 것.
행동 데이터에 "왜(Why)"라는 맥락을 부여하는 해석 프레임을 만드는 것.

Part 02 — Architecture

RXR은 4개의 레이어로 만들어져 있다

데이터를 수집하고 → 해석하고 → 심리를 분석하고 → 온오프를 통합하는 구조입니다.

🏪 Bracket = 매장이 곧 센서

매장 자체가 데이터 수집 장치입니다. CCTV + POS + 서베이 + DID(디지털 사이니지)로 고객의 행동을 자동 기록합니다. 쿠키가 사라진 시대에, 오프라인 공간이 유일한 1st party 데이터 채널이 됩니다.

🧠 RXR Engine = 해석의 틀

수집된 데이터를 "인게이지먼트 레벨"로 변환합니다. 단순 숫자가 아니라, 고객이 브랜드와 얼마나 깊은 관계를 맺고 있는지를 L0~L7 8단계로 분류합니다.

🔍 2-Layer = 심리 분석

리뷰나 서베이 텍스트를 분석합니다. "무엇을 말했는가"(Content)뿐 아니라 "어떻게 말했는가"(Psyche)까지 읽어서, 진심인지 과장인지를 판별합니다.

🌐 OMO = 온오프 통합

오프라인에서의 감성적 소비와 온라인에서의 이성적 소비를 하나로 연결합니다. 같은 사람이 채널에 따라 어떻게 다르게 행동하는지를 통합 프로필로 만듭니다.

Mental Cost Engagement

고객은 8단계 계단을 오른다

매 단계마다 심리적 비용(멘탈코스트)이 높아집니다. 지나가는 것과 추천하는 것은 완전히 다른 관계입니다.

비유로 이해하기

소개팅을 생각해보세요. 눈이 마주치는 것(L0) → 다가가서 말을 거는 것(L2) → 2시간 대화하는 것(L3) → 연락처를 교환하는 것(L4) → 다시 만나는 것(L5) → 연인이 되는 것(L6) → 친구들에게 소개하는 것(L7). 단계마다 심리적 비용이 완전히 다릅니다.

Core Concepts

RXR만의 핵심 개념 3가지

⏱

DQS
Dwell Quality Score

같은 5분이라도 가치가 다릅니다.

바쁜 직장인의 5분 = 강한 관심 (평소 1.5분 대비 3.3배)

여유로운 주말 방문자의 5분 = 보통 수준 (평소 4분 대비 1.25배)

"얼마나"가 아니라 "그 사람 치고 얼마나"가 진짜 지표입니다.

✍

RQL
Review Quality Level

같은 "추천합니다"라도 깊이가 다릅니다.

Q1: "좋아요" (극저)
Q3: "테린느가 특히 좋았고, 2층이 인상적" (중)
Q5: "꼭 가보세요. 예약 어렵지만 그만한 가치가..." (극고)

후기의 "양"이 아니라 "질적 깊이"를 측정합니다.

📖

변환 테이블
Behavior-Psychology Table

서베이 + 행동 데이터를 교차해서 만드는 "사전"입니다.

예) 서베이에서 "추천하겠다"고 답한 사람들의 행동 패턴: 체류 4분+, 3구역 이상 탐색

이 사전이 쌓이면, 서베이 없이도 행동만 보고 심리를 추론할 수 있습니다.

RXR의 핵심 지적 자산입니다.

Part 03 — Algorithm

2-Layer: "뭘 말했는가" + "어떻게 말했는가"

사람의 말에는 의식적으로 선택한 내용과, 무의식적으로 드러나는 심리가 동시에 존재합니다.

비유로 이해하기

누군가 "이 카페 분위기 좋고 맛있었어요"라고 말했다면,
Content Layer는 "분위기"와 "맛"이라는 주제를 읽고,
Psyche Layer는 "나"가 아닌 "이 카페"를 주어로 쓴 것에서 약간의 거리감을 읽습니다.

반면 "여기 진짜 내 취향이라 또 가고 싶다"라면,
Content는 같은 긍정이지만, Psyche는 "여기 + 내 + 또"에서 강한 개인적 몰입을 읽습니다.

Content Layer

"무엇을 말했는가" — 의식적 반응

  • 토픽 추출 (어떤 주제를 언급했는가)
  • 감성 분석 (긍정 / 부정 / 중립)
  • 키워드 빈도 & 연관 분석
  • 구체성 점수 (고유명사 비율)
  • 분석 깊이 (비교/인과 표현)
+

Psyche Layer

"어떻게 말했는가" — 무의식적 심리

  • 대명사 패턴 (나/우리/여기 → 관여 방식)
  • 조사 패턴 ("이/가" 발견형 vs "은/는" 평가형)
  • Authenticity — 진정성 점수
  • Clout — 추천 확신도
  • Analytical — 이성적 vs 감성적
  • Tone — 감정 깊이
핵심 포인트

같은 "추천합니다"라도 —
Authenticity 78 + Clout 72 = 진짜 옹호자
Authenticity 28 + Clout 85 = 의심 필요 (과장/조작 패턴)

Content Layer만으로는 둘 다 "긍정적 추천"으로 보입니다.
Psyche Layer가 있어야 진심과 거짓을 구분할 수 있습니다.

Sincerity Gate

3단계 필터로 "진짜 목소리"만 남긴다

같은 데이터를 3개 Gate로 통과시키면, 숫자 뒤에 숨은 진짜 의미가 드러납니다.

이클립스 월드 파일럿 실증 결과

• 협찬 8건이 긍정률을 5%p 부풀림 (46% → 41%)
• 협찬의 Authenticity 47.6 vs 자발적 58.1 (차이 10.5점)
• 협찬은 긍정률 88%, 부정 0% — 비현실적 편향
• Sincerity Gate 적용으로 과장된 긍정 26건이 걸러짐

Part 04 — Difference

기존 솔루션과 뭐가 다른가?

글로벌 리테일테크 6개사 + 컨설팅 3개사를 비교 분석한 결과,
RXR만 커버하는 영역이 2개 있습니다.

Before vs After

같은 팝업, 완전히 다른 리포트

BEFORE — 기존 리포트

  • 📌 총 방문자: 10,000명
  • 📌 전환율: 8.3%
  • 📌 평균 체류: 12분
  • 📌 매출: 34,500,000원
  • 📌 SNS 언급: 320건
  • "...그래서 어쩌라고?"

AFTER — RXR 리포트

  • 📌 총 방문자: 10,000명, 전환율: 8.3%
  • Brand Advocacy Score: 72 (옹호자 비율 상위)
  • 핵심 팬: 34% / 구경꾼: 66%
  • 3주차 Freshness Index 하락 감지 → 매출 하락 선행 신호
  • 20대는 방문 48%이나 구매 기여 31%. 30대가 진짜 매출원
  • 미니멀 세그먼트 객단가 42,000원, 디저트+굿즈 세트 선호
  • → 전략 제안: 4주차 콘텐츠 리프레시 + 30대 타겟 프로모션
감정 선행지표

RXR은 매출이 떨어진 후에 원인을 찾는 게 아닙니다.
SNS 리뷰의 Freshness Index(신선함 지표)가 하락하면, 실제 매출 하락보다 2~3주 먼저 감지됩니다.
감정이 먼저 변하고, 행동이 따라옵니다. RXR은 이 감정 선행지표를 읽습니다.

Glossary

핵심 용어 사전

DQS Dwell Quality Score. 체류시간의 "질"을 측정하는 복합 지표. (실제 체류 / 기대 체류) × 밀도 × 세그먼트 보정
RQL Review Quality Level. 후기의 질적 깊이 5단계 (Q1 별점만 ~ Q5 추천형 서사)
FWA Function Word Analysis. 기능어(대명사, 조사, 접속사) 패턴으로 무의식적 심리를 읽는 분석 방법
2-Layer Content Layer(뭘 말했는가) + Psyche Layer(어떻게 말했는가)를 결합한 이중 텍스트 분석 알고리즘
Sincerity Gate 3단계 데이터 정제 필터. G1(전체) → G2(협찬 제외) → G3(Auth 60+). 진심만 남기는 구조
AI C.C AI Customer Category. 시각+행동+거래+텍스트 데이터를 통합한 자동 고객 세그먼테이션
멘탈코스트 심리적 비용. 지나가기(극저) → 진입(중) → 구매(고) → 추천(극고). 단계별 관계 깊이의 척도
변환 테이블 서베이와 행동 데이터를 교차해서 만드는 "행동 → 심리" 추론 사전. RXR의 핵심 지적 자산
OMO Online Merges with Offline. 오프라인(감성적 소비)과 온라인(이성적 소비)을 하나로 연결하는 데이터 통합
Bracket 매장 자체가 데이터 수집 장치인 리테일 미디어 스토어. RXR의 물리적 인프라 레이어

기술은 오픈소스일 뿐이다.
분석과 해석의 프레임이 자산이다.
