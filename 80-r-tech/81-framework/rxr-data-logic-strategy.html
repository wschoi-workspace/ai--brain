RXR Data Logic Strategy

RXR 데이터 로직 전략적 차별화

기존 리테일 측정(POS) · 리테일테크(메이아이) · 기능어분석(FWA)을 비교하고,
RXR의 4축 데이터 프레임워크가 왜 더 큰 가능성을 가지는지 전략적으로 정리

2026.04 · Project RENT · R-lab 분석솔루션

1 핵심 비교: 4가지 접근법의 커버리지

질문: 오프라인 리테일 경험의 성과를 어떻게 측정하는가?
기존 방식들은 각각 일부 축만 측정한다. RXR의 차별점은 4개 축 전체를 하나의 프레임워크로 통합하고, 여기에 기능어분석(FWA)까지 결합한다는 것이다.
측정 영역 전통 POS 메이아이(mAsh) NPS/설문 RXR 4축 RXR + FWA
Traffic (유입) X O 피플카운팅 X O CCTV+시선인식 O
Behavior (행동) X O 동선/체류 X O Zone별 체류/동선 O
Conversion (매출) O 매출만 X POS 미연동 X O POS 통합 O
Feedback (인식/확산) X X △ 의식적 응답만 O SNS+설문 O + 기능어분석
무의식적 심리/진정성 X X X 편향 존재 X O FWA 독점 영역
가짜 리뷰 / 진정성 필터 X X X X O LIWC 기반
브랜드 심리적 거리 측정 X X △ X O 대명사/Clout
4축 통합 인사이트 X X X O O + 심리 레이어
핵심: 메이아이는 축 ①②(유입/행동)에 강하고, POS는 축 ③(매출)만 본다. 설문은 축 ④(인식)의 "의식적 응답"만 잡는다.
RXR은 4축 전체를 통합하고, 기능어분석(FWA)을 더해 설문/리뷰에서 "말하지 않은 심리"까지 포착하는 유일한 구조다.

3 방식별 상세 비교

A. 전통적 리테일 측정 (POS 매출 중심)

측정 범위

매출액, 객단가, 판매 수량, 시간대별 매출
"구매한 사람"만 측정 가능

블라인드 스팟

방문했으나 구매하지 않은 고객 = 보이지 않음
이탈 원인, 매장 내 행동 = 파악 불가
왜 샀는지/왜 안 샀는지 = 추정만 가능

B. 리테일테크 — 메이아이(mAsh)

강점

기존 CCTV만 활용 (추가 장비 불필요)
Re-ID 정확도 92% (학계 66% 대비)
히트맵/트래픽맵/저니맵 3종 동시 제공
300+ 매장 공급, 재계약률 90%
CES 2024 AI 혁신상

한계

POS 매출 데이터 미연동 — "누가 샀는지" 모름
감정/만족도 측정 불가 — 영상으로는 심리 파악 어려움
온-오프 통합 부재 — SNS/온라인 행동 추적 불가
매장 외부 여정 사각지대 — 방문 전/후 행동 미파악
"데이터를 어떻게 활용해야 할지 모르겠다" 고객 반응

C. NPS / 설문 기반 측정

강점

고객 만족도, 추천 의향, 브랜드 인지도 직접 측정
브랜드 퍼널(인지→경험→선호→구매→추천) 추적

근본적 한계

설문 응답 편향 — 사회적 바람직성, 기억 왜곡
의식적 응답만 — "왜 좋았는지" 말할 수 없는 심리는 놓침
실시간 불가 — 사후 수집만 가능
실제 행동과 괴리 — "재방문 의향 높음" ≠ 실제 재방문

정리: POS는 "결과만", 메이아이는 "행동만", 설문은 "의식적 응답만" 본다.
세 가지 모두 "고객이 무의식적으로 느끼는 진짜 감정과 브랜드 인식"은 측정하지 못한다.

4 기능어분석(FWA)이 여는 5번째 축: 무의식 심리 레이어

기능어분석(Function Word Analysis)은 텍스트 내 대명사, 조사, 접속사 등 "기능어"의 사용 패턴을 분석하여 화자의 무의식적 심리 상태를 포착하는 기법이다.
핵심: 기능어는 의도적 조작이 거의 불가능하므로, 설문/리뷰에서 "말하지 않은 진짜 감정"을 드러낸다.

RXR 축 ④(Feedback)에 FWA가 추가하는 것

기존 축 ④ (SNS + 설문)→축 ④ + FWA (강화된 버전)
SNS 언급량 / 감성 분석 (긍/부정) → 감성 분석 + 무의식적 심리 구조
LIWC Authenticity로 리뷰 진정성 측정
대명사 패턴으로 브랜드 심리적 거리 측정
키워드 클러스터 ("공간", "분위기") → 내용어 + 기능어 통합 분석
"무엇을 말했는가" + "어떻게 말했는가"
조사 패턴("이/가" vs "은/는")으로 새로움 vs 익숙함 측정
설문 NPS / 재방문 의향 → 설문 응답 + 개방형 응답 FWA
Authenticity 점수로 "진짜 의견" vs "면접관 효과" 구분
Clout 점수로 자발적 추천 의향의 강도 측정
가짜 리뷰 필터링 없음 → 기능어 패턴으로 가짜 리뷰 자동 탐지
1인칭 과다 + 긍정어 과잉 = 비진정 신호
4개 피처로 79% 정확도

FWA가 RXR에 추가하는 4대 측정 지표

Analytical

소비자가 브랜드를 이성적으로 평가하는지, 감성적으로 반응하는지

Clout

소비자가 브랜드에 대해 자신감 있게 추천하는지, 조심스러운지

Authenticity

리뷰/응답이 진정한 의견인지, 사회적 바람직성 편향인지

Emotional Tone

단순 긍/부정을 넘어 감정의 깊이와 구조를 측정

5 RXR만의 3대 차별화 포인트

차별화 1

4축 통합 = 하나의 스토리

메이아이는 "사람이 어디서 멈췄는지"까지. POS는 "얼마 팔렸는지"까지.

RXR은 "봤다 → 들어왔다 → 머물렀다 → 샀다 → 공유했다" 전체를 하나의 퍼널로 연결.

이것이 IR/제안서에서 "데이터 프로덕트"로 기능하는 이유.

차별화 2

기능어분석 = 심리의 정량화

설문은 "좋았어요"만 잡는다.
FWA는 "얼마나 진심으로 좋았는지"를 수치화한다.

가나초콜릿 케이스에서 실증:
제품→공간 전환 시 Clout 35→65, Authenticity 65→45로 변화.

이런 정량 데이터는 업계 최초.

차별화 3

AI 인사이트 자동화 = 의사결정 엔진

메이아이: 데이터만 준다. "어떻게 활용하지?" 고객 반응.

RXR: 데이터 → KPI → 패턴 탐지 → 우선순위화 → AI 문장화 → 액션 제안까지 자동.

"단순 대시보드가 아니라 운영 의사결정을 자동으로 만들어주는 인터페이스"

6 AI 인사이트 엔진에 기능어분석 통합

기존 인사이트 생성 로직 (4단계)

1
데이터 정규화 — CCTV/POS/설문/SNS를 날짜/시간 기준으로 통합
2
KPI 계산 — Traffic/Behavior/Conversion/Feedback 각 축의 핵심 지표 산출
3
이상치/패턴 탐지 — 증감률, Z-score, 지표 간 상관관계로 "의미 있는 변화만" 선별
4
문장화 + 점수화 — Impact × Confidence × Relevance 스코어링 → 상위 3~5개만 출력

FWA 통합 후 확장된 로직 (6단계)

1
데이터 정규화 — 기존 4소스 + SNS/리뷰 원문 텍스트 + 설문 개방형 응답
2
KPI 계산 — 기존 KPI + LIWC 4대 변수(Analytical/Clout/Authenticity/Tone) + 대명사 비율 + 조사 패턴
3
진정성 필터링 — NEW! 리뷰 Authenticity 점수로 가짜/과장 리뷰 제거 → 클린 데이터 확보
4
심층 패턴 탐지 — 기존 상관관계 + FWA 교차 분석
예: "체류시간↑ + 구매↓" + "리뷰 Analytical↑ Clout↓" = "관심은 높지만 확신 부족"
5
점수화 + 우선순위 — FWA 기반 인사이트에 가중치 추가 (무의식 패턴 = 높은 Impact)
6
AI 문장화 — 기존 템플릿 + 심리 해석 템플릿 추가
"소비자의 리뷰 언어에서 Clout 점수가 하락하고 있어, 브랜드 추천 확신이 약해지고 있는 것으로 보입니다."

FWA 통합 인사이트 예시

기존 인사이트 (FWA 없이)

"SNS 긍정 비율은 81%로 높게 유지됐습니다. 브랜드 반응이 긍정적입니다."

FWA 강화 인사이트

"SNS 긍정 비율은 81%로 높으나, 리뷰 텍스트의 Authenticity 점수가 42로 낮아 사회적 바람직성 편향이 존재할 가능성이 있습니다. Clout 점수도 38로, 자발적 추천보다는 단순 감상에 가까운 반응입니다. 진정한 브랜드 옹호(advocacy) 전환을 위해 제품 체험 터치포인트 강화가 필요합니다."

핵심 차이: 기존 방식은 "긍정 81% = 좋다"로 끝나지만, FWA는 "그 81%가 얼마나 진심인지"까지 파고든다. 이것이 IR/제안서에서 "다른 솔루션과 차원이 다른 인사이트"로 포지셔닝되는 근거.

8 전략적 포지셔닝 — RXR은 어디에 서는가

시장 포지셔닝 맵
