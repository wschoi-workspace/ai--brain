RXR 2-Layer Case Study

가나초콜릿 vs 가나초콜릿하우스
동일 브랜드, 완전히 다른 심리 구조

같은 "가나" 브랜드의 제품 리뷰와 공간 리뷰를 RXR 2-Layer 프레임워크(Content Layer + Psyche Layer)로 분석하여, 단일 레이어 분석이 놓치는 심리적 전환을 실증한다.

2026.04 · Project RENT · R-lab · 리뷰 데이터 기반 2-Layer 실증 분석

1 분석 대상 & 데이터 개요

A 가나초콜릿 (제품)

유형: 판형 초콜릿 (FMCG)
역사: 1975~ (51년)
누적 매출: 1.3조 원, 66억 갑
포지셔닝: "국민 초콜릿"
수집 리뷰: 커뮤니티/블로그/쇼핑몰 기반

B 가나초콜릿하우스 (공간)

유형: 체험형 팝업스토어/디저트 카페
역사: 2022~ (시즌3까지)
시즌3 규모: 150평, 구매전환율 95%+
포지셔닝: "가나, 디저트가 되다"
수집 리뷰: 블로그/SNS/리뷰플랫폼 기반

분석 프레임: 수집된 리뷰 텍스트를 RXR 2-Layer 알고리즘에 투입.
Layer 1 (Content) — n-gram, 감성분류, 토픽 클러스터링으로 "무엇을 말했는가" 분석
Layer 2 (Psyche) — LIWC 4대 변수 + 대명사/조사/어미 패턴으로 "어떻게 말했는가" 분석
Layer 3 (Synthesis) — What × How 교차 분석으로 "왜 다른 반응인가" 진단

읽는 법: 아래 교차 분석에서 각 리뷰 유형에 패턴 ID가 부여된다. 패턴 ID를 보면 "이 리뷰가 어떤 심리 상태이고, 어떤 전략으로 대응해야 하는지"가 즉시 파악된다. 이것이 단순 감성 분석(긍정/부정)과 2-Layer 분석의 결정적 차이.

4-1. 제품 리뷰 — 패턴 매칭

리뷰 유형L1 (Content)L2 (Psyche)패턴 ID2-Layer 인사이트
솔직한 팬 긍정 / "맛있다", "맞다" Auth 높음, Clout 낮음 P-03 관심은 있지만 추천 의지 없음. "난 좋아하는데 남한테 권하기는..." — 개인적 만족에 머무름
분석적 비판자 부정 / "팜유", "싸구려" Auth 높음, Analytical 높음 P-03 변형 진심 불만 (Zone C). 실제 품질 문제 인지. 무시하면 안 됨
추억형 소비자 긍정 / "어릴 때", "추억" Auth 높음, Tone 중립, 시제: 과거 P-06 전조 노스탤지어 의존. 현재 브랜드 경험이 아닌 과거 기억에 의존 — 새로운 경험 없이 점진적 이탈 가능
제품 진단: L1만 보면 "긍정 48%, 그럭저럭" — L2를 더하면 긍정 리뷰조차 추천 의지가 없고(Clout 35), 과거 추억에 의존하고 있음이 드러남. 브랜드가 현재 시점에서 새로운 긍정 경험을 생산하지 못하고 있다.

4-2. 공간 리뷰 — 패턴 매칭

리뷰 유형L1 (Content)L2 (Psyche)패턴 ID2-Layer 인사이트
인스타 소비형 긍정 90%+ / "예쁘다", "사진" Auth 낮음, Clout 낮음, Tone 높음 P-02 바이럴 =/= 충성. SNS 공유는 하지만 재방문 의향 약함. 인스타에 올리고 싶었을 뿐
기대 위반형 긍정 / "생각보다 좋았다", "감동" Auth 높음, Clout 높음, Freshness 높음 P-01 진짜 팬 전환. "제품 팝업인 줄 알았는데 진심인 공간" — 스키마 위반이 진정한 긍정 반응 생성
관계 소비형 긍정 / "같이 가자", "데이트" Clout 높음, Distance 낮음(우리/여기), 시제: 미래 P-01 변형 행동 전환 완료. 단순 감상을 넘어 동반자에게 행동을 촉구 — 가장 높은 전환 가치
실망형 부정 / "별거 없다", "아쉽다" Auth 높음, Tone 낮음 Zone C 진심 불만. 소수지만 Auth 높아 실제 문제 반영. 특히 "초콜릿 특색 부족"은 제품-공간 연결 약점 시사
공간 진단: 긍정 86% 중 분류하면 —
• 인스타 소비 (P-02): 약 40~45% — Auth 낮음, Clout 낮음. 바이럴은 되지만 재방문 가치 낮음
• 진짜 팬 (P-01): 약 30~35% — Auth+Clout 모두 높음. 진정한 브랜드 옹호자
• 관계 소비 (P-01 변형): 약 15~20% — 행동 전환까지 완료. 최고 가치 세그먼트
Layer 1의 "긍정 86%"는 하나의 숫자지만, Layer 2를 추가하면 3개의 전혀 다른 세그먼트가 드러난다.

5 핵심 발견: 5대 심리 축의 체계적 전환

동일 브랜드 "가나"가 제품→공간으로 확장될 때, 소비자의 언어 구조에서 5개 심리 축이 동시에 전환된다. 이 전환은 Layer 1(내용 분석)만으로는 "토픽이 다르다" 정도만 보이지만, Layer 2(심리 분석)를 추가해야 "왜 다른지", "그것이 비즈니스에 무엇을 의미하는지"가 보인다.
심리 축 제품 → 공간 기능어 증거 비즈니스 의미
1. 자아 축
"나" → "여기/우리"		 개인적 경험 → 사회적 경험 1인칭 단수 ↓
장소 지시어+복수형 ↑		 공간이 소비를 "공유 가능한 경험"으로 전환 — 바이럴 구조 형성
2. 시간 축
과거 → 현재+미래		 노스탤지어 → 현재 몰입 과거 시제 ↓
현재형+청유형(-자) ↑		 브랜드가 "기억"에서 "현재 경험"으로 리포지셔닝 — 노스탤지어 의존 탈피
3. 감각 축
미각 → 시각		 입안 감각 → 공간 감각 미각/촉각 단어 ↓
시각/공간 단어 ↑		 근접 감각(Proximal)→원격 감각(Distal) 전환 — SNS 공유에 최적화된 감각 구조
4. 인지 축
분석 → 감동		 이성적 평가 → 감성적 반응 Analytical 65→45
인과 접속사 ↓, 감탄사 ↑		 분석적 비교 대상→감정적 몰입 대상 — 가격 경쟁에서 가치 경쟁으로
5. 행동 축
평가 → 추천		 수동적 소비자 → 능동적 추천자 Clout 35→65
청유형 어미 ↑		 소비자가 자발적 영업사원으로 전환 — 마케팅 비용 구조 변화
이 전환 데이터의 가치: "공간이 좋다"는 누구나 알 수 있지만, "5개 심리 축이 동시에 전환된다"는 정량 데이터는 RXR 2-Layer만 제공할 수 있다. 이것이 IR/제안서에서 "감각적이었습니다" 대신 "Clout 35→65, Freshness 0.35→0.78, 감각 축 Proximal→Distal 전환"이라고 말할 수 있는 근거.

6 전략적 시사점 — 2-Layer가 만드는 차이

6-1. Layer 1만 vs 2-Layer 통합 — 인사이트 품질 비교

질문Layer 1만 답변2-Layer 통합 답변
공간이 성공했나? "긍정 86%, 성공" "긍정 86% 중 진짜 팬은 30~35%. 나머지 40~45%는 인스타 소비형(Auth 낮음)으로 재방문 가치 낮음"
제품 브랜드는 건강한가? "긍정 48%, 부정 30%, 보통" "긍정 리뷰도 Clout 35 — 추천 의지 없음. 긍정의 대부분이 과거 추억 기반(시제 과거 우세)이라 현재 경험 생산력 부족"
공간이 제품 대비 나은 이유? "감성 점수가 더 높다" "5개 심리 축(자아/시간/감각/인지/행동)이 동시에 전환되며 소비자를 평가자에서 추천자로 변환"
다음 팝업에서 뭘 바꿔야? "긍정 유지하면 된다" "인스타 소비(P-02)를 진짜 팬(P-01)으로 전환하려면 체험 깊이 강화 필요. 포토존만으로는 Auth 올릴 수 없음. 제품 터치포인트(DIY/테이스팅)를 동선 핵심에 배치"
시즌4를 해야 하나? "긍정이니 해도 됨" "Freshness 0.78은 '아직 신선'이지만, 시즌 반복 시 하락 필연. 주간 리프레시 캘린더와 시즌별 차별화 콘텐츠 필수. Freshness 모니터링으로 소진 시점 사전 감지"

6-2. 가나초콜릿하우스를 위한 세그먼트별 전략

인스타 소비형 (40~45%)

L1: 긍정 / "예쁘다", "사진"
L2: Auth 낮음, Clout 낮음

전략: 포토존 뒤에 제품 체험 동선 배치. "사진 찍고 → 시식 → 구매"로 연결. SNS 공유 시 체험 쿠폰 제공하여 바이럴을 체험으로 전환

진짜 팬 (30~35%)

L1: 긍정 / "감동", "진심"
L2: Auth 높음, Clout 높음

전략: VIP 프로그램(시즌 프리오픈, 한정 메뉴), 리뷰어 앰배서더 프로그램. 이 세그먼트의 리뷰가 다른 소비자의 P-02→P-01 전환을 유도

관계 소비형 (15~20%)

L1: 긍정 / "같이", "데이트"
L2: Clout 높음, Distance 낮음

전략: 2인/그룹 전용 메뉴, 커플/가족 체험 코스. "함께 만드는 초콜릿" DIY를 핵심 프로그램으로 — 관계 기억에 브랜드 각인

★ 핵심 요약

Layer 1만의 결론

  • 제품: 긍정 48%, 보통
  • 공간: 긍정 86%, 대성공
  • "공간이 제품보다 좋다"
  • 전략? "계속 하면 된다"

2-Layer 통합의 결론

  • 제품: 긍정도 Clout 35, 추천 의지 없음. 과거 추억 의존
  • 공간: 긍정 86% 중 진짜 팬 30~35%, 인스타 소비 40~45%
  • 제품→공간 전환 시 5개 심리 축 동시 전환 실증
  • 전략: 세그먼트별 차별화 + Freshness 모니터링
