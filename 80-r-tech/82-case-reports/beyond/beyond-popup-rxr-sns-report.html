1. 버즈 규모 — 진짜 유효 반응은?

BEFORE — 기존 도구
총 251건 mentions
기존 도구는 검색 결과 전체를 1건=1건으로 카운트. 보도자료, 리그램, 노이즈, 협찬이 모두 동일 가치로 합산됨. "251건 mentions"이라는 숫자 하나만 제공.
→
AFTER — RXR
Gate 3 유효 149건 (59%)
251건 중 102건(41%)이 노이즈/보도/협찬/Auth 미달로 탈락. 실제로 진심이 담긴 반응은 149건.
📊 최고 Auth 88 — "4月 | 내가 좋아하는 것만 하면 뭐 어때" (05.03, 서사형 Q5, 개인 경험+구체적 디테일 풍부)
📊 최저 Auth 10 — "3월 맞벌이 부부 재테크" (06.01, 이벤트 무관 콘텐츠, 브랜드만 언급)
→ 251건의 41%가 허수. 기존 도구의 "251건"과 RXR의 "149건"은 전혀 다른 이야기를 말한다. 협찬·보도·노이즈를 걸러내면 실제 소비자 반응의 규모와 질이 동시에 보인다.

2. 감성 분석 — 긍정이 정말 긍정인가?

BEFORE — 기존 도구
긍정/중립/부정 3단계
기존 3단계: 긍정 38%, 중립 50%, 부정 2%
"긍정이 많으니 캠페인 성공"이라는 단순 결론.
→
AFTER — RXR
4단계 + "혼합"이 핵심
긍정 37% / 혼합 48% / 부정 1% / 중립 12%
기존 "중립 50%"의 대부분이 실은 "혼합"(48%) — 긍정+부정이 공존하는 균형 잡힌 리뷰.
📊 혼합 사례 — "공병 가져가면 리필 제품으로 바꿔줘서 좋았는데, 종류가 더 다양했으면" (긍정+건설적 피드백)
📊 위장 긍정 — 협찬 포스트: 느낌표 15개+, 극단 긍정어 7개 → Auth 33 (기존 도구에서는 "긍정"으로 분류)
→ 혼합 48%는 오히려 좋은 신호. "좋긴 한데 더 이런 게 있었으면"이라는 건설적 피드백은 진지하게 경험한 사람들의 신호. 기존 도구가 이를 "중립"으로 묻어버리면 가장 가치 있는 피드백을 놓친다.

8. Sincerity Gate — 진심 필터링

Gate 1 (G 제거) — 249건 | 긍정 37% | Auth 64.3
Gate 2 (+CEF 제거) — 179건 | 긍정 37% | Auth 66.9
Gate 3 (+Auth60+) — 149건 | 긍정 38% | Auth 73.7
Gate 1→3 변화: 건수 249→149 (-40%), 긍정률 37%→38% (+1%p), Auth 64.3→73.7 (+9.4점)
비욘드의 특이점: 긍정률이 Gate를 거쳐도 거의 변하지 않음(+1%p). 새로(-12%p)와 극명한 대비. 이는 과장 긍정이 적고, 진짜 긍정이 대부분이라는 뜻.

제언: 다음 팝업에서 무엇을 바꿀 것인가?

🧴
헤줘 라인 체험 강화
헤줘 토픽 4%로 거의 부재. 세정 라인 체험을 팝업 핵심 동선에 배치 — 미니 세면대+향 테스트+리필 시연으로 제품 인지 확대.
♻️
공병 교환 → 상설화
공병 수거가 방문 동기의 핵심. 1회성 팝업이 아닌 네이처컬렉션 매장 상설 리필 스테이션으로 전환하면 Auth 유지 가능.
💬
혼합 감성 피드백 반영
"종류가 더 다양했으면" 등 건설적 피드백을 다음 팝업에 직접 반영. "여러분의 의견을 반영했습니다" 커뮤니케이션으로 참여감 제고.
