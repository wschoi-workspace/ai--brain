I. 시장현황 및 문제점 (Market Status & Problem)
1. 문제 정의 (Problem)
쿠키리스(Cookieless) 시대 도래로 디지털 광고의 '아이덴티티' 상실 — 연간 $10B+ 규모의 광고 비효율 발생
전체 리테일 매출의 80%는 오프라인에서 발생하나, 오프라인 행동 데이터의 디지털 활용률은 0%
오프라인 유동인구는 파편화된 채 '식별 가능한 취향 데이터'로 전환되지 못하고 블랙박스로 남아 있음
가장 순수한 소비자 데이터(발자국·손길)를 자산화·상품화한 플레이어가 부재
2. 시장 현황 (Why Now)
글로벌 리테일 미디어 시장의 폭발적 성장과 쿠키리스 전환이 동시 진행
퍼스트파티 데이터 CPM 프리미엄: 서드파티 대비 2~5배
퍼스트파티 기반 광고 전환율 향상: +23% (Google 발표)
글로벌 리테일 미디어 시장: $120B+ (2025) / CAGR 25%+
3. 시사점 (Implication)
오프라인의 '발자국·손길'은 조작 불가능한 가장 순수한 퍼스트파티 데이터 소스
이 데이터를 표준화된 하드웨어·AI로 수집·자산화하고 온라인 리타겟팅·커머스와 연결하는 풀스택 인프라 필요
㈜알엑스알은 이 문제를 해결하는 Physical Ad-Tech Network를 구축

II. 해결방안 및 세부내용 (Solution & Technology)
1. 연구개발 요약 (Solution)
가. 솔루션 개요
오프라인 기반 데이터-커머스 풀스택(Full-stack) 플랫폼 개발
= Ad-Tech(광고 지면) + SaaS(데이터 구독) + Commerce(OMO 결제)
오프라인의 '발자국'과 '손길'을 조작 불가능한 퍼스트파티 데이터로 전환
표준화 하드웨어 'Bracket' + 지능형 'Physical AI'로 오프라인 블랙박스를 데이터화
수집된 데이터를 전용 OMO 커머스 앱과 연결하여 인지→경험→구매→재구매 전 여정을 데이터로 통제
나. 핵심 차별점
국내 유일의 '오프라인 취향 데이터 → 온라인 구매 전환' 풀퍼널 커버 모델
Trade Desk(타겟팅) + Walmart Connect(오프라인 지면) + Criteo(리테일 미디어)를 결합한 유일한 풀스택
원천 특허 3건 확보: 온-오프라인 상관관계 분석 / LAS AI / Bracket 모듈 디자인
다. 정체성
우리는 공간 기획사가 아닌, '도시의 유동인구를 데이터 자산으로 바꾸는 애드테크(Ad-Tech) 기업'
The Google AdSense for Offline — 오프라인 광고 성과 측정의 De-facto Standard 지향

2. 연구개발 내용 및 목표 (Technology & Goal)
가. 4대 핵심 기술 모듈
① Bracket Node (표준화 H/W + 엣지 컴퓨팅)
표준화된 모듈형 하드웨어 — 48시간 내 설치 완료, AI 카메라 + 엣지 컴퓨팅 내장
유휴 오프라인 공간을 지능형 미디어 노드로 변환, 전국 네트워크 확장 구조
② LAS Engine (Lifestyle AI Segmentation)
실시간 외형·행동 분석으로 취향 태그 자동 생성 (비식별 처리)
고객별 AI 페르소나 프로필 구축 — 마케팅 시장에서 가장 희소성 높은 데이터 자산
③ RXR Data Bridge (오프라인↔온라인 커넥터)
Layer1 오프라인 비식별 데이터 → Layer2 모바일 AD-ID 매칭(Wi-Fi/BLE/앱 동의)
→ Layer3 온라인 리타겟팅(Meta·Google·카카오 DSP) — 주요 DSP API 연동 완료
④ OMO Attribution (특허 알고리즘)
오프라인 방문 → 온라인 구매의 인과관계를 수학적으로 증명 (Incrementality 측정)
광고주에게 ROAS를 숫자로 보여주는 유일한 오프라인 솔루션
