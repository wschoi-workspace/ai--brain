1. 문제 정의 (Problem)
  • 쿠키리스(Cookieless) 시대 도래로 디지털 광고의 '아이덴티티' 상실 — 연간 $10B+ 규모의 광고 비효율 발생
  • 전체 리테일 매출의 80%는 오프라인에서 발생하나, 오프라인 행동 데이터의 디지털 활용률은 0%
  • 오프라인 유동인구는 파편화된 채 '식별 가능한 취향 데이터'로 전환되지 못하고 블랙박스로 남아 있음
  • 가장 순수한 소비자 데이터(발자국·손길)를 자산화·상품화한 플레이어가 부재

1 Bracket Node
표준화 H/W + 엣지 컴퓨팅
  • 표준화된 모듈형 하드웨어 — 48시간 내 설치 완료, 전국 네트워크 확장 구조
  • AI 카메라 + 엣지 컴퓨팅 내장, 원본 영상 현장 처리 (개인정보 보호)
  • 유휴 오프라인 공간을 지능형 미디어 노드(Retail Data Node)로 변환
2 LAS Engine
Lifestyle AI Segmentation — 실시간 취향 태그 생성
  • 실시간 외형·행동 분석으로 취향 태그(taxonomy) 자동 생성
  • 비식별 처리로 개인정보 이슈 없이 고객별 AI 페르소나 프로필 구축
  • 행동 벡터 군집화: 목적형·탐색형·경험형·비교형·즉흥구매형 등 세그먼트 분류
3 RXR Data Bridge
오프라인 ↔ 온라인 커넥터 (3-Layer Architecture)
  • Layer 1: 오프라인 비식별 행동 데이터 수집 (시선·접근·진입·체류·구매의도)
  • Layer 2: 모바일 AD-ID 매칭 (Wi-Fi Probe·BLE·앱 동의 기반)
  • Layer 3: 온라인 리타겟팅 연동 (Meta·Google·카카오 DSP API)
4 OMO Attribution
특허 알고리즘 — 오프라인 방문 → 온라인 구매의 인과관계 증명
  • A/B 테스트 기반 점진적 기여도(Incrementality) 측정
  • Physical Media ROAS를 숫자로 증명하는 유일한 오프라인 솔루션
  • Archi-Data 결합: 공간 구조·콘텐츠 조건별 전환 기여도 분해
