종합 인사이트

1. 총 노출 규모: Social 253만 + Non-Social 3,516만 = 약 3,770만 Reach
4일간 팝업으로 약 3,770만 도달. 광고 환산가(AVE) 약 414만원. 단, 노이즈(Eclipse IDE 등) 포함 가능.
2. UGC 비율 86% — 자발적 콘텐츠가 압도적
532건 중 455건이 User Generated Content. 브랜드/미디어 발 콘텐츠(77건) 대비 6배. 팝업이 자발적 콘텐츠 생산을 유도하는 데 성공.
3. 인스타그램 104건 — 가장 중요한 미분석 데이터
네이버 블로그(자체 크롤링)에서 못 잡은 인스타 데이터 104건이 Brand24에서 확보됨. 이 데이터에 RXR 2-Layer를 적용하면 팝업 반응의 전체 그림 완성.
4. 4/5에 이상 스파이크(64건) — 노이즈 확인 필요
Brand24 AI Insight에 따르면 "Eclipse mint 소르비톨 부작용" 관련 X(트위터) 바이럴 + "Eclipse IDE" 노이즈 혼재. 이클립스 팝업과 무관한 데이터를 필터링해야 정확한 분석 가능.
5. 블로그 vs SNS 시차 = 측정 타이밍이 다르다
인스타/X: 당일 반응 → 즉각적 감성/분위기 포착
블로그: 1~2일 후 → 숙고된 경험 서술 → FWA 분석에 더 적합
→ 채널별 분석 목적이 다름. 둘 다 필요.
