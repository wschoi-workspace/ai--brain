### 2-4. 핵심 패턴 발견

**1) 사전 버즈가 본 행사보다 압도적으로 컸다**
- 팝업 전(~3/4) 209건 vs 팝업 기간(3/12~15) 46건
- 2/28~3/4 기간에 **일평균 38건**으로 최대 버즈 → 신제품 출시 보도자료 + 인플루언서 사전 시딩 효과로 추정
- 팝업 자체보다 **신제품 출시 마케팅**이 더 큰 버즈를 만든 구조

**2) 사전예약 오픈(3/7)에 두 번째 스파이크**
- 3/7에 11건으로 사전홍보 기간 최대
- 예약 방법/혜택/꿀팁 정리 포스트 집중

**3) 팝업 2~3일차에 후기 피크**
- 첫날(3/12) 3건 → 2일차(3/13) 15건 → 3일차(3/14) 22건
- 방문 후 하루 지나서 블로그 후기를 작성하는 패턴

**4) 팝업 종료 후 급감**
- 3/16 이후 일평균 1건 수준으로 급감
- 오프라인 이벤트의 전형적 특성 — 종료 후 버즈 유지력 약함

## 5. 시사점 — RXR 분석 관점

### 5-1. 팝업 vs 신제품 출시 — 버즈 기여도 분리 필요

이 케이스에서는 **팝업스토어 단독 효과를 측정하기 어렵다.**
- 신제품 '포도향' 출시와 팝업이 동시에 진행
- 팝업 전 버즈의 58%(121/209건)가 이미 팝업 언급 포함
- → **신제품 출시 버즈 vs 팝업 버즈를 분리**하려면 키워드 세부 분류 또는 본문 토픽 분석 필요

### 5-2. 블로그만으로는 부족 — 인스타그램 데이터 필수

네이버 블로그 299건은 전체 버즈의 일부:
- 팝업 방문자의 즉시 반응은 **인스타그램 스토리/피드**에 집중
- 블로그는 "정리형/후기형" 포스트 중심 → 1~2일 딜레이
- **Brand24으로 인스타 데이터를 추가하면 전체 그림 완성**

### 5-3. 2-Layer 분석 적용 포인트

현재 수집된 데이터에 RXR 2-Layer를 적용하면:
- **Content Layer**: 토픽 분류 (공간 체험 / 제품 맛 / 굿즈 / 포토존 / 게임), 감성 극성
- **Psyche Layer**: 방문 후기의 진정성(Authenticity) — 협찬 포스트 vs 자발적 후기 구분
- **RQL**: 후기 깊이 등급 (Q1 한줄 → Q5 서사형) 자동 분류
