핵심 인사이트

1. 사전 버즈가 본 행사보다 4.5배 컸다
팝업 전(~3/4) 209건 vs 팝업 기간(3/12~15) 46건. 2/28~3/4에 일평균 38건으로 최대 버즈. 신제품 '포도향' 출시 보도자료 + 인플루언서 사전 시딩 효과로 추정. 팝업 자체보다 신제품 출시 마케팅이 더 큰 버즈를 만든 구조.
2. 사전예약 오픈(3/7)에 두 번째 스파이크
3/7에 11건으로 사전홍보 기간 최대. 예약 방법/혜택/꿀팁 정리 포스트 집중. 사전예약 시스템이 자체적으로 콘텐츠 생산을 유도하는 구조.
3. 팝업 2~3일차에 후기 피크
첫날(3/12) 3건 → 2일차(3/13) 15건 → 3일차(3/14) 22건. 방문 후 하루 뒤에 블로그 후기를 작성하는 패턴. 인스타는 즉시 반응이므로 패턴이 다를 것으로 예상.
4. 팝업 종료 후 급감 — 버즈 유지력 약함
3/16 이후 일평균 1건 수준으로 급감. 오프라인 이벤트의 전형적 특성. 장기 버즈를 원한다면 팝업 종료 후 콘텐츠 전략이 별도로 필요.
5. 팝업 vs 신제품 — 버즈 기여도 분리 필요
신제품 '포도향' 출시와 팝업이 동시 진행되어 팝업 단독 효과를 측정하기 어려운 구조. 키워드 세부 분류 또는 본문 토픽 분석(2-Layer)으로 분리 필요.
