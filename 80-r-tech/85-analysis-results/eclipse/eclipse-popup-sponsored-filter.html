추가 발견: 팝업 기간(3/12~15)에는 협찬이 0건

팝업 기간 46건 중 협찬 = 0건!

협찬 8건은 모두 팝업 전 사전홍보 기간(~3/11)에 집중.
실제 팝업 진행 기간의 후기는 100% 자발적 → 팝업 기간 데이터는 필터링 없이도 순수 반응.

이것은 "사전 시딩(협찬) → 본 행사(자발적 반응)" 구조를 보여줌.
협찬의 역할은 팝업 자체의 인지도를 높이는 것이었고, 실제 경험 후기는 자발적으로 발생.

각 Gate에서 다른 질문에 답한다:
• Gate 1: "팝업이 시장에 얼마나 노출됐는가?" → 양적 규모
• Gate 2: "유료 홍보를 빼면 실제 반응이 얼마인가?" → 순수 효과
• Gate 3: "진심으로 좋았던 사람이 얼마인가?" → 브랜드 옹호자 규모

이 3단계를 같이 보여주는 것이 RXR의 차별화.
기존 도구는 Gate 1만 보여줌. RXR은 Gate 1→2→3를 관통하면서 "숫자 뒤의 진짜 의미"를 드러냄.
