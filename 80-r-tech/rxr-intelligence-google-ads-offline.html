수익모델: 단계별 확장

SaaS 구독에서 시작해 플랫폼 거래 수수료로 진화

Phase 1
측정 SaaS
매장별 퍼널 측정 대시보드. 월 구독료.
확실한 초기 매출 — 팝업/쇼룸 즉시 적용
Phase 2
성과 컨설팅
데이터 기반 공간 최적화 제안. 프로젝트비 + 성과보수.
진입률 3%→5% = 매출 67% 증가 증명
Phase 3
오디언스 데이터

Phase 4
미디어 플래닝
"이 세그먼트에게, 이 장소에서, 이 시간에 노출" 캠페인 설계. 집행 수수료(Take Rate).

Google이 증명한 공식

검색 데이터가 쌓일수록 광고 정확도가 올라갔다 → 광고주가 더 모였다 → 데이터가 더 쌓였다.

RXR도 동일: 매장 데이터 축적 → 예측 정확도 상승 → 더 많은 브랜드 참여 → 더 많은 노드 확보

양면 시장 (Two-sided Market)

양쪽 모두 RXR의 '공통 측정 언어'가 있어야 거래가 성립한다

공급측: 매장 / 공간

  • 내 매장 앞 사람이 누군지 알고 싶다
  • 왜 안 들어오는지 알고 싶다
  • 공간을 미디어로 수익화하고 싶다
  • 임대료 이상의 가치를 증명하고 싶다
↔

수요측: 브랜드 / 광고주

  • 타깃이 어디를 다니는지 알고 싶다
  • 어디에 팝업/매장을 열면 될지 알고 싶다
  • 오프라인 노출을 측정 가능하게 사고 싶다
  • 오프라인 ROAS를 증명하고 싶다

프로젝트 렌트의 구조적 이점

대부분의 리테일테크 스타트업은 매장을 확보하지 못해 데이터를 모으기 어렵다.
프로젝트 렌트는 직접 팝업/쇼룸을 운영하기 때문에:

• 설치 동의 불필요 — 자사 공간에서 즉시 검증
• POS/CRM 데이터 직접 접근
• 공간 설계 변경 → 효과 측정 반복 가능
• 실제 운영 경험이 Archi-Data 품질을 보장
