01 · Strategy

5대 설득 원칙 — 모든 페이지에 관통

정부 위탁용역 제안서에서 반복적으로 나타나는 구조적 패턴. 이 5가지가 관통하면 설득력 있는 제안서가 된다.

01
정책 정합성 우선
"우리가 잘하는 것"이 아니라 "당신이 풀려는 문제를 우리가 푼다"의 순서. 발주처의 상위 정책 목표를 먼저 읽고, 제안이 그 정책을 실현하는 구조임을 증명.
02
HIRE 프레임 (JTBD)
발주처가 이 사업을 '고용(HIRE)'하려는 것을 F(기능)/E(정서)/S(사회) 3축으로 정의. 이것이 제안 전체의 평가 기준.
03
문제 → 해결 인과 구조
전년도/기존 사업의 구조적 한계를 지적하고, 올해 제안이 그 한계를 정확히 해소하는 인과관계 명시. "진열은 있었지만 경험은 없었다" 패턴.
04
리스크 → 자산 1:1 매칭
사업의 구조적 리스크를 정의하고, 각 리스크를 제안사의 압도적 자산에 1:1 매칭. 컨소시엄이면 "왜 이 조합인가"의 답.
05
검증 가능한 성과 체계
정부사업은 "잘했다"의 증거가 필요. KPI·EMV·BEI 등 정량 지표를 제안 단계에서 제시하고 측정 방법론까지 포함. "성과보고서를 어떻게 쓸 것인가"를 역산하여 제안을 설계한다.
02 · Strategy

SCQA + HIRE — 제안서 작성 전 설계

제안서를 쓰기 전에 이 두 프레임을 먼저 완성해야 한다. 이것이 제안서 전체의 논리 뼈대가 된다.

SCQA — 심사위원 머릿속 흐름 설계

S (Situation) — 해당 분야의 글로벌/국내 현실. 팩트와 숫자.
C (Complication) — 한국에서의 인지 격차 또는 구조적 문제.
Q (Question) — 이 격차를 어떻게 메우는가?
A (Answer) — 우리 컨셉의 핵심 한 줄.
섬세한식탁: S(세계 휴양지는 모두 섬) → C(한국 섬은 "낙후·시골" 인지 격차) → Q(어떻게 메우나?) → A(거친 주민의 손이 빚어낸 정교함으로)
더셀렉츠: S(K-패션 글로벌 부상) → C(신생 브랜드 해외 데뷔의 3대 리스크) → Q(동시에 해결하는 구조?) → A(리스크별 전담사 1:1 매칭 컨소시엄)

HIRE (F/E/S) — 발주처가 고용하려는 것

F (Functional) — 기능적으로 달성해야 할 것. "방문객이 경험하고 재방문하게 만든다"
E (Emotional) — 정서적으로 전환해야 할 것. "브랜드 이미지를 '낙후'에서 '가고 싶은 곳'으로"
S (Social) — 사회적으로 보여줘야 할 것. "국가 브랜드 행사, 정책 성과 보고"
F(할 수 있게) → E(느끼게) → S(보이게) 3단 상승. 이 3축이 제안서 Phase 4의 모든 페이지에 연결되어야 한다.
SCQA와 HIRE의 관계:
SCQA의 A(답)가 HIRE의 F를 충족해야 한다. SCQA가 "왜 이 사업인가"의 답이라면, HIRE는 "이 사업의 성공은 무엇인가"의 정의. 둘 다 없이 제안서를 쓰면 '자사 자랑 나열'이 된다.

04 · Strategy + Execution

챕터별 전략 + 실행 통합 명세

각 챕터마다 "왜 이 순서인가(전략)"와 "몇 장으로 어떻게 만드는가(실행)"를 하나로 병합.

표지
Opening Shot — 컨셉 선언
MUST 1장 고정 · 절대 늘리지 않는다
"이 제안서를 한 문장으로 요약하면?" — 첫 페이지에서 심사위원의 프레임을 설정
컨셉 한 줄 + 풀블리드 실사cover 레이아웃

패턴: [기존 방식의 한계] + [우리의 전환 명제] — "X가 아니라 Y다" 대비 구조
섬세한식탁: "그동안 섬 홍보는 '진열'에 그쳤습니다. 섬세한 식탁은 한 끼의 '경험'을 제공합니다."
더셀렉츠: "매장을 운영하지 않습니다. 매장을 미디어로 운용합니다."
풀블리드 실사 + 스크림(어두운 오버레이) + 워드마크. 컨셉이 5장째(s06)에서야 논리적으로 설명되지만, 표지에서 먼저 선언한다.
01 제안배경
"왜 지금 이 사업인가"
MUST 3~6P · 전체의 18%
발주처의 정책 과제를 먼저 읽고 → 시장의 빈틈을 짚고 → 재정의한 뒤 → 컨셉을 제시. 자사 자랑이 아니라 정책부터.
① 간지 (28P 이상일 때만)divider
Chapter 01 · Background. 28P 이상이면 풀페이지 간지, 20P 이하면 생략하고 상단 정보바로 대체.
② 사업환경 (정책 + 시장)1P · 카드+스탯
발주 기관의 상위 정책 방향 + 시장 데이터·트렌드를 1장에 압축. 정부가 왜 이 사업에 예산을 배정했는지의 맥락과 기회의 크기.
[상위 정책] → [본 사업의 위치] → [국가적 타이밍/이벤트] → [숫자로 기회 크기 증명]
섬세한식탁: KIDI 섬 지역 특성화 + 섬의 날·섬박람회 / 더셀렉츠: KOCCA 콘텐츠IP + 한불수교 140주년
50P 이상 대형 제안서에서만 정책/시장을 분리(각 1P). 40P 이하는 반드시 1장에 통합.
③ 기존 한계 (The Gap)1P · 대조 분할
기회가 있지만 해결되지 않은 구조적 문제 3~4가지. 전년도 사업의 한계를 분석하면 가장 설득력 있다.
패턴: [전년도 팩트] → [구조적 한계 지적] → [이 리스크를 풀 수 있는 구조가 필요]
섬세한식탁: 해결점 4개 (진열뿐/충동없음/주인공없음/다음없음)
더셀렉츠: 리스크 3개 (BX일관성/동시운영복잡성/파리현지실행)
④ 팝업의 재정의 + 추진방향1P · flow+사이클
문제 해결의 방향성 + 매니페스토를 1장에 통합. NOT→IS→BECOMES 3단 전환 또는 3축 프레임.
항상 3축 구조. 심사위원이 외울 수 있는 단위.
섬세한식탁: 경험설계 × 사람을무대에 × 측정매뉴얼 / 더셀렉츠: 운영안정성 × 미디어밸류 × 타이밍서사
50P 이상이면 매니페스토(NOT/IS/BECOMES)와 추진방향을 분리 가능. 40P 이하는 반드시 1장 통합.
⑤ 본 제안 컨셉1P · 공식 도식+chips
SCQA의 A(답)를 시각화. 컨셉 공식(A×B×C=X) + 톤 가드레일(❌NOT / ✅YES) + 카피 패밀리.
컨셉은 5장째(s06)에 등장 — 정책을 먼저 읽은 뒤라야 설득된다. 표지에서 선언하지만, 여기서 논리적으로 증명.
섬세한식탁: 캐릭터(주민) × 공간(섬) × 의식(체험) = 환상
배경의 정석: 사업환경(②) → 기존 한계(③) → 재정의+방향(④) → 컨셉(⑤)
이 순서가 뒤바뀌면 "우리가 하고 싶은 것"이 되고, 이 순서를 지키면 "발주처가 원하는 것"이 된다.
02 일반현황
"우리가 적임자다"
MUST 2~5P · 전체의 11%
자격 증명은 3장 안에. 정체성 한 줄 → 왜 차별적인가 → 증거. 본 사업과 무관한 실적은 버린다.
① 회사현황 + 사업분야1P · 스탯+카드
법인 기본정보 + 정체성 한 줄 + 3대 사업축. 컨소시엄이면 각 사 요약 + 역할 분담 1줄 표.
정부 배점표 "제안사 신뢰도" 직접 대응. 형식적이지만 누락 시 감점.
28P 이하에서는 회사현황+사업분야를 반드시 1장에 통합. 50P 이상 또는 컨소시엄이면 분리 가능.
② 운영모델 차별성1P · 단 분할
회사의 핵심 역량을 01 제안배경의 "구조적 리스크"에 1:1로 연결. "우리가 잘하는 것" 나열이 아니라 "앞서 정의한 문제를 왜 우리가 풀 수 있는가"의 구조.
[01 제안배경의 리스크 N개] = [우리 자산 N개] — 대칭 구조
더셀렉츠: 리스크①→프로젝트렌트 / 리스크②→마인드디자인 / 리스크③→Le Project P
③ 주요실적1P · 2단 리스트
최근 3년 수행 실적 (건수·금액·발주처). 공공·글로벌 실적 2라인을 1장에 병합. 본 사업과 결 맞는 실적만 선별.
배점표 "사업수행 능력" 직접 대응. 유사사업 실적은 가산점 핵심.
50P 이상이면 성공사례(레퍼런스)를 2~5P 추가 — 이미지 + 성과 수치. 28P 이하면 실적 리스트 1장으로 끝.
03 사업계획
"어떻게 실행하나" — ★ 평가의 무게중심
MUST 11~25P · 전체의 57%
제안서의 본체. 배점 비중 최대. 페이지를 여기에 몰아준다.
① 간지divider
Chapter 03 · Project Plan. 01과 03에만 풀 간지를 넣는다.
② 과업분석 (HIRE/JTBD)1P · 카드+표
발주처가 이 사업으로 달성하려는 것을 F/E/S 3축으로 정리. RFP의 행간을 읽어 "진짜 원하는 것"을 명시.
02 SCQA+HIRE에서 설계한 프레임을 여기서 시각화. F(할 수 있게) → E(느끼게) → S(보이게) 3단 상승.
③ 전년도/기존 분석1P (50P 이상일 때)
전년도 사업의 성과와 한계를 객관적 분석. "잘한 점" 인정 + "구조적 한계" 지적.
[전년 팩트] → [구조적 한계] → [올해 해법 연결]. 01 제안배경의 ③과 중복되므로, 28P 이하에서는 01에 흡수.
28P 이하: 01 배경의 "기존 한계"에 통합하여 별도 페이지 불필요. 40P 이상: 독립 페이지로 분리.
④ 프로젝트 기획 (3축)1P · 매트릭스
01 배경의 "추진방향 3축"을 구체 실행 계획으로 전환. 각 축별 실행 방법 + 기대 효과.
⑤ 핵심 체험 / 매트릭스1~2P · ★시그니처 도식
실제 방문자/참여자가 경험하는 내용의 상세. 프로그램 매트릭스 + 상품 라인업 + 수익 환원 구조.
섬세한식탁: 38섬 = 10코스 풀코스 매트릭스 / 더셀렉츠: 11개 브랜드 2주 릴레이 BX 변환 SOP
패턴: [매트릭스/리스트] → [운영 시나리오(하루 단위)] → [수익/환원 구조]
⑥ 시그니처1P · scene(실사)
컨셉 전체의 무게중심이 되는 1가지. 이 1장이 심사위원 기억에 남아야 한다.
섬세한식탁: 비진도 해녀가 차린 한 끼 / 더셀렉츠: K-Layering 공간 미디어
⑦ 거점별 운영 (거점 수 × 3~4P)scene+KPI
각 거점: 개요(1P) → 특화연출(1~2P) → 모객+KPI(1P). 동일 슬로건, 톤만 미세 조정.
섬세한식탁: 여수 3P(박람회 본점) + 성수 4P(도심 7일 영업)
더셀렉츠: 도쿄 5P(파르코 릴레이) + 파리 4P(라파예트 연합전)
패턴: [거점 A 특성→운영 방식] vs [거점 B 특성→운영 방식] 대조
연출컷(scene+실사)이 집중 배치되는 구간. 필요 실사 컷 수 = 거점별 scene 장수.
⑧ 현장 운영체계1P · 표/체크리스트
스태프 구성 + 시간대 체크리스트 + PM 중심 통합관리 체계.
⑨ 마케팅·홍보 계획1~4P
영상 제작 + 온라인 마케팅 + KOL/미디어 + 굿즈. 채널별 예산 배분까지 명시.
패턴: [채널별 전략] → [예산 배분] → [KPI 연결] → [성과 측정 방법론]
28P 이하: 마케팅 1P로 압축. 40P 이상: 영상/SNS/KOL/굿즈 분리 가능.
⑩ 행동데이터 / 성과측정 (RXR)1P · 파이프라인 도식
경험을 데이터 자산으로 — 성과 측정 IP. EMV·BEI·CVR 등 정량 측정 방법론.
원칙 05 "검증 가능한 성과 체계"의 실체. 경쟁사 차별 결정타.
⑪ 안전관리1P · steps/표
폭염·재난·식품 3중 매뉴얼 + 안전관리책임자 실명·이력 + 중대재해처벌법 준수안.
형식적이라도 반드시 포함. 누락 시 감점 확률 높음. 책임자 실명 기재가 가산 포인트.
⑫ 무드보드 / 디자인 톤1P · 팔레트+소재
컬러 팔레트 + 타이포 규정 + 소재·텍스처 + 레퍼런스 3~5개 + 금지 항목.
패턴: [A × B] 크로스 무드 공식 + 소재 리스트 + 서체 규정 + 금지 목록
섬세한식탁: 파인다이닝 톤다운 × 정관식 정갈함 / 더셀렉츠: 한국 앤틱 × K-패션 콘트라스트
⑬ 제안 핵심 정리 + 차별점1P (40P 이상일 때)
사업계획 전체를 1P로 압축 요약. 컨셉 반복 + 차별화 3~5개 불릿.
28P 이하: 클로징에 흡수. 40P 이상: 독립 페이지.
사업계획의 리듬:
큰 그림(②~⑤) → 시그니처 1장(⑥) → 거점별 연출(⑦) → 운영 인프라(⑧~⑪) → 톤 마감(⑫) → 정리(⑬)
이 순서가 "과업 이해 → 컨셉 증명 → 실행 디테일 → 관리 → 마감"의 자연스러운 하강 곡선.
04 운영관리
"일정과 예산"
MUST 2~4P · 전체의 7%
"실행을 믿을 수 있는가?" — 심사위원의 리스크 불안을 해소하는 구간
① 사업단계 · 일정1P · gantt
월 단위 간트차트. 준비기→실행기→사후기 + 마일스톤(착수·중간·최종 보고) 표시.
RFP의 사업기간과 정확히 대응. 중간보고·최종보고 시점 명시 필수.
② 예산 + 인력 + 성과품1P · budget bar
총 사업비 항목별 배분 + 투입 인력 구성 + 산출물 리스트. 정부 정산 기준(증빙 5종, 영세율 등)을 이해하고 있음을 보여야 한다.
정부 정산 경험이 있음을 암시하는 디테일이 핵심.
50P 이상이면 예산/인력/성과품을 각 1P. 28P 이하는 1장에 통합. 추가제안도 이 장에 흡수(Rule 2).
③ 거버넌스 (10억 이상)1P (조건부)

10억 이상 또는 다수 이해관계자 사업에서만 독립 페이지. 그 외는 ②에 흡수.
05 추가제안 / 클로징
"기대 이상의 가치 + 마지막 한 문장"

가산점 영역 + 비전. 사업의 확장 가능성을 제시하고, 컨셉을 반복하며 원형 구조를 완성.
① 스케일업 / 글로벌 출구0~2P
본 사업 이후의 확장 시나리오. 커머스 연계, 해외 진출, 연간 프로그램화 등.
발주처가 "내년에도 이 업체에 맡기고 싶다"고 느끼게.
28P 이하: 04 예산 슬라이드에 흡수(Rule 2). 40P 이상: 독립 1~2P.
② 클로징1P · 대형 카피
표지 컨셉을 반복하며 원형 구조 완성. "From Display to Experience" 같은 원라이너.

입찰 전 코어리뷰 체크리스트

제안서를 쓰기 전에 내부에서 답해야 할 전략 질문들. 이 문서는 제안서 안에 들어가지 않는다.

GO/NO-GO 판단

1. 역량 일치: RFP 과업이 우리 핵심 역량과 80% 이상 일치하는가?
2. 정량 점수: 수행실적 + 경영상태 합산 예상 점수는? (정량 만점 = 10점 기준)
3. 자격 요건: 직접생산확인증명서·중소기업확인서 등 필수 서류 발급 가능 ETA
4. 단독 vs 컨소시엄: 자격 충족 시 단독이 디폴트. 미충족 시 공동수급 백업 준비.
5. 일정 리스크: 마감까지 잔여 영업일 대비 제안서 완성 가능성
6. 경쟁 구도: 예상 경쟁사 파악, 우리의 차별화 포인트 2~3개 확인

자체 평가 점수표 (정성 80점 기준)

항목배점자체 예상보완 포인트
기획·제작 능력20__/20컨셉 워드플레이 + 시각 시안 + 무드보드 완성도
모객 능력15__/15후크 메커니즘 + 예약제/줄세움 디테일 + SNS 전략
운영 능력15__/15매뉴얼 보유 + 현장 운영 자산 + 인력 명단·교육훈련안
사업 이해도10__/10발주처 정책 본질을 한 줄로 통역할 수 있는가
성과 관리10__/10정량 KPI + 측정 방법론 + 경쟁사 차별
안전 관리10__/10중대재해처벌법 준수안 + 책임자 실명·이력
제안서에 넣지 않는 것: 자체평가 점수, 자격 분기 시나리오(단독 vs 컨소시엄), D-Day 타임라인, GO/NO-GO 판단 — 이것들은 코어리뷰(내부 전략 문서)에만. 제안서에는 RFP 정성평가 항목에 대응하는 콘텐츠만 넣는다.

제안배경 상세 가이드 (01 챕터) — 3건 교차 분석

사업환경 페이지 작성 공식

Formula
정책 배경 → 시장 규모 → 전차년도 실적 → 올해 타이밍
이 순서가 뒤바뀌면 "우리 이야기"가 되고, 이 순서를 지키면 "국가 이야기"가 된다
숫자 표현: 카드형 KPI 패턴시각화 필수
대형 숫자를 카드(박스)형으로 나열. 100K+, 80+, 200+, 30+ 패턴으로 한 눈에 규모 파악.
숫자는 문장 안에 묻히면 효과가 반감. 독립 카드로 분리하여 시각적 앵커 역할.
신기술융합 패턴121P 대형
4년간 여정(2021→2024) 타임라인 + 글로벌 시장 CAGR(연평균 성장률) + 정부 준비지수 1위 인용
[역사적 맥락] → [시장 성장 증명] → [한국의 위치 = 기회] → [경주 관광객 4,700만 = 개최지 정당성]
섬세한식탁 패턴32P 압축
세계 휴양지 = 섬이라는 보편적 사실 + 한국 섬의 인지 격차(낙후·시골) = 기회
[글로벌 트렌드] → [한국의 갭] → [이 갭 = 사업 기회]
더셀렉츠 패턴50P 표준
파르코 역대 1위 실적 선언 + K-뷰티→K-패션 확장 트렌드 = 다음 기회
[자사 실적 = 검증] → [산업 확장 트렌드] → [우리가 그 확장의 적임자]

기존 한계 페이지 작성 공식

Tone Control
"까지 않으면서 한계를 드러내는" 톤 조절법
전년도를 비판하면 발주처 자존심을 건드린다. 빈틈을 지적하되 진화를 인정한다.
핵심 패턴모든 규모 공통
[잘한 점 인정] + [그러나 구조적 한계] + [왜 올해 다르게 해야 하는가]
비판이 아닌 "진화 제안". 발주처가 "맞아, 우리도 그게 아쉬웠어"라고 고개 끄덕이게.
신기술융합 사례121P
"22년까지 B2C 중심 → 23년부터 B2B2C로 변경" = 진화를 인정하면서 한계를 구조적으로 지적
연도별 변화를 타임라인으로 보여주며, 각 연도의 진화 포인트와 남은 과제를 병렬 배치.
섬세한식탁 사례32P
"진열은 있었지만 경험은 없었다" = 전년도 비판이 아닌 빈틈 지적. 문장 하나로 구조적 한계를 선언.
NOT→IS 대비 구조를 활용하여 한계를 시각적으로 보여준다.

추진방향 작성 공식

Structure
항상 3~5축 구조 + 영문 네이밍 필수
심사위원이 기억할 수 있는 키워드 = 제안의 무기
신기술융합 4축 구조121P 대형
공간: SplitSync (B2C+B2B 공간 분리 동기화)
프로그램: Interact&Impact (체험→비즈니스 연결)
홍보: K-Chain Reaction (K-POP IP → 방송 → SNS 연쇄 확산)
운영: Zero Barrier (14개국 언어 + 이동지원 + 체류지원)
각 축의 영문 네이밍 = 심사위원이 PT 후에도 기억하는 키워드. 한글 설명만으로는 기억에 남지 않는다.
차별화 전략 시각화필수
각 축의 1줄 요약을 차트/다이어그램으로 도식화. 텍스트 나열이 아닌 구조도로 보여야 심사위원이 "체계적"이라고 느낀다.
사이클 다이어그램, 매트릭스, 또는 4분면 차트 중 축 수에 맞는 형식 선택.
제안배경 3단 공식 요약:
① 사업환경 = "국가·시장이 이 방향으로 가고 있다" (정책+숫자)
② 기존 한계 = "그런데 여기에 빈틈이 있다" (톤 조절 = 인정+지적)
③ 추진방향 = "그래서 이 3~5축으로 간다" (영문 네이밍 + 도식화)

일반현황 상세 가이드 (02 챕터) — 자격 증명의 기술

컨소시엄 구성, 인력 조직도, 전차년도 수행 경험 등 "우리가 적임자"의 증거를 체계적으로 배치.

제안사 소개 공식

Formula
대표사 + 참여사(컨소시엄) = 각 사의 고유 역할 1줄
컨소시엄 구성 자체가 "왜 이 조합인가"의 답이어야 한다
신기술융합 컨소시엄 패턴121P
대표사: BTL 28년 No.1 — 현장 운영 총괄
참여사 1: 디지털 헤리티지 전문 — 기술 콘텐츠 연출
참여사 2: VFX/사운드 전문 — 몰입형 체험 구현
각 참여사가 01 제안배경의 "추진방향 축"과 1:1 대응. 조직 구성 자체가 전략의 실체.
인력 조직도 표준 패턴1P
총감독(연출) → PM(관리) → 실무팀장 → 분야별 책임자
각 포지션에 실명 + 핵심 경력 1줄. PM과 총감독의 역할 분리가 핵심.
조직도는 트리 형태로 1장. 핵심 인물 3~5명은 프로필 카드(사진+경력+본 사업 역할) 배치.
주요 수상/인증 표기가산점
정부 표창장, 우수 수행사 선정, ISO 인증 등. 정부 기관으로부터 받은 표창이 가산점의 핵심.
수상 이력은 "자랑"이 아니라 "발주처 입장에서의 안심 장치". 유사 사업 수상이 최고 가치.

전차년도 수행 경험 = 최강 무기

Weapon
"3년 연속 수행사" = 사업 이해도 만점급 근거
전차년도 수행 경험이 있다면 반드시 별도 섹션으로 강조
연도별 성과 숫자 나열1P · 타임라인
연도별 핵심 수치를 타임라인으로 나열:
바이어 수 / 미팅 건수 / 상담 금액 / 방문객 수 / 전년 대비 성장률
숫자의 우상향 추세가 핵심. "매년 성장하고 있다" = "내년에도 성장시킬 수 있다"의 암시.
B2B 전문 네트워크 Pool별도 1P 할애 가치
축적된 바이어·파트너·기관 네트워크 DB를 시각화. 국가별, 산업별, 관계 깊이별로 분류. "올해 새로 모집"이 아니라 "이미 보유"하고 있음을 증명.
네트워크 맵 또는 국가별 테이블로 1P. 숫자(바이어 수, 국가 수, 누적 상담 금액)를 카드형 KPI로.
일반현황의 핵심: "우리가 잘하는 것" 나열이 아니라 "01 제안배경에서 정의한 문제를 왜 우리가 풀 수 있는가"의 증거 제시. 전차년도 수행사라면 이것 하나로 02 챕터의 70%를 커버한다.

사업계획 상세 가이드 (03 챕터) — 평가의 무게중심

121P 대형 제안서에서 발견된 장소 선정·콘텐츠 상세화·B2B 프로그램·마케팅의 상세 공식.

장소 선정 평가 매트릭스

Matrix
5축 평가 — 주관적 판단을 객관적 프레임워크로
장소를 "주어진 것"이 아니라 "제안하는 것"으로 전환
5축 평가 체계1P · 레이더 차트
① 관람객 접근성 — 대중교통, 주차, 도보 동선
② VIP 접근성 — 공항·KTX 거리, 숙박, 의전 동선
③ 콘텐츠 적합성 — 공간 규모, 천장고, 전력, 장비 반입
④ 비즈니스 용이성 — 미팅룸, 네트워킹 공간, 동시통역 부스
⑤ 장소 사용성 — 허가 난이도, 비용, 동시 사용 가능성
25점 만점 레이더 차트로 후보지 비교. "왜 이 장소인가"를 감이 아닌 데이터로 설득.
SplitSync 구조 (대형 사업)신기술융합 패턴
B2C 메인 장소 + B2B 서브 장소 = 공간 분리 동기화. 같은 도시 내 2개 장소를 연계하여 관객 동선과 비즈니스 동선을 분리.
장소 배치도(지도) + 두 장소 간 이동 경로 + 셔틀/교통편 명시.
추가 제안 장소가산점 전략
RFP에 명시된 장소 외에 "추가 제안 장소"를 포함. 장소를 "주어진 것"으로만 쓰지 않고, 더 나은 대안을 "제안하는 것"으로 격상.
"발주처가 생각하지 못한 장소"를 제안 = 사업 이해도의 증거. 단, 반드시 5축 평가로 정당화.

콘텐츠/프로그램 상세화 공식

Format
존(Zone)별 반복 포맷 — 패턴 인식의 힘
심사위원이 패턴을 인식하면 이후 페이지를 빠르게 스캔 가능
존별 설명 반복 포맷존 수 × 1~2P
존이름 → 핵심문구 (1줄 서브카피) → Highlight Point (3~5개 불릿) → Design Elements (소재·장비·기술) → 참가기업+콘텐츠명 (리스트)
동일 포맷 반복 = 처음 2~3개를 꼼꼼히 읽으면 나머지는 구조를 신뢰하고 빠르게 넘김. (패턴 10: 반복 포맷의 힘)
시그니처/슈퍼IP 상세화2~3P · 별도 심화
단계별 체험 방식 4단계:
① 입장/준비 — 기대감 조성, 사전 안내
② 핵심 체험 — 몰입, 인터랙션
③ 심화(해설/투어) — 맥락 부여, 스토리
④ 클라이맥스/전환 — 감정 피크, 행동 유도(SNS 공유, 구매)
기존 제안서의 "프로그램 나열"과 차별. 관객의 심리 여정을 설계하면 심사위원도 "체험한 듯" 읽는다. (패턴 7: 단계별 체험 설계)
기술 스펙 명시각 존별 포함
LED 해상도(P2.6 이하), 음향 채널 수(Dolby Atmos 7.1.4), HMD 사양(Meta Quest 3 등), 프로젝터 루멘, 센서 종류까지 구체 명시.
기술 스펙 = "실행 가능성"의 증거. 장비 수량 + 단가 + 설치 면적까지 포함하면 예산 정합성까지 증명.

B2B 프로그램 체계 (대형 사업 전용)

B2B
바이어 유형 세분화 + 3차 배포 프로세스
10억+ 대형 사업에서만 독립 섹션. 소형 사업은 03⑤ 매트릭스에 흡수.
바이어 유형 세분화 매트릭스1P · 4분면
Type A: 콘텐츠 구매자 (방송사, OTT, 배급사)
Type B: 기술 협업 파트너 (XR/AI 기업, 플랫폼)
Type C: 산학연 (대학, 연구소, 공공기관)
Type D: 투자사·엑셀러레이터 (VC, 정부 펀드)
바이어를 한 덩어리로 다루면 "진짜 B2B를 아는가?"라는 의심. 유형별 세분화 = B2B 전문성 증명.
국가별 산업 발전 4단계1P · 계층도
초기시장 (동남아 일부) → 성장기 (중동, 중남미) → 선도 (일본, 유럽) → 글로벌 리더 (미국, 중국)
각 단계별 맞춤 바이어 유형 + 접근 전략이 달라야 한다.
세계 지도 위에 국가별 색 구분 + 각 단계의 바이어 프로파일 카드.
바이어 프로파일 3차 배포프로세스 도식
Longlist (200+) → Shortlist (80+) → Finalist (30+)
각 단계의 선별 기준 + 접촉 방법 + 일정 명시.
숫자가 단계별로 줄어드는 깔때기(Funnel) 구조 = "체계적으로 바이어를 선별한다"의 증거.
피칭데이 포맷시간표
7분 피칭 + 3분 Q&A = 10분/팀. 하루 최대 30팀. 피칭 심사위원 구성 + 평가 기준 + 매칭 알고리즘까지 명시.
피칭데이 타임테이블 + 공간 배치도(피칭룸, 네트워킹 라운지, 1:1 미팅룸) 포함.

마케팅 계획 상세 공식

Marketing
연쇄 확산(K-Chain Reaction) 구조
강력한 IP → 방송매체 → SNS → 바이럴의 4단계 확산
K-POP 슈퍼IP 협업이슈화 핵심
글로벌 팬덤 보유 IP와의 공식 협업 = 초기 관심 폭발. IP 선정 기준(팬덤 규모, 해외 비중, 콘텐츠 연계성)까지 명시.
주요 방송 매체 연계도달범위 확장
지상파/케이블 방송 협업 = 비팬덤 대중까지 도달. 방송 포맷(특집 다큐, 생중계, 매거진) 별 기대 도달 수치 제시.
채널별 전략 분리1P · 채널 매트릭스
SNS: Z세대 도달 (TikTok/릴스 중심, 챌린지 기반)
산업 채널: B2B 타깃 브랜딩 (LinkedIn, 업계 미디어, 뉴스레터)
B2C와 B2B의 마케팅 채널을 분리하여 제시 = "두 타깃을 동시에 다룰 수 있다"의 증명.
연쇄 확산 플로우1P · 플로우 차트
강력한 IP → 방송매체 픽업 → SNS 2차 확산 → UGC 바이럴 → 미디어 밸류 측정(EMV)
플로우 차트로 도식화 + 각 단계의 예상 도달 수(Reach) 수치 병기.
사업계획 57%의 비밀: 장소(왜 여기인가) → 콘텐츠(무엇을 보여주나) → B2B(비즈니스 성과) → 마케팅(어떻게 알리나). 이 4블록이 유기적으로 연결되면 "실행할 수 있겠다"는 확신이 된다.

운영관리 상세 가이드 (04 챕터) — 신뢰의 마지막 퍼즐

안전관리·예산·다국어 대응 등 "이 팀에게 맡기면 사고 없이 끝낼 수 있다"의 증거.

안전관리 표준 문구 가이드

Safety
3중 매뉴얼 + 법률 준수 + 실명 기재
형식적이지만 누락 시 감점. 실명 기재 = 가산점.
3중 매뉴얼 체계1P · 표
① 폭염/재난 매뉴얼 — 기상 단계별 대응(주의→경고→위기), 대피 동선, 긴급 연락망
② 식품 안전 매뉴얼 — HACCP 기반, 냉장 체인, 알레르기 대응, 식중독 예방
③ 시설 안전 매뉴얼 — 구조물 하중, 전기 안전, 소방 설비, 비상구 확보
섬세한식탁 패턴: 식품 안전이 특히 중요한 사업은 식품 매뉴얼을 별도 심화.
중대재해처벌법 준수필수 언급
중대재해처벌법 시행령 제4조 제9호 준수안 명시. 안전보건관리체계 구축 + 위험성 평가 + 안전교육 이수 증빙.
법률 조항 번호까지 명시 = "아는 업체"라는 신뢰. 단순 "안전에 최선"은 감점.
안전관리책임자 실명·이력가산점 핵심
안전관리책임자의 실명 + 자격증 + 유사사업 안전관리 경력. 단순 "담당자 배치 예정"이 아닌 실명 기재가 가산점.

예산 슬라이드 작성 공식

Budget
RFP 비목 준수 + 정산 경험 암시
예산은 "신뢰"의 영역. 비목 구조가 정부 기준과 다르면 즉시 감점.
표준 비목 구조1P · 예산표
인건비 (내부인력) / 경비 (외주·장비·운영) / 일반관리비 (간접비) / 이윤
RFP에 명시된 비목 구조를 그대로 따라야 한다. 자의적 비목 신설 금지.
비목 구조 = 발주처의 정산 양식. 이것을 맞추는 것 자체가 "정부사업 경험"의 증거.
정산 증빙 5종 이해 표시신뢰 구축
세금계산서, 거래명세서, 이체증, 견적서, 검수확인서 — 이 5종을 "준비 가능합니다" 또는 "기존 정산 경험"으로 암시.
직접 나열하기보다, "정산 정합성 확보를 위한 증빙 관리 체계 운영 중"으로 1줄 언급.
추가제안 흡수 전략Rule 2 적용
추가제안(05 챕터)의 스케일업/글로벌 출구를 예산장 하단에 "글로벌 출구" 또는 "스케일업 비전"으로 흡수. 독립 챕터 없이도 가산점 확보.
운영관리의 본질: "실행을 믿을 수 있는가?"에 대한 답. 안전(사고 없을 것) + 예산(돈을 아는 것) + 장벽 해소(세심한 것) = 3종 세트가 갖춰지면 심사위원의 리스크 불안이 해소된다.

Evaluation Matrix		 5축 평가 + 점수화 + 레이더 차트
