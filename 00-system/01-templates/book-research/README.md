# 도서 리서치 루틴 템플릿

정기 도서 추천 리서치를 **토큰 효율적으로** 진행하기 위한 3단계 템플릿입니다.

## 배경

이전 방식(전체 카테고리 × 책마다 1000자 매력글 일괄 생성)은 5시간 블록 한도(12M 토큰)를 초과했습니다.
이 템플릿은 **Depth 분할 전략**으로 이 문제를 해결합니다.

## 3단계 워크플로우

```
┌─────────────────────────────────────────────────────────┐
│  Step 1. 수집 (Collect)                                 │
│  → 카테고리별 책 리스트 + 메타데이터만                  │
│  → 300자 이내 짧은 써머리                               │
│  → 예상 토큰: 전체의 20%                                │
└─────────────────────────────────────────────────────────┘
                       ↓
         [사람이 CSV 보면서 ★ 표시로 선정]
                       ↓
┌─────────────────────────────────────────────────────────┐
│  Step 2. 선정 (Select)                                  │
│  → 선정된 책만 리스트로 옮김                            │
│  → Claude가 선정 이유 한 줄 추가 (선택)                 │
└─────────────────────────────────────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────────────┐
│  Step 3. 상세 작성 (Enrich)                             │
│  → 선정된 책만 상세 추천사/매력글 작성                  │
│  → 블록 나눠 실행 (한 블록당 10~20권)                   │
│  → 예상 토큰: 전체의 30~50%                             │
└─────────────────────────────────────────────────────────┘
```

## 파일 구성

| 파일 | 용도 |
|------|------|
| `01-collect.md` | Step 1 가이드 - 리스트 수집 + 짧은 써머리 |
| `02-select.md` | Step 2 가이드 - 선정 규칙 |
| `03-enrich.md` | Step 3 가이드 - 상세 추천사 작성 |
| `book-list-template.csv` | Step 1 출력 CSV 템플릿 |
| `selected-books-template.csv` | Step 2 출력 CSV 템플릿 |
| `final-recommendations-template.md` | Step 3 최종 결과물 템플릿 |

## 블록 관리 (5시간 한도)

| 작업 | 권장 분할 |
|------|-----------|
| Step 1 (수집) | 카테고리당 1블록 - 한 번에 2~3 카테고리 |
| Step 2 (선정) | 사람 작업 - 토큰 거의 안 씀 |
| Step 3 (상세) | **한 블록당 최대 20권** - 1000자 매력글 기준 |

## 사용법

1. **프로젝트 생성**: `10-projects/YY-book-research-YYYY-MM/` 폴더 생성
2. **템플릿 복사**: 이 폴더의 파일들을 프로젝트로 복사
3. **Step 1 실행**: `/thinking-partner` 또는 Claude에게 `01-collect.md` 참조해서 실행 요청
4. **CSV 확인 후 선정**
5. **Step 3 실행**: 블록 나눠서 (하루 1~2블록)

## 토큰 절감 비교

| 방식 | 예상 토큰 | 블록 수 |
|------|-----------|---------|
| 기존 (일괄 1000자) | 12M+ | 1블록 한도 초과 |
| 새 방식 (3단계) | 3~5M | 2~3블록 분산 |
