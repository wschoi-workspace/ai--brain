기법 비교

Etching 에칭

원리: 왁스 도포 → 바늘로 그림 → 산 부식
선 특성: 자유로운 곡선, 스케치 느낌
톤 표현: 해칭 밀도 (1~3단계 교차)
대표 작가: 렘브란트, 고야

5-layer continuous lines + min_radius guarantee

Engraving 인그레이빙

원리: 뷔랭으로 금속 직접 조각
선 특성: 정밀한 평행선, 기계적 정돈감
톤 표현: 선 굵기 + 간격 변조
대표 용도: 지폐, 우표, 공식 초상화

Parallel line width modulation

Mezzotint 메조틴트

원리: 로커로 전면 거칠게 → 버니셔로 밝은 부분 매끄럽게
선 특성: 선 없음 — 점/면 기반 톤
톤 표현: 연속적 그라데이션 (가장 풍부)
대표 작가: 루트비히 폰 지겐

Multi-scale grain + gamma tone mapping
