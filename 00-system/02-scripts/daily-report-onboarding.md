# 일일업무보고 봇 — 직원 온보딩

봇: **@Dailywork_report_bot** · 링크 `https://t.me/Dailywork_report_bot`

## 온보딩 대상 (미학습 5명)
배성원(사업기획)·전제훈(공간)·김가은(공간)·김예진(공간)·김도영(사업기획)
> 이미 사용 중: 양은정·김준호·양지혜·정예은·조나연 / 대표·감사 제외

## 직원에게 보낼 안내문 (텔레그램 복붙용)

```
📋 일일업무보고 봇 안내

오늘부터 하루 업무를 봇으로 간단히 보고합니다.
정리·요약은 봇이 알아서 해드려요. 매일 퇴근 전 1~2분이면 됩니다.

[처음 시작 — 1분, 딱 한 번만]
1) 봇 열기 → https://t.me/Dailywork_report_bot
   (안 열리면 텔레그램에서 @Dailywork_report_bot 검색)
2) /report 입력
3) 이름만 한 번 입력 (예: 김가은) — 명부에 있어 다음부턴 자동 인식
4) 오늘 한 일을 생각나는 대로 쭉 입력

→ 봇이 7가지(한 일·산출물·의미·결정필요·지원필요·다음·회고)로
   정리해서 본인과 팀장에게 보내드립니다.

부족한 부분은 봇이 1가지만 가볍게 되물어요. 편하게 쓰시면 됩니다 🙏
```

## 운영 메모
- 텔레그램 ID는 **직원이 봇에 첫 메시지를 보내야** 확보됨(대표가 미리 등록 불가) → 위 안내문 전달이 유일한 경로.
- 첫 `/report`에서 이름 입력 1회 → `register_telegram_id`로 자동 학습 → 이후 이름 입력 생략.
- **학습현황 확인**: `arisa-employees.json`의 `by_telegram_id`에 5명이 추가되면 완료. 미접속자에게 리마인드.
- C레벨(배성원·전제훈)도 대상 — 빈도 부담 시 대표가 조정.
