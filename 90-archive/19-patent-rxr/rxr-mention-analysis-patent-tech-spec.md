## 2. 기술 분야

본 발명은 소셜미디어 및 온라인 리뷰 텍스트의 자연어처리(NLP) 및 감성분석 분야에 관한 것으로, 보다 구체적으로는 마케팅 성과측정을 위해 소비자의 온라인 반응에서 감정적 품질, 진정성, 인지적 몰입도를 **다층 심리언어학적 분석**으로 정량화하는 시스템 및 방법에 관한 것이다.

---

## 3. 배경 기술 (종래 기술의 문제점)

### 3-1. 종래 기술 1: 멘션 건수 기반 측정

소셜미디어 모니터링 도구(Brand24, Mention, Brandwatch 등)는 브랜드가 언급된 **건수(volume)**와 **도달 범위(reach)**를 핵심 지표로 사용한다. 그러나 "281건 언급되었다"는 숫자는 해당 언급이 긍정적인지, 진정성 있는 체험에서 비롯된 것인지, 얼마나 깊이 있는 인게이지먼트인지를 전혀 구분하지 못한다.

**문제점:** 협찬 리뷰 100건과 자발적 체험 리뷰 10건이 동일한 가치로 합산됨.

### 3-2. 종래 기술 2: 키워드 매칭 기반 감성분석

기존 감성분석 기술은 사전에 정의된 긍정/부정 키워드 사전과 텍스트를 대조하여 감성을 분류한다.

**문제점:**
- 간접 표현을 처리하지 못함: "나쁘진 않은데..."는 키워드 매칭으로는 "부정"이 아닌 것으로 판별되지만, 화용론적으로는 약한 부정의 의미를 가짐
- "완전 대박 최고!"(과잉 긍정)와 "꼼꼼히 사용해본 결과 만족스러웠다"(절제된 긍정)의 진정성 차이를 구분하지 못함
- 문맥 없이 단어 단위로만 분석하여 "충치 예방에 좋다는 건 아니지만"을 긍정으로 오분류할 수 있음

### 3-3. 종래 기술 3: 협찬 콘텐츠 탐지

기존 협찬 탐지 기술은 "#광고", "제공받", "협찬" 등 명시적 키워드의 존재 여부에 의존한다.

**문제점:**
- 키워드를 사용하지 않는 우회적 협찬 콘텐츠를 감지하지 못함
- 협찬이라도 진정성 있는 리뷰와 형식적인 리뷰를 구분하지 못함 (협찬 내 진심 스펙트럼 존재)

### 3-4. 종래 기술 4: 리뷰 품질 평가

기존 리뷰 품질 평가는 글자수, 이미지 수, 별점 등 양적 지표에 의존한다.

**문제점:**
- 300자 미만이지만 "근데 아쉬운 점은... 그래서 결론적으로 ~라면 추천"처럼 논증 구조가 있는 분석적 리뷰를 과소평가
- 1,000자이지만 동일한 내용을 반복하는 리뷰를 과대평가

### 3-5. 종래 기술 5: LIWC (Linguistic Inquiry and Word Count)

Pennebaker가 개발한 LIWC는 텍스트의 기능어 패턴을 분석하여 심리적 상태를 추론하는 도구이다. 학술 연구에서 널리 사용되나, 한국어 온라인 리뷰 분석에 직접 적용하기에는 한계가 있다.

**문제점:**
- 영어 중심 설계로 한국어의 교착어 특성(조사, 어미 활용)을 충분히 반영하지 못함
- 마케팅 성과측정이라는 실용적 목적에 맞는 통합 파이프라인이 부재
- 감성분석, 진정성 평가, 협찬 탐지 등 마케팅 특화 기능과 연계되지 않음

## 4. 발명이 해결하고자 하는 과제

본 발명은 상기 종래 기술의 문제점을 해결하기 위해 다음의 과제를 해결하고자 한다.

1. **감정적 품질의 정량화:** 단순 건수가 아닌, 각 멘션의 감정적 품질(진정성, 감정 강도, 인지적 깊이)을 0~100 점수로 수치화
2. **다층 분석의 통합:** 내용(Content), 심리(Psyche), 언어학(Linguistic) 3개 레이어의 분석 결과를 하나의 통합 점수 체계로 결합
3. **이중 교차검증:** 키워드 패턴 기반 지표와 언어학적 구조 기반 지표를 독립적으로 산출하여 상호 검증하고, 괴리 발생 시 자동 플래깅
4. **한국어 특화:** 한국어의 교착어 특성(조사 "이/가" vs "은/는", 어미 활용, 문체 레지스터)을 활용한 분석
5. **협찬 콘텐츠의 정교한 처리:** 협찬을 일괄 제거하지 않고, 협찬 내에서의 진심 스펙트럼을 분석

## 5. 발명의 구성 (상세한 기술 설명)

### 5-1. 시스템 전체 구성도

본 발명의 시스템은 4개의 처리 단계(Stage)로 구성된다.

```
┌─────────────────────────────────────────────────────────────────┐
│                    RXR Mention Analysis System                   │
├─────────┬──────────────┬──────────────────────┬─────────────────┤
│ Stage A │   Stage B    │      Stage C         │    Stage D      │
│ 데이터  │  진정성      │   3-Layer            │   수치화 &      │
│ 수집부  │  필터부      │   분석 엔진          │   리포트부      │
│         │              │                      │                 │
│ ·검색   │ ·Content     │ ·Content Layer       │ ·통계 집계      │
│  API    │  Class       │  (무엇을)            │ ·시각화         │
│ ·본문   │  A~G 분류    │ ·Psyche Layer        │  리포트         │
│  크롤링 │ ·Trust       │  (어떻게)            │ ·마케팅         │
│ ·필터링 │  Score       │ ·Linguistic Layer    │  인사이트       │
│         │  산출        │  (언어학적으로)      │  도출           │
│         │              │ ·심리-언어 브릿지    │                 │
│         │              │ ·교차검증            │                 │
└─────────┴──────────────┴──────────────────────┴─────────────────┘
```

### 5-2. Stage A: 데이터 수집부

#### 5-2-1. 검색 및 수집

본 발명의 데이터 수집부는 소셜미디어 플랫폼의 검색 API를 통해 특정 브랜드명 및 관련 키워드에 대한 멘션 데이터를 자동 수집한다.

**수집 대상 데이터:**
- 포스트 제목, 본문 텍스트, 작성일, 작성자, URL
- 검색 키워드: 브랜드명, 별칭, 제품명, 관련 해시태그

**수집 방법:**
- 키워드별 관련도순(sim) + 최신순(date) 병행 검색으로 포괄성 확보
- 링크(URL) 기준 중복 제거
- 모바일 버전 페이지에서 본문 텍스트 추출 (복수의 HTML 구조 패턴 순차 시도)

#### 5-2-2. 3단계 필터링

수집된 데이터에 대해 순차적으로 3단계 필터링을 적용한다:

1. **날짜 필터:** 지정된 분석 기간 내의 포스트만 선별
2. **관련성 필터:** 브랜드명 또는 별칭이 제목/본문에 포함된 포스트만 선별
3. **노이즈 제외:** 사전 정의된 제외 키워드(예: 원료 성분 설명, 학술 논문 등)가 포함된 포스트 제외

---

### 5-3. Stage B: 진정성 필터부 (Sincerity Filter)

#### 5-3-1. Content Class 분류 (A~G 7등급)

수집된 모든 포스트를 텍스트 패턴 분석을 통해 7개 등급으로 자동 분류한다.

| 등급 | 분류명 | 분류 기준 | 후속 처리 |
|:---:|--------|----------|----------|
| **A** | 진성후기 | 개인 경험 신호 2건 이상 AND 구체적 디테일 3건 이상 동시 충족 | 3-Layer 분석 핵심 대상 |
| **B** | 일반포스트 | 특별 플래그 없음, 브랜드 관련 내용 포함 | 3-Layer 분석 대상 |
| **C** | 협찬/체험단 | 협찬 마커("제공받", "#광고", "체험단" 등) 감지 | 3-Layer 분석 (진심 스펙트럼 분석) |
| **D** | 나열형정보 | 제목에 "총정리", "리스트", "모음", "BEST" 포함 | 버즈량만 카운트 |
| **E** | 리그램/단순공유 | "리그램" 마커 또는 비해시태그 텍스트 30자 미만 | 버즈량만 카운트 |
| **F** | 비즈니스/보도 | 보도자료 문구 3건 이상 매칭 또는 비즈니스 신호 2건 이상 | 제외 또는 카운트만 |
| **G** | 노이즈 | 브랜드 키워드 미포함 (다의어 문제 등) | 제외 |

**본 발명의 특징적 구성:** C등급(협찬)을 제거하지 않고 분석 대상에 포함한다. 이는 협찬 콘텐츠 내에서도 진정성 스펙트럼이 존재한다는 연구 결과(King et al., 2014의 eWOM 생성 동기 분류)에 기반한다.

**개인 경험 신호 예시:** "먹어봤", "씹어봤", "사봤", "써봤", "다녀", "친구랑", "엄마"
**구체적 디테일 신호 예시:** "원", "개", "알", "통", "편의점", "마트", 구체적 가격/수량/장소

#### 5-3-2. Trust Score 산출 (0~100)

각 포스트에 대해 기본 100점에서 플래그별 가감점을 적용하여 신뢰도 점수를 산출한다.

**감점 요인:**

| 플래그 | 감점 | 감지 조건 |
|--------|:---:|----------|
| REGRAM | -30 | "리그램", "퍼옴", "repost" 포함 |
| PRESS_COPY | -35 | 보도자료 핵심 문구 3건 이상 매칭 |
| BIZ_SUSPECT | -25 | 비즈니스 신호("공식", "판매처", "구매링크") 2건 이상 |
| LISTING | -20 | 제목에 "총정리/리스트/모음" 포함 |
| SPONSORED | -15 | 협찬 마커 감지 |
| OFF_TOPIC | -30 | 브랜드 키워드 미포함 |
| TOO_SHORT | -20 | 전체 텍스트 20자 미만 |

**가점 요인:**

| 플래그 | 가점 | 감지 조건 |
|--------|:---:|----------|
| PERSONAL_EXP | +15 | 개인 경험 신호 2건 이상 |
| DETAILED | +10 | 구체적 디테일 3건 이상 |

**판정 기준:** 70점 이상 = 신뢰 가능, 40~69점 = 주의, 39점 이하 = 필터 대상

### 5-4. Stage C: 3-Layer 분석 엔진

Content Class **A, B, C**에 해당하는 포스트에 대해 3개 레이어의 분석을 수행한다.

#### 5-4-1. Layer 1: Content Layer (무엇을 말했는가)

**(a) 토픽 분석**

브랜드/카테고리별로 사전 정의된 토픽 사전의 키워드 빈도를 계측하여 각 포스트의 주요 토픽을 자동 분류한다. 포스트당 상위 3개 토픽을 추출하고, 전체 포스트의 토픽 분포를 집계한다.

**(b) 4-way 감성 분류**

긍정/부정 키워드 사전과의 매칭 결과에 기반하여 4단계로 분류한다:

| 분류 | 조건 |
|------|------|
| 긍정 | 긍정 키워드 수 > 부정 키워드 수 + 2 |
| 부정 | 부정 키워드 수 > 긍정 키워드 수 |
| **혼합** | 긍정 키워드와 부정 키워드가 모두 존재 |
| 중립 | 긍정·부정 키워드 모두 부재 또는 극소 |

**본 발명의 특징적 구성:** 기존 3분류(긍정/부정/중립)에 "혼합"을 추가. 혼합 감성은 장단점을 모두 고려하는 분석적 사고(ELM 중심경로 처리)의 행동적 증거이며, 정보 밀도가 높은 양질 리뷰의 신호이다.

**(c) RQL (Review Quality Level)**

리뷰의 품질을 5등급으로 분류하고, 등급별 가중치를 부여하여 리뷰 자산값을 산출한다.

| 등급 | 조건 | 가중치 |
|:---:|------|:---:|
| Q5 서사형 | 500단어 이상 AND 디테일 3건 이상 AND 개인경험 1건 이상 | ×2.0 |
| Q4 분석형 | 300단어 이상 AND 디테일 2건 이상 | ×1.5 |
| Q3 경험형 | 150단어 이상 AND (디테일 1건 이상 OR 개인경험 1건 이상) | ×1.0 |
| Q2 감상형 | 50단어 이상 | ×0.5 |
| Q1 간단형 | 50단어 미만 | ×0.2 |

**리뷰 자산값** = Σ(각 등급 건수 × 해당 가중치)

#### 5-4-2. Layer 2: Psyche Layer (어떻게 말했는가)

**(a) 대명사 패턴 분석**

텍스트 내 대명사 사용 패턴을 정규표현식으로 추출하여 심리적 관여도를 측정한다.

| 패턴 | 정규표현식 | 심리학적 의미 |
|------|-----------|-------------|
| 1인칭 단수 | `나는\|내가\|저는\|제가\|난\s\|전\s` | 심리적 소유감 — 개인적 경험의 주체적 서술 |
| 1인칭 복수 | `우리\|같이\|함께\|다같이` | 사회적 경험 공유 — 집단 동일시 |
| 장소 참조 | `여기\|이곳\|이 공간\|이 매장` | 공간적 몰입 |

**(b) Freshness Index (정보 신선도)**

한국어 조사 "이/가"와 "은/는"의 사용 비율로 산출한다.

```
Freshness Index = "이/가" 출현수 / ("이/가" 출현수 + "은/는" 출현수) × 100
```

**이론적 근거 — 해석수준이론(Construal Level Theory, Trope & Liberman):**
- "이/가"(주격 초점 조사): 새로운 정보의 도입, 저수준 해석(구체적·감각적 처리)을 반영
- "은/는"(주제 조사): 이미 알려진 정보의 설명, 고수준 해석(추상적·스키마 통합)을 반영
- Freshness가 높을수록 대상을 아직 감각적으로 처리하고 있다는 의미
- 시간 경과에 따라 감각 기억이 의미 기억으로 전환되면서 Freshness가 자연 하락하는 패턴은 기억 체계 이론(Memory Systems Theory)과 일치

**(c) Authenticity 산출 (0~100)**

기본 60점에서 다음 요인에 따라 가감점한다:

| 요인 | 보정 | 심리학적 근거 |
|------|:---:|-------------|
| 협찬 감지 | -15 | 외재적 동기에 의한 인지 부조화(Festinger) |
| 과잉 긍정어 5개 초과 | 초과분 × -3 | 비진정한 설득 전략 신호 |
| 느낌표 10개 초과 | 초과분 × -2 | 과잉 감정 표현 |
| 개인경험 2건 이상 | +10 | 직접 체험의 행동적 증거 |
| 구체적 디테일 3건 이상 | +10 | 실제 경험의 기억 기반 세부사항 |
| 긍정+부정 공존(균형 평가) | +15 | 정교화가능성모델(ELM) 중심경로 처리 신호 |
| 범위 | 10~95 | |

**(d) Clout 산출 (0~100)**

추천 의도의 강도를 수치화한다. 기본 40점에서 가감점:

| 요인 | 보정 | 심리학적 근거 |
|------|:---:|-------------|
| 추천 표현("추천", "강추") | 건당 +15 | 직접적 구매 촉진 의도 |
| 복수형 > 1인칭 | +10 | 사회적 증거 활용 |
| 조건부 추천("~라면 강추") | +10 | 높은 신뢰도 — 맹목적 추천 아닌 조건 명시 |
| 예방 초점("안 사면 후회") | +8 | 손실 회피(Loss Aversion) 활용 |
| 정체성 선언("~덕후", "~팬") | +12 | 사회적 정체성(Tajfel)에 기반한 지속적 옹호 |
| 범위 | 10~95 | |

#### 5-4-3. Layer 3: Linguistic Layer (언어학적으로 어떻게 말했는가)

**본 발명의 핵심적 신규 구성. 기존 감성분석 기술에 존재하지 않는 6차원 언어학적 분석 레이어.**

**(a) 차원 1: 형태소 분포 분석**

텍스트의 감각 어휘(미각, 촉각, 후각, 체화 감각) 밀도를 측정한다.

```
감각어 밀도 = 감각 어휘 출현 수 / 전체 단어 수 × 100
```

**감각 어휘 사전 구성 (카테고리별 커스터마이징):**
- 미각: 달달, 상큼, 시원, 청량, 달콤, 씁쓸
- 촉각: 탱글, 쫀득, 부드러, 딱딱, 식감
- 후각: 향, 냄새, 향기
- 체화 감각: 개운, 상쾌, 깔끔, 기분좋, 입안

**판정:** 감각어 밀도 > 3/100단어 = 직접 체험 가능성 높음. 실제 경험한 사람은 신체 감각을 구체적으로 서술하는 경향이 있으며, 이는 체화된 인지(Embodied Cognition) 이론에 부합한다.

**(b) 차원 2: 기능어 패턴 분석**

James Pennebaker의 기능어분석 프레임워크를 한국어에 적용한다.

**분석 대상 기능어:**

| 기능어 유형 | 정규표현식 | 심리학적 해석 |
|------------|-----------|-------------|
| 1인칭 단수 비율 | `나는\|내가\|저는\|제가\|난\s\|전\s\|나의\|내\s` / 전체단어 | 5~15% = 자연스러운 자기표현. 과다(>30%) = 과잉 진정성 연출 의심 |
| 1인칭 복수 | `우리\|같이\|함께\|다같이` | 사회적 경험 공유 수준 |
| 독자 지향 | `여러분\|당신\|분들` | 설득 의도 — Clout와 연관 |
| 장소 조사 "에서" | `에서` | 체험 장소 언급 빈도 |
| 비교 조사 "보다" | `보다` | 경쟁제품 대비 평가 빈도 |

**이론적 근거 — Pennebaker "The Secret Life of Pronouns":**
진짜 리뷰는 1인칭 단수가 자연스러운 비율(5~15%)로 분포. 가짜 리뷰는 1인칭을 과다 사용(진정성 연출 시도)하거나 극도로 적음(개인 경험 부재). Newman & Pennebaker(2003)의 "Lying Words" 연구에서 기능어 사용 패턴이 기만(deception) 탐지에 유효함을 실증.

**(c) 차원 3: 확신도 스펙트럼**

화자의 인식론적 확실성(epistemic certainty)을 측정한다.

**확신 마커:** "확실히", "분명히", "무조건", "진짜", "당연히", "틀림없이"
**완화 마커(헤징):** "좀", "약간", "~것 같아요", "~인 듯", "~편이에요", "~느낌", "~같은데"

```
확신도 비율 = 확신 마커 수 / (확신 마커 수 + 완화 마커 수) × 100
```

**판정 기준 및 보정:**

| 확신도 비율 | 해석 | Auth 보정 |
|:---:|------|:---:|
| 80~100% | 강한 확신 — 열렬한 팬 또는 과잉 긍정 의심 | 과잉긍정과 교차: 둘 다 높으면 -5 |
| 40~79% | 적절한 확신 — 경험 기반 자신감 | +3 |
| 20~39% | 균형잡힌 헤징 — 분석적 사고 | +5 (가장 진정성 높은 구간) |
| 0~19% | 과도한 불확실 — 경험 부재 또는 회피 | -3 |

**이론적 근거:** ELM(정교화가능성모델)에서 중심경로 처리는 확신과 불확실 사이의 균형에서 발생. 극단적 확신은 주변경로(감정적 "좋아요!") 또는 인위적 강조의 신호.

**(d) 차원 4: 담화 표지 분석**

리뷰의 논증 구조를 담화 표지(discourse marker)의 유형과 빈도로 측정한다.

| 담화 표지 유형 | 한국어 마커 | 인지적 의미 |
|--------------|-----------|------------|
| 대조 | "근데", "그런데", "다만", "아쉬운 점은", "하지만", "반면" | 장단점 모두 고려 — ELM 중심경로 |
| 조건 | "~라면", "~경우에", "좋아하시면", "원하시면" | 맥락 인식 — 높은 신뢰도 추천 |
| 인과 | "그래서", "덕분에", "때문에", "그러니까" | 인과적 추론 — 깊은 처리 |

**Discourse Complexity Score (0~100):**
```
Discourse Score = min(대조 마커 수 × 15 + 조건 마커 수 × 12 + 인과 마커 수 × 10, 100)
```

**보정 규칙:**
- Discourse Score ≥ 40이면서 기존 RQL이 Q3인 경우 → Q4로 승격 (양적 기준에 질적 기준 추가)
- Discourse Score ≥ 60이면 Auth +5

**(e) 차원 5: 화용론적 분석**

표면 의미와 실제 의미의 괴리를 감지하여 감성 분류를 정밀화한다.

| 화용론 패턴 | 감지 방법 | 표면 의미 | 실제 의미 | 감성 재분류 |
|------------|----------|---------|---------|-----------|
| 간접 부정 | `나쁘진\s*않` | 중립 | 약한 부정 | 중립 → 혼합 |
| 축소어법(litotes) | `안\s*나쁘` | 이중 부정 | 약한 긍정 | 중립 → 혼합(긍정 기울) |
| 수사적 질문 | `안\s*\S+\s*수\s*있` | 질문 | 강한 긍정 | 감정 강도 상향 |
| 조건부 추천 | `좋아하\S*면\s*(추천\|강추)` | 제한적 추천 | 높은 신뢰도 | Clout +10 |
| 암묵적 비교 | `다른\s*껌\S*\s*(다르\|달리)` | 비교 | 차별화 인지 | 토픽 보강 |

**(f) 차원 6: 문체 레지스터 분석**

한국어의 문체 등급(speech level)을 분류하고, 레지스터 일관성을 측정한다.

| 레지스터 | 감지 패턴 (어미) | 의미 |
|---------|---------------|------|
| 합쇼체 (최존칭) | "-ㅂ니다", "-습니다", "-됩니다" | 공식적 문체 — 보도자료/PR 의심 |
| 해요체 (보통존칭) | "-어요", "-아요", "-에요", "-거든요", "-더라고요" | 표준 리뷰 문체 |
| 해체 (비격식) | "ㅋㅋ", "ㅎㅎ", "ㅠㅠ", "-임", "-음" | 친밀/동료 지향 문체 |

**레지스터 일관성 점수:**
```
각 문장의 레지스터를 판별 → 최빈 레지스터 비율 산출 (0~100%)
```

**보정 규칙:**
- 일관성 > 95% AND 합쇼체 = **템플릿/PR 콘텐츠 의심** → Trust Score -10
- 일관성 60~85% = **자연스러운 레지스터 전환** → Auth +3 (실제 사람은 문체를 자연스럽게 섞음)

**이론적 근거:** 스타일로메트리(Stylometry) 연구에서 기능어/문체 패턴은 "언어적 지문"으로 기능하며, 인위적으로 작성된 텍스트는 자연 발생 텍스트와 구별 가능한 문체적 특징을 보인다.

### 5-5. 복합 지표 및 교차검증

#### 5-5-1. LAI (Linguistic Authenticity Index)

Psyche Layer의 Auth와 **독립적으로** 산출되는 순수 언어학적 진정성 지수. 키워드 매칭이 아닌 **언어 구조의 자연스러움**을 기준으로 진정성을 평가한다.

```
LAI = (sensory_density_norm × 0.30) +
      (hedging_naturalness × 0.20) +
      (register_variability × 0.20) +
      (temporal_proximity × 0.15) +
      (first_person_naturalness × 0.15)

결과값 범위: 0~100
```

**각 구성요소 상세:**

| 구성요소 | 산출 방법 | 0점 (부자연) | 1점 (자연) |
|---------|----------|------------|----------|
| sensory_density_norm | 감각어 밀도 / 10 (최대 1.0) | 감각어 전무 | 10/100단어 이상 |
| hedging_naturalness | 확신도 20~60%이면 1.0, 벗어나면 감소 | 극단적 확신 또는 극단적 불확실 | 자연스러운 확신-불확실 혼합 |
| register_variability | 1 - |일관성 - 75| / 25 (최대 1.0) | 극단적 일관(100%) 또는 극단적 불일관 | 75% 근처의 자연 전환 |
| temporal_proximity | 시간근접 표지 존재 시 1.0 | "오늘","방금","아까" 등 부재 | 최근 체험 표지 존재 |
| first_person_naturalness | 1인칭 비율 5~15%이면 1.0, 벗어나면 감소 | 0% 또는 30% 이상 | 자연스러운 자기표현 범위 |

#### 5-5-2. EDS (Engagement Depth Score)

리뷰어의 인지적 몰입 수준을 측정한다. 리뷰를 작성할 때 얼마나 깊이 사고했는지를 수치화.

```
EDS = (metaphor_density × 0.25) +
      (discourse_complexity × 0.25) +
      (qualification_density × 0.25) +
      (specific_detail_density × 0.25)

결과값 범위: 0~100
```

| 구성요소 | 측정 내용 | 의미 |
|---------|----------|------|
| metaphor_density | 비유/은유 표현("~처럼", "~같은", "~느낌") 수 / 문장 수 | 깊은 인지 처리의 증거 |
| discourse_complexity | Discourse Score / 100 | 논증 구조의 복잡성 |
| qualification_density | 한정 표지("개인적으로", "제 입장에서", "취향 차이") 수 / 문장 수 | 자기인식·메타인지 수준 |
| specific_detail_density | 구체적 수치/장소/시간 표현 수 / 문장 수 | 정보 밀도 |

#### 5-5-3. 이중 교차검증 메커니즘

Auth(Layer 2, 키워드 패턴 기반)와 LAI(Layer 3, 언어학적 구조 기반)를 독립 산출한 후, 두 지표의 괴리 정도에 따라 자동 판정한다.

| |LAI - Auth| | 판정 | 조치 |
|:---:|------|------|
| ≤ 10 | **일치** — 높은 신뢰도 | 그대로 사용 |
| 11 ~ 20 | **경미한 불일치** — 정상 범위 | 참고 표시 |
| > 20 | **유의미한 괴리** — 검토 필요 | 자동 플래깅 |

**괴리 유형별 해석:**

| 유형 | 조건 | 해석 |
|------|------|------|
| AUTH_UNDERESTIMATE | LAI >> Auth | 언어학적으로 진정성 높으나 키워드 패턴이 낮음 → 독특한 표현 스타일로 Auth가 과소평가된 가능성 |
| POSSIBLE_CRAFTED | LAI << Auth | 키워드 패턴은 진정성 높아 보이나 언어 구조가 부자연스러움 → 정교하게 제작된 협찬 리뷰 의심 |

#### 5-5-4. 심리학-언어학 브릿지 보정 테이블

Linguistic Layer 분석 결과로 Psyche Layer 점수를 보정하는 규칙 집합:

| 언어학 피처 | 조건 | 보정 대상 | 보정값 | 심리학적 근거 |
|------------|------|----------|:---:|-------------|
| 감각어 밀도 | >3/100단어 | Auth | +8 | 직접 체험의 신체적 증거 (체화 인지) |
| 1인칭 비율 | 5~15% | Auth | +5 | 심리적 소유감의 자연적 표현 (Pennebaker) |
| 확신도 | 20~60% | Auth | +5 | ELM 중심경로 처리 신호 |
| 과잉 확신 + 과잉 긍정 | 확신도 >80% AND 과잉긍정어 >5 | Auth | -5 | 인위적 감정 표현 의심 |
| 시간근접 표지 | "오늘","방금" 존재 | Freshness | +5 | CLT 저수준 해석 — 최근 체험 |
| 합쇼체 일관 | 일관성 >95% AND formal | Trust | -10 | 템플릿 문체의 구조적 증거 |
| 자연 레지스터 전환 | 일관성 60~85% | Auth | +3 | 실제 인간 문체의 자연적 변동 |
| 대조 표지 2개 이상 | 담화 복잡성 | Auth | +5 | 분석적 사고의 담화적 증거 |
| Discourse Score ≥40 | Q3 리뷰인 경우 | RQL | Q3→Q4 승격 | 양적 기준 미달이나 질적 기준 충족 |
| 간접 부정 감지 | 패턴 매칭 | Sentiment | 중립→혼합 | 화용론적 함축 반영 |
| 조건부 추천 | 패턴 매칭 | Clout | +10 | 조건 명시 = 높은 신뢰도 |

### 5-6. Stage D: 수치화 및 리포트부

분석 결과를 집계하여 다음의 출력물을 생성한다:

2. **통계 요약:** 감성 분포, 토픽 분포, RQL 분포, Psyche Layer 평균, Linguistic Layer 지표

4. **기존 방식 대비 비교:** 2-Layer만 적용한 결과와 3-Layer 적용 결과를 병렬 표시

| 효과 항목 | 수치 |
|----------|------|
| Auth 보정 발생 비율 | 58.1% (978건/1,685건)에서 상향 보정 |
| 평균 Auth 변화 | 71.3 → 74.6 (+3.4점) |
| RQL 승격 건수 | 31건 (Q3→Q4, 담화 복잡성 기반) |
| 합쇼체 템플릿 탐지 | 458건 (27.2%) — 기존 키워드 방식으로는 미탐지 |
| 감각 풍부 리뷰 식별 | 200건 (11.9%) — 핵심 체험 리뷰 자동 식별 |

### 6-2. 정성적 효과

1. **진정성 평가의 정밀도 향상:** 단일 지표(Auth)가 아닌 이중 교차검증(Auth + LAI)으로 과대/과소 평가 자동 감지
2. **리뷰 깊이의 질적 평가:** 글자수가 아닌 담화 구조로 "짧지만 깊이 있는 리뷰"를 정확히 평가
3. **협찬 콘텐츠의 정교한 탐지:** 키워드에 의존하지 않고 문체 레지스터 일관성으로 구조적 탐지
4. **마케팅 인사이트의 심화:** 단순 긍정/부정 비율을 넘어 "어떤 감각으로 경험했는가", "얼마나 깊이 고민했는가", "어떤 문체로 말했는가"까지 파악
5. **카테고리 범용성:** config 기반 설계로 FMCG, 화장품, F&B, 팝업스토어 등 다양한 브랜드에 적용 가능

## 7. 청구항 초안 (변리사 검토용)

### 독립항 1 (시스템)

온라인 멘션의 감정적 인게이지먼트를 정량화하는 시스템에 있어서,

특정 브랜드에 대한 온라인 멘션 데이터를 수집하는 **데이터 수집부**;

수집된 멘션 데이터를 텍스트 패턴 분석에 기반하여 복수의 진정성 등급으로 분류하고, 등급별 신뢰도 점수를 산출하는 **진정성 필터부**;

상기 진정성 필터부에서 분석 대상으로 판정된 멘션에 대해,
(a) 토픽, 감성, 리뷰 품질을 분석하는 Content Layer,
(b) 대명사 패턴, 조사 비율, 진정성 점수, 추천 강도를 분석하는 Psyche Layer, 및
(c) 형태소 분포, 기능어 패턴, 확신도 스펙트럼, 담화 표지, 화용론적 패턴, 문체 레지스터를 분석하는 Linguistic Layer
를 포함하는 **3-Layer 분석 엔진**;

상기 Psyche Layer에서 산출된 제1 진정성 지표(Auth)와 상기 Linguistic Layer에서 독립적으로 산출된 제2 진정성 지표(LAI)의 괴리를 감지하는 **교차검증부**; 및

상기 분석 결과를 수치화하여 리포트를 생성하는 **리포트 생성부**

를 포함하는 것을 특징으로 하는 온라인 멘션 감정적 인게이지먼트 정량화 시스템.

### 독립항 2 (방법)

온라인 멘션의 감정적 인게이지먼트를 정량화하는 방법에 있어서,

(a) 특정 브랜드에 대한 온라인 멘션 데이터를 수집하는 단계;
(b) 수집된 데이터를 텍스트 패턴에 기반하여 복수의 진정성 등급으로 분류하는 단계;
(c) 분석 대상 멘션에 대해, 내용(Content), 심리(Psyche), 언어학(Linguistic) 3개 레이어로 분석하는 단계;
(d) 상기 심리 레이어에서 키워드 패턴 기반으로 산출한 제1 진정성 지표와, 상기 언어학 레이어에서 언어 구조 기반으로 독립 산출한 제2 진정성 지표를 교차검증하는 단계; 및
(e) 상기 언어학 레이어의 분석 결과에 기반하여 상기 심리 레이어의 점수를 보정하는 단계

를 포함하는 것을 특징으로 하는 온라인 멘션 감정적 인게이지먼트 정량화 방법.

### 종속항 후보

3. 제1항에 있어서, 상기 Linguistic Layer는 한국어 조사 "이/가"와 "은/는"의 사용 비율에 기반하여 정보 신선도(Freshness Index)를 산출하는 것을 특징으로 하는 시스템.

4. 제1항에 있어서, 상기 Linguistic Layer는 문체 레지스터(합쇼체/해요체/해체) 분류 및 레지스터 일관성 점수를 산출하여, 일관성이 기준값 이상이면서 격식체인 경우 템플릿 콘텐츠로 판정하는 것을 특징으로 하는 시스템.

5. 제1항에 있어서, 상기 진정성 필터부는 협찬 콘텐츠를 분석 대상에서 제거하지 않고, 협찬 내 진정성 스펙트럼을 분석하는 것을 특징으로 하는 시스템.

6. 제2항에 있어서, 상기 (c) 단계의 언어학 레이어 분석은 확신 마커와 완화 마커의 비율에 기반한 확신도 스펙트럼 분석을 포함하며, 확신도가 소정 범위 내일 때 진정성 점수를 상향 보정하는 것을 특징으로 하는 방법.

7. 제2항에 있어서, 상기 (c) 단계는 담화 표지(대조, 조건, 인과)의 빈도에 기반한 담화 복잡성 점수를 산출하고, 해당 점수가 기준값 이상인 경우 리뷰 품질 등급을 승격시키는 것을 특징으로 하는 방법.

8. 제2항에 있어서, 상기 (c) 단계는 화용론적 패턴(간접 부정, 축소어법, 수사적 질문)을 감지하여 감성 분류를 재조정하는 것을 특징으로 하는 방법.

9. 제1항에 있어서, 상기 3-Layer 분석 엔진은 인지적 몰입도 점수(EDS)를 추가로 산출하며, 상기 EDS는 은유 밀도, 담화 복잡성, 한정 표지 밀도, 구체적 디테일 밀도의 가중합으로 산출되는 것을 특징으로 하는 시스템.
