1. 기술 분야

본 발명은 소셜미디어 및 온라인 리뷰 텍스트의 자연어처리(NLP) 및 감성분석 분야에 관한 것으로, 보다 구체적으로는 온라인상의 소비자 반응(멘션)에서 감정적 품질, 진정성(authenticity), 인지적 몰입도(engagement depth)를 다층 심리언어학적 분석으로 정량화하여 마케팅 성과를 측정하는 시스템 및 방법에 관한 것이다.

2. 배경 기술

2-1. 종래 기술 1: 멘션 건수 기반 측정

소셜미디어 모니터링 도구(Brand24, Mention, Brandwatch 등)는 브랜드가 언급된 건수(volume)와 도달 범위(reach)를 핵심 지표로 사용한다. 그러나 "281건 언급되었다"는 숫자는 해당 언급의 감정적 품질, 진정성, 인게이지먼트 깊이를 전혀 구분하지 못한다. 협찬 리뷰 100건과 자발적 체험 리뷰 10건이 동일한 가치로 합산되는 문제가 있다.

2-2. 종래 기술 2: 키워드 매칭 기반 감성분석

기존 감성분석 기술은 사전 정의된 긍정/부정 키워드 사전과 텍스트를 대조하여 감성을 분류한다. 그러나 이 방법은 다음의 한계를 가진다:

  • 간접 표현 미처리: "나쁘진 않은데..."는 키워드 매칭으로 부정이 아닌 것으로 판별되지만, 화용론적으로는 약한 부정의 의미를 가짐
  • 감정 강도 미구분: "완전 대박 최고!"(과잉 긍정)와 "꼼꼼히 사용해본 결과 만족스러웠다"(절제된 긍정)의 진정성 차이를 구분하지 못함
  • 문맥 부재: 단어 단위 분석으로 "충치 예방에 좋다는 건 아니지만"을 긍정으로 오분류할 가능성

2-3. 종래 기술 3: 협찬 콘텐츠 탐지

기존 협찬 탐지 기술은 "#광고", "제공받", "협찬" 등 명시적 키워드의 존재 여부에 의존한다. 키워드를 사용하지 않는 우회적 협찬 콘텐츠를 감지하지 못하며, 협찬이라도 진정성 있는 리뷰와 형식적인 리뷰를 구분하지 못한다.

2-4. 종래 기술 4: 리뷰 품질 평가

기존 리뷰 품질 평가는 글자수, 이미지 수, 별점 등 양적 지표에 의존한다. 300자 미만이지만 논증 구조가 있는 분석적 리뷰를 과소평가하고, 1,000자이지만 동일 내용을 반복하는 리뷰를 과대평가하는 문제가 있다.

2-5. 종래 기술 5: LIWC 기반 텍스트 분석

Pennebaker가 개발한 LIWC(Linguistic Inquiry and Word Count)는 텍스트의 기능어 패턴을 분석하여 심리적 상태를 추론하는 학술 도구이다. 그러나 영어 중심 설계로 한국어의 교착어 특성(조사, 어미 활용)을 충분히 반영하지 못하며, 마케팅 성과측정이라는 실용적 목적에 맞는 통합 파이프라인이 부재하다.

3. 해결하고자 하는 과제

본 발명은 상기 종래 기술의 문제점을 해결하기 위해 다음의 과제를 해결하고자 한다.

  1. 단순 건수가 아닌, 각 멘션의 감정적 품질(진정성, 감정 강도, 인지적 깊이)을 0~100 점수로 수치화
  2. 내용(Content), 심리(Psyche), 언어학(Linguistic) 3개 레이어의 분석 결과를 통합하는 점수 체계
  3. 키워드 패턴 기반 지표와 언어학적 구조 기반 지표를 독립적으로 산출하여 이중 교차검증
  4. 한국어의 교착어 특성(조사 "이/가" vs "은/는", 어미 활용, 문체 레지스터)을 활용한 분석
  5. 협찬 콘텐츠를 일괄 제거하지 않고, 협찬 내 진심 스펙트럼을 분석

4. 과제의 해결 수단

본 발명은 온라인 멘션의 감정적 인게이지먼트를 정량화하기 위해, 데이터 수집부, 진정성 필터부, 3-Layer 분석 엔진(Content Layer, Psyche Layer, Linguistic Layer), 교차검증부, 리포트 생성부를 포함하는 시스템을 제공한다.

특히, 상기 3-Layer 분석 엔진의 Linguistic Layer는 형태소 분포, 기능어 패턴, 확신도 스펙트럼, 담화 표지, 화용론적 패턴, 문체 레지스터의 6차원 언어학적 분석을 수행하여, Psyche Layer의 진정성 지표(Auth)와 독립적인 제2 진정성 지표(LAI)를 산출하고, 두 지표의 괴리를 자동 감지하는 것을 특징으로 한다.

5. 발명의 상세한 설명

5-1. 시스템 전체 구성

본 발명의 시스템은 4개의 처리 단계(Stage A~D)로 구성된다.

Stage구성부기능출력
A데이터 수집부검색 API, 3단계 필터링, 본문 추출원시 데이터(JSON)
B진정성 필터부Content Class A~G 분류, Trust Score 산출등급별 분류 데이터
C3-Layer 분석 엔진Content + Psyche + Linguistic 분석, 교차검증, 브릿지 보정종합 분석 데이터
D수치화/리포트부통계 집계, 시각화, Before/After 비교리포트

5-2. Stage A: 데이터 수집부

5-2-1. 검색 및 수집

소셜미디어 플랫폼의 검색 API를 통해 브랜드명 및 관련 키워드에 대한 멘션 데이터를 자동 수집한다. 키워드별로 관련도순(sim)과 최신순(date)을 병행 검색하여 포괄성을 확보하며, URL 기준으로 중복을 제거한다.

5-2-2. 3단계 필터링

  1. 날짜 필터: 지정된 분석 기간 내의 포스트만 선별
  2. 관련성 필터: 브랜드명 또는 별칭이 제목/본문에 포함된 포스트만 선별
  3. 노이즈 제외: 사전 정의된 제외 키워드가 포함된 포스트 제외

5-2-3. 본문 추출

복수의 HTML 구조 패턴을 순차 시도하여 본문 텍스트를 추출한다. 특정 패턴으로 추출 실패 시 다음 패턴을 자동 시도하는 폴백 메커니즘을 적용한다.

5-3. Stage B: 진정성 필터부 (Sincerity Filter)

5-3-1. Content Class 분류 (A~G)

수집된 모든 포스트를 텍스트 패턴 분석을 통해 7개 등급으로 자동 분류한다.

등급분류명분류 기준후속 처리
A진성후기개인 경험 신호 2건+ AND 구체적 디테일 3건+3-Layer 핵심 대상
B일반포스트특별 플래그 없음, 브랜드 관련3-Layer 분석 대상
C협찬/체험단협찬 마커 감지3-Layer 분석 (진심 스펙트럼)
D나열형정보제목에 "총정리/리스트/모음" 포함버즈량만 카운트
E리그램/단순공유극히 짧은 텍스트 또는 공유 마커버즈량만 카운트
F비즈니스/보도보도자료 문구 매칭 또는 비즈니스 신호제외
G노이즈브랜드 키워드 미포함제외
핵심 설계 원칙: C등급(협찬)을 제거하지 않고 분석 대상에 포함한다. 협찬 콘텐츠 내에서도 진정성 스펙트럼이 존재하며(Auth 10~63 범위), 이를 구분하는 것이 본 발명의 차별점이다.

5-3-2. Trust Score 산출 (0~100)

기본 100점에서 플래그별 가감점을 적용한다.

플래그점수감지 조건
REGRAM-30"리그램", "퍼옴", "repost"
PRESS_COPY-35보도자료 문구 3건+ 매칭
BIZ_SUSPECT-25비즈니스 신호 2건+

PERSONAL_EXP+15개인 경험 신호 2건+

5-4. Stage C: 3-Layer 분석 엔진

Content Class A, B, C에 해당하는 포스트에 대해 3개 레이어의 분석을 수행한다.

5-4-1. Layer 1: Content Layer

(a) 토픽 분석: 브랜드별 커스텀 토픽 사전의 키워드 빈도를 계측하여 포스트당 상위 3개 토픽을 추출한다.

(b) 4-way 감성 분류: 긍정/부정 키워드 공존 시 "혼합"으로 분류하여, 기존 3분류 대비 정보 밀도가 높은 리뷰를 구분한다.

분류조건
긍정긍정 키워드 수 > 부정 키워드 수 + 2
부정부정 키워드 수 > 긍정 키워드 수
혼합긍정·부정 키워드 모두 존재
중립긍정·부정 모두 부재 또는 극소

(c) RQL (Review Quality Level): 5등급 + 가중치 체계

등급조건가중치
Q5 서사형500단어+ AND 디테일 3건+ AND 경험 1건+×2.0
Q4 분석형300단어+ AND 디테일 2건+×1.5
Q3 경험형150단어+ AND (디테일 1건+ OR 경험 1건+)×1.0
Q2 감상형50단어+×0.5
Q1 간단형50단어 미만×0.2
리뷰 자산값 = Σ(각 등급 건수 × 해당 가중치)

5-4-2. Layer 2: Psyche Layer

(a) Freshness Index: 한국어 조사 "이/가"(초점/발견)와 "은/는"(주제/설명)의 비율로 정보 신선도를 산출한다.

Freshness Index = "이/가" 출현수 / ("이/가" + "은/는") × 100

이론적 근거: 해석수준이론(Construal Level Theory). "이/가"는 새로운 정보 도입(저수준 해석), "은/는"은 이미 알려진 정보 설명(고수준 해석)을 반영한다.

(b) Authenticity (제1 진정성 지표, 0~100):

Auth = 60(기본)
  - 협찬 감지 시 -15
  - 과잉긍정어 5개 초과 시: 초과분 × -3
  - 느낌표 10개 초과 시: 초과분 × -2
  + 개인경험 2건+ 시 +10
  + 구체적 디테일 3건+ 시 +10
  + 균형 평가(긍정+부정 공존) 시 +15
범위: 10~95

(c) Clout (추천 강도, 0~100):

Clout = 40(기본)
  + 추천 표현("추천","강추") 건당 +15
  + 조건부 추천("~라면 추천") +10
  + 예방 초점("안 사면 후회") +8
  + 정체성 선언("~팬","~덕후") +12
범위: 10~95

5-4-3. Layer 3: Linguistic Layer ★ 핵심 신규 구성

본 발명의 핵심적 차별화 요소. 기존 감성분석에 존재하지 않는 6차원 언어학적 분석을 수행한다.

차원 1 — 형태소 분포: 감각 어휘(미각/촉각/후각/체화 감각) 밀도 측정

감각어 밀도 = 감각 어휘 수 / 전체 단어 수 × 100
판정: > 3/100단어 = 직접 체험 가능성 높음 → Auth +8

차원 2 — 기능어 패턴: Pennebaker LIWC의 한국어 적용. 1인칭 대명사 비율(5~15% = 자연 범위), 조사 패턴, 어미 분석.

차원 3 — 확신도 스펙트럼: 확신 마커("확실히","진짜") vs 완화 마커("것 같아요","좀")의 비율

확신도 비율 = 확신 마커 수 / (확신 마커 수 + 완화 마커 수) × 100
확신도해석Auth 보정
80~100%강한 확신 — 과잉 긍정 의심과잉긍정 교차 시 -5
40~79%적절한 확신+3
20~39%균형잡힌 헤징 (가장 진정성 높음)+5
0~19%과도한 불확실-3

차원 4 — 담화 표지: 대조("근데","다만"), 조건("~라면"), 인과("그래서") 마커의 빈도 분석

Discourse Score = min(대조 × 15 + 조건 × 12 + 인과 × 10, 100)
Score ≥ 40이면서 Q3 → Q4 승격

차원 5 — 화용론: 간접부정("나쁘진 않은데"), 축소어법("안 나쁘다"), 수사적 질문 감지 → 감성 재분류

차원 6 — 문체 레지스터: 해요체/해체/합쇼체 분류 + 일관성 점수

레지스터 일관성 = 최빈 레지스터 문장 수 / 전체 문장 수 × 100
일관성 >95% AND 합쇼체 = 템플릿 의심 → Trust -10
일관성 60~85% = 자연 전환 → Auth +3

5-4-4. 복합 지표

LAI (Linguistic Authenticity Index, 제2 진정성 지표):

LAI = (감각어밀도정규화 × 0.30) + (헤징자연도 × 0.20) + (레지스터변동성 × 0.20) + (시간근접성 × 0.15) + (1인칭자연도 × 0.15)
범위: 0~100

EDS (Engagement Depth Score, 인지적 몰입도):

EDS = (은유밀도 × 0.25) + (담화복잡성 × 0.25) + (한정표지밀도 × 0.25) + (구체적디테일밀도 × 0.25)
범위: 0~100

5-4-5. 이중 교차검증

Auth(Layer 2)와 LAI(Layer 3)를 독립적으로 산출한 후, 두 지표의 괴리를 자동 감지한다.

|Auth - LAI|판정조치
≤ 10일치그대로 사용
11~20경미한 불일치참고 표시
> 20유의미한 괴리자동 플래깅 + 수동 검토

괴리 유형: LAI >> Auth = 독특한 표현으로 Auth 과소평가(AUTH_UNDERESTIMATE) / LAI << Auth = 정교한 협찬 리뷰 의심(POSSIBLE_CRAFTED)

  2. 담화 구조 기반 리뷰 깊이 평가로 "짧지만 깊이 있는 리뷰" 정확 식별

7. 특허 청구범위

독립항
[청구항 1] 온라인 멘션의 감정적 인게이지먼트를 정량화하는 시스템에 있어서,

특정 브랜드에 대한 온라인 멘션 데이터를 수집하는 데이터 수집부;

수집된 멘션 데이터를 텍스트 패턴 분석에 기반하여 복수의 진정성 등급(A~G)으로 분류하고, 등급별 신뢰도 점수(Trust Score)를 산출하되, 협찬으로 분류된 멘션을 분석 대상에서 제외하지 않고 진심 스펙트럼 분석 대상으로 포함하는 진정성 필터부;

상기 진정성 필터부에서 분석 대상으로 판정된 멘션에 대해,
(a) 토픽, 감성(긍정/부정/혼합/중립 4-way), 리뷰 품질 등급(RQL)을 분석하는 Content Layer,
(b) 대명사 패턴, 조사 비율 기반 정보 신선도(Freshness Index), 키워드 패턴 기반 제1 진정성 지표(Auth), 추천 강도(Clout)를 분석하는 Psyche Layer, 및
(c) 형태소 분포, 기능어 패턴, 확신도 스펙트럼, 담화 표지, 화용론적 패턴, 문체 레지스터를 포함하는 6차원 언어학적 분석을 수행하고, 상기 분석 결과로부터 제2 진정성 지표(LAI) 및 인지적 몰입도 지표(EDS)를 산출하는 Linguistic Layer
를 포함하는 3-Layer 분석 엔진;

상기 Psyche Layer에서 산출된 제1 진정성 지표(Auth)와 상기 Linguistic Layer에서 독립적으로 산출된 제2 진정성 지표(LAI)의 차이를 산출하고, 상기 차이가 소정의 임계값을 초과하는 경우 해당 멘션을 자동 플래깅하는 교차검증부; 및

상기 Linguistic Layer의 분석 결과에 기반하여 상기 Psyche Layer의 점수를 보정하는 브릿지 보정부;

를 포함하는 것을 특징으로 하는 온라인 멘션 감정적 인게이지먼트 정량화 시스템.
[청구항 2] 온라인 멘션의 감정적 인게이지먼트를 정량화하는 방법에 있어서,

(a) 특정 브랜드에 대한 온라인 멘션 데이터를 수집하는 단계;
(b) 수집된 데이터를 텍스트 패턴에 기반하여 복수의 진정성 등급으로 분류하되, 협찬으로 분류된 멘션을 제거하지 않고 분석 대상에 포함하는 단계;
(c) 분석 대상 멘션에 대해, 내용(Content), 심리(Psyche), 언어학(Linguistic) 3개 레이어로 분석하는 단계;
(d) 상기 심리 레이어에서 키워드 패턴 기반으로 산출한 제1 진정성 지표(Auth)와, 상기 언어학 레이어에서 언어 구조 기반으로 독립 산출한 제2 진정성 지표(LAI)를 비교하여 괴리를 감지하는 교차검증 단계; 및
(e) 상기 언어학 레이어의 분석 결과에 기반하여 상기 심리 레이어의 점수를 보정하는 브릿지 보정 단계

를 포함하는 것을 특징으로 하는 온라인 멘션 감정적 인게이지먼트 정량화 방법.
종속항
[청구항 3] 제1항에 있어서, 상기 Psyche Layer의 정보 신선도(Freshness Index)는, 한국어 주격 조사 "이/가"의 출현 빈도와 주제 조사 "은/는"의 출현 빈도의 비율로 산출되며, 해석수준이론(Construal Level Theory)에 기반하여 "이/가" 비율이 높을수록 정보 신선도가 높은 것으로 판정하는 것을 특징으로 하는 시스템.
[청구항 4] 제1항에 있어서, 상기 Linguistic Layer의 문체 레지스터 분석은, 각 문장의 어미 패턴에 기반하여 해당 문장의 문체를 격식체(합쇼체), 보통존칭(해요체), 비격식체(해체) 중 하나로 분류하고, 전체 문장에서 최빈 문체의 비율로 레지스터 일관성 점수를 산출하되, 일관성이 소정의 기준값 이상이면서 격식체인 경우 템플릿 콘텐츠로 판정하여 신뢰도 점수를 감점하는 것을 특징으로 하는 시스템.
[청구항 5] 제1항에 있어서, 상기 Linguistic Layer의 확신도 스펙트럼 분석은, 텍스트 내 확신 마커와 완화 마커(헤징)의 비율을 산출하고, 상기 비율이 제1 범위(20~60%) 내일 때 제1 진정성 지표를 상향 보정하고, 상기 비율이 제2 범위(80% 이상)이면서 과잉 긍정 표현이 소정 수 이상 존재하는 경우 제1 진정성 지표를 하향 보정하는 것을 특징으로 하는 시스템.
[청구항 6] 제2항에 있어서, 상기 (c) 단계의 언어학 레이어 분석은 담화 표지(대조, 조건, 인과)의 빈도에 기반한 담화 복잡성 점수를 산출하고, 해당 점수가 소정의 기준값 이상인 경우 리뷰 품질 등급을 상위 등급으로 승격시키는 것을 특징으로 하는 방법.
[청구항 7] 제2항에 있어서, 상기 (c) 단계의 언어학 레이어 분석은 화용론적 패턴(간접 부정, 축소어법, 수사적 질문)을 감지하여 감성 분류를 재조정하는 것을 특징으로 하는 방법.
[청구항 8] 제1항에 있어서, 상기 제2 진정성 지표(LAI)는 감각어 밀도 정규화, 헤징 자연도, 레지스터 변동성, 시간 근접성, 1인칭 자연도의 가중합으로 산출되는 것을 특징으로 하는 시스템.
[청구항 9] 제1항에 있어서, 상기 인지적 몰입도 지표(EDS)는 은유/비유 밀도, 담화 복잡성, 한정 표지 밀도, 구체적 디테일 밀도의 가중합으로 산출되는 것을 특징으로 하는 시스템.
[청구항 10] 제1항에 있어서, 상기 교차검증부는, 제1 진정성 지표(Auth)와 제2 진정성 지표(LAI)의 차이가 소정의 임계값을 초과하는 경우, 상기 LAI가 Auth보다 높으면 "과소평가(AUTH_UNDERESTIMATE)"로, 상기 Auth가 LAI보다 높으면 "제작 의심(POSSIBLE_CRAFTED)"으로 유형을 분류하여 플래깅하는 것을 특징으로 하는 시스템.

9. 실시예

본 발명의 시스템을 롯데웰푸드의 "자이리톨" 브랜드에 적용한 실시예를 설명한다.

9-1. 데이터 수집

네이버 블로그 검색 API를 통해 6개 키워드("자이리톨 후기", "자이리톨 껌 추천", "자이리톨 리뷰", "롯데 자이리톨", "자이리톨 맛", "자이리톨 효과")로 최근 30일간의 데이터를 수집하였다. 총 8,227건을 수집하고, 3단계 필터링 후 1,961건이 분석 대상으로 선별되었다.

9-4. Linguistic Layer 발견

  • 문체 레지스터: 해요체 41.5%, 합쇼체 27.2%, 해체 18.8% — 합쇼체 비율이 예상보다 높아, 치과/건강 정보 블로그가 브랜드 인식의 상당 부분을 형성하고 있음을 발견
