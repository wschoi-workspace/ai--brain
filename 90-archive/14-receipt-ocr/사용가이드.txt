════════════════════════════════════════════
  영수증 OCR 변환기 사용 가이드
════════════════════════════════════════════

■ 프로그램 소개

영수증 사진을 폴더에 모아두고 실행 버튼 한 번으로
Excel 파일(.xlsx)로 자동 정리해주는 프로그램입니다.


■ 추출되는 정보 (7개 항목)

 1. 날짜       - 거래일자
 2. 상호       - 매장 이름
 3. 사업자번호 - 123-45-67890 형식
 4. 총금액     - 최종 결제 금액
 5. 결제수단   - 카드 / 현금 / 기타
 6. 주소       - 매장 주소
 7. 원본파일명 - 어느 사진에서 추출됐는지 추적용


■ 지원 이미지 포맷

  .jpg, .jpeg, .png, .webp, .heic (아이폰 사진)


════════════════════════════════════════════
  사용 방법
════════════════════════════════════════════

[ Step 1. 영수증 사진 준비 ]

- 영수증을 스마트폰으로 촬영하거나 스캔합니다.
- 영수증이 프레임 안에 모두 들어오도록 찍습니다.
- 구겨지거나 흐린 영수증은 인식률이 떨어집니다.
- 사진들을 PC의 아무 폴더에나 모아둡니다.

  예시)  C:\Users\홍길동\Desktop\3월영수증\
          ├── receipt_001.jpg
          ├── receipt_002.jpg
          ├── receipt_003.jpg
          └── ...


[ Step 2. 프로그램 실행 ]

1) 'ReceiptOCR.exe' 파일을 더블클릭합니다.

2) 프로그램 창이 열립니다.

   ┌──────────────────────────────────────┐
   │  영수증 OCR → Excel 변환기           │
   ├──────────────────────────────────────┤
   │ 영수증 폴더: [(폴더를 선택하세요)]   │
   │            [찾아보기]                │
   │                                      │
   │        [ 변환 시작 ]                 │
   │                                      │
   │ 진행률: ░░░░░░░░░░  0/0              │
   │                                      │
   │ 로그:                                │
   │ ┌──────────────────────────────────┐ │
   │ │                                  │ │
   │ └──────────────────────────────────┘ │
   │                                      │
   │      [결과 Excel 열기]               │
   └──────────────────────────────────────┘


[ Step 3. 폴더 선택 ]

1) "찾아보기" 버튼을 클릭합니다.

2) 영수증 사진이 들어있는 폴더를 선택합니다.

3) 선택하면 폴더 아래에 이미지 개수가 표시됩니다.
   예) "이미지 파일 15개 발견 (.jpg, .png, ...)"


[ Step 4. 변환 실행 ]

1) "변환 시작" 버튼을 클릭합니다.

2) 진행률이 실시간으로 업데이트됩니다.
   - 각 영수증마다 약 2~5초 소요
   - 100장이면 약 5~10분 소요

3) 로그 창에 처리 결과가 표시됩니다.
   ✓ 표시 = 성공
   ✗ 표시 = 실패 (오류 메시지 포함)


[ Step 5. 결과 확인 ]

1) 변환 완료 시 아래 메시지가 표시됩니다.
   "완료: 성공 15건 / 실패 0건"

2) Excel 파일은 영수증 폴더 안에 자동 저장됩니다.
   파일명: receipts_20260405_1430.xlsx
          (날짜_시간 형식)

3) "결과 Excel 열기" 버튼을 클릭하면 바로 열립니다.

■ 시트 구조

시트명: "영수증_정리"

┌────────┬─────────┬──────────┬─────────┬────────┬─────┬──────────┬────┐
│ 날짜   │ 상호    │ 사업자번호│ 총금액 │ 결제수단│주소│원본파일명│오류│
├────────┼─────────┼──────────┼─────────┼────────┼─────┼──────────┼────┤
│2026-04-│스타벅스 │123-45-...│ 15,000 │ 카드   │...  │001.jpg   │    │
│2026-04-│GS25     │456-78-...│  3,500 │ 현금   │...  │002.jpg   │    │
└────────┴─────────┴──────────┴─────────┴────────┴─────┴──────────┴────┘

■ 특징

- 첫 행(헤더)은 고정되어 스크롤 시에도 보입니다.
- 금액은 천단위 구분자(,)가 자동 적용됩니다.
- 실패한 영수증은 붉은색 글자로 표시되며,
  "오류" 열에 실패 사유가 기록됩니다.


════════════════════════════════════════════
  자주 묻는 질문 (FAQ)
════════════════════════════════════════════

Q1. 한 번에 몇 장까지 처리 가능한가요?
    → 권장 100장 이하. 그 이상도 되지만 시간이 오래 걸립니다.

Q2. 인터넷 없이도 작동하나요?
    → 아니요. OCR 처리를 위해 인터넷 연결 필수입니다.

Q3. 인식 실패한 영수증은 어떻게 되나요?
    → Excel 파일에 빨간색으로 기록되며, "오류" 열에 사유가
      표시됩니다. 해당 영수증은 수기 입력하시면 됩니다.

Q4. 같은 영수증을 두 번 처리하면 어떻게 되나요?
    → 실행할 때마다 새 Excel 파일이 만들어집니다
      (파일명에 시간이 포함됨).

Q5. Excel 파일을 수정해도 되나요?
    → 네. 자유롭게 수정/삭제/정렬 가능합니다.

Q6. 프로그램이 안 뜹니다.
    → 아래 "문제 해결" 섹션 참고.

Q7. .env 파일은 뭔가요? 삭제해도 되나요?
    → API 키가 들어있는 설정 파일입니다.
      절대 삭제하면 안 됩니다. (프로그램 작동 안 함)


════════════════════════════════════════════
  문제 해결
════════════════════════════════════════════

[ 문제 1 ] 더블클릭했는데 아무것도 안 뜸

원인: Windows Defender가 차단했을 수 있습니다.
해결:
  1) "Windows에서 PC를 보호했습니다" 메시지 확인
  2) "자세한 정보" 클릭
  3) "실행" 버튼 클릭
  (또는 담당자에게 문의하여 예외 등록)


[ 문제 2 ] "API 키 없음" 오류

원인: .env 파일이 .exe 옆에 없거나 삭제됨
해결:
  1) ReceiptOCR.exe가 있는 폴더 확인
  2) 같은 폴더에 .env 파일이 있는지 확인
  3) 없으면 담당자에게 재요청


[ 문제 3 ] 네트워크 오류

원인: 인터넷 연결 문제 또는 회사 방화벽
해결:
  1) 인터넷 연결 확인 (브라우저 접속 테스트)
  2) IT팀에 다음 도메인 접근 허용 요청:
     generativelanguage.googleapis.com


[ 문제 4 ] 인식률이 너무 낮음

원인: 이미지 품질 문제
해결:
  1) 영수증이 선명하게 찍혔는지 확인
  2) 영수증 전체가 프레임에 들어왔는지 확인
  3) 그림자/빛 반사 없는지 확인
  4) 흐린 영수증은 다시 촬영 권장


[ 문제 5 ] 처리 속도가 느림

원인: 이미지 크기가 너무 크거나 네트워크 느림
해결:
  1) 스마트폰 카메라 고화질 모드 해제 고려
  2) 한 번에 50장 이하로 나눠서 처리

  주의사항
════════════════════════════════════════════

■ API 사용료 안내

- 이 프로그램은 Google Gemini API를 사용합니다.
- 영수증 100장당 약 300~700원의 비용이 발생합니다.
- 회사에서 공용 API 키를 사용 중이므로 개인 결제는 없습니다.
- 과도한 사용(하루 수천 건 등)은 자제해주세요.

■ 보안 사항

- .env 파일(API 키)을 외부에 공유하지 마세요.
- 이 프로그램을 회사 외부에 배포하지 마세요.
- 개인정보가 포함된 영수증 처리 시 주의하세요.

■ 데이터 처리

- OCR 처리를 위해 영수증 이미지가 Google 서버로 전송됩니다.
- 민감한 문서는 별도 승인 후 사용하세요.
