## 봉은사 출력본 PDF 전수 재생성 + /print-version 스킬 고도화

### 페이지 분리 버그 수정
- 기존: 10/13개 파일에서 슬라이드가 합쳐져 출력됨 (15슬→7p 등)
- 원인: `@media print`에 `height` 고정 없이 `min-height`만 → 짧은 슬라이드 합산
- 수정: `@page{size:1280px 720px;margin:0}` + `height:720px!important` + `prefer_css_page_size=True`
- 결과: 13개 전체 슬라이드 수 = PDF 페이지 수 완벽 일치

### 스킬 업데이트
- **기본 규격 신설**: 가로형(Landscape) 16:9 = 1280×720px (별도 요청 없으면 항상 가로)
- **PDF 필수 생성**: 출력본 = HTML + PDF 쌍으로 항상 생성
- **스크립트 수정**: convert-to-print.py에 @page 중복 제거 + 새 PRINT_CSS 적용
- 메모리 2곳 반영 (feedback_print_version_workflow + feedback_slide_proposal_layout)

### 드롭박스 정리
- `Dropbox/.../봉은사프로젝트/중간산출물/출력본/` 폴더에 HTML 13개 + PDF 13개 = 26개 파일
