#### 2. RFP 분석 Phase 1 — 계약/과업/미팅 종합
- 과업지시서(PDF) + 추가합의서(DOCX) + 미팅 회의자료 종합 분석
- **수익 baseline 확인: 월 3.5억 (연 42억)**
- **문화집회시설 = 음식조리 불가** (과태료) 확인
- 3년 공실 (누적 기회비용 ~126억)
- 추가합의서 리스크 헷지 5건 확인 (2배 배상→계약금 이하, 자동승인 10일 등)
- 잔존 리스크: 지체상금(일 0.25%) 미헷지
- 라빌 건물은 봉은사 리뉴얼과 **별개 → 장기 투자 가능** (사용자 확인)
- 전략적 질문 9개 도출 (미팅에서 7건 확인 완료)

#### 3. RFP 분석 Phase 2 — 건축도면 · 법적 · 개발 심층
- 건축 도면 4장 분석: 무주 대공간 300~350평, **6~8m 통층 천고**, 전시 최적
- 수리견적 2사: 에이비씨건설 28.56억 / 케희건설 27~41.5억
- 10.27 법난기념관(879석 공연장) → 더 라빌은 전시/판매/체험에 집중
- 법적 종합: 전통사찰법 제6조 = 최대 허들, 건축법 용도변경 = 신고만으로 가능
- 세법: 직영 시 고유목적사업준비금 50% 손금산입
- 삼성동 개발: 2031년 유동인구 50~70만 (현재 31.6만 대비 2배)

#### 4. 아이데이션 — 법적 범위 × 가능성 맵
- 법적 허용 3단계: 🟢보수적 / 🟡긍정적(유권해석) / 🔵옵션(구조설계)
- 종교시설 BM **22개 패턴** 도출 (글로벌 레퍼런스)
- 시나리오 3안:
  - A(문화거점): 20~30억 — baseline 미달, 즉시 가능
  - **B(전통문화복합): 46~65억** — baseline 초과, F&B가 게임체인저
  - C(프리미엄허브): 70~92억 — 구조설계 필요
- 핵심 발견: "제약이 곧 경쟁 해자" — 허용 범위 안에서 제대로 하면 경쟁자 없음
- 권고: A→B→C 단계적 접근
