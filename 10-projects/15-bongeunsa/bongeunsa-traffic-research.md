## 1. 봉은사 방문객 데이터

### 1-1. 일평균/연간 방문객 수

| 구분 | 수치 | 비고 |
|------|------|------|
| **일평균 방문객** | **1만 명 이상** | 신도 + 일반 관람객 합산 (나무위키/봉은사 공식) |
| **일평균 외국인 관광객** | 200~300명 | 별도 집계 |
| **연간 방문객 (추정)** | **약 365~400만 명** | 일 1만 명 x 365일 기준 추정. 주말/행사일 가중 시 상향 |

- **출처**: [봉은사 - 나무위키](https://namu.wiki/w/%EB%B4%89%EC%9D%80%EC%82%AC), [봉은사 공식 사이트](http://www.bongeunsa.org/)
- **참고**: 봉은사는 입장료 무료이므로 정확한 매표 기반 통계는 없음. 위 수치는 봉은사 측 및 관광 관련 자료에서 반복적으로 인용되는 수치

### 1-2. 계절별/시기별 변동

| 시기 | 방문객 특성 | 추정 규모 |
|------|------------|-----------|
| **봄 (3~5월)** | 벚꽃/매화 개화기 + 부처님오신날 | 연중 최고 피크. 일 2~3만 명 추정 |
| **여름 (6~8월)** | 템플스테이 성수기, 일반 방문 감소 | 평시 대비 소폭 감소 |
| **가을 (9~11월)** | 단풍 + 국화전시, 관광 수요 증가 | 봄에 준하는 수준 |
| **겨울 (12~2월)** | 신년 해맞이/새해 기도 | 설/연말 스팟 급증, 평시 감소 |
| **주말 vs 평일** | 주말 방문객이 평일 대비 1.5~2배 | 주말 1.5~2만 명 추정 |

### 1-3. 주요 행사 시 방문객 급증

| 행사 | 시기 | 추정 방문객 | 비고 |
|------|------|------------|------|
| **부처님오신날** | 음력 4월 8일 (양력 5월) | **3~5만 명/일** | 봉은사 단독. 서울 전체 연등회는 100만 명 이상 |
| **연등회 (전통등 전시)** | 5월 중 약 2주간 | **일 2~3만 명** | 봉은사에서도 전통등 전시회 개최 |
| **제야의 종 타종/신년 기도** | 12/31~1/1 | **2~3만 명** | 새해 해맞이 |
| **봉은사 국화축제** | 10~11월 | **일 1.5~2만 명** | 도심 속 가을 행사 |
| **초하루법회** | 매월 음력 1일 | **3,000~5,000명** | 정기 법회 중 최대 규모 |

- **출처**: [연등회 공식](http://www.llf.or.kr/), [기록으로 만나는 대한민국 - 석가탄신일](https://theme.archives.go.kr/next/koreaOfRecord/buddhasDay.do)
- **참고**: 서울 연등회 전체(조계사~동대문 행렬)는 30만 명 이상 참여/관람, 부처님오신날 전체 축제 참여자는 100만 명 이상 (2016년 기준)

## 3. 주변 직장인 트래픽

### 3-1. GBD (강남업무지구) 개요

봉은사는 서울 3대 오피스 권역 중 하나인 **GBD(Gangnam Business District)** 핵심부에 위치. GBD는 강남대로와 테헤란로를 축으로 강남구, 서초구, 송파구 일부를 포함.

### 3-2. 직장인 유동인구

| 구분 | 수치 | 출처/비고 |
|------|------|-----------|
| **테헤란로 상권 일평균 유동인구** | **약 65.8만 명** | 2020.1 기준, 스트레이트뉴스 |
| **코엑스/삼성역 앞 월평균 유동인구** | **약 500만 명** | 코엑스 공식 |
| **삼성역 일 승하차 인원** | 약 10~12만 명/일 | 4개 역(삼성/선릉/역삼/강남) 합산 58.3만 명 (2017) |
| **GBD 오피스 공실률** | 약 2% | 2024~2025년 기준, 초저공실 |

- **출처**: [강남 테헤란로 상권 유동인구 - 스트레이트뉴스](https://www.straightnews.co.kr/news/articleView.html?idxno=71035), [반론보도닷컴 - 직장인 유동인구](https://www.banronbodo.com/news/articleView.html?idxno=23420)

### 3-5. 점심/퇴근 후 유동인구 패턴

- **점심시간 (11:30~13:30)**: 테헤란로 상권 매출의 84.9%가 주중 발생. 포스코센터~샹제리제센터~선정릉 일대 식당 밀집 구역이 "사람들로 미어터짐"

- **주중 vs 주말 매출 비율**: 주중 84.9% vs 주말 15.1% (테헤란로 상권 기준)

- 강남구는 서울 25개 자치구 중 **가구소득 1위** 지역
- 강남구 거주 인구 약 55만 명, 주간 활동 인구(직장인 포함)는 이보다 훨씬 많음
- 서울시 빅데이터 캠퍼스에서 카드 매출 데이터 기반 동별 소비 패턴 확인 가능

## 5. 지방 방문객

### 5-1. 코엑스/삼성동 방문객 현황

| 구분 | 수치 | 비고 |
|------|------|------|
| **코엑스 일평균 방문객** | 평일 14만 명 / 주말 25만 명 | 코엑스몰 + 전시장 합산 |
| **코엑스몰 일평균** | 약 15만 명 | 쇼핑몰 단독 |
| **코엑스 연간 전시/행사** | 200건 전시 + 2,000건 회의/행사 | 전시 참관객 상당수가 지방에서 방문 |
| **연간 방문객 (추정)** | **약 5,000~7,000만 명** | 일 14~25만 명 x 365일 기준 |

### 5-2. 지방 방문객 비율 추정

코엑스/봉은사 일대 지방 방문객 비율에 대한 공식 통계는 제한적이나, 다음 근거로 추정 가능:

| 추정 근거 | 내용 |
|-----------|------|
| **코엑스 전시/컨벤션** | 전국 단위 전시회(서울모터쇼, 메가쇼, 서울카페쇼 등) 참관객의 30~40%가 수도권 외 지방 거주자로 추정 |
| **한국관광데이터랩 기준** | 서울 주요 관광지 방문자의 거주지 분포 확인 가능 (이동통신 데이터 기반) |
| **코엑스-봉은사 묶음 방문 패턴** | 코엑스 전시 참관 + 봉은사 관람을 함께 하는 경우 빈번. 도보 3분 거리 |
| **지방 관광객 서울 방문 시 선호 지역** | 명동(85.9%), 홍대(52.8%), **강남(45.9%)** 순 (2023 외래관광객 기준이나 내국인도 유사 패턴) |

### 5-3. 코엑스-봉은사 묶음 방문 패턴

```
[전형적 방문 동선]

지방 방문객 (전시/비즈니스 목적)
  → 삼성역 도착
  → 코엑스 전시 참관 (2~4시간)
  → 코엑스몰 쇼핑/식사 (1~2시간)
  → 봉은사 산책/관람 (30분~1시간)
  → 선정릉 산책 (선택)

지방 방문객 (관광/여가 목적)
  → 강남/삼성 일대 쇼핑
  → 봉은사 관광 (사찰/문화 체험)
  → 코엑스 별마당도서관/아쿠아리움
  → 청담/압구정 이동
```

## 종합 요약 (Key Takeaways)

### 트래픽 규모 종합

| 카테고리 | 일평균 | 연간 추정 |
|----------|--------|-----------|
| 봉은사 방문객 | 1만+ 명 | 365~400만 명 |
| 코엑스 유동인구 | 14~25만 명 | 5,000~7,000만 명 |
| 테헤란로 유동인구 | 약 66만 명 | - |
| 삼성역 승하차 | 약 10~12만 명 | - |
| 강남구 종사자(직장인) | - | 약 75~80만 명 (상주) |

### 핵심 인사이트

1. **도심 최고 밀집 트래픽**: 봉은사는 코엑스(일 14~25만), 테헤란로(일 66만)와 인접한 "서울에서 가장 유동인구가 많은 사찰"
2. **직장인 배후 수요 풍부**: GBD 핵심부 위치, 강남구 종사자 75~80만 명의 점심/퇴근 후 잠재 고객
3. **소비력 최상위**: 강남구는 서울 25개 구 중 가구소득 1위, 전국 평균 대비 1.5배 이상 소비력
4. **종교+관광+비즈니스 복합 트래픽**: 신도(1만+/일) + 관광객 + 전시 참관객 + 직장인이 혼합
5. **GBC 효과 기대**: 현대차그룹 GBC 완공(2028 예상) 시 추가 1~2만 명 상주 직장인 유입
6. **계절성 존재하나 기저 트래픽 탄탄**: 종교 행사/사계절 자연경관으로 연중 일정 수준 유지

### 데이터 한계 및 추가 확인 필요 사항

- 봉은사 정확한 방문객 수는 매표 기반이 아니므로 공식 통계 부재. 한국관광데이터랩의 이동통신 기반 데이터 확인 권장
- 강남구 종사자수 최신 수치는 [강남구 2024년 사업체조사 보고서](https://www.gangnam.go.kr/board/B_000643/1058712/view.do)에서 확인 가능
- 코엑스-봉은사 묶음 방문 비율은 한국관광데이터랩의 방문자 이동경로 분석으로 확인 가능
- 강남구 동별 카드 매출 데이터는 [서울시 빅데이터 캠퍼스](https://bigdata.seoul.go.kr/)에서 상세 조회 가능
