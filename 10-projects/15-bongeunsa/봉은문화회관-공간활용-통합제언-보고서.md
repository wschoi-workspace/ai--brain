## 1. 법적 프레임워크 요약

봉은문화회관은 **세 개의 법이 동시에 적용**되는 특수한 공간이다. 어떤 활용 방안이든 이 세 법의 **교집합** 안에 있어야 한다.

### 1-1. 세 법의 역할과 관계

```
┌─────────────────────────────────────────────────┐
│  건축법 (가장 유연)                                │
│  문화집회시설(4군) → 하위군 변경 = 신고만으로 가능     │
│                                                   │
│  ┌─────────────────────────────────────────┐      │
│  │  도시공원법 (중간)                         │      │
│  │  역사공원 내: 교양시설(전시)·편익시설(음식점) OK  │      │
│  │  독립적 판매시설은 공원시설 미해당 위험          │      │
│  │                                           │      │
│  │  ┌─────────────────────────────────┐      │      │
│  │  │  전통사찰법 (가장 제약적)          │      │      │
│  │  │  보존구역 내 영업행위 원칙 금지     │      │      │
│  │  │  허용: 불교·전통문화 상품, 전통다원  │      │      │
│  │  │       신도 편의시설(교육·포교)     │      │      │
│  │  │                                 │      │      │
│  │  │  ★ 이 안이 실제 사업 가능 영역 ★  │      │      │
│  │  └─────────────────────────────────┘      │      │
│  └─────────────────────────────────────────┘      │
└─────────────────────────────────────────────────┘
```

### 1-2. 법별 핵심 조항

| 법령 | 핵심 조항 | 봉은문화회관에의 적용 |
|------|---------|-------------------|
| **건축법** 시행령 제14조 | 4군(문화집회)→5군(영업)/7군(근린): **신고** | 용도변경 절차 자체는 간단 |
| **전통사찰법** 제6조 | 보존구역 내 영업행위 **원칙 금지** | 모든 상업 활동의 상위 규제 |
| **전통사찰법** 시행령 제7조 | 예외 허용: 불교문화상품, 전통문화상품, **전통다원**, 신도편의시설 | 허용 범위 내에서만 상업 가능 |
| **도시공원법** 시행규칙 별표1 | 역사공원 내 교양시설(문화예술회관), 편익시설(음식점) 허용 | 전시·F&B는 가능, 독립 판매시설은 위험 |
| **전통사찰법** 제9조 | 외부 임대 시 **서울시장 허가 + 조계종 승인** 필요 | 직영이 법적으로 가장 안전 |

### 1-3. 결론: 제약이 곧 차별화

일반 상업시설로의 전환은 법적으로 불가능하다. 그러나 이 제약은 역설적으로 **경쟁 해자(moat)**를 만든다. "강남 한복판에서 1,200년 사찰이 직접 운영하는 프리미엄 문화공간"은 어떤 상업시설도 흉내낼 수 없는 포지셔닝이다.

### 2-2. 다섯 가지 활용 가능 영역

위 매트릭스에서 도출된 **법적으로 가능한 5대 활용 영역**:

1. **전통다원·사찰음식 F&B** — 시행령 제7조 "전통다원의 운영"
2. **전시·체험 뮤지엄** — 문화집회시설 본래 용도 + 포교·교육 목적
3. **전통문화 상품 판매** — 시행령 제7조 "불교·전통문화 관련 상품"
4. **명상·웰니스 프로그램** — 시행령 제7조 "신도 편의시설(수행·교육·포교)"
5. **대관·포럼·기업 프로그램** — 집회장 기존 용도 활용

이하 각 영역별로 **구체적 제언 + 실재하는 레퍼런스 케이스 3건**을 제시한다.

## 3. 제언 1: 전통다원·사찰음식 F&B

### 3-1. 방향

시행령 제7조의 **"전통다원의 운영"**은 봉은문화회관에서 가장 확실한 수익원이자, 법적으로 가장 안전한 상업 활동이다. "전통다원"의 해석을 최대한 넓혀, 사찰음식 파인다이닝부터 전통차 카페까지 **다층 F&B 구조**를 구성한다.

### 3-2. 제안 구성

| 공간 | 콘셉트 | 예상 면적 | 예상 객단가 |
|------|--------|---------|---------|
| **봉은사 사찰음식 레스토랑** | 예약제 코스 다이닝, 사찰 장류 활용, 계절 메뉴 | 300~500㎡ | 50,000~100,000원 |
| **전통다원** | 프리미엄 전통차 + 한과·떡, 다도 체험 겸용 | 200~300㎡ | 15,000~30,000원 |
| **사찰음식 체험 키친** | 쿠킹클래스, 발효식품 워크숍, 외국인 체험 | 150~200㎡ | 80,000~150,000원 |

### 3-3. 레퍼런스 케이스

### 3-4. 봉은사 적용 시 차별화 포인트

- **발우공양과의 차별화**: 봉은사 자체 장류(된장·간장·고추장)와 텃밭 식재료 활용, "봉은사 오리지널" 레시피 개발
- **강남 프리미엄**: 삼성동 입지 → 기업 접대·외국인 관광·강남 고소득층 타겟으로 더 높은 객단가 가능
- **공간 스케일**: 발우공양(소규모 룸)과 달리, 8,868㎡ 공간을 활용한 **대형 전통다원 + 쿠킹클래스 + 발효식품 체험관** 통합 운영

## 4. 제언 2: 전시·체험 뮤지엄

### 4-1. 방향

문화집회시설의 **본래 용도를 살리면서** 봉은사의 1,200년 역사, 불교미술, 전통문화를 전시 콘텐츠로 활용한다. 전통 박물관이 아닌 **몰입형·체험형 전시**로 설계하여 MZ세대와 외국인 관광객의 방문을 유도한다.

### 4-2. 제안 구성

| 공간 | 콘셉트 | 예상 면적 | 예상 입장료 |
|------|--------|---------|---------|
| **상설전시관** | 봉은사 1,200년 역사 + 불교미술 + 디지털 미디어아트 융합 | 2,000~3,000㎡ | 15,000~20,000원 |
| **기획전시실** | 시즌별 기획전 (현대미술×불교, K-Culture×전통 등) | 800~1,200㎡ | 전시별 상이 |
| **체험 전시존** | 인터랙티브 명상 체험, 사찰 건축 VR, 경전 필사 체험 | 500~800㎡ | 상설전 포함 |

### 4-3. 레퍼런스 케이스

#### 케이스 1: 아르떼뮤지엄 (제주·강릉·여수·부산)

| 항목 | 내용 |
|------|------|
| **운영** | 디스트릭트(d'strict) |
| **규모** | 제주: 4,600㎡, 10개 전시관 / 강릉: 4,950㎡, 11개 테마 |
| **입장료** | 성인 17,000원, 청소년 13,000원, 어린이 10,000원 |
| **성과** | 제주 누적 200만명, 강릉 1년 100만명 돌파. 2021년 연매출 360억 중 **240억(67%)이 전시관 매출** |
| **수익구조** | 입장료 162억 + 상품판매 21억 + 체험 F&B |

## 8. 별도 정리: 국내 종교시설 상업화 케이스

### 8-1. 왜 별도로 정리하는가

봉은사 사업의 의사결정권자(주지스님, 총무원)를 설득할 때, **다른 종교시설이 이미 상업적 운영을 하고 있다는 선례**는 강력한 근거가 된다. 특히 기독교(교회/성당)는 전통사찰법 같은 특별법이 없어 더 자유롭게 상업 활동을 하고 있으며, 이 사례들은 "종교공간의 상업화가 사회적으로 수용되고 있다"는 증거다.

### 8-3. 교회 vs 사찰 — 상업화 법적 환경 비교

| 항목 | 교회(기독교) | 사찰(불교) |
|------|-----------|---------|
| **특별법 적용** | ❌ 없음 | ✅ 전통사찰법 (영업행위 제한) |
| **건축법 용도변경** | 신고/허가로 자유롭게 가능 | 동일하게 가능하나 전통사찰법이 상위 차단 |
| **영업 범위** | 사실상 제한 없음 | 불교·전통문화 상품, 전통다원, 신도편의시설만 |
| **세법** | 수익사업 = 법인세 과세 | 동일 (수익사업 과세) |
| **임대** | 자유 (교회법인 이사회 결의) | 서울시장 허가 + 조계종 총무원 승인 필요 |
| **사회적 인식** | 교회 상업화에 대한 비판적 시선 존재 | 사찰의 전통문화 활동은 비교적 긍정적 |

### 8-4. 교회 사례에서 봉은사가 배울 것

1. **명동성당 1898**: 종교 정체성을 유지하면서 상업공간을 운영하는 것이 가능하며, 사회적으로 수용됨
2. **소망교회 세금 추징**: 수익사업 미신고의 리스크 → 봉은사는 **사업 개시 전 세무 설계를 완벽히** 해야 함
3. **교회 카페 불법영업 70%**: 대형교회 약 70%가 용도변경·사업자등록 없이 카페를 운영 중이라는 조사 결과 → 봉은사는 **반드시 합법적 절차를 밟아** 리스크를 원천 차단해야 함
4. **사랑의교회 복합시설**: 대규모 종교시설 내 다목적 공간 운영의 운영 노하우 참고

## 10. 실행을 위한 선결 과제

### 10-1. 법적 선결 과제 (최우선)

| 순서 | 과제 | 대상 기관 | 목적 |
|------|------|---------|------|
| **1** | **문체부 종무실 유권해석 요청** | 문화체육관광부 종무실 | "전통다원" 범위(사찰음식 파인다이닝 포함 여부), "전통문화 관련 상품" 범위, "신도 편의시설" 범위 확정 |
| **2** | 조계종 총무원 사전 동의 | 대한불교조계종 총무원 | 사업 방향에 대한 종단 차원의 지지 확보 |
| **3** | 서울시 보존구역 행위 허가 사전상담 | 서울시 문화체육관광국 | 전통사찰법 제9조의2 의제허가 가능 범위 확인 |

| **5** | 역사공원 시설 분류 사전협의 | 서울시 공원녹지과 | 판매시설의 공원시설 해당 여부 |

### 10-2. 사업 구조 선결 과제

| 과제 | 내용 |
|------|------|
| **운영 주체 결정** | 봉은사 직영 vs 위탁운영 vs 하이브리드. 직영이 법적으로 가장 안전하나 운영 전문성 필요 |
| **세무 설계** | 수익사업 개시신고, 구분회계 체계, 고유목적사업준비금(50%) 적립 전략 |
| **리모델링 범위 확정** | 내부 인테리어 vs 구조 변경. 국가유산기본법 현상변경 허가 필요 여부 확인 |
| **투자비 산정** | 리모델링 + 전시 콘텐츠 제작 + 초기 운영비. 사업타당성 분석의 핵심 변수 |

### 10-3. 유권해석이 사업의 천장을 결정한다

```
유권해석 넓음 (발우공양 수준 허용)
  → 사찰음식 파인다이닝 + 전통문화 편집숍 + 기업 프로그램
  → 예상 연매출 70~90억

유권해석 중간 (전통 범위 엄격 해석)
  → 전통차 다원 + 불교용품 숍 + 전시 + 명상
  → 예상 연매출 40~60억

유권해석 좁음 (최소 범위만 허용)
  → 전시 + 소규모 다원 + 기념품 수준
  → 예상 연매출 20~30억
```

**따라서 문체부 유권해석 요청이 이 프로젝트의 가장 중요한 첫 번째 액션이다.**
