# 봉은문화회관 수익사업 세법 및 사업구조 법적 검토

> **작성일**: 2026-05-01
> **대상**: 봉은사(대한불교조계종) 소유 봉은문화회관
> **사업 계획**: 전시 / 판매 / F&B / 체험 프로그램 복합운영
> **면책**: 본 문서는 법률 자문이 아닌 참고용 리서치 자료입니다. 실제 의사결정 전 세무사/변호사 자문을 권장합니다.

---

## 1. 종교법인의 수익사업 과세 (법인세법)

### 1.1 봉은사의 법적 지위

- **대한불교조계종 소속 전통사찰**: 비영리 종교법인
- 법인세법상 **비영리내국법인**에 해당 (법인세법 제1조 제2호)
- 비영리법인은 **수익사업에서 발생한 소득에 대해서만** 법인세 납부의무를 부담 (법인세법 제3조 제3항)
- 고유목적사업(종교활동, 포교, 복지사업 등)에서 발생하는 소득에는 법인세 비과세

### 1.2 고유목적사업 vs 수익사업 구분 기준

**고유목적사업** (비과세):
- 종교 의식, 포교, 법회, 수행
- 사회복지사업, 교육사업 (무상 또는 실비)
- 후원금, 보시, 신도 회비 수입
- 고유목적사업에 무상으로 제공되는 용역

**수익사업** (과세):
- 법인세법 제4조 제3항, 시행령 제2조에서 열거
- 한국표준산업분류에 따른 사업 중 수익이 발생하는 사업
- 대가성이 있고, 실비 이상의 이익을 창출하며, 지속적/반복적으로 수행하는 사업

**구분 판단 기준** (대법원 판례):
- 사업의 **계속성/반복성** 여부
- **대가성** 존재 여부 (실비 초과 수익 발생)
- 사업의 **독립적 영리 목적** 존재 여부
- "비영리 내국법인은 수익사업 또는 수입에서 생긴 소득에 대해서만 법인세를 납부할 의무가 있다" (대법원)

### 1.3 봉은문화회관 사업별 수익사업 해당 여부

| 사업 유형 | 수익사업 해당 여부 | 근거 |
|-----------|-------------------|------|
| **전시 입장료 수입** | **과세 가능성 높음** | 유상 입장료 → 도매소매업/서비스업 해당 |
| **상품 판매 (기념품, 전통문화 상품)** | **과세** | 도매소매업에 해당, 지속적/대가성 있는 판매 |
| **F&B (카페, 레스토랑)** | **과세** | 음식점업에 해당, 명확한 수익사업 |
| **체험 프로그램 (유상)** | **과세 가능성 높음** | 교육서비스업 해당, 실비 초과 수강료 부과 시 |
| **체험 프로그램 (무상/실비)** | **비과세 가능** | 포교/교화 목적의 종교행위로 인정될 경우 |

> **핵심 판단**: 봉은문화회관에서 계획하는 전시/판매/F&B/유상 체험프로그램은 대부분 **법인세법상 수익사업에 해당**하여 법인세 과세 대상이 됨

### 1.4 법인세 세율 (2025년 현행 / 2026년 이후)

비영리법인의 수익사업 소득에는 영리법인과 동일한 법인세율이 적용됩니다.

| 과세표준 | 2025년 세율 | 2026년 이후 세율 |
|---------|------------|-----------------|
| 2억 원 이하 | 9% | 10% |
| 2억 초과 ~ 200억 이하 | 19% | 20% |
| 200억 초과 ~ 3,000억 이하 | 21% | 22% |
| 3,000억 초과 | 24% | 25% |

> 봉은문화회관 수익사업 규모를 고려하면 **9~19% (2025년 기준)** 구간 적용이 예상됨

### 1.5 고유목적사업준비금 (법인세법 제29조) -- 핵심 절세 수단

수익사업에서 발생한 소득을 고유목적사업(종교활동)에 사용하기 위해 적립하는 준비금으로, **손금에 산입**하여 법인세 부담을 경감할 수 있습니다.

#### 손금산입 한도율

| 소득 유형 | 한도율 | 비고 |
|-----------|--------|------|
| **이자소득** (비영업대금 이익 제외) | **100%** | 전액 손금산입 가능 |
| **배당소득** (상속증여세 과세 제외) | **100%** | 전액 손금산입 가능 |
| **기타 수익사업 소득** | **50%** | 종교법인 일반 적용 |

> **종교법인(봉은사)은 수익사업 소득의 50%까지 고유목적사업준비금으로 손금산입 가능**
> 즉, 수익사업 소득의 50%에 대해서만 법인세를 납부하게 됨

#### 참고: 더 유리한 한도율 적용 가능 법인

| 법인 유형 | 한도율 |
|-----------|--------|
| 사립학교법인, 산학협력단 | **100%** |
| 사회복지법인 | **100%** |
| 박물관/미술관 | **100%** |
| 공익법인 (장학금 50% 이상 지출) | **80%** |
| **종교법인 (봉은사)** | **50%** |

#### 사용 기한 및 환입

- **사용 기한**: 손금산입한 사업연도 종료일 이후 **5년 이내** 고유목적사업에 사용
- **사용 범위**: 고유목적사업 고정자산 취득, 인건비, 기금적립, 지정기부금 등
- **미사용 시**: 5년 이내 미사용 잔액은 익금산입 + **이자상당액 추가 납부**
  - 이자상당액 = 법인세액 차액 x 기간(일수) x 3/10,000
- **결산상 손금 계상 필수**: 세무조정만으로는 불가, 반드시 결산서에 비용 계상

### 1.6 수익사업 개시 시 신고 의무

1. **사업자등록**: 수익사업 개시일로부터 **20일 이내** 관할 세무서에 사업자등록 신청
2. **수익사업 개시신고**: 사업 개시일로부터 **2개월 이내** 수익사업 개시신고서 + 대차대조표를 납세지 관할 세무서장에게 제출
3. **구분회계**: 고유목적사업과 수익사업의 **분리 결산** 필수 → 재합산하여 법인세 산출
4. **법인세 신고**: 사업연도 종료일로부터 3개월 이내 법인세 신고/납부

## 2. 부가가치세

### 2.1 종교법인의 부가가치세 면세 범위

**근거 법령**: 부가가치세법 제26조 제1항 제18호, 시행령 제45조

> "종교, 자선, 학술, 구호 그 밖의 공익을 목적으로 하는 단체가 공급하는 재화 또는 용역으로서 대통령령으로 정하는 것"은 부가가치세 면세

**면세 요건** (부가가치세법 시행령 제45조):
- 주무관청의 허가/인가를 받거나 등록된 단체일 것
- 단체의 **고유 사업목적**을 위하여 공급할 것
- **일시적으로 공급**하거나 **실비 또는 무상**으로 공급할 것

### 2.2 사업 유형별 부가가치세 과세/면제 판단

| 사업 유형 | 부가가치세 | 판단 근거 |
|-----------|-----------|-----------|
| **전시 입장료 (종교/문화교육 목적)** | **면세 가능** | 고유목적(포교/교화) + 실비 수준이면 면세 |
| **전시 입장료 (상업적 전시)** | **과세** | 영리 목적의 지속적 유상 공급 |
| **상품 판매 (전통문화 상품, 기념품)** | **과세** | 재화의 지속적/반복적 유상 공급 → 도소매업 |
| **F&B (카페, 레스토랑)** | **과세** | 음식점업으로 부가가치세 과세 대상 |
| **체험 프로그램 (유상 수강료)** | **과세 가능성 높음** | 실비 초과 수강료 → 교육서비스업 과세 |
| **체험 프로그램 (무상/실비)** | **면세 가능** | 종교 교화 목적 + 실비/무상이면 면세 |
| **공간 임대료** | **과세** | 부동산 임대업 → 부가가치세 과세 |

### 2.3 사업자등록 필요 여부

- **수익사업(과세사업) 영위 시**: 반드시 **사업자등록** 필요
- 과세사업과 면세사업을 동시에 운영하면 **겸영사업자**로 등록
- 면세사업만 있는 경우에도 사업장현황신고 의무 (매년 2월 10일까지)
- 미신고 시 가산세 부과

### 2.4 실무적 유의사항

- 종교단체가 상품을 판매하는 것이 **일시적/실비 공급**인지, **지속적/영리 목적 공급**인지에 따라 면세 여부가 달라짐
- 봉은문화회관의 경우 전시/판매/F&B를 **상시 운영**하는 구조이므로, 대부분 **부가가치세 과세** 대상
- 과세사업자로 등록하면 매입세액 공제 가능 (시설투자, 인테리어, 원재료 등)

## 3. 부동산 관련 세금

### 3.1 재산세 면제 및 과세전환

**근거 법령**: 지방세특례제한법 제22조

**면제 원칙**:
> "종교 및 제사를 목적으로 하는 단체가 과세기준일 현재 해당 사업에 **직접 사용하는 부동산**은 재산세를 면제"

**과세전환 상황**:

| 사용 용도 | 재산세 | 비고 |
|-----------|--------|------|
| 종교 의식, 법당, 수행 공간 | **면제** | 고유목적사업 직접 사용 |
| **전시/판매/F&B 등 수익사업** | **과세** | 수익사업 사용 = 직접 사용 불인정 |
| **외부업체에 임대** | **과세** | 임대 = 직접 사용 아님 |
| 혼합 사용 (일부 종교 + 일부 수익) | **면적 비율로 안분** | 수익사업 사용 면적분 과세 |

> **봉은문화회관을 수익사업에 사용할 경우, 해당 부분의 재산세 면제 혜택이 상실됨**

### 3.2 취득세

**근거 법령**: 지방세특례제한법 제50조

- 종교단체가 종교사업 목적으로 취득한 부동산: **취득세 면제**
- **추징 사유**:
  - 취득일로부터 **3년** 이내 해당 용도로 미사용 시
  - **2년 미만** 사용 후 매각/증여/용도변경 시
  - 수익사업 등 **다른 용도로 사용** 시

> 봉은문화회관이 기존에 취득세를 면제받았다면, 수익사업 용도로 전환 시 **면제받은 취득세 추징 가능성** 검토 필요
> 다만 과거 웨딩홀 임대 운영 이력이 있으므로, 이미 과세 전환 처리되었을 가능성도 있음

### 3.3 종합부동산세

- 종교법인 소유 부동산 중 **수익사업에 사용되는 부분**: 종합부동산세 과세 대상 가능
- 법인 소유 주택이 아닌 상업용 건물은 별도합산과세 대상
- 법인의 종합부동산세: 공제금액 적용 제한, 높은 세율 적용

### 3.4 임대소득 과세

- 봉은사가 봉은문화회관을 외부업체에 임대하는 경우:
  - **법인세**: 부동산 임대소득으로 수익사업 과세 (법인세법 제4조)
  - **부가가치세**: 부동산 임대용역 = 부가가치세 과세 (10%)
  - **재산세**: 임대 부분 = 면제 불가, 과세 전환

## 4. 운영주체별 법적 구조 비교

### 4.1 A안: 봉은사 직영 (종교법인이 직접 수익사업)

**구조**: 봉은사가 수익사업에 대해 사업자등록 → 직접 운영

| 항목 | 내용 |
|------|------|
| **법인세** | 수익사업 소득에 법인세 과세 (9~19%) |
| **고유목적사업준비금** | 수익사업 소득의 **50%** 손금산입 가능 → 실효세율 절반 수준 |
| **부가가치세** | 과세사업에 대해 부가세 납부 의무 |
| **재산세** | 수익사업 사용 부분 재산세 과세 전환 |
| **장점** | 수익의 직접 귀속, 고유목적사업준비금 활용 가능, 의사결정 신속 |
| **단점** | 종교법인의 세무/회계 부담 증가, 구분회계 필수, 수익사업 리스크가 법인에 직접 귀속 |
| **인허가** | 봉은사 명의로 사업자등록, 식품위생법 영업신고/허가 등 |
| **리스크** | 수익사업 실패 시 종교법인 재산에 영향, 종단 내부 규율 확인 필요 |

### 4.2 B안: 외부업체에 위탁 운영

**구조**: 봉은사가 공간/시설 소유, 외부 전문업체가 운영 위탁(management contract)

| 항목 | 내용 |
|------|------|
| **봉은사 과세** | 위탁수수료 수입 → 수익사업 소득으로 법인세 과세 |
| **운영업체 과세** | 운영업체가 별도 사업자로 매출/부가세/법인세(또는 소득세) 납부 |
| **재산세** | 위탁운영은 직접사용이 아닐 수 있어 **과세 전환 가능** |
| **장점** | 전문성 활용, 운영 리스크 분산, 봉은사 세무 부담 경감 |
| **단점** | 수익 분배 구조 복잡, 브랜드 관리 필요, 위탁계약 조건에 따른 리스크 |
| **인허가** | 운영업체가 영업허가/신고 담당, 봉은사는 공간 제공 |
| **리스크** | 위탁계약 구조에 따라 '실질적 임대'로 판단될 가능성 |

### 4.3 C안: 별도 법인 설립

**구조**: 봉은사(또는 조계종)가 출자하여 별도 영리/비영리법인 설립 → 해당 법인이 운영

| 항목 | 내용 |
|------|------|
| **별도법인 과세** | 독립 법인으로 법인세/부가세 납부 |
| **봉은사 과세** | 출자배당, 임대료 등 수입에 대해 과세 |
| **재산세** | 봉은사가 임대 → 과세 전환 / 별도법인에 출자(양도) → 취득세 발생 |
| **장점** | 수익사업 리스크 완전 분리, 전문 경영 가능, 종교법인 이미지 보호 |
| **단점** | 법인 설립/운영 비용, 이중과세 가능성(법인세+배당소득세), 고유목적사업준비금 활용 불가 |
| **법인 유형 옵션** | |
| - 주식회사 | 가장 일반적, 영리 목적, 이중과세 |
| - 사회적기업 | 고용노동부 인증 시 세제 혜택, 사회적 목적 부합 시 |
| - 공익법인 | 설립 요건 까다로움, 수익사업 가능하나 제약 있음 |
| **리스크** | 종단 규율/내부 승인 절차 복잡, 부동산 이전 시 취득세/양도세 발생 |

### 4.4 D안: 공간 임대 (봉은사 = 임대인, 외부업체 = 임차인)

**구조**: 봉은사가 봉은문화회관을 외부업체에 임대, 임차인이 독립적으로 사업 운영

| 항목 | 내용 |
|------|------|
| **봉은사 과세** | 임대소득 = 수익사업 → 법인세 과세 |
| **부가가치세** | 임대료에 부가가치세 10% 부과 (봉은사가 납부) |
| **재산세** | 임대 = 직접사용 아님 → **재산세 과세 전환** |
| **취득세** | 기존 면제분 **추징 가능** |
| **장점** | 가장 단순한 구조, 안정적 임대수입, 운영 리스크 없음 |
| **단점** | 수익 한정(임대료), 사업 방향 통제력 제한, 재산세/취득세 면제 혜택 상실 |
| **인허가** | 임차인이 모든 영업허가/신고 담당 |
| **리스크** | 임차인의 사업 부진/이미지 리스크, 전통사찰보존법상 부동산 대여 허가 필요 |

### 4.5 운영주체별 비교 요약

| 비교 항목 | A. 직영 | B. 위탁 | C. 별도법인 | D. 임대 |
|-----------|---------|---------|------------|---------|
| **세금 부담** | 중 (준비금 활용) | 중~저 | 높음 (이중과세) | 낮음~중 |
| **고유목적사업준비금** | O (50%) | O (50%) | X | O (50%) |
| **운영 리스크** | 높음 | 중 | 낮음 | 낮음 |
| **수익 극대화** | 높음 | 중 | 중 | 낮음 |
| **관리 부담** | 높음 | 중 | 중 | 낮음 |
| **사업 통제력** | 완전 | 높음 | 중 | 낮음 |
| **재산세 영향** | 과세 전환 | 과세 전환 | 과세 전환 | 과세 전환 |
| **전통사찰보존법 허가** | 불요/간소 | 계약 구조별 판단 | 출자/양도 시 허가 | **허가 필수** |

## 5. 전통사찰 관련 특별 법령

### 5.1 전통사찰의 보존 및 지원에 관한 법률

봉은사는 **전통사찰**로 등록되어 있으므로 이 법률의 적용을 받습니다.

#### 제9조 (허가 사항) -- 경내지 행위 제한

전통사찰의 주지가 다음 행위를 하려면 **시/도지사 허가** 필요:
- 동산/부동산의 **대여 또는 담보 제공**
- 경내지에서 건조물 **신축/증축/개축/철거**

**부동산 양도 시**: 소속 대표단체(조계종) 대표자 승인서 + **문화체육관광부장관 허가** 필요

**허가 기준** (다음 중 하나 이상 충족):
- 전통사찰의 이용 가치 향상
- 주민 편의 제공 등 공공용 목적
- 전통문화 보급/활용 기여
- 공익사업

#### 제12조 (사업) -- 수익사업 범위 제한

> "전통사찰의 주지는 **불교 목적의 범위**에서 포교사업, 사회복지사업, 그 밖의 공익사업과 그 **부대사업**을 할 수 있다"

**시사점**:
- 봉은문화회관의 수익사업은 **"불교 목적 범위 내의 부대사업"**으로 구성해야 함
- 전통문화 콘텐츠 전시/판매, 불교 체험 프로그램 등은 부합 가능
- 순수 상업적 F&B는 "부대사업"으로의 포섭 여부 검토 필요

#### 봉은문화회관 관련 유의사항

- 건물 **용도변경** 시: 시/도지사 허가 필요 (경내지 내 건조물 변경)
- 공간 **임대** 시: 부동산 대여 허가 필요
- 시설 **리모델링** 시: 건축법상 허가/신고 + 전통사찰보존법 허가 이중 적용

### 5.2 문화유산의 보존 및 활용에 관한 법률 (구 문화재보호법)

- 봉은사 경내에 **지정문화재**가 있는 경우, 현상변경허가 필요
- 문화재 보호구역 내 건축행위 제한
- 봉은문화회관이 문화재 보호구역 내인지 확인 필요
- 현상변경 허가를 받은 경우 전통사찰보존법상 허가를 받은 것으로 간주 (법 연계)

## 6. 기타 사업 관련 법령

### 6.1 식품위생법 -- F&B 운영 시

**영업 유형별 인허가**:

| 업종 | 인허가 유형 | 관할 |
|------|-----------|------|
| 카페 (음료/경양식) | **휴게음식점 영업신고** | 관할 구청장 |
| 레스토랑 (일반 음식) | **일반음식점 영업신고** | 관할 구청장 |
| 주류 제공 | **일반음식점 + 주류판매면허** | 구청 + 세무서 |
| 식품 제조/판매 | **식품제조가공업 등록** | 식약처/구청 |

**필수 요건**:
- 조리장, 객석, 급수시설, 화장실 등 **시설기준** 충족
- 식품위생교육 이수 (신규 영업 전)
- 건축물 용도가 **식품접객업** 가능 용도인지 확인
- 미신고 영업 시: **3년 이하 징역 또는 3,000만 원 이하 벌금** (식품위생법 제97조)

**봉은문화회관 유의사항**:
- 기존 웨딩홀 용도에서 F&B 용도로의 **건축물 용도변경** 필요 가능성
- 종교시설 내 영업허가 가능 여부는 건축물 용도와 관련
- 배수, 환기, 방화 등 건축법 기준 충족 필요

### 6.2 관광진흥법 -- 체험 프로그램 운영 시

**관광사업 등록이 필요한 경우**:
- 관광객이용시설업 (전문 관광시설)
- 한옥체험업, 관광편의시설업 등

**등록이 불필요한 경우**:
- 종교법인이 자체적으로 운영하는 **교육/문화 체험 프로그램**
- 일시적/비영리적 성격의 프로그램
- 사찰의 **템플스테이**: 한국불교문화사업단 통해 운영, 별도 관광사업 등록 불요

**봉은문화회관 시사점**:
- 단순 문화체험/교육 프로그램: 관광사업 등록 **불요**
- 숙박을 포함하는 관광 체험 패키지: **관광사업 등록 검토** 필요
- 외국인 대상 관광 상품화 시: 관광편의시설업 등록 검토

### 6.3 소비자보호/공정거래 -- 상품 판매 시

**적용 법령**:
- **전자상거래법**: 온라인 판매 시 적용 (통신판매업 신고, 청약철회, 표시광고)
- **소비자기본법**: 소비자 권익 보호, 환불 규정
- **표시/광고법**: 허위/과장 광고 금지
- **제조물책임법**: 판매 상품의 결함으로 인한 손해배상 책임

**실무 체크리스트**:
- 상품 가격 표시, 원산지 표시 의무
- 환불/교환 정책 수립 및 고지
- 온라인 판매 시 통신판매업 신고 (관할 구청)
- PG사 결제 시스템 도입 시 관련 규정 준수

### 6.4 건축법/소방법

- 건축물 용도변경: **건축법 제19조**에 따른 허가/신고
- 다중이용시설 안전관리: **다중이용시설 안전관리에 관한 특별법**
- 소방시설 설치/관리: 불특정 다수 출입 시설 → 소방안전관리자 선임

## 7. 종합 권고사항

### 7.1 운영구조 선택 시 고려 요소

1. **수익 규모 전망**: 소규모 → A안(직영), 대규모 → B안(위탁) 또는 C안(별도법인)
2. **전통사찰보존법 허가 용이성**: A안(직영) > B안(위탁) > D안(임대)
3. **세금 효율**: A안(고유목적사업준비금 50%) ≥ B안 > D안 > C안
4. **리스크 관리**: D안(임대) > C안(별도법인) > B안(위탁) > A안(직영)

### 7.2 실무 추진 순서 (권장)

1. **조계종 내부 승인** 절차 확인 및 사전 협의
2. **전통사찰보존법** 행위허가 사전 상담 (서울시)
3. **건축물 용도변경** 가능 여부 확인 (강남구청)
4. 세무사/회계사 자문: **최적 운영구조 설계** 및 세무 시뮬레이션
5. **사업자등록** 및 영업허가/신고
6. 구분회계 시스템 구축

### 7.3 추가 확인 필요 사항

- [ ] 봉은문화회관 건물의 현재 **건축물대장상 용도** 확인
- [ ] 봉은사 경내지 범위 및 **문화재보호구역** 해당 여부 확인
- [ ] 과거 웨딩홀 운영 시 **취득세/재산세 과세 이력** 확인
- [ ] 조계종 종헌/종법상 **수익사업 관련 내부 규정** 확인
- [ ] 서울시 문화재과 **전통사찰보존법 사전 상담**
- [ ] 강남구 세무과 **재산세 과세전환 범위** 사전 확인
