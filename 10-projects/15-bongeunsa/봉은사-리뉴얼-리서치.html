2-1. 추진 배경

  • 1971년 도시공원 지정 → 44년간 건축 규제 (신축/증축 불가)

  • 주지 원명 스님 부임 후 1년 8개월간 문화재위원 설득
"경내 설치된 가건물을 볼 때마다 부끄럽고 죄를 짓는 기분이었다. 40여 년간 이루지 못한 숙원사업이다." — 원학 스님 (전 주지, 중창불사 최초 발의자)

핵심 발견: 문화회관(구 웨딩홀)의 위치

마스터플랜 2단계에서 "문화회관(결혼식장) 반환"이 명시되어 있습니다. 원래 계획은 "신도 기도공간으로 전환"이었으나, 이것이 우리 프로젝트의 대상 건물입니다.

즉, 봉은사 공식 계획에는 이 건물을 "기도공간"으로 활용하려 했으나, 현재 "복합문화공간"으로의 전환을 검토 중인 것으로 보입니다. 이는 원래 계획보다 더 야심찬 방향이며, 불교 브랜딩 + 수익화까지 고려한다는 점에서 마스터플랜의 업그레이드 버전이라 할 수 있습니다.

3. 주지 원명 스님의 비전과 인터뷰

"수행자는 이 땅 어디에 서든 수행자로 남아야 한다." — 원명 스님 (봉은사 주지)

취임 후 첫 번째 결정: 산문(사찰 정문)을 밤 12시까지 개방하고 불을 밝혀 "도심 속 쉼터"로 전환.

5-1. 건물의 전략적 가치

  • 규모: 1,500평 = 명동1898의 약 3~4배. 층별로 다른 컨셉 배치 가능
  • 구조: 웨딩홀 특성상 대형 홀 + 높은 천고 + 파티션 유연성 보유 추정
  • 입지: 봉은사역 + 코엑스 도보권. 강남 핵심 상권 인접
  • 부지비용: 봉은사 소유 → 0원 (명동1898과 동일 구조)
  • 독점성: 강남 유일의 불교문화시설이 될 수 있는 위치

6. 잘한 점 — 복합문화공간 관점

1. 콘텐츠 인프라가 이미 탄탄

템플스테이, 사찰음식, 참선, 다도, 인경, 불교대학 — 다른 곳에서는 10년 걸릴 콘텐츠 라인업이 이미 검증되어 있음. 이것을 복합문화공간에 "이식"하면 되는 구조.

2. 사찰음식 → K-Food 브랜딩 선제 투자

연구소 설립, 밀키트 개발, 글로벌 수출 구상까지 — F&B를 단순 체험이 아닌 "상품"으로 발전시키려는 의지가 명확함. 복합문화공간의 시그니처 F&B로 즉시 활용 가능.

3. 외국인 유입 이미 검증

일일 300명+ 외국인 방문 (템플스테이 50~120명). 코엑스 인접이라는 입지와 결합하면 자연적 관광 유입 동선이 형성되어 있음.

4. 주지 스님의 개방적 리더십

산문 자정 개방, 불교대학원 설립, 사찰음식 상품화 — 보수적인 사찰 문화에서 "세상과 소통"을 실행에 옮긴 리더. 복합문화공간 추진에 필요한 리더십 기반이 있음.

5. 문화유산의 국가 지정 추진력

수륙재·생전예수재를 국가무형문화재/서울시무형문화재로 끌어올린 실적 → "봉은사 = 살아있는 무형문화재의 보고"라는 스토리텔링 자산.

7. 결핍 — 복합문화공간 관점

1. 공간 브랜딩 전무

명동1898은 "1898"이라는 숫자 하나로 장소 역사를 압축. 봉은사에는 이런 "사찰을 넘어서는 공간 브랜드"가 없음. 시설 명칭이 "전통문화체험관 1관, 2관"처럼 기능적임.

→ 필요: 복합문화공간의 고유 브랜드 네이밍 + VI(Visual Identity) 설계

2. 수익화 모델 부재

현재 수익원은 사실상 불전함 + 템플스테이 참가비뿐. 기념품숍, 에디터리얼 커머스, F&B 매장, 임대 수익 등 상업적 수익 구조가 거의 설계되어 있지 않음.

→ 필요: 명동1898의 임대+직영+주차 모델을 참고하되, 더 적극적 수익화 설계

3. 디지털 마케팅 / SNS 전략 부재

공식 웹사이트가 2000년대 UI 수준. 인스타그램·유튜브 등 MZ세대 접점이 거의 없음. 코엑스 관광객이 봉은사까지 들어오는 이유가 "우연히 보여서"에 가까움.

→ 필요: 인스타그래머블 공간 설계 + 디지털 채널 전략

4. MZ세대 / 비신자 접점 약함

템플스테이는 "의도적으로 찾아오는 사람"만 경험. 카페·서점·갤러리 같은 "가벼운 접점"이 없어서 비신자가 우연히 들르고 불교 문화에 자연스럽게 노출되는 경로가 없음.

→ 필요: Low-barrier 진입점 (카페, 서점, 명상 라운지 등) 설계

5. 서비스 기획 / UX 설계 부재

프로그램은 있지만 "고객 여정(Customer Journey)"이 설계되어 있지 않음. 방문객이 어떤 순서로 어떤 경험을 하고, 무엇을 사고, 어떤 기억을 안고 가는지에 대한 서비스 디자인이 없음.

→ 필요: 서비스 블루프린트 + 고객 여정 맵 설계

6. 사찰음식 상품화 실행 속도

2021년 연구소 개소했으나, 밀키트·홈쇼핑·해외수출 등 구체적 상용화 성과가 아직 공개되지 않음. 구상은 좋으나 실행이 따라오고 있는지 확인 필요.

→ 필요: 사찰음식 F&B를 복합문화공간의 앵커 테넌트로 구체화

9. 기획자 관점 종합 의견

핵심 판단

봉은사 리뉴얼 계획(중창불사)은 "건물 인프라"에 집중되어 있고, "서비스 기획"과 "브랜딩"은 사실상 부재합니다.

  • 있는 것: 마스터플랜(3단계), 콘텐츠 자산(템플스테이·사찰음식·참선), 리더십(원명 스님), 부지(구 웨딩홀 5층), 외국인 유입 동선
  • 없는 것: 공간 브랜드, 수익 모델, 디지털 전략, 서비스 디자인, MZ 접점, 기념품/리테일, 입점 테넌트 큐레이션

이것은 우리 프로젝트의 기회입니다. "없는 것"을 채우는 것이 곧 마스터플랜의 빈 칸을 완성하는 일이 됩니다.
