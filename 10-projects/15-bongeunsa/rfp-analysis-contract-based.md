> 분석일: 2026-06-09 | 분석 대상: 계약서 원문(12p) + 추가합의서(1p)
> 순수 문서 기반 분석 — 미팅 메모, 외부 리서치 배제

## 1. 계약 구조 총괄

### 1-1. 3중 문서 체계

```
[추가합의서] ← 최우선 적용 (제6조)
    ↑ 상충 시 우선
[계약서 본문] (제1조~제20조)
    ↑ 구체화
[과업지시서] (붙임2, I~V장)
```

**효력 순위**: 추가합의서 > 계약서 > 과업지시서 (추가합의서 제6조 명시)

| 부제 | 전통문화콘텐츠·상품 기반 상설전시 및 판매 운영 타당성 검토와 사업 계획 | 표지 |

## 2. 금전 흐름 분석

### 2-1. 지급 구조

| 구분 | 비율 | 금액 | 조건 | 근거 |
|------|------|------|------|------|
| 착수금 | 50% | 2,500만원 | 계약 체결 후 1주일 이내 | 제5조 |
| 잔금 | 50% | 2,500만원 | 최종 결과물 제출 → 갑 검수 하자 없음 승인 후 14일 이내 | 제5조 |

### 2-2. 리스크 노출 금액

| 리스크 | 원 계약 | 추가합의서 적용 후 | 변화 |
|--------|---------|---------------|------|
| 위반 시 배상 | 계약금 2배 = **1억원** (제13조 2항) | 총 계약금 = **5,000만원** 한도 (추가합의서 제4조) | **50% 감소** |
| 지체상금 | 일 0.25% = 일 12.5만원 | 변동 없음 (미헷지) | - |
| 90일 최대 지체상금 | 22.5% = 1,125만원 | 변동 없음 | - |
| 배상 범위 | 전체 손해 | 직접 손해만 (특별손해·일실이익 면책) | **대폭 축소** |

### 2-3. 잔금 확보 메커니즘

```
결과물 제출 → 10일 경과 → 서면 불합격 미통지 시 → 자동 승인 (추가합의서 제3조)
                ↓ 서면 불합격 통지 시
          수정 요구 (과업지시서 범위 내만 가능) → 재제출 → 최종 검수 완료
```

**핵심**: 추가합의서 제3조가 잔금 미지급 리스크의 최대 방어선. 주관적 불만족으로는 승인 거부 불가.

## 3. 산출물 정의 (추가합의서 기준)

### 3-1. 2단계 산출물 구조

추가합의서 제1조가 과업지시서의 9개 챕터를 2단계로 재구조화:

#### 중간인도물 (전략 방향성 보고서)

| 항목 | 과업지시서 대응 챕터 |
|------|-----------------|
| 시장 및 타겟 분석, 입지 여건 분석 결과 | Ch.2(현황조사), Ch.4(타깃 이용자) |
| 국내외 유사 사례 분석을 통한 시사점 도출 | Ch.3(유사사례 조사·분석) |
| 공간 브랜드 핵심 콘셉트 및 전략적 방향성 제안 | Ch.4(콘셉트 및 방향 설정) |

#### 최종결과물 (실행 계획 및 사업성 검토 보고서)

| 항목 | 과업지시서 대응 챕터 |
|------|-----------------|
| 3개 이상 공간 활용 시나리오 + 장단점 분석 | Ch.5(활용 대안 도출) |
| 운영 모델(직영/위탁/협업) + 단계별 추진 로드맵 | Ch.6(사업화·운영모델), Ch.8(실행전략) |
| 5개년 예상 수지분석 및 경제성 검토 | Ch.7(사업타당성·수익성 분석) |
| 온·오프라인 마케팅 및 홍보 계획 | Ch.6(홍보계획 수립) |

#### 최종 납품물 (과업지시서 V장)

- 최종보고서 **5부** + 성과품 파일 USB **5개**
- 요약보고서 **5부** + 성과품 파일 USB **5개**
- 발표자료 (과업지시서 III-9)

### 3-2. 콘텐츠 작성 권한

| 영역 | 원 계약/과업지시서 | 추가합의서 적용 후 |
|------|---------------|--------------|
| 콘텐츠 원고 작성 | 전문가 자문 + **발주처 승인** 후 작성 | "을"이 전문적으로 구성, 마케팅적 실효성 최우선 |
| 전문가 자문 | 의무적 수렴 | 필요 시 상호 협의, 추가 비용은 갑 부담 |
| 학술적 검증 | 관련 분야 전문가 자문 필수 | 전략 컨설팅 특성에 맞게 조정 |

**시사점**: 추가합의서가 콘텐츠 작성의 자율성을 확보. 학술적 자문에 종속되지 않고, 마케팅·전략 관점에서 주도적 구성 가능.

## 6. 과업 수행 지침 요약 (과업지시서 IV장)

### 6-1. 일반사항

- 연구내용·방법은 연구진이 기획, 지침은 발주기관과 협의 후 최종 결정
- **전문가·교수 등의 의견 수렴** 필요 (단, 추가합의서로 의무성 완화)
- 변경/불분명 사항은 발주처와 협의 결정
- 과업지시서 미명기 사항은 과업 목적 범위 내 상호 협의 시행

### 6-2. 과업수행 규칙

- **공신력 있는 최신 자료** 활용, 출처·근거 명시
- 콘텐츠는 관람 대상의 이해력·습득력 고려, 전시기획의도에 부합
- 콘텐츠 원고는 관람객 수준에 맞게 작성 (단, 추가합의서로 을의 자율성 확보)
- 선정 콘텐츠는 공간 설치·운영과의 연계성·통일성·조화 고려

### 6-3. 보안 의무

- 과업 착수 시 대표자 및 연구진 **인적사항 + 보안각서** 제출
- **보안관리 책임자** 지명
- 별도 보관함 비치, 관리책임자 지정
- 모든 자료는 과업수행 목적으로만 사용, 발주처 미승인 시 제공·대여 불가
- 특허권·저작권 분쟁 시 과업수행기관 전적 책임

### 6-4. 기간연장

- 계약만료일 **15일 전**까지 연장신청서 제출
- 연장 비용은 **과업수행기관 부담** 원칙

## 12. 핵심 문제 정의 (계약서 기반)

### 12-1. 과업지시서가 정의한 문제

> "봉은사 내 복합문화공간에 대한 활용 여건을 조사·분석하고, 전시 및 팝업 스토어 중심의 콘텐츠 기획, 전시물 및 운영요소 확보 방안, 홍보계획, 예상 수지분석, 사업타당성 및 수익성 검토를 통해 실행 가능한 최적의 공간 활용방안과 콘셉트를 제안"

### 12-2. 문제 구조화

| 질문 | 답변 (계약서 기반) |
|------|-------------|
| **무엇이** 부족한가? | 8,868㎡ 유휴공간의 활용 전략과 수익 모델 |
| **왜** 필요한가? | "전통문화를 국민들이 향유할 수 있는 열린 장" + "봉은사 대외적 위상강화" + "전통문화산업 진흥의 전진기지" |
| **어떻게** 접근해야 하는가? | 사례분석 → 콘셉트 설정 → 대안 도출(3안+) → 경제성 비교 → 최적안 선정 → 실행 로드맵 |
| **얼마나** 구체적이어야 하는가? | 5개년 사업계획서 + 5개년 경제성 분석 보고서 수준 |
| **누구에게** 설득해야 하는가? | 봉은사 내부 의사결정자 (과업지시서에 명시 안 됨, 계약서 대표: 진우) |

### 12-3. 과업지시서가 명시한 핵심 원칙

1. **"나열식 전시를 지양"** — 단순 진열 금지, 체험형·스토리텔링형 접근
2. **"전통문화와 현대적 상품의 전시 및 판매"** — 전통 + 현대 융합이 핵심 콘셉트
3. **"차별화된 콘텐츠를 가진 전시관"** — 봉은문화회관만의 독자적 콘텐츠
4. **"전통문화의 발전에 이바지"** — 상업 목적만이 아닌 문화적 사명
5. **"봉은사의 대외적 위상강화"** — 브랜드 가치 제고
