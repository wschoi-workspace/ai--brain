## 1. 사업 가치 재정의 (JTBD)

### 1-1. 표면적 사업 목적

🔍 "K-패션 디자이너 브랜드 16개의 도쿄(파르코)·파리(갤러리 라파예트) 글로벌 팝업스토어 운영 및 홍보·마케팅 위탁"

### 1-2. JTBD 3차원 재정의

| 차원 | 표면 | 재정의 |
|------|------|--------|
| **Functional Job** | 16개 브랜드 팝업 운영 + 홍보 | **K-패션 신생 브랜드가 글로벌 럭셔리 유통 채널에서 "팔리는 브랜드"로 인정받는 첫 번째 경험을 만들어주는 것** |
| **Emotional Job** | 성과 보고서에 넣을 실적 | **"레스티어(편집샵)에서 파르코·라파예트(메이저 백화점)로 점프시켰다"는 스케일업 서사** — 콘진원 IP사업팀의 존재 가치를 증명하는 사업 |
| **Social Job** | 언론 노출, 바이럴 | **"한국 패션이 갤러리 라파예트에 입성했다"는 한 줄 — 한불수교 140주년 맥락에서 국가 브랜딩 수준의 상징성** |

### 1-3. 재정의된 가치 제안

> **이 사업은 "팝업 운영"이 아니라, K-패션 디자이너 브랜드의 "글로벌 First Impression"을 설계하는 사업이다.**
> 매장에서 물건을 파는 것이 아니라, 파르코와 라파예트라는 무대 위에서 브랜드의 글로벌 가치를 증명하고, 그 증명이 미디어를 통해 확산되어 "팔릴 수밖에 없는 구조"를 만드는 것이다.

💡 **So What**: 이 재정의가 제안서 전체의 톤을 결정한다.
- ❌ "운영을 잘 하겠습니다" → ✅ "브랜드의 글로벌 가치를 증명하겠습니다"
- ❌ "홍보를 많이 하겠습니다" → ✅ "팔릴 수밖에 없는 구조를 설계하겠습니다"

## 2. 타겟 가치 사슬 (Value Chain)

### 2-1. 가치 전달 경로 (Logic Model + CDJ)

```
[투입] 18.2억 / 4사 컨소시엄 / 12개월
  ↓
[활동] 도쿄 11회 릴레이 팝업 + 파리 3주 메가 팝업 + 온라인 PR
  ↓
[산출 Output]
  ├── 도쿄: 11개 브랜드 × 2주 릴레이 팝업 at 파르코 시부야
  ├── 파리: 5개 브랜드 × 3주 팝업 at 갤러리 라파예트 오스만
  ├── PR: 2거점 PR 이벤트 + 인플루언서 시딩 + 앰배서더 캠페인
  └── 콘텐츠: SNS/미디어 에셋 제작·배포
  ↓
[성과 Outcome]
  ├── 브랜드 인지도: 도쿄·파리 패션 커뮤니티 내 K-패션 인지율 상승
  ├── 구매 전환: 팝업 매출 + 후속 바이어 미팅/오더
  ├── 미디어 확산: 글로벌 패션 미디어 노출 + SNS 바이럴
  └── 유통 관계: 파르코·라파예트 바이어와의 정규 입점 논의 기반
  ↓
[영향 Impact]
  ├── K-패션 생태계: 신생 디자이너 브랜드의 글로벌 진출 경로 표준화
  ├── 콘진원 정책: "더셀렉츠" 사업 모델의 제도화 → 연간 정규사업화
  └── 국가 브랜딩: 한불수교 140주년 맥락에서 K-컬처 → K-패션 확장
```

**CDJ 매핑**:

| 단계 | CDJ 관점 | 이 사업에서의 설계 |
|------|---------|------------------|
| **산출** | 인지(Awareness) | 파르코/라파예트라는 "무대" 자체가 인지 채널 — 오프라인 매장이 미디어 |
| **성과** | 고려→참여(Consider→Engage) | 인플루언서 시딩 → 현지 패션피플 매장 방문 → 구매/SNS 공유 |
| **영향** | 옹호(Advocate) | 현지 바이어·미디어·인플루언서가 "K-패션은 진짜다"를 자발적으로 확산 |

### 2-2. 확산 구조 (Growth Flywheel)

```
[팝업 경험]                    [글로벌 미디어 노출]
    ↓                              ↑
[현지 방문객 감동]    ←→    [인플루언서/KOL 콘텐츠]
    ↓                              ↑
[자발적 SNS 공유]      →     [K-패션 인지도 확대]
    ↓                              ↑
[바이어 관심·오더]     →     [정규 입점 논의]
    ↓
[브랜드 글로벌 매출] → [더셀렉츠 다음 시즌 참여 브랜드 증가]
```

**Cialdini 원칙 매핑**:

| Flywheel 단계 | Cialdini 원칙 | 설계 포인트 |
|-------------|-------------|-----------|
| 팝업 경험 → 감동 | **상호성** | 무료 입장 + 체험형 컨텐츠 → "이 브랜드에 보답하고 싶다" (구매/공유) |
| 감동 → SNS 공유 | **사회적 증거** | "파르코 역대 팝업 1위" 같은 수치가 다음 방문객을 끌어옴 |
| KOL 콘텐츠 → 인지 | **권위** | 파리·도쿄 탑티어 인플루언서의 추천 = 글로벌 패션 커뮤니티 검증 |
| 인지 → 바이어 관심 | **희소성** | "2주 한정 릴레이" → 지금 아니면 못 보는 구조 |
| 정규 입점 논의 | **통합/소속감** | "더셀렉츠 패밀리" 브랜드 커뮤니티 형성 → 매 시즌 재참여 |

## 3. 기대효과 구조화 (4-Layer Impact Model)

### Layer 1: 직접 성과 (Direct Output) — 정량

| 구분 | 지표 | 산출 근거 | 목표치 |
|------|------|---------|--------|
| **도쿄 매출** | 팝업 매출 (11개 브랜드 합산) | 🔍 더현대1차 30억/11개 브랜드, 2차 유사 규모 → 파르코 이전 시 80~100% 수준 | **25~30억원** |
| **파리 매출** | 팝업 매출 (5개 브랜드 합산) | 💡 라파예트 객단가 높으나 규모 작음. 파르코 대비 50~60% | **12~18억원** |
| **인플루언서 리치** | KOL/시딩 도달 합산 | 💡 도쿄 10인(평균 50만 팔로워) + 파리 10인(평균 30만) | **800만 reach 이상** |
| **미디어 노출** | 글로벌 패션 미디어 게재 | 💡 도쿄 Vogue JP, WWD JP 등 + 파리 Vogue FR, Le Figaro 등 | **글로벌 미디어 30건 이상** |
| **PR 이벤트** | 오프닝/클로징 이벤트 참석자 | 🔍 파리 오프닝 200인 + 클로징 100인 (제안요청서 명시) | **도쿄·파리 합산 500인 이상** |
| **콘텐츠 생산** | SNS 에셋 + 영상 | 💡 브랜드별 10종 × 16개 = 160종 + 총괄 영상 2편 | **162종 이상** |

> **산출 근거 원칙**: 더현대 글로벌 1차(30억/11개) 실적을 앵커로, 파르코 이전·라파예트 신규의 변수를 반영하여 보수적 추정.

### Layer 2: 행동 변화 (Behavioral Outcome) — 정량+정성

| 구분 | 지표 | 측정 방법 | 기대 수준 |
|------|------|---------|----------|
| **브랜드 인지도** | 참여 브랜드의 도쿄·파리 내 검색량 변화 | Google Trends + SNS 멘션 분석 (RXR) | Before 대비 **300%+ 증가** |
| **구매 전환율** | 팝업 방문객 → 구매 전환 | POS 데이터 (더현대 글로벌 제공) | **전환율 15~25%** (더현대1차 기준) |
| **바이럴 지수** | UGC 생성량 + SNS 공유 | 해시태그 트래킹 (#TheSelects #더셀렉츠) | **UGC 3,000건 이상** (2거점 합산) |
| **바이어 전환** | 팝업 → 정규입점/오더 연결 건수 | 후속 바이어 미팅 기록 | **3개 브랜드 이상 정규입점 논의** |
| **참여자 만족도** | 참여 디자이너 브랜드 만족도 | 사후 설문 (NPS 방식) | **NPS 60점 이상** |
| **미디어 센티먼트** | K-패션 관련 보도 논조 | RXR Sentiment 분석 | **긍정 비율 85% 이상** |

> 💡 **So What**: Layer 2가 Layer 1과 다른 점은 "행동이 바뀌었는가"를 본다는 것. 매출(L1)은 나왔지만 검색량이 안 올랐다면 "일회성 구매"일 뿐. 검색량+UGC+바이어 전환이 함께 올라야 "글로벌 인지도 구축"이 증명된다.

### Layer 3: 파급 효과 (Ripple Impact) — 정성+추정 정량

**경제적 파급**:
- **직접 매출**: 37~48억원 (도쿄 25~30 + 파리 12~18)
- **간접 경제효과**: 항공·숙박·물류·현지 고용 등 사업 집행에 따른 파생 소비 → 추정 **10~15억원**
- **후속 오더 매출**: 정규입점/재오더 시 연간 추정 **5~10억원** (3개 브랜드 기준)
- **K-패션 생태계 파급**: 16개 브랜드의 글로벌 매출 증가분 (간접 기여) → 산출 어려우나 **생태계 확장 효과**

**사회적 파급**:
- **신생 디자이너 인큐베이션**: 16개 브랜드가 글로벌 럭셔리 유통에서 검증받는 경험 → 국내 패션 산업 인력 양성 효과
- **한불/한일 문화 교류**: 한불수교 140주년 + 한일 패션 교류 → 문화 외교 기여
- **지역 브랜딩**: "K-패션 = 파르코에서 팔리는 것" → 국가 브랜드 가치 상승

**정책적 파급**:
- **사업 모델 제도화**: 더셀렉츠가 성공하면 콘진원 연간 정규사업으로 제도화 가능
- **타 콘텐츠 분야 확산**: K-패션 모델을 K-뷰티, K-디자인, K-푸드에 적용 가능
- **후속 사업 파이프라인**: 2027 시즌 확대 (뉴욕, 밀라노 등) → 사업 지속 성장

### Layer 4: 지속가능성 (Sustainability)

**자체 수익 모델**:
- **팝업 매출 수수료**: 더현대 글로벌이 유통 마진을 가져가지만, 위탁사 운영비는 매출 규모에 비례하여 정당화
- **브랜드 매니지먼트 피**: 팝업 종료 후에도 정규입점 브랜드 관리 → 연간 리테이너 모델
- **RXR 분석 리포트 판매**: 각 브랜드에 맞춤 성과 분석 리포트 제공 → IP 라이선싱

**인프라 잔존가치**:
- **KOL 네트워크**: 도쿄·파리 인플루언서 관계 → 다음 시즌·다른 사업에 재활용
- **운영 SOP**: 11회 릴레이 운영 매뉴얼 → 2027 시즌 즉시 적용
- **RXR 해외 데이터**: 글로벌 SNS 분석 모델 → 다른 공공사업에도 적용 가능
- **더현대 글로벌 관계**: 유통 파트너십 → 후속 사업의 인프라

**조직 역량 내재화**:
- **콘진원 담당자 역량**: 12개월 협업 과정에서 글로벌 팝업 운영 노하우 이전
- **브랜드 자체 역량**: 16개 디자이너 브랜드가 해외 진출 프로세스 체득
- **성과 측정 체계**: RXR 기반 성과보고 프레임워크 → 콘진원 표준 모델화

## 4. 평가위원 설득 전략

### 4-1. SCQA 구조

```
S (Situation):
  더셀렉츠는 2025년 도쿄 레스티어(하이엔드 편집샵)에서 성공적으로 론칭.
  2026년은 파르코 시부야(메이저 백화점) + 갤러리 라파예트(글로벌 럭셔리)로
  채널을 급상승시키는 스케일업 연도.

C (Complication):
  레스티어(8개 브랜드, 1회)와 파르코+라파예트(16개 브랜드, 11+5회)는
  복잡성이 완전히 다른 사업. 2거점 동시 운영 + 12개월 장기 + 대규모 PR.
  "잘 운영하는 것"이 아니라 "글로벌 미디어 이벤트를 만들어야" 하는 사업.

Q (Question):
  어떻게 하면 16개 K-패션 브랜드의 "글로벌 First Impression"을
  도쿄와 파리에서 동시에, 미디어로 확산 가능한 구조로 설계할 수 있는가?

A (Answer) — Rule of Three:
  1. RETAIL MEDIA: 매장을 미디어 자산으로 운용 (프로젝트렌트)
  2. SCALE OPERATION: 11+5 릴레이의 복잡성을 시스템으로 제어 (마인드디자인)
  3. NATIVE EXECUTION: 두 거점 모두 현지 정규법인이 실행 (Le Project P + 일본사)
```

### 4-2. Cialdini 제안서 매핑

| Cialdini 원칙 | 이 제안서에서의 적용 |
|-------------|-------------------|
| **권위** | 더현대 글로벌 1차 30억/파르코 역대1위 실적 인용. Le Project P 한불 140주년 기수행 |
| **사회적 증거** | "파르코 역대 팝업 1위", "100+ 팝업 실적" → 숫자로 증명 |
| **일관성** | "매장을 미디어로" 키워드를 제안서 전체에 반복. RETAIL MEDIA 일관된 톤 |
| **호감** | 제안서 디자인 품질 — R스타일가이드(청보라 #6C5CE7, Pretendard) 적용 |
| **희소성** | "한불 140주년 기수행 파트너는 Le Project P뿐" → 대체 불가능한 자산 |
| **통합** | "콘진원의 K-패션 비전을 함께 만들어갑니다" — 파트너 톤, 수탁자 톤 아님 |

## 6. 전략 제언 요약

### 핵심 전략 3개 (Pyramid 구조)

**핵심 메시지**: *"K-패션의 글로벌 First Impression을 설계하는 미디어 컨소시엄"*

1. **RETAIL MEDIA** — 매장을 미디어 자산으로 운용하여, 팝업이 끝나도 브랜드 가치가 남는 구조
2. **SCALE OPERATION** — 11+5 릴레이를 시스템으로 제어하여, 16개 브랜드 모두 균등한 품질 보장
3. **NATIVE EXECUTION** — 파리·도쿄 모두 현지 정규법인이 실행하여, 진짜 "네이티브" 마케팅

### 경쟁사가 놓칠 포인트

1. **한불 140주년 기수행**: Le Project P만 가진 자산. 이것 하나로 파리 섹션 30점 중 5~10점 차이 가능
2. **RXR 측정 가능한 성과보고**: 경쟁사는 "노출수"로 보고하지만, 우리는 "감정·행동변화"까지 측정
3. **ROI 프레이밍**: 경쟁사는 "예산을 잘 쓰겠다"지만, 우리는 "18.2억으로 48억 매출 + 글로벌 인프라"

### 실행 우선순위

| 우선순위 | 액션 | 기한 |
|---------|------|------|
| 1 | 일본 파트너사 확정 (컨소시엄 완성) | 즉시 |
