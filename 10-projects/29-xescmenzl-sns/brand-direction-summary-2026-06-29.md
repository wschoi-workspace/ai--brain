## I. 브랜드 코어

### 한 문장 정의
**"장인의 기술로 한국의 일상을 다시 존중하는 브랜드"**

### 패러다임 전환

| 기존 접근 | SESC MENSES |
|---|---|
| 부대찌개를 **고급화** | 부대찌개를 **장인의 언어로 번역** |
| 비싼 재료를 넣는다 | 재료의 존재 이유를 다시 정의한다 |
| 샤퀴테리 브랜드 | **Craft Reinterpretation Brand** |
| 기술(Craft) 중심 | **존중(Respect)** 중심 |

### 브랜드 계층

```
SESC MENSES — House of Charcuterie
"A House That Translates Craft into Culture"

         │
    European Craft (출발점)
         │
    Translation (번역 = 브랜드의 행위)
         │
    Korean Food Culture (도착점)
         │
    ─────────────────────────
    │         │         │
   Budae    Future    Future
   by SESC  Korean    Products
   MENSES   Classics
```

- 샤퀴테리 = 존중을 표현하는 **기술**
- 부대찌개 = 그 존중을 한국적으로 번역한 **첫 번째 작품**
- 향후 김치찌개, 순대, 전골, 샌드위치 등 = **같은 철학의 다음 작품들**

### 내러티브 엔진 — "생존에서 장인으로"

샤퀴테리도 부대찌개도 **같은 곳**에서 출발했다: **생존**.

- 유럽: 생존의 기술 → 시간이 흘러 → **장인의 미식**으로 진화
- 한국: 생존의 기술 → 시간이 흘러 → **대중음식으로 고착** (푸짐함·가성비 이미지)

세스크멘스의 질문:
> "부대찌개도 샤퀴테리처럼 장인의 기술로 다시 해석될 수 있지 않은가?"

핵심 문장:
> **"생존의 기술은 시간이 지나 장인의 기술이 됩니다. 샤퀴테리가 그랬고, 치즈가 그랬고, 와인이 그랬습니다. 우리는 부대찌개도 그렇게 될 수 있다고 믿습니다."**

## II. 식사 설계 — 메뉴가 아니라 하루의 시나리오

### 핵심 원칙
점심과 저녁은 **완전히 다른 비즈니스**다.

| | Lunch: The First Meal | Dinner: The Last Table |
|---|---|---|
| **시간 전략** | 시간을 **절약**해주는 브랜드 | 시간을 **보내게** 만드는 브랜드 |
| **고객이 주문하는 것** | 부대찌개 | 세스크멘스의 **세계관** |
| **브랜드 역할** | 브랜드를 **만나는** 시간 | 브랜드를 **이해하는** 시간 |
| **설계 원리** | 한 가지를 가장 완벽하게 | 메뉴판이 아니라 **코스의 흐름** |
| **KPI** | 다시 생각나는 점심 | 세계관 경험 완성 |

### Lunch — The First Meal

> "좋은 하루는 좋은 첫 끼에서 시작됩니다."

**구성 (4품)**
| 메뉴 | 설계 의도 |
|---|---|
| **Signature Korean Charcuterie Stew** | 브랜드 대표작. 모든 햄·소시지가 이 찌개를 위해 설계된 샤퀴테리 |
| **Pot Rice (솥밥)** | "탄수화물이 아니라 첫 끼에 대한 존중." 매일 새로 짓는다 |
| **Seasonal Side** | 계절 피클/절임/장아찌 — 국물 밸런스 |
| **Today's Charcuterie** | 한 점. 점심에도 샤퀴테리 브랜드임을 각인 |

**경험 플로우**
```
도착 → 5분 이내 제공 → 15~20분 식사 → 커피 없이도 따뜻한 점심 → 오후 시작
```

**메뉴 개발 방향**
- 선택지 최소화. 빠른 의사결정 + 높은 완성도
- 햄·소시지는 범용이 아닌 **부대찌개 전용 설계** (지방비율, 훈연, 염도 조정)
- 솥밥은 국물과의 페어링 관점에서 품종·수분·취사 방식 역설계
- Today's Charcuterie는 원가 부담 없이 브랜드 정체성 전달하는 **미니멀 터치포인트**

### Dinner — The Last Table

> "좋은 하루는 좋은 식탁에서 완성됩니다."

**코스 흐름 (5단계)**
| 코스명 | 메뉴 | 역할 |
|---|---|---|
| **Begin** | Charcuterie Board | 오늘의 숙성 햄·테린·파테·계절 샤퀴테리 + 와인 |
| **Fire** | Grilled Sausage | 부위별·숙성별·훈연별 테이스팅. 기술력이 드러나는 순간 |
| **Stew** | Korean Charcuterie Stew | 점심과 같은 음식, 역할이 다름 — 테이블의 중심, 함께 나누는 음식 |
| **Finish** | Pot Rice + Finish Course | 남은 국물에 리조토/치즈/계란/볶음밥 택1 — 장인의 스튜를 끝까지 즐기는 마지막 코스 |
| **Pairing** | Wine/Beer/Highball/Seasonal | 샤퀴테리 기준 큐레이션 |

**시그니처 메뉴 2종**
| 메뉴 | 설계 의도 |
|---|---|
| **Chef's Sausage** | 매주/매달/계절 변경. "이 메뉴 하나 때문에 다시 오는 이유가 생긴다" |
| **Charcuterie Flight** | 와인 플라이트처럼 소시지 4종(Classic→Smoked→Aged→Chef's Special) 소량 테이스팅. 세스크멘스만 가능 |

**메뉴 개발 방향**
- Grilled Sausage 라인업: Classic / Smoked / Aged / Seasonal 4종 상시 + Chef's Special 회전
- Finish Course: 일반 볶음밥이 아닌 "마지막 코스" 격으로 설계 (리조토 = 양식 마감, 계란 = 한식 마감 등 선택에 의미 부여)
- Charcuterie Board: 플레이팅이 "장면"이 되어야 함 — 사진 한 장으로 브랜드 세계관 전달
- 페어링은 와인 외 맥주·하이볼 포함 — 접근성과 깊이의 양립

## III. 가격 전략 원칙

**가격이 아니라 경험의 깊이를 달리한다.**

- Lunch: 단일 세트 구성. 합리적 가격대에서 **완성도로 차별화**
- Dinner: 코스 흐름 중심. "비싼 저녁"이 아니라 "경험하는 저녁"으로 가격 정당성 확보
- 같은 공간, 같은 대표 메뉴로 **전혀 다른 두 개의 비즈니스 모델** 운영

## IV. 장인의 방식 — 메뉴 개발 철학

> "장인은 비싼 재료를 쓰는 사람이 아니라, 재료의 존재 이유를 다시 정의하는 사람이다."

| 요소 | 접근 |
|---|---|
| 햄 | 부대찌개를 위해 **새로 만든다** |
| 소시지 | 부대찌개를 위해 **새로 설계한다** |
| 지방 비율 | 국물과의 밸런스를 위해 **조정한다** |
| 훈연 | 찌개 안에서의 향 변화를 고려해 **바꾼다** |
| 염도 | 국물 전체의 균형을 위해 **다시 맞춘다** |
| 육수 | 부용 기반으로 **다시 설계한다** |
| 밥 | 국물과의 페어링으로 **다시 설계한다** |

모든 것은 "더 비싸게 만들기 위해서"가 아니라, **부대찌개라는 한 그릇을 완성하기 위해서**.

## V. Manifesto

> 우리는 부대찌개를 고급화하지 않습니다.
> 우리는 부대찌개를 존중합니다.
>
> 생존을 위해 태어난 음식에도,
> 장인의 기술은 스며들 수 있다고 믿습니다.
>
> 좋은 샤퀴테리가 시간을 존중하듯,
> 좋은 부대찌개도 재료와 시간을 존중해야 합니다.
>
> 그래서 우리는 햄을 다시 만들고,
> 소시지를 다시 설계하고,
> 국물을 다시 끓이고,
> 밥을 다시 짓습니다.
>
> 우리가 만드는 것은 새로운 음식이 아닙니다.
> 한국의 스튜를 장인의 언어로 다시 번역하는 일입니다.
