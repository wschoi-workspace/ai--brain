CBC Title 24 (California Building Code)
0/5
Part 2 Building Code 적용 범위 확인 (IBC 2021 기반 + CA amendments) ▸
  • Occupancy Classification 확인 — 전시/문화시설은 A-3(Assembly) 해당
  • Type of Construction(I~V) 확인: 화재등급별 구조 요건
  • Occupant Load 산출: 전시공간 기준 면적(sq ft) ÷ 30 = 최대 수용인원
  • 기존 건물 개조(Alteration) 시 적용 범위: 전체 소급 vs 개조 부분만 판단
Part 6 Energy Code — 조명 LPD(Lighting Power Density) 기준 확인 ▸
  • LPD(Lighting Power Density) 허용치: 전시공간 Category별 W/sq ft 제한 확인
  • 조명 제어 요건: 자동 차단(auto-off), 점유감지(occupancy sensor) 설치
  • Title 24 에너지 서류(CF-1R/CF-2R/CF-3R) 제출 필수 여부 확인
  • LED 전환 시 기존 대비 에너지 절감 문서화 필요
Part 11 Chapter 11B — California Accessibility (ADA보다 엄격) 확인 ▸
  • California Accessibility는 연방 ADA보다 엄격 — 항상 CBC 11B 기준 우선
  • CASp(Certified Access Specialist) 검사 권장: 선제 검사 시 소송 방어에 유리
  • 주요 차이: 통로(ADA 36" vs CBC 44"), 카운터 높이, 표지판 규격
  • CBC 11B 비준수 시 Unruh Act $4,000/건 자동 적용
Part 12 Fire Code 적용 사항 확인 ▸
  • 인테리어 변경이 Fire Code 신고 트리거에 해당하는지 확인
  • LAFD와 사전 협의 필요 여부 판단
  • 기존 스프링클러/감지기 시스템에 영향 없는지 확인
  • 비상 대피로(Means of Egress) 변경 시 별도 Plan Check 필요
Structural: 전시물 하중 검토 (suspended displays, wall-mounted) ▸
  • 석고보드 벽 하중 한계: 일반 15lb/anchor, 블로킹 시 50lb+
  • 천장 매달기(Suspended): 기존 구조 하중 검토 + 구조 엔지니어 계산서 필요 여부
  • 바닥 Live Load 확인: 일반 상업공간 100 psf
  • 지진 고려: Seismic Zone 4(LA) — 전시물 전도 방지 앵커링 필수
ADA 핵심 수치 요약
0/6
통로 36" 최소 (44" 권장, CBC 11B: 44" 요구) ▸
  • ADA §403.5.1: 최소 36"(914mm), 24" 구간만 32"까지 축소 허용
  • CBC 11B: 점유 50명 초과 시 44"(1118mm) 요구 — LA 문화시설 대부분 해당
  • 전시물 배치 시 통로 폭 재측정 필수: 임시 전시물 통로 침범 빈번
  • 양방향 통행 구간은 60" 권장
출입문 32" clear opening ▸
  • ADA §404.2.3: 문 90도 열었을 때 순수 통과 폭 32"(813mm) 이상
  • 측정: 문틀~문짝 사이 거리(문 두께 제외)
  • 문 닫히는 힘: 5 lbs 이하(fire door 제외), 닫힘 시간 최소 5초
  • 도어 하드웨어: 레버형 필수(둥근 손잡이 불가), 한 손 조작
휠체어 회전 60" 직경 ▸
  • ADA §304.3: 180° 회전 시 60"(1524mm) 직경 원형 공간
  • 대안: T자형 회전공간 60"×36" 사용 가능(§304.3.2)
  • 막다른 구간(dead-end)에 반드시 회전 공간 확보
  • 전시 레이아웃 변경 시마다 회전 공간 재검증
경사로 1:12 기울기, 36" 너비 ▸
  • ADA §405.2: 최대 경사 1:12(8.33%), 30" 상승 시 최소 30ft 경사로
  • 너비: 최소 36", CBC 11B에서는 48" 권장
  • 60"×60" 평탄 랜딩: 경사로 상·하단 + 30ft마다
  • 핸드레일: 높이 34-38", 양면 설치, 상·하단 12" 연장
전시 디스플레이 높이 15–48" (forward reach range) ▸
  • ADA §308: Forward reach range 15-48"(381-1219mm)
  • 장애물 없을 때: 15-48", 장애물 깊이 20" 이내 시 최대 48"
  • 장애물 깊이 20-25" → 최대 높이 44"
  • 핵심 전시물 중심선은 36-39"(휠체어 눈높이 최적) 배치 권장
→ 상세 매뉴얼 참조 ref: ada-compliance-manual.html ▸
  • ADA 전체 수치 테이블, 전시공간 특수 규정, 리스크 매트릭스는 별도 ADA Manual 참고
  • CBC 11B가 ADA보다 엄격한 항목 비교표 포함
  • 설계/시공/최종검수 3단계 체크리스트도 ADA Manual에 포함
Fire Code (LAFD)
0/6
내장재 등급 확인: Class A (FSI 0–25) 필수 ▸
  • ASTM E84 Tunnel Test: FSI 0-25 = Class A / Class B: 26-75 / Class C: 76-200
  • 전시공간은 Class A 필수
  • 공급업체에 ASTM E84 시험성적서 원본 요청 — 인증서 사본 불가
  • 페인트도 내장재에 포함: 내화 등급 페인트 사용 시 인증서 확보
스프링클러 시스템 기존 상태 확인 ▸
  • 기존 스프링클러 위치·커버리지 확인: 전시 구조물이 살수 방해 여부
  • 높이 변경(천장 내림 등) 시 헤드 재배치 필요 여부
  • 전시 조명 열원이 헤드에 영향 없는지 확인 (최소 18" 이격)
  • 인테리어 작업 시 배관 파손 주의
비상구 수량 확인 (49인 이하=1개, 50+=2개) ▸
  • LAMC §57.33.17: 49인 이하=1개, 50-499인=2개, 500+=3개
  • Occupant Load 산출: 전시면적(sq ft) ÷ 30
  • 출구 간 이격거리: 대각선 길이의 1/2 이상(스프링클러 있으면 1/3)
  • 비상 출구 폭: 인원 × 0.2"/person(최소 36")
비상구 표시등(Exit Sign) — illuminated, battery backup ▸
  • 항시 점등(illuminated), 정전 시 90분 이상 배터리 백업
  • 설치 위치: 출구 위 6'8"~7'0", 바닥에서 보이는 경로 표시
  • 문자 높이: 최소 6", 조명식 또는 자발광
  • 경로 표시(directional): 출구까지 경로가 불명확한 곳에 추가
소화기 위치 및 종류 확인 (75ft travel distance max) ▸
  • 설치 간격: 이동거리 75ft(22.8m) 이내에 1대
  • 종류: Class ABC, 최소 2A:10B:C 등급
  • 설치 높이: 바닥에서 손잡이까지 3.5-5ft(ADA 접근 가능)
  • 월 1회 육안 점검, 연 1회 전문 점검, 6년마다 내부 점검
비상 조명(Emergency Lighting) 설치 여부 ▸
  • 정전 시 자동 점등, 최소 90분 지속
  • 바닥면 평균 1 fc(foot-candle), 최소 0.1 fc
  • 비상 탈출로 전 구간 커버, 전시장 내부 포함
  • 월 1회 30초 테스트, 연 1회 90분 풀 테스트
Zoning / Land Use
0/3
현재 Zoning 용도지역 확인 (Commercial/Cultural) ▸
  • LADBS ZIMAS(Zoning Information & Map Access System)에서 현재 zoning 확인
  • 문화원 건물 용도: C(Commercial) 또는 PF(Public Facility) 확인
  • 전시 용도가 현재 zoning에서 허용 use인지 확인
  • CUP(Conditional Use Permit) 존재 여부 확인
용도변경(Change of Use) 필요 여부 확인 ▸
  • 기존→전시/갤러리 변경 시 Change of Use permit 필요 여부
  • Occupancy Classification 변경(B→A-3 등) 시 추가 요건 발생 가능
  • 용도변경 없어도 Occupant Load 증가 시 비상구/스프링클러 재검토
  • 문화원이 이미 문화시설로 등록되어 있으면 변경 불필요 가능성
주차 요건 확인 (ADA accessible parking) ▸
  • LA 주차 기준: Assembly use 기준 좌석/면적당 주차대수
  • ADA 주차: 1-25대 → 1대 accessible, 26-50대 → 2대
  • Accessible 주차: 8ft 폭 + 5ft aisle, 표지판 높이 60"
  • Van accessible: 최소 1대, 11ft + 5ft aisle

  • 일반적으로 valuation의 0.5-2% 수준
  • 추가: LAFD review fee, Accessibility review fee, 기계/전기 별도

Plan Check 제출서류
0/7
설계 도면(Floor Plan) — 1/4" = 1'-0" scale ▸
  • 축척: 1/4" = 1'-0"(1:48) 또는 1/8" = 1'-0"(1:96)
  • 필수 표기: 치수, 문 위치·방향, 벽체 유형, 재료 표시
  • 기존 vs 신규: Demolition Plan + New Plan 구분 표기
  • 비상구 경로, Occupant Load, 출구 너비 표기
전기 도면(Electrical Plan) ▸
  • 조명 배치, 콘센트·스위치 위치, 회로 배정
  • 패널 스케줄: 회로별 부하(amperage), 브레이커 크기
  • 특수 전원: 영상장비, 인터랙티브 설치물 전용 회로
  • 비상 조명 및 Exit Sign 위치 표기
기계 도면(Mechanical Plan) — 해당 시 ▸
  • HVAC 덕트 변경·신규 공조 설비 추가 시 필수
  • 기존 HVAC 영향 없으면 생략 가능
  • 환기량: ASHRAE 62.1 기준, 전시장 cfm/person
  • 소음 등급: 갤러리 NC 25-30 권장
내장재 사양서(Finish Schedule) with fire ratings ▸
  • 각 실/구역별 바닥·벽·천장 재료 및 색상 명기
  • Fire Rating 포함: 각 재료의 ASTM E84 결과(FSI + SDI)
  • 제조사·제품번호·인증 번호 기재
  • 마감 두께, 접착제 종류, 시공 방법 포함
Title 24 Energy Compliance Forms ▸
  • CF-1R(설계 적합), CF-2R(시공 적합), CF-3R(검증) 양식
  • 에너지 계산서(소프트웨어 출력물) 첨부
  • 조명만 변경 시 Lighting Only 축약 양식 사용 가능
  • HERS rater 필요 여부 확인(CF-3R 작성자)
Accessibility Compliance (CASp report 권장) ▸
  • CASp 사전 검사 강력 권장
  • CASp 보고서: ADA 소송 시 120일 유예 + 합리적 항변
  • 검사 비용: $2,000-$5,000(규모에 따라)
  • Plan Check 시 CASp 보고서 첨부 → Accessibility review 간소화
Structural calculations (해당 시) ▸
  • 필요: 벽체 철거/신설, 천장 매달기 하중, 중량 전시물 설치
  • PE 또는 SE 면허 보유자 작성 필수
  • Seismic 계산 포함: LA는 Seismic Design Category D
  • 경미한 비내력벽 파티션은 구조계산서 불필요
Inspection 타임라인
0/6
Rough-in Inspection 일정 예약 ▸
  • 전기·배관·기계 배선/배관을 벽/천장 안에 넣기 전 검사
  • 예약: LADBS 온라인/전화, 최소 1 working day 전
  • 검사관 현장 방문, 불합격 시 보정 후 재검사(re-inspection fee)
  • 검사 통과 없이 벽 닫으면 다시 열어야 함
Framing Inspection ▸
  • 벽체 프레이밍 완료 후, 마감재 설치 전
  • 스터드 간격(16" o.c.), 헤더 사이즈, 블로킹 위치
  • 비내력벽도 화재 차단(fire blocking) 확인
  • 파티션 높이·고정 방식 확인
Electrical Inspection (rough & final) ▸
  • Rough: 배선 경로, 와이어 규격, 박스 설치, 접지
  • Final: 기기 설치 후, 패널 라벨링, GFCI 작동, 전압 측정
  • 조명 제어 시스템 시운전(commissioning) 포함
  • AFCI 필요 구역 확인(주거 아니면 대부분 불필요)
Mechanical Inspection (해당 시) ▸
  • HVAC 덕트 변경·신규 장비 설치 시에만 해당
  • 기존 시스템 유지(덕트 연장 없음)이면 생략 가능
  • 검사: 덕트 연결, 절연, 접근 패널, 환기량
  • 제어 시스템(thermostat, 타이머) 작동 확인
Drywall/Lath Inspection (before closing walls) ▸
  • 석고보드 설치 후, 테이핑/머딩 전 검사
  • 보드 두께(5/8" fire-rated where required), 나사 간격, 이음새
  • 방화벽 관통부(penetration) 방화 실링 확인
  • 이 검사 통과 전 조인트 컴파운드 작업 금지
Final Inspection 일정 확인 ▸
  • 모든 시공 완료 후 종합 검사, C of O 발급 전 필수
  • 사전 준비: 모든 중간 검사 통과, 패널 라벨링, 안전장치 작동
  • 소방(LAFD) 별도 final sign-off 필요할 수 있음
  • 불합격 시 보정 항목 리스트 발급, 보정 후 재검사
