# KCCLA Interior Compliance — Research Brief

## 프로젝트 개요
- **클라이언트**: LA 한국문화원 (Korean Cultural Center Los Angeles)
- **프로젝트 유형**: 전시 인테리어 시공
- **관할**: City of Los Angeles, State of California
- **적용 법규**: CBC Title 24 (2022→2025 cycle), ADA 2010 Standards, LAFD Fire Code, OSHA

## 핵심 법규 체계

### 1. California Building Code (CBC) Title 24
- Part 1: Administrative
- Part 2: Building Code (IBC 기반 + CA amendments)
- Part 6: Energy Code
- Part 11: **California Access Compliance (Chapter 11B)** — ADA보다 엄격
- Part 12: Fire Code

### 2. ADA 2010 Standards for Accessible Design
- Title III: Public Accommodations (문화시설 해당)
- 28 CFR Part 36
- CBC Ch.11B가 더 엄격한 경우 CBC 우선

### 3. LAFD Fire Code
- LAMC Chapter 5, Article 7
- Interior finish: Class A (Flame Spread 0-25)
- 스프링클러: 상업공간 5,000 sq ft 이상
- 비상구: 49인 이하 1개, 50+ 2개

### 4. Permit 체계 (LADBS)
- Counter Permit: 소규모 (전기/조명 교체)
- Standard Plan Check: 구조변경, MEP
- Express Permit: 경미한 인테리어

### 5. 문화원 특수 고려사항
- 정부(연방/한국) 소유 건물 시설관리 규정
- 외부 시공업체 보안 출입 절차
- COI (Certificate of Insurance) 필수 제출
- 원상복구(Restoration) 기준 명확화 필요
- 정부 자산 훼손 시 책임 소재
