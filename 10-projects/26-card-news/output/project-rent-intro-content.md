# 프로젝트 렌트 회사소개 — 카드뉴스 12장 초안

핵심 메시지: 우리는 광고를 측정하는 회사가 아니라, 사람의 행동을 측정하는 회사다.
타깃: 일반 대중·잠재 파트너(SNS 피드) · 톤: 브랜드 스토리(중강) · 규격: 1080×1350(4:5) 캐러셀
크레딧: by Project Rent

> 서사: **던지기(A) → 우리의 답(B) → 증명(C) → 비전(D)**, 4섹션 × 3장.
> 청보라 `#6C5CE7`는 카드당 강조 1곳에만. 다크 배경 + 크림 텍스트.

## SECTION A — 질문을 던진다

### 카드 01 — 표지
- 라벨: PROJECT RENT
- 헤드카피: **매장은 미디어다**
- 서브카피: 성수동을 만든 회사가 던지는, 다음 질문
- 본문: (없음 — 표지는 한 문장의 무게로)
- 강조: "미디어" (청보라)
🔎 R-Voice: 단언 1문장. 통념(매장=판매처)을 재정의(매장=미디어). De-AI ✅

### 카드 02 — 세 회사의 대구
- 라벨: A · 우리가 보는 것
- 헤드카피: 온라인은 *무엇을 클릭했는지* 안다. 그런데 *왜 움직였는지*는?
- 본문:
  - Google은 사람들이 **무엇을 검색하는지** 알아냈다.
  - Amazon은 사람들이 **무엇을 구매하는지** 알아냈다.
  - 우리는 묻는다. 사람들이 **왜 움직이는지**, 그건 누가 아는가.
- 강조: "왜 움직이는지"
🔎 R-Voice: 수사적 질문으로 연다(규칙3). 3단 귀납·대구 리듬. ✅

### 카드 03 — 문제
- 라벨: A · 지금의 풍경
- 헤드카피: 광고는 넘치는데, *사람은 이해되지 않는다*
- 본문:
  - 피드를 채운 건 광고다. 그런데 그 광고가 누구에게 닿았는지, 왜 멈춰 섰는지는 아무도 모른다.
  - 클릭은 숫자로 남지만, 마음은 숫자로 남지 않는다.
  - 더 많이 사도, 사람은 더 멀어진다.
- 강조: "사람은 이해되지 않는다"
🔎 R-Voice: 단문 단언 + 대조(클릭↔마음). 부풀리기 없음. ✅

## SECTION B — 우리의 답

### 카드 04 — 공백
- 라벨: B · 빈자리
- 헤드카피: 소비의 *80%*는 오프라인에서 일어난다
- 본문:
  - 우리는 화면 밖에서 만지고, 머물고, 고른다. 삶의 대부분은 여전히 매장에서 벌어진다.
  - 그런데 데이터는 온라인에서만 쌓인다.
  - 가장 큰 행동이, 가장 적게 기록된다. 이 빈자리가 기회다.
- 강조: "80%"
- 출처주: 오프라인 소매 비중 ~80% (이커머스 침투율 16~20%, DC360·Statista)
🔎 R-Voice: 귀납(장면→원리). 대조(큰 행동↔적은 기록). ✅

### 카드 05 — 통찰 (Peak)
- 라벨: B · 핵심
- 헤드카피: **검색어는 속일 수 있지만, 발자국은 거짓말하지 않는다**
- 본문:
  - 사람은 검색창 앞에서 자신을 꾸민다. 하지만 두 발은 정직하다.
  - 무엇 앞에서 멈췄는지, 얼마나 머물렀는지, 무엇을 집어 들었는지.
  - 행동은 해명하지 않는다. 그저 보여준다.
- 강조: 헤드카피 전체 (다크 강조 슬라이드)
🔎 R-Voice: 대조·재정의의 정점. 서정적 단언. 가장 강한 카드. ✅

### 카드 06 — 솔루션
- 라벨: B · 우리의 방식
- 헤드카피: 그래서 우리는 매장을 *미디어*로 바꾼다
- 본문:
  - 광고 → 방문 → 체류 → 반응 → 구매. 흩어져 있던 이 길을 하나로 잇는다.
  - 매장은 더 이상 물건을 파는 곳이 아니라, 사람의 행동이 데이터로 남는 곳이 된다.
  - 공간이 곧 미디어다.
- 강조: "미디어"
🔎 R-Voice: 재정의(파는 곳→데이터가 남는 곳). 연쇄 정의. ✅

## SECTION C — 이미 증명했다

### 카드 07 — 차별점
- 라벨: C · 우리가 다른 이유
- 헤드카피: 우리는 데이터를 *분석*하지 않는다. 데이터가 *생기는 현장*을 운영한다
- 본문:
  - 대부분은 남이 만든 데이터를 들여다본다. 우리는 데이터가 태어나는 자리를 직접 연다.
  - 350개가 넘는 팝업, 8개 직영 거점.
  - 빌려온 숫자가 아니라, 우리가 만든 숫자다.
- 강조: "현장을 운영한다"
- 출처주: 350+ 팝업 누적, 직영 8거점 (자사 운영 실적)
🔎 R-Voice: 대조(분석↔운영, 빌린 숫자↔만든 숫자). ✅

### 카드 08 — 성수동 스토리
- 라벨: C · 시작
- 헤드카피: *6.5평* 빈 공간에서, 성수동의 시대가 열렸다
- 본문:
  - 2019년, 작은 공간 하나에 가나초콜릿하우스를 세웠다.
  - 사람들이 줄을 섰고, 거리가 바뀌었다. 분기당 서너 개였던 팝업이, 수십 개로 늘었다.
  - 우리는 매장을 연 게 아니라, 동네의 문법을 바꿨다.
- 강조: "6.5평"
- 출처주: 가나초콜릿하우스 2019(실명 노출 OK), 성수 팝업 확산 분기점 (기업 프로필)
🔎 R-Voice: 귀납(구체 장면→원리). 단언 마무리. ✅

### 카드 09 — 성과
- 라벨: C · 증거
- 헤드카피: 광고비는 *사라지는 비용*이 아니라, *쌓이는 자산*이 된다
- 본문:
  - 가나초콜릿 — 마케팅 효율 **4,500%**.
  - 글로벌 IT 브랜드 — 매장 활성화 **+44%**.
  - 한 라이프스타일 브랜드 — 재방문 희망 **100%**.
  - 그리고 6년 연속 흑자. 매출 40억. 6명이 만든 결과다.
- 강조: "쌓이는 자산"
- 출처주: 수치는 자사 데이터 기준. 4,500%=가나초콜릿(실명) / +44%=삼성스토어→"글로벌 IT"로 익명화 / 100%=비욘드→"라이프스타일 브랜드"로 익명화. 측정 정의는 캡션 각주.
🔎 R-Voice: 재정의(비용↔자산). 숫자 귀납 후 단언. ✅

## SECTION D — 우리가 가는 곳

### 카드 10 — 일하는 법
- 라벨: D · 우리는 누구인가
- 헤드카피: 우리는 컨설턴트가 아니라 *운영자*다
- 본문:
  - 보고서를 쓰는 대신, 문을 열고 직접 운영한다.
  - 공간, 운영, F&B, 판매, 데이터까지 한 팀이 끝까지 책임진다.
  - 여섯 명 전원이 AI로 일한다. 작게, 그러나 깊게.
- 강조: "운영자"
🔎 R-Voice: 대조(컨설턴트↔운영자, 보고서↔문을 연다). ✅

### 카드 11 — 비전
- 라벨: D · 우리가 만드는 것
- 헤드카피: 브랜드가 *발견되고 성장하는* 인프라를 만든다
- 본문:
  - 우리의 꿈은 성수동을 한 권의 잡지 같은 거리로 만드는 것.
  - 한 페이지를 넘기면, 다른 브랜드의 이야기가 펼쳐진다.
  - 그 페이지를 서울에서 도쿄로, 파리로 넓혀간다.
- 강조: "발견되고 성장하는"
- 출처주: 글로벌 확장 도쿄·파리 (더셀렉츠 KOCCA 실적 기반)
🔎 R-Voice: 서정적 비유(거리=잡지). 단언. ✅

- 라벨: PROJECT RENT

- 강조: "사람의 행동을 측정하는 회사"

## SNS 본문 (캡션)

매장은 미디어다.

성수동의 팝업 시대를 연 프로젝트 렌트는, 매장을 '파는 곳'이 아니라 '사람의 행동이 데이터로 남는 곳'으로 다시 정의합니다.
