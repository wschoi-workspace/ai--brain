# 현대자동차 RXR — 차량 실내 공기질·냄새·공조 인식 분석

> **RXR SNS Analysis Solution** · by Project Rent · 분석일 2026-07-17

## 배경
현대차 R&D POC(2026-07, 공기·공조 주제)와 연계한 RXR 인식 분석. 차량 실내의 **냄새·공조·공기질**에 대한 실사용자의 진짜 목소리를 수집·정제(광고 전량 제외)해 카테고리화하고 **욕구(Needs) 관점**으로 재정의한다.

> ⚠️ **분석 범위 주의**: 특정 브랜드가 아니라 **차량 전반(브랜드 무관)** 시장 인식을 분석하고, 그 위에 12개 브랜드 태깅을 얹었다. "현대자동차 RXR"은 이 자산의 **귀속 프로젝트명**이며, 데이터 자체는 현대차 한정이 아니다.

## 데이터 파이프라인 (RXR 4-Gate)
```
네이버 블로그 30,398건
  → 최근 180일 18,210
  → 차량×공기 주제 관련 12,289
  → 유효후기(A/B/C) 10,588
  → Gate3: 협찬 제외 + 진정성 60+  9,865
  → Gate4: 업체홍보·커머스·비차량 강화 제외  3,586  ← 최종 진성후기
+ 보배드림·클리앙 커뮤니티 실사용 발화 23건 (정성 근거)
```
- **핵심 발견**: 표준 협찬 마커 필터(Gate1→3)로는 감성이 거의 안 변함. 진짜 노이즈는 "#광고" 표기 없는 **세차·디테일링·필터판매 콘텐츠**(Gate4에서 −6,279건).

## 핵심 결과 3줄
1. **냄새 축 부정 54%** = 최대 페인포인트. 정체는 곰팡이·쉰내(부정 55%), 원인은 습기→**에바(증발기, 부정 70%)**
2. **미세먼지·필터만 유일 충족**(긍정 42%) — 냄새는 여전히 사후 대응(에바크리닝 부정 77%)뿐 = '대처의 역설'
