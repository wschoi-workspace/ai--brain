# 직무별 AI 실습 예제집 — 리서치 · 자료정리 · 보고서

> 4개 직무군 × 업무흐름 3단계(리서치 → 자료정리 → 보고서) 매트릭스.
> 각 블록: **상황 → 복붙 프롬프트 → 기대 산출물 → 검증 한 마디 → 더 깊이(출처)**
> 전제: 각자 자기 Claude Code(공유 워크스페이스 없이). 프롬프트는 그대로 복사해 쓰고, 대괄호 `[ ]`만 내 상황으로 바꾸면 됩니다.

**프롬프트 4요소** — 좋은 지시는 ① 맥락 ② 원하는 것 ③ 출력형태 ④ 예시. 아래 예제는 이 틀을 따릅니다.

## 🎨 공간/브랜드 디자인

### 🔍 리서치 — 레퍼런스 3곳, 디자인 시스템으로 뜯어보기
**상황**: 새 프로젝트 무드를 잡으려 레퍼런스를 모았는데, "예쁘다"에서 멈추고 언어화가 안 된다.

```
레퍼런스로 본 공간/브랜드 웹사이트 3곳이야: [URL1] [URL2] [URL3]
각각의 디자인 시스템을 같은 기준으로 뜯어서 비교표로 만들어줘.
기준: 컬러 팔레트(hex), 타이포(폰트 계열·위계), 레이아웃·여백 리듬, 소재·질감 무드, 톤 키워드 3개.
표로 정리하고, 맨 아래 '우리가 가져올 점 / 피할 점'을 항목별로 적어줘.
```

- **산출물**: 3열 비교표 + 차용/회피 포인트
- **검증**: `각 항목이 실제 그 사이트에서 확인되는지, 추측한 건 표시해줘`
- **더 깊이**: `web-crawler-ocr`(URL 본문·이미지 추출), 이미지가 핵심이면 스크린샷 첨부 후 분석

### 🗂 자료정리 — 현장답사 사진·메모 → 무드보드 컨셉
**상황**: 답사 다녀와 사진 수십 장과 단편 메모만 남았다. 컨셉 키워드로 묶이지 않는다.

```
현장답사 메모와 관찰을 붙여넣을게. [메모 텍스트 붙여넣기 / 사진 첨부]
이걸 (1) 반복되는 공간 경험 키워드 5개로 묶고
(2) 각 키워드별 대표 관찰 1~2개를 근거로 달고
(3) 이걸 관통하는 컨셉 문장 후보 3개를 만들어줘.
형식: 키워드 → 근거 → 한 줄 해석.
```

- **산출물**: 컨셉 키워드 맵 + 컨셉 문장 후보 3
- **검증**: `내 메모에 없는 내용을 지어내지 말고, 근거 없는 키워드는 빼줘`
- **더 깊이**: 답사 녹음이 있으면 `transcript-organizer`로 먼저 텍스트화

### 📑 보고서 — 공간 컨셉 제안 슬라이드 1장
**상황**: 컨셉을 클라이언트에게 한 장으로 보여줘야 하는데 PPT 손이 오래 걸린다.

```
아래 컨셉을 제안용 슬라이드 1장(HTML)으로 만들어줘.
컨셉: [컨셉 문장]  핵심 키워드: [3개]  무드 컬러: [hex 또는 느낌]
구성: 큰 컨셉 카피 + 키워드 3개 + 무드 설명 2~3줄 + 하단 출처/크레딧.
16:9, 다크 톤, Pretendard. 완성되면 스크린샷으로 보여줘.
```

- **산출물**: 제안 슬라이드 HTML → PNG
- **검증**: `클라이언트 입장에서 한눈에 컨셉이 읽히는지, 군더더기 문구 빼줘`
- **더 깊이**: `playwright`(HTML→이미지), 여러 장이면 `card-news`/슬라이드 패턴

## 📋 기획 · AE · PM

### 🔍 리서치 — 경쟁사 3곳 90일 동향 비교
**상황**: 제안 들어가기 전 경쟁 지형을 빠르게 잡아야 하는데 웹서핑만 며칠째다.

```
[우리 카테고리/시장]에서 경쟁사 3곳 [A] [B] [C]을 같은 틀로 비교해줘.
항목: 핵심 제품·가격대, 타깃 고객, 포지셔닝 한 줄, 최근 동향(가능하면 최근 90일), SNS/채널 활동, 강점·약점.
표로 만들고, 맨 아래 '빈 자리(아무도 안 차지한 포지션)' 3가지를 제안해줘.
출처가 불확실한 항목은 [확인필요]로 표시해.
```

- **산출물**: 경쟁사 비교표 + 빈 포지션 제안
- **검증**: `각 수치·주장의 출처를 옆에 달고, 추정이면 추정이라고 표시해줘`
- **더 깊이**: 깊은 리서치는 `deep-research`(다출처+사실검증), 브랜드 SNS는 `brand-analysis`

### 🗂 자료정리 — 인터뷰 녹취 → 니즈 코딩
**상황**: 사용자 인터뷰 5건 녹취를 풀었는데, 공통 니즈로 정리하는 데 반나절이 걸린다.

```
사용자 인터뷰 녹취를 붙여넣을게. [녹취 텍스트]
(1) 반복되는 니즈/페인포인트를 카테고리 5개로 묶고
(2) 각 카테고리에 대표 인용구를 원문 그대로 1~2개 달고
(3) 우리 제안의 기회 포인트를 카테고리별 한 줄로 적어줘.
실제 발화에 있는 것만 써. 없으면 빈칸으로 둬.
```

- **산출물**: 니즈 카테고리 + 대표 인용 + 기회 포인트
- **검증**: `말하지 않은 걸 채워넣지 말고, 인용은 원문과 일치하는지 확인해줘`
- **더 깊이**: 녹취 자동정리 `transcript-organizer`, 정성분석 심화 `sensemaking`

### 📑 보고서 — 제안서 초안 + 평가관점 셀프체크
**상황**: 제안서 뼈대 잡는 데 하루, 내용 채우는 데 며칠. 게다가 내 눈으론 약점이 안 보인다.

```
이 프로젝트로 제안서 초안을 만들어줘.
배경: [상황]  목표: [목표]  우리 강점: [강점]  대상/심사: [누구]
구성: 문제정의 → 우리 해석 → 해결안 → 실행계획 → 기대효과, 각 섹션 3~5줄.
초안을 만든 뒤, '심사위원 입장'에서 약한 부분 3가지와 보완 질문을 따로 정리해줘.
```

- **산출물**: 제안서 초안 + 심사 관점 약점·보완점
- **검증**: `우리가 안 가진 강점을 부풀리지 않았는지, 과장 표현 걸러줘`
- **더 깊이**: 전략 검증 대화 `thinking-partner`, 정부제안은 `gov-proposal`/`rfp-analysis`, 덱은 `consulting-report`

## 🏪 매장운영관리

### 🔍 리서치·기획 — 우리 매장 마케팅 전략 + 콘텐츠 기획
**상황**: 마케팅을 감으로 하고 있다. 우리 상권·고객에 맞는 전략과 콘텐츠 흐름이 필요하다.

```
우리 매장 마케팅 전략을 짜줘.
매장: [업종/위치/콘셉트]  주 고객: [연령·성향]  강점: [차별점]  목표: [재방문/신규/객단가 중]
(1) 핵심 타깃 한 줄 정의 (2) 어울리는 채널 2~3개 (3) 채널별 메시지 방향
(4) 지금 당장 해볼 액션 3가지. 표와 짧은 설명으로.
```
이어서 콘텐츠 기획:
```
위 전략에 맞춰 다음 한 달 SNS 콘텐츠 캘린더 짜줘.
주차별로: 1주 매장·메뉴 소개, 2주 활용·팁, 3주 고객·후기, 4주 이벤트·시즌.
주당 게시물 주제 2개씩 + 간단한 캡션 톤 가이드.
```

- **산출물**: 마케팅 전략 한 장 + 4주 콘텐츠 캘린더
- **검증**: `우리 매장 규모로 실제 실행 가능한 것만 남기고, 과한 건 빼줘`
- **더 깊이**: 카드뉴스 제작·예약 `card-news`, 반응 분석 `rxr-mention-analysis`

### 🗂 자료정리·분석 — 매출 분석하기
**상황**: POS 매출 엑셀은 쌓이는데, 잘 팔리는 패턴과 신호를 못 읽는다.

```
이 매출 엑셀(첨부) 분석해줘.
(1) 기간·요일·시간대별 매출 흐름 (2) 잘 팔리는·안 팔리는 메뉴 Top/Bottom 5
(3) 객단가 추이 (4) 눈에 띄는 변화·신호 3가지.
표 + 한 줄 인사이트로. 숫자 빠지거나 잘못 더한 거 없는지 확인해줘.
```
더 깊게:
```
이 매출에서 '왜 그런지' 가설을 3개 세워줘. (예: 비 오는 날 매출↓, 신메뉴 효과 등)
각 가설을 확인하려면 어떤 데이터를 더 보면 되는지도 알려줘.
```

- **산출물**: 매출 분석표 + 인사이트·가설
- **검증**: `합계와 증감률을 한 번 더 검산하고, 추정한 건 추정이라고 표시해줘`
- **더 깊이**: 엑셀 변환 `excel-to-csv`, 대시보드화 `dashboard-prd`

### 📑 시스템 만들기 — 운영 매뉴얼 / 상품관리 / 정산관리
**상황**: 매뉴얼·상품·정산을 매번 머리와 수기로. 한 번 틀을 만들면 계속 같은 품질로 굴러간다.

```
우리 매장 운영 매뉴얼 만들어줘.
업무: [오픈·마감·응대·위생 등 떠오르는 대로]
직원이 바로 보고 따라할 수 있게 체크리스트 형태로, 빠진 항목은 나한테 질문해줘.
```
상품관리:
```
우리 상품 리스트(첨부 또는 붙여넣기)로 상품관리표 만들어줘.
열: 상품명·원가·판매가·마진율·재고·발주점(이하면 발주). 발주 필요 항목은 표시해줘.
```
정산관리:
```
일/월 정산표 만들어줘.
입력: 매출(현금/카드)·매입·고정비·기타비용.
순익과 비용 비율 자동계산, 매달 같은 양식으로 쓸 수 있게. 완성되면 PDF로 저장해줘.
```

- **산출물**: 운영 매뉴얼 체크리스트 / 상품관리표 / 정산표(PDF)
- **검증**: `합계·마진율·순익 계산이 맞는지 확인해줘`
- **더 깊이**: 반복되면 스킬로(EP05), HTML→PDF `playwright`, 대시보드 `dashboard-prd`

## 📣 마케팅

### 🔍 리서치 — SNS 버즈·경쟁사 콘텐츠 모니터링
**상황**: 캠페인/팝업 반응이 "좋아요 수"로만 보이고, 진짜 정서가 안 잡힌다.

```
[브랜드/캠페인]에 대한 SNS 반응을 분석해줘.
[자료] 아래 후기/댓글 텍스트를 붙여넣을게: [텍스트 또는 URL]
(1) 긍정/부정 비율 (2) 반복되는 칭찬·불만 키워드 (3) 협찬·광고로 보이는 거품 후기 구분
(4) 한 줄 총평. 표와 짧은 코멘트로.
```
경쟁사 콘텐츠면:
```
경쟁사 [A]의 최근 인스타/블로그 콘텐츠 흐름을 정리해줘: 주제 패턴, 톤, 잘 먹힌 포맷, 우리가 비어 있는 주제.
```

- **산출물**: 버즈 분석 요약 / 경쟁 콘텐츠 맵
- **검증**: `광고성 후기를 진짜 반응으로 섞지 않았는지 다시 봐줘`
- **더 깊이**: 상시 멘션 `rxr-mention-analysis`, 팝업/이벤트 `rxr-sns-analysis`, 종합 진단 `brand-analysis`

### 🗂 자료정리 — 설문 CSV → 인사이트 / 콘텐츠 캘린더
**상황**: 설문 응답 수백 건 CSV, 또는 머릿속에만 있는 콘텐츠 일정.

```
방문객/고객 설문 결과(CSV 첨부) 분석해줘.
(1) 인구통계 분포 (2) 가장 좋았던 요소 Top3 (3) 핵심 불만 3개 + 대표 코멘트
(4) 표면 만족도와 실제 재방문·추천 의향의 격차. 표 + 한 줄 인사이트.
```
콘텐츠 캘린더면:
```
다음 한 달 SNS 콘텐츠 캘린더 짜줘.
브랜드: [한 줄]  채널: [인스타/블로그]  주요 이벤트: [있으면]
주차별로: 1주 정보성, 2주 튜토리얼/활용, 3주 고객사례, 4주 성과·시즌. 각 주 게시물 주제 2개씩.
```

- **산출물**: 설문 인사이트 / 4주 콘텐츠 캘린더
- **검증**: `불만·격차를 좋게 포장하지 말고 있는 그대로 보여줘`
- **더 깊이**: 설문 설계·분석 `rxr-survey-analysis`, CSV 변환 `excel-to-csv`

### 📑 보고서 — 카드뉴스 + 캠페인 성과 리포트
**상황**: 카드뉴스 외주·캔바 작업에 시간이 들고, 성과 보고는 매번 새로 만든다.

```
카드뉴스 만들어줘. 주제: [주제]  키포인트: [3~5개]  톤: [발랄/차분/전문]
인스타 1080×1080, 8장 구성(표지+본문+마무리). 우리 톤은 [브랜드 한 줄].
완성되면 스크린샷으로 보여줘.
```
성과 리포트면:
```
이번 캠페인 성과 데이터(첨부)로 리포트 한 장 만들어줘.
도달·참여·전환 핵심 지표, 전월/목표 대비, 잘된 것·아쉬운 것, 다음 액션 3개.
```

- **산출물**: 카드뉴스 PNG / 캠페인 성과 리포트
- **검증**: `지표 계산과 전월 대비가 맞는지 확인해줘`
- **더 깊이**: 카드뉴스→예약발행 `card-news`(Buffer), 기획 심리 점검 `psychology-critic`

## 공통 — 어떤 직무든 통하는 3습관
