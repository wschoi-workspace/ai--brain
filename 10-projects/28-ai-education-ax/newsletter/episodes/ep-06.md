📺 *AX 뉴스레터 EP06*
*디자인 몰라도 — HTML로 비주얼 만들기*
━━━━━━━━━━━━━━━━
카드뉴스 한 세트 외주 주면 수십만 원, 직접 만들면 캔바와 씨름하다 반나절. 그 일을 AI가 *설계부터 이미지까지* 해줍니다.

비결은 HTML이에요. 어렵게 들리지만 — AI가 다 짜줍니다. 우리는 _주제만_ 던지면 됩니다.

💡 *흐름은 3단계*
1. 구조 설계
`인스타 카드뉴스 8장 구조 짜줘. 주제는 "여름 신메뉴".`
2. 디자인 생성
`1080×1080 비율로 HTML 만들어줘. 베이지+코랄, Pretendard 폰트.`
3. 이미지로 저장
`이거 스크린샷으로 캡처해줘.` → 인스타에 바로 올릴 PNG 완성

🔁 *OSMU — 한 번 만들어 여러 곳에*
같은 내용을 카드뉴스로, 슬라이드로, 한 장 요약으로. 소스 하나로 여러 채널에 씁니다. _One Source Multi Use._

📐 *왜 HTML이냐*
파워포인트·캔바는 사람이 손으로 끌어야 하지만, HTML은 AI가 글로 짜고 자동으로 그림으로 바꿉니다. 수정도 "코랄 좀 더 진하게" 한 마디면 끝이에요.

🎯 *이번 주 실습 (20분)*
실제 우리 가게/업무 주제로 카드뉴스 1장 만들어 보세요:
1. `__ 주제로 카드뉴스 1장 HTML로 만들어줘`
2. 브라우저에서 열어 확인
3. `스크린샷으로 저장해줘` → 이미지 확보

💼 *내 직무라면 — 한 장으로 만들 산출물*
🎨 디자인: 공간 컨셉 제안 슬라이드
📋 기획: 제안서 표지·핵심 1장
🏪 운영: 정산표·매출 리포트
📣 마케팅: 인스타 카드뉴스

🧭 리서치·정리를 거친 결과를 *보고서 한 장*으로 맺는 단계. 흐름의 마무리예요.
📂 직무별 보고서 프롬프트 → *직무별 예제집*

✅ *체크*
□ 카드뉴스/슬라이드 1장 생성
□ 색·문구 1회 수정 요청
□ 이미지(PNG)로 저장
