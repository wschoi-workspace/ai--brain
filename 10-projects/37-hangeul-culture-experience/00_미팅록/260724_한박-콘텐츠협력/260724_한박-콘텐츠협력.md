# LA 한국문화원 K-Zone — 한글박물관 콘텐츠 협력 미팅

## [1] 개요 / 배경

LA 한국문화원 및 코리아센터 리뉴얼(K-Zone 구축)에 한글박물관 콘텐츠를 배치하는 건에 대한 실무 협의. 한글박물관 증축으로 사업비가 대폭 삭감된 상황에서, 기획 확정 후 공간에 맞는 콘텐츠를 제안하는 순서로 진행하기로 합의. Project Rent 측에서 체험형 콘텐츠 기획안을 공유하며 방향성을 논의.

| 키 | 값 |
|---|---|
| 일시 | 2026-07-24 |
| 참석 | 한글박물관 과장님 / Project Rent 측 |
| 공간 | LA 한국문화원 1층(상설 전시) + 코리아센터(팝업/체험형) |
| 오프닝 목표 | 2026년 9월 말 (가장 긍정적 시나리오) |
| 한박 가용 예산 | 콘텐츠 제작비 약 3,000~4,000만 원 |
| 시공 현황 | 퓨저빌리티(Feasibility) 체크 중, 7월 초 방향성 합의 → 8월 초중순 상세 기획 확정 예정 |

## [2] Summary

LA 한국문화원 K-Zone 리뉴얼에 **한글 콘텐츠를 어떤 형태로 배치할 것인가**를 논의한 기획 미팅. 핵심 합의는 **"기획이 먼저, 콘텐츠 제안은 후순위"** — 공간 기획과 배치가 확정된 후 한글박물관이 해당 공간에 맞는 콘텐츠를 제안하는 순서로 진행하기로 함. 한글박물관 증축에 따른 **화재보험 비용 증가로 사업비가 절반 이상 삭감**된 상태이며, 가용 예산은 콘텐츠 제작비 약 **3~4천만 원** 수준. Project Rent 측은 **체험형·인터랙티브 콘텐츠**(이름 한글 변환 → 도장/티셔츠/포스터 → 의미 발견 → 테이크아웃)를 중심으로 한 기획 방향을 공유했으며, 한글박물관 측은 **전시 콘텐츠·교육 키트·영상 소스 제공 가능성**을 확인. 문화원장(이해돈 원장)의 인터랙션 강화 요구, LA 한인 10만 명 타겟 병행, 한글날(10/9) 연계 오프닝 시즌 등이 주요 맥락.

## [3] 본문 — 안건별 정리

### A. 공간 구조 및 콘텐츠 배치 전략

**두 개의 공간이 각기 다른 역할로 운영 예정:**

| 공간 | 성격 | 콘텐츠 방향 | 운영 주기 |
|---|---|---|---|
| 한국문화원 1층 | 상설 전시관 (28년 올림픽까지 유지) | 클래식한 전시, 프리미엄/프라이빗 클래스, 면대면 서비스 | 시즌별 주제 교체 (인프라 상설, 콘텐츠 시즌널) |
| 코리아센터 | 팝업 플레이스 | 체험형·이벤트성, 기업 행사, 관련 부서 행사 | 상설 개념 없음, 지속적 팝업 운영 |

- 한국문화원은 천장이 낮고 공간이 좁아 **대규모 체험은 불가** → 프리미엄 서비스·프라이빗 클래스로 한정
- 코리아센터는 번잡한 체험(결과물 제작 등)을 수용하는 공간으로 활용
- 두 공간을 **하나의 주제로 연계**하여 동선 유도 검토 중 (전시 관람 → 체험 → 기념물 테이크아웃)
- 문화원 → 코리아센터 도보 약 20~30m, 물리적 연계 가능하나 외벽 활용은 제약 있음

**문화원 내 전시 구성 (4개 분야 혼합):**

| 분야 | 배치 방향 |
|---|---|
| 한글 | 첫 시즌 메인 테마 (한글날 연계), 문화원 중심 |
| 한지/가구 | 상품 개발 쪽으로 전시 |
| 한복 | 굿즈·상품 전시 위주 |
| K-콘텐츠 | 코리아센터 쪽에서 소화 |
| 푸드 | 비중 약하게, 라이프스타일 베이스 |

- 기존 한옥 구조물: 끝에 위치한 것은 **전시 쇼케이스**로 활용, 가운데 것은 철거
- 가변성 확보를 위해 새로 설계 중 (기존 공간은 가변성 거의 없었음)
- 시즌 메인 테마가 바뀔 때 관련 콘텐츠를 릴레이트해서 확장 (예: 한글 시즌에 한지 일부 포함)

### B. 콘텐츠 제공 방식 — "기획 먼저, 제안 후순위"

한글박물관 측 핵심 입장:

1. **지금 뭘 하겠다고 미리 드리는 것보다 공간 기획이 나온 후 제안하는 게 적중률이 높음**
2. 너무 불가능한 범위로 제안하게 되는 리스크 방지
3. 문화원 사업이므로 외교행낭을 통한 운송비 절감 가능
4. 가용 콘텐츠 풀이 넓지 않으므로 **보유 리스트를 먼저 공유 → 기획사가 그 안에서 선택**하는 방식이 효율적

한글박물관 제공 가능 콘텐츠:
- **전시품**: 기존 양질의 전시 콘텐츠 (시즌별 교체 가능)
- **교육 키트**: 연구교육과에서 개발한 외국인 체험 키트 (공예품 + 한글 만들기, 비용만 들이면 수량 무제한 제작·운송 가능)
- **영상 소스**: 실감 영상 보유 (단, 설명형 콘텐츠이지 연출 영상은 아님)
- **미디어아트 소스**: 인천공항 미디어 파사드에 사용한 한글 영상 (옥스퍼드 사전 등재 단어 기반, 타이포그래피 작가 협업)
- **한글 상품**: 이멘탈리오 등 해외 진출 상품 보유, 다만 올해 신규 개발분은 시점이 안 맞을 수 있음 → 기존 개발 상품 활용, 신규분은 내년 시즌 교체 시 반영
- **작가 작품**: 별도 비용 발생 + 작가 협의 필요

### C. 체험형 콘텐츠 기획 방향 (Project Rent 측 제안)

**핵심 전환: "교육"에서 "경험(Experience)"으로**

> 언어를 배우는 건 어려운 일. 해봤더니 재밌고 → 이쁘고 → 쿨한 것 같아 — 이 엔트리 레벨의 관심 유도가 목표.

**체험 여정 5단계:**

| 단계 | 내용 | 실행 형태 (참고) |
|---|---|---|
| 1. 이름 쓰기 | 자기 이름을 한글로 변환 | AI 활용 또는 수기, 실행 방식 미확정 |
| 2. 만들기 | 한글 이름을 결과물로 제작 | 도장 새기기, 티셔츠 프린트, 조합형 블록 |
| 3. 아트화 | 한글 이름의 비주얼라이제이션 | AI 기반 아티스틱 변환, 타입 비주얼 프린트 |
| 4. 의미 발견 | 이름의 한국어 의미 탐색 | Emily→별/하늘, Michelle→강 등 의미 매핑 + 감정 표현 소개 |
| 5. 테이크아웃 | 결과물을 기념품으로 가져감 | 인비테이션/보딩패스/패스포트 형태, "한국을 와봤으면 좋겠다" 메시지 |

**확장 가능 콘텐츠:**
- 블록 놀이 (한글 조합)
- 포스터 제작
- K-POP 가사 만들기 (소비 콘텐츠 연결)
- 한글 패턴 프로젝션 매핑 (기둥 구조물 활용, 미디어아트)

### D. 타겟 및 방문자 분석

| 타겟 | 특성 | 고려사항 |
|---|---|---|
| 외국인 일반 | 한국에 관심 높은 편 (문화원 방문자 특성) | 이미 자기 이름 한글로 쓸 수 있는 수준일 수 있음 → 단순 이름 쓰기로는 부족 |
| 필드트립 학생 | 단체 방문 빈번, 세종학당 수강생 포함 | 동남아 학생 그룹핑 자체를 포기하는 추세 |
| LA 한인 (10만 명) | 핵심 비중 타겟으로 설정 | 자기 외국인 친구 데려오고 싶어지는 퀄리티 필요 |

- 문화원장(이해돈 원장) 의견: **체류형·체험형 콘텐츠 강화 필수** — 현재 1시간 이상 체류할 이유가 없어 수업 수강자도 없는 상태
- 해외 한인들의 피드백: "K-컬쳐를 고급 문화로 끌어올려야 할 타이밍. 커머셜만 하면 안 됨"
- 커머셜 구조도 요구사항에 포함: 미국 현지 라이센스 기반 판매 가능성까지 테스팅

### E. 미디어·영상 콘텐츠 활용

- 한글박물관 보유 영상은 **설명형(사람이 보는 내용 있는 영상)** — 미디어아트용 연출 영상은 아님
- 기획사 측에서 **프로젝션 매핑 활용** 검토 중 (공간이 좁아 물리적 전시물 대비 효과적)
- 인천공항 미디어 파사드 영상: 한글 타이포 + 옥스퍼드 사전 등재 한국어 단어 기반 — **리사이징·리터칭하여 활용 가능성 검토 요청**
- 한글박물관은 원본 소스의 변형 사용을 원하지 않음 → 소스 컨디션 확인 후 부분 발췌·스트럭처 재구성 가능 여부 별도 협의 필요
- 상설관 내 기둥 구조물에 한글 패턴 프로젝션 매핑 고려 중

### F. 예산 및 일정

**예산 현실:**
- 한글박물관 증축에 따른 화재보험 비용 급증 → 사업비 **반 이상 삭감**
- 가용 예산: 콘텐츠 제작·개발 지원 약 **3,000~4,000만 원**
- 문화원 사업이므로 외교행낭 운송 활용 가능 → 운송비 절감
- 작가 작품 활용 시 별도 비용 + 작가 협의 필요
- 커머셜 구조: 사업자를 끼고 미국 현지 판매 라이센스까지 고려 중 (상부 요구사항)
- 미국 현지 시공: 라이센스 없이 쓰려면 비용 폭등 → 난이도별 분리·한국 업체 투입 검토

**일정:**

| 마일스톤 | 시점 |
|---|---|
| 기획 완료 (드래프트) | 1주 내 (7월 말) |
| 퓨저빌리티 체크 + 방향성 합의 | 7월 초 (이미 진행 중) |
| 전체 콘텐츠 라인업 미팅 | 기획 드래프트 후 즉시 |
| 상세 기획 확정 | 기획 후 2~2.5주 (8월 초중순) |
| 한박 콘텐츠 배치 제안 | 공간 확정 후 |
| 기관 내부 행정 처리 | 콘텐츠 확정 후 (절대 시간 필요) |
| 오프닝 | 9월 말 (가장 긍정적 시나리오) |

## 미해결 · 후속 확인

- **코리아센터 vs 문화원 콘텐츠 최종 배분**: 기획 확정 후 결정. 체험 파트가 문화원 1층에서 소화 안 되면 코리아센터로 이동 가능
