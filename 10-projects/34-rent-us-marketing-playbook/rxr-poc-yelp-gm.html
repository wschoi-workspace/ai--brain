Sincerity Filter 영어 적용 핵심 발견

  • 경험 신호("I've been a customer for 5 years")와 디테일 신호("$60/month → $300/month")가 한국어보다 더 명시적으로 나타남 — 필터 정확도 향상
  • 영어 리뷰의 기간 명시 습관("for 4 years", "since the update")이 Trust Score 산출에 강력한 추가 피처
  • 한국어의 "제공받았습니다" vs 영어의 이해관계 신호는 구조가 다름 — 영어는 비교 사이트 맥락에서 이해관계 감지가 핵심
  • Fake 리뷰 의심 패턴: "Most 5-star reviews appeared planted, using buzzwords" — App Store 리뷰어가 직접 지적

06PoC 검증 — RXR 영어 적용 가능성

검증 항목결과상세
Sincerity Filter A~G 분류 PASS 영어 마커 패턴으로 7등급 분류 유의미하게 작동. 특히 경험 신호("I've used for X years")가 한국어보다 명시적.
Trust Score 산출 PASS 기간 명시, 가격 디테일, 구체적 기능 언급이 Score 변별력 제공. 평균 68/100.
Content Layer: 토픽 분류 PASS 7개 토픽으로 자연스럽게 클러스터링. 복수 태깅도 문제없이 적용.
Content Layer: 4-Way 감성 PASS 긍정/혼합/중립/부정 분류 정상. 영어의 "not bad" 같은 이중부정 처리도 가능.
Psyche Layer: Authenticity PASS 감정 변화 서사("was great → now terrible")가 Auth 산출의 강력한 피처.
Psyche Layer: Clout PASS 추천/반추천 강도가 영어에서 더 직접적("Don't bother", "money hungry"). 변별력 높음.
Psyche Layer: Freshness PARTIAL 한국어 조사 기반 측정 → 영어에서는 동사 시제/발견형 구문으로 전환 필요. 작동하나 캘리브레이션 필요.
RQL 등급 산출 PASS Q1~Q5 분류 정상. 영어 리뷰가 한국어보다 평균 RQL 약간 높음 (구조적 서술 경향).
Auth × 감성 교차 분석 PASS 긍정=낮은Auth / 부정·혼합=높은Auth 패턴이 한국어 분석과 동일하게 나타남. 이론적 일관성 확인.
수집 병목 해결 PARTIAL WebSearch+WebFetch로 38건 수집 성공. G2/Capterra 403 차단 → API 또는 크롤러 필요.

PoC 결론: 8/10 검증 항목 PASS

  • RXR 핵심 엔진(Sincerity Filter + Content Layer + Psyche Layer)은 영어에서도 유의미하게 작동 — 프레임워크의 범용성 확인
  • Freshness Index는 언어 구조적 차이로 인해 캘리브레이션 필요 (한국어 조사 → 영어 동사 시제/구문 전환)
  • 데이터 수집이 유일한 실질적 병목 — G2/Capterra API 연동 또는 Reddit API 크롤러 구축으로 해결 가능
  • Auth × 감성 교차 패턴이 한국어·영어에서 동일하게 나타남 → RXR의 이론적 기반(ELM, eWOM)의 문화 초월적 타당성 시사

07마케팅 인사이트 — Yelp Guest Manager 시장 포지셔닝

프로젝트 렌트 관점: 기회 영역

  • Vendor Lock-in 반감이 시장 전반에 형성 — "탈 Yelp" 니즈 존재
  • 레스토랑 오너의 데이터 주권 의식 강함 ("guest data not owned by restaurants")
  • 가격 투명성에 대한 강한 기대 — quote-based pricing 불신
  • 경험 설계·측정 방법론은 Yelp가 제공하지 못하는 영역 — 프로젝트 렌트의 차별화 포인트
  • OpenTable 대비 저가 포지셔닝($163/mo 절감)이 Yelp GM의 강점 → 가격이 아닌 가치 기반 경쟁 필요

시장 리스크 · 유의점

  • Yelp AI Host(24/7 자동 예약) 등 기술 투자 가속 — 경쟁 진입장벽 상승
  • Yelp 생태계 의존도가 높은 레스토랑은 전환 비용 높음
  • 미국 레스토랑 시장의 리뷰 플랫폼 의존 구조 — 단순 SaaS 대체가 아닌 생태계 재설계 필요
  • App Store 리뷰의 부정률(53%)은 생존자 편향 고려 필요 — 만족 유저는 리뷰를 잘 안 씀
  • Resy(AmEx 인수)의 프리미엄 포지셔닝이 upscale 세그먼트 선점 중
